Automatisierte Client-Konfiguration

Wenn Ihr Mac OS X 10.6-Computer mit einem Netzwerk und nicht mit einer Verzeichnis-Domain verbunden ist, verwendet er Bonjour zur Erkennung von Mac OS X 10.6-Servercomputern.

Ermittelt der Mac dabei einen Mac OS X 10.6-Servercomputer, der als Open Directory-Master dient, der wiederum mindestens einen automatisch konfigurierbaren Dienst ausführt (z. B. AFP, SMB, VPN, Mail, iCal, iChat, Benachrichtigungen) sowie einen Benutzer-Account in der Verzeichnis-Domain besitzt, dessen Kurz- oder Langname mit dem des derzeit angemeldeten Benutzers des Mac OS X 10.6-Computers übereinstimmt, wird auf Ihrem Mac OS X 10.6-Computer eine Einladung angezeigt.

In der Einladung werden Assistenten für die Konfiguration der Verbindung zu einem Verzeichnisserver angeboten. Sie können die Einladung, eine Verbindung zu einem Server herzustellen, annehmen oder ablehnen, oder sich für eine Verbindung zu einem späteren Zeitpunkt entscheiden.

Wenn Ihr Computer mehrere Mac OS X 10.6-Verzeichnisserver erkennt, zu denen Sie eine Verbindung herstellen können, werden diese Server in einem Einblendmenü der Einladung aufgelistet. Wenn Sie auf einen Server zugreifen, wird der Assistent geöffnet und verbindet Ihre Programme mit den Diensten, die der Verzeichnisserver ausführt, und konfiguriert sie.

Gehen Sie wie folgt vor, um mithilfe des automatisierten Assistenten eine Verbindung zu einer Verzeichnis-Domain herzustellen:

  1. Klicken Sie, wenn der Assistent angezeigt wird, auf "Dienste konfigurieren".

  2. Wählen Sie aus dem Einblendmenü "Server" die Verzeichnis-Domain aus, auf die Sie zugreifen möchten.

    In Ihrem Netzwerk sind möglicherweise mehrere Verzeichnis-Domains verfügbar.

    Befindet sich der gewünschte Server nicht in der Liste, geben Sie den Namen oder die IP-Adresse des Servers in das Feld "Server" ein und klicken Sie auf "OK".

  3. Geben Sie den Benutzernamen und das Kennwort eines Accounts ein, der die erforderlichen Berechtigungen besitzt, einen Computer mit der Verzeichnis-Domain zu verbinden, auf die Sie zugreifen.

    Wenn Sie diese Informationen nicht zur Hand haben, wenden Sie sich an Ihren Verzeichnisadministrator.

  4. Geben Sie im Anschluss daran das Kennwort für den Benutzer-Account ein, der im Bereich "Geben Sie das Kennwort für den Account benutzername auf diesem Computer" angezeigt wird.

    Der Assistent ändert das Kennwort des Verzeichnis-Accounts des Benutzers auf dem Server entsprechend dem Kennwort des lokalen Benutzer-Accounts.

  5. Klicken Sie auf "Weiter".

  6. Klicken Sie auf "Fortfahren", wenn das Dienstprogramm "UpgradeUser" alle Schritte ausgeführt hat.

  7. Wenn eine Meldung angezeigt wird, in der die konfigurierten Dienste genannt und Sie zum Abmelden aufgefordert werden, klicken Sie auf "Abmelden".

    Wenn Sie sich anschließend wieder anmelden, können Sie die neuen Dienste verwenden.
