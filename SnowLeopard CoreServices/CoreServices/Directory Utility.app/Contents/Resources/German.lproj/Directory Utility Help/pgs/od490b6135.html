Informationen über die Verzeichnisdienste

Das Programm "Verzeichnisdienste" legt fest, welche Verzeichnisdienste (Directory Services) von einem Mac OS X-Computer verwendet werden und wie dieser die Verbindung zu bestimmten Verzeichnis-Domains herstellt. Das Programm "Verzeichnisdienste" definiert außerdem Suchpfade für die Suche nach Identifikations- und Kontaktinformationen in bestimmten Verzeichnis-Domains.

Einrichten der Verzeichnisdienste

Sie können das Programm "Verzeichnisdienste" lokal auf Ihrem Computer verwenden, um die Verzeichnisdienste auf Ihrem Computer einzurichten und festzulegen, wie der Zugriff auf diese Dienste erfolgen soll. Das Programm "Verzeichnisdienste" bietet aber auch die Möglichkeit, den Zugriff auf die Verzeichnisdienste auf Ihrem Computer netzwerkgestützt per Fernzugriff zu konfigurieren. Darüber hinaus können Sie mit dem Programm "Verzeichnisdienste" den root-Account aktivieren und das Kennwort für den root-Account ändern.

Konfigurieren des Zugriffs auf Dienste

Mit dem Programm "Verzeichnisdienste" können Sie unterschiedliche Dienste aktivieren und deaktivieren, auf die Sie mit Mac OS X über ein Netzwerk zugreifen können. Zu diesen Diensten zählen Verzeichnisdienste, die Zugriff auf Benutzer-Accounts und andere Verwaltungsdaten gewähren, die in Verzeichnis-Domains gesichert sind.

Konfigurieren des Zugriffs auf Verzeichnisse

Mit dem Programm "Verzeichnisdienste" können Sie Ihren Computer so einrichten, dass er auf Benutzer-Accounts und andere administrative Informationen zugreifen kann, die in speziellen LDAP-Verzeichnissen, in einer Active Directory-Domain oder einer NIS-Domain gespeichert sind.

Einrichten von Suchpfaden

Mit dem Programm "Verzeichnisdienste" können Sie Suchpfade festlegen und bestimmen, welche Bereiche und Verzeichnisse Mac OS X im Zuge der Identifizierung durchsuchen soll, um Benutzeranmeldedaten und andere administrative Daten aus den Verzeichnis-Domains abzurufen. Ähnliche Suchpfade können Sie für die Identifizierung von Kontakten angeben, d. h. Sie können festlegen, wo Mac OS X suchen soll, um Namen, Adressen und andere Informationen über Kontakte aus den Verzeichnis-Domains abzurufen.
