Vorbereiten eines schreibgeschützten LDAP-Verzeichnisses für Mac OS X

Wenn Sie möchten, dass ein Mac OS X-Computer administrative Daten aus einem schreibgeschützten LDAP-Verzeichnis erhält, müssen die Daten in dem von Mac OS X benötigten Format vorhanden sein. Es kann sein, dass Sie Daten im schreibgeschützten LDAP-Verzeichnis hinzufügen, ändern oder umstrukturieren müssen.

Da Mac OS X keine Daten in ein schreibgeschütztes Verzeichnis schreiben kann, müssen Sie die Änderungen mithilfe anderer Dienstprogramme vornehmen. Diese Dienstprogramme müssen sich auf dem Server befinden, der das schreibgeschützte LDAP-Verzeichnis enthält.

Gehen Sie wie folgt vor, um ein schreibgeschütztes LDAP-Verzeichnis für Mac OS X vorzubereiten:

  1. Begeben Sie sich zu dem Server, der Host des schreibgeschützten LDAP-Verzeichnisses ist. Konfigurieren Sie den Server so, dass er die LDAP-basierte Identifizierung und Kennwortüberprüfung unterstützt.

  2. Ändern Sie gegebenenfalls die Objektklassen und Attribute des LDAP-Verzeichnisses, damit die von Mac OS X benötigten Daten bereitgestellt werden.

Ausführliche Angaben zu den von Mac OS X-Verzeichnisdiensten benötigten Daten finden Sie im Handbuch "Open Directory - Administration".
