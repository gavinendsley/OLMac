Definieren von Suchpfaden für das lokale Verzeichnis

Sie können im Programm "Verzeichnisdienste" die Suchpfade eines Mac OS X-Computers für die Benutzeridentifizierung und die Suche nach Kontakten so konfigurieren, dass nur das lokale Verzeichnis des Computers verwendet wird.

Ein auf das lokale Verzeichnis reduzierter Suchpfad schränkt den Zugriff eines Computers auf Informationen zur Identifizierung und andere administrative Daten ein.

Wenn Sie den Suchpfad eines Computers für die Benutzeridentifizierung auf das lokale Verzeichnis beschränken, können sich nur Benutzer mit lokalen Accounts an dem betreffenden Computer anmelden.

Gehen Sie wie folgt vor, um einen Suchpfad auf die lokale Verzeichnis-Domain (auf das lokale Verzeichnis) zu beschränken:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Suchpfad" und wählen Sie einen Suchpfad aus:

    • Identifizierung: In diesem Bereich wird der Suchpfad angezeigt, der für die Benutzeridentifizierung und die meisten administrativen Aufgaben verwendet wird.

    • Kontakte: In diesem Bereich wird der Suchpfad angezeigt, der für Kontaktinformationen im Programm "Adressbuch" und anderen Programmen verwendet wird.

  7. Wählen Sie "Lokales Verzeichnis" aus dem Einblendmenü "Suchen" aus. Klicken Sie anschließend auf "Anwenden".
