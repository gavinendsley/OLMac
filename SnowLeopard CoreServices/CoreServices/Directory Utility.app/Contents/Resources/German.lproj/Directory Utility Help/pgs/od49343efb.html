Konfigurieren des Zugriffs auf Verzeichnisse

Mit dem Programm "Verzeichnisdienste" (in der Systemeinstellung "Benutzer") können Sie konfigurieren und festlegen, wie ein Computer mit Mac OS X oder ein Server mit Mac OS X Server auf Verzeichnis-Domains zugreift.

Zugreifen auf LDAP-Verzeichnisse

Mit dem Programm "Verzeichnisdienste" können Sie einen Computer mit Mac OS X oder einen Server mit Mac OS X Server so konfigurieren, dass er auf bestimmte LDAP-Verzeichnisse zugreift (z. B. auch auf das LDAP-Verzeichnis eines Open Directory-Masters von Mac OS X Server). Klicken Sie auf die unten aufgelisteten Themen, wenn Sie weitere Informationen wünschen.

Verwenden von BSD-Konfigurationsdateien

Mithilfe von Open Directory können administrative Daten aus BSD-Konfigurationsdateien (z. B. aus der Datei "/etc/master.passwd") abgerufen werden. Diese Dateien werden auch als "lokale BSD-Dateien" bezeichnet. Im Programm "Verzeichnisdienste" können Sie den Zugriff auf solche BSD-Konfigurationsdateien konfigurieren. Klicken Sie auf die unten aufgelisteten Themen, wenn Sie weitere Informationen wünschen.
