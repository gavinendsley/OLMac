Ändern des Kennworts für die Identifizierung einer LDAP-Verbindung

Mithilfe des Programms "Verzeichnisdienste" können Sie eine identifizierte LDAP-Verbindung so aktualisieren, dass ein neues Kennwort verwendet wird, das auf dem LDAP-Server geändert wurde. (Wenn das Kennwort für die Identifizierung der LDAP-Verbindung auf dem Server geändert wird, müssen anschließend alle Computer aktualisiert werden, die eine identifizierte Verbindung zu dem betreffenden LDAP-Server unterhalten.)

Gehen Sie wie folgt vor, um das Kennwort für eine LDAP-Verbindung zu ändern:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Wählen Sie in der Liste der Dienste "LDAPv3" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  8. Wenn die Liste der Serverkonfigurationen ausgeblendet ist, klicken Sie auf "Optionen einblenden".

  9. Wählen Sie aus der Liste eine Serverkonfiguration aus und klicken Sie auf "Bearbeiten".

  10. Klicken Sie auf "Sicherheit" und ändern Sie die Einstellung "Kennwort".

    • Lesen Sie die Hinweise unter Identifizieren einer LDAP-Verbindung, wenn die Einstellung "Kennwort" grau dargestellt wird, weil die Option "Beim Verbindungsaufbau identifizieren" deaktiviert ist.

    • Wenn die Einstellung "Kennwort" grau dargestellt wird, weil die Option "Eingebunden in das Verzeichnis als" ausgewählt ist (aber grau dargestellt wird), so bedeutet dies, dass die Identifizierung der Verbindung nicht anhand des Benutzerkennworts erfolgt. Die Verbindung verwendet stattdessen einen identifizierten Computereintrag für die vertrauenswürdige Bindung.
