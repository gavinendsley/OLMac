Trennen vom Active Directory-Server

Wenn der Computer die ADC-Komponente (Active Directory Connector) des Programms "Verzeichnisdienste" verwendet, um eine Bindung zu einem Active Directory-Server herzustellen, können Sie den Computer vom Active Directory-Server trennen.

Sie können die Bindung sofort trennen, wenn der Computer keinen Kontakt zum Server herstellen kann oder der Computereintrag vom Server gelöscht wurde.

Gehen Sie wie folgt vor, um den Computer vom Active Directory-Server zu trennen:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Wählen Sie in der Liste der Dienste "Active Directory" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  8. Klicken Sie auf "Trennen". Identifizieren Sie sich als ein Benutzer, der berechtigt ist, eine Verbindung zur Active Directory-Domain zu trennen. Klicken Sie dann auf "OK".

    Wenn Sie eine Warnmeldung erhalten, dass die Anmeldedaten nicht akzeptiert wurden oder der Computer keine Verbindung zu Active Directory herstellen konnte, klicken Sie auf "Verbindung sofort trennen", um die Verbindung sofort zu unterbrechen.

    Wenn Sie die Bindung sofort trennen, enthält Active Directory dennoch weiterhin einen Computereintrag für diesen Computer. Benachrichtigen Sie in einem solchen Fall den Active Directory-Administrator, damit er den Computereintrag löscht.

  9. Deaktivieren Sie im Bereich "Dienste" die Active Directory-Einstellung "Aktivieren" und klicken Sie auf "Anwenden".
