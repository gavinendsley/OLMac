Verwenden erweiterter Einstellungen für das Programm "Verzeichnisdienste"

Im Programm "Verzeichnisdienste" werden die Verzeichnisdienste aufgeführt, auf die Mac OS X zugreifen kann. Mithilfe der aufgelisteten Verzeichnisdienste ist es Mac OS X möglich, auf Benutzerinformationen und andere administrative Daten zuzugreifen, die in Verzeichnis-Domains gespeichert sind.

Sie können den Zugriff auf jeden Verzeichnisdienst individuell aktivieren oder deaktivieren. Wenn Sie im Programm "Verzeichnisdienste" einen Dienst deaktivieren, so hat dies zur Folge, dass Mac OS X nicht mehr auf den betreffenden Verzeichnisdienst zugreift.

  • Aktivieren oder Deaktivieren des Active Directory-Diensts

  • Aktivieren oder Deaktivieren von LDAP-Verzeichnisdiensten
