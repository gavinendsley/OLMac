Angeben eines bevorzugten Active Directory-Servers

Auf einem Computer, der so konfiguriert ist, dass er die ADC-Komponente (Active Directory Connector) des Programms "Verzeichnisdienste" verwendet, können Sie den DNS-Namen des Servers festlegen, auf dessen Active Directory-Domain der Computer standardmäßig zugreifen soll.

Wenn der Server irgendwann nicht mehr zur Verfügung steht, wechselt die ADC-Komponente zu einem anderen, nahe gelegenen Server in der Gesamtstruktur.

Wenn diese Option deaktiviert ist, ermittelt die ADC-Komponente die nächstgelegene Active Directory-Domain in der Gesamtstruktur.

Gehen Sie wie folgt vor, um einen Server anzugeben, auf den die ADC-Komponente bevorzugt zugreifen soll:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf "Dienste".

  6. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  7. Wählen Sie in der Liste der Dienste "Active Directory" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  8. Wenn die erweiterten Optionen ausgeblendet sind, klicken Sie auf "Erweiterte Optionen einblenden".

  9. Klicken Sie auf "Administration".

  10. Wählen Sie "Bevorzugter Domain-Server" aus und geben Sie den DNS-Namen des Active Directory-Servers ein.

  11. Klicken Sie auf "OK".
