Steuern der Identifizierung aus allen Domains in der Active Directory-Gesamtstruktur

Auf einem Computer, der so konfiguriert ist, dass er die ADC-Komponente (Active Directory Connector) des Programms "Verzeichnisdienste" verwendet, können Sie Benutzern von allen Domains in der Active Directory-Gesamtstruktur die Identifizierung ermöglichen. Sie können die Identifizierung aber auch auf Benutzer aus einzelnen Domains beschränken.

Gehen Sie wie folgt vor, um zu steuern, ob sich Benutzer von allen Domains in der Gesamtstruktur aus identifizieren können:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  8. Wählen Sie in der Liste der Dienste "Active Directory" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  9. Wenn die erweiterten Optionen ausgeblendet sind, klicken Sie auf "Erweiterte Optionen einblenden".

  10. Klicken Sie auf "Administration".

  11. Wählen Sie "Identifizierung aus jeder Domain in der Gesamtstruktur ermöglichen" aus.

    Wenn Sie "Identifizierung aus jeder Domain in der Gesamtstruktur ermöglichen" auswählen, können Sie die Active Directory-Gesamtstruktur zu den eigenen Suchpfaden des Computers für Identifizierung und Kontakte hinzufügen.

    Wenn Sie eine Active Directory-Gesamtstruktur zu einem eigenen Suchpfad hinzufügen, wird die Gesamtstruktur in der Liste der verfügbaren Verzeichnis-Domains wie folgt angezeigt: "/Active Directory/All Domains". (Dies ist die Standardeinstellung.)

    Wenn Sie "Identifizierung aus jeder Domain in der Gesamtstruktur ermöglichen" deaktivieren, können Sie den eigenen Suchpfaden des Computers einzelne Active Directory-Domains für Identifizierung und Kontakte hinzufügen.

    Wenn Sie Active Directory-Domains zu einem eigenen Suchpfad hinzufügen, wird jede Active Directory-Domain in der Liste der verfügbaren Verzeichnis-Domains separat angezeigt.

  12. Klicken Sie auf "OK".

  13. Ändern Sie den eigenen Suchpfad im Bereich "Identifizierung" und im Bereich "Kontakte", nachdem Sie die Option "Identifizierung aus jeder Domain in der Gesamtstruktur ermöglichen" ausgewählt haben, damit die Active Directory-Gesamtstruktur bzw. ausgewählte Domains berücksichtigt werden.

    Weitere Informationen über das Ändern eines eigenen Suchpfads finden Sie unter Definieren eigener Suchpfade.
