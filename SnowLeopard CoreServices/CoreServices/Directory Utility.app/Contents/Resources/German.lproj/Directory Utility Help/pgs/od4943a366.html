Zuordnen der UID zu einem Active Directory-Attribut

Auf einem Computer, der so konfiguriert ist, dass er die ADC-Komponente (Active Directory Connector) des Programms "Verzeichnisdienste" verwendet, können Sie ein Active Directory-Attribut definieren, das dem Attribut der eindeutigen Mac OS X-Benutzer-ID (UID) zugeordnet werden soll.

Normalerweise muss das Active Directory-Schema um ein Attribut erweitert werden, das für die Zuordnung zur UID geeignet ist.

  • Wenn der Active Directory-Administrator das Active Directory-Schema erweitert, indem er die Microsoft-Dienste für UNIX installiert, können Sie die UID dem Attribut "msSFU-30-Uid-Number" zuordnen.

  • Wenn der Active Directory-Administrator das Active Directory-Schema manuell so erweitert, dass es das Attribut "RFC 2307" enthält, können Sie die UID dem Attribut "uidNumber" zuordnen.

  • Wenn der Active Directory-Administrator das Active Directory-Schema manuell so erweitert, dass es das Mac OS X-Attribut "UniqueID" enthält, können Sie die UID diesem Attribut zuordnen.

Wenn die UID-Zuordnung deaktiviert ist, generiert die ADC-Komponente eine UID auf Basis des standardmäßigen GUID-Attributs von Active Directory.

ACHTUNG: Wenn Sie die Zuordnung der UID zu einem späteren Zeitpunkt ändern, können Benutzer den Zugriff auf zuvor erstellte Dateien verlieren.
Gehen Sie wie folgt vor, um die UID einem Attribut in einem erweiterten Active Directory-Schema zuzuordnen:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Wählen Sie in der Liste der Dienste "Active Directory" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  8. Wenn die erweiterten Optionen ausgeblendet sind, klicken Sie auf "Erweiterte Optionen einblenden".

  9. Klicken Sie auf "Pfade".

  10. Wählen Sie dann "Eindeutige Kennung (UID) auf Attribut zuordnen" und geben Sie den Namen des Active Directory-Attributs ein, das Sie der UID zuordnen wollen.

  11. Klicken Sie auf "OK".
