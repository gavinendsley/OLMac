Ändern einer Konfiguration für den Zugriff auf ein LDAP-Verzeichnis

Sie können mit dem Programm "Verzeichnisdienste" die Einstellungen einer Konfiguration für ein LDAP-Verzeichnis jederzeit ändern. Die Konfigurationseinstellungen geben an, auf welche Weise Open Directory auf ein bestimmtes LDAPv3- oder LDAPv2-Verzeichnis zugreift.

Eine von DHCP bereitgestellte LDAP-Konfiguration kann nicht geändert werden. Aus diesem Grund wird eine Konfiguration dieses Typs in der Liste der LDAP-Konfigurationen grau dargestellt.

Gehen Sie wie folgt vor, um eine Konfiguration für den Zugriff auf ein LDAP-Verzeichnis zu bearbeiten:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Wählen Sie in der Liste der Dienste "LDAPv3" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  8. Wenn die Liste der Serverkonfigurationen ausgeblendet ist, klicken Sie auf "Optionen einblenden".

  9. Ändern Sie bei Bedarf die folgenden Einstellungen:

    • Aktiv: Klicken Sie auf ein Feld, um den Zugriff auf den jeweiligen LDAP-Verzeichnisserver zu aktivieren bzw. zu deaktivieren.

    • Konfigurationsname: Wählen Sie zum Bearbeiten einen Konfigurationsnamen durch Doppelklicken aus.

    • Servername oder IP-Adresse: Wählen Sie zum Ändern einen Servernamen oder eine IP-Adresse durch Doppelklicken aus.

    • LDAP-Pfade: Wählen Sie eine Vorlage aus dem Einblendmenü aus, geben Sie das Suchbeginnsuffix für das LDAP-Verzeichnis ein und klicken Sie auf "OK".

      Wenn Sie eine Vorlage wählen, müssen Sie ein Suchbeginnsuffix eingeben, da ohne dieses Suffix der Computer nicht in der Lage ist, Informationen im LDAP-Verzeichnis zu finden. Im Normalfall wird das Suffix für den Suchbeginn vom DNS-Namen des Servers abgeleitet. Für einen Server mit dem DNS-Namen ods.example.com lautet das Suffix beispielsweise "dc=ods,dc=example,dc=com".

      Wenn Sie sich nicht für eine Vorlage, sondern für die Einstellung "Vom Server" entscheiden, ist das Suffix für den Suchbeginn nicht erforderlich. In diesem Fall geht Open Directory davon aus, dass das Suchbeginnsuffix der ersten Ebene des LDAP-Verzeichnisses entspricht.

      Wenn Sie sich für "Eigene" entscheiden, müssen Sie die Zuordnungen zwischen den Datensatztypen und Attributen von Mac OS X auf der einen Seite und den Klassen und Attributen des LDAP-Verzeichnisses, zu dem Sie die Verbindung herstellen, auf der anderen Seite festlegen. Weitere Informationen finden Sie unter Konfigurieren von LDAP-Suchen und -Zuordnungen.

    • SSL: Klicken Sie auf dieses Markierungsfeld, um die verschlüsselte Kommunikation auf der Basis des Protokolls SSL zu aktivieren oder zu deaktivieren. Fragen Sie vor Auswahl des Markierungsfelds "SSL" Ihren Open Directory-Administrator, ob eine SSL-Verschlüsselung erforderlich ist.

  10. Klicken Sie zum Ändern der folgenden Standardeinstellungen für diese LDAP-Konfiguration auf "Bearbeiten", um die Optionen für die ausgewählte LDAP-Konfiguration einzublenden. Nehmen Sie dann die Änderungen vor und klicken Sie auf "OK", wenn alle Änderungen wunschgemäß festgelegt sind.

    • Klicken Sie auf "Verbindung", um die Optionen für die Zeitüberschreitung festzulegen, einen eigenen Port anzugeben, Serververweise zu ignorieren oder die Verwendung des LDAPv2-Protokolls (schreibgeschützt) zu erzwingen. Weitere Informationen finden Sie unter Ändern der Verbindungseinstellungen für ein LDAP-Verzeichnis.

    • Klicken Sie auf "Suche & Pfade", um die Such- und Zuordnungseinstellungen für einen LDAP-Server festzulegen. Weitere Informationen finden Sie unter Einrichten einer vertrauenswürdigen Bindung für ein LDAP-Verzeichnis.

    • Klicken Sie auf "Sicherheit", um (anstelle einer vertrauenswürdigen Bindung) eine identifizierte Verbindung und andere für die Sicherheitsrichtlinien relevante Optionen einzurichten. Weitere Informationen finden Sie unter Ändern der Sicherheitsrichtlinie für eine LDAP-Verbindung.

    • Klicken Sie auf "Einbinden", wenn Sie eine vertrauenswürdige Bindung einrichten wollen. Klicken Sie auf "Trennen", wenn Sie eine vertrauenswürdige Bindung beenden wollen. (Wenn das jeweilige LDAP-Verzeichnis keine vertrauenswürdige Bindung unterstützt, werden diese Tasten möglicherweise nicht angezeigt.) Weitere Informationen finden Sie unter Einrichten einer vertrauenswürdigen Bindung für ein LDAP-Verzeichnis.

  11. Klicken Sie auf "OK", um die Änderungen an der Konfiguration für den Zugriff auf ein LDAP-Verzeichnis abzuschließen.
