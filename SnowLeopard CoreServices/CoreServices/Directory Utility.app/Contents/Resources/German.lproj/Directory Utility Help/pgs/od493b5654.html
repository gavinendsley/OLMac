Füllen von LDAP-Verzeichnissen mit Daten für Mac OS X

Nach dem Konfigurieren des Zugriffs auf LDAP-Verzeichnis-Domains und dem Einrichten der Datenzuordnung können Sie die LDAP-Verzeichnisse mit Datensätzen (Einträgen) und Daten für Mac OS X füllen. Für LDAP-Verzeichnisse, die eine Verwaltung per Fernzugriff (Schreib-/Lesezugriff) zulassen, können Sie den Arbeitsgruppenmanager von Mac OS X Server wie folgt verwenden:

  • Geben Sie die Netzwerkvolumes und gemeinsam genutzte Domains an, die für Benutzer in der Übersicht über die vorhandenen Netzwerkverbindungen automatisch aktiviert werden sollen (diese werden angezeigt, wenn die Benutzer in der Seitenleiste eines Finder-Fensters auf "Netzwerk" klicken).

    Verwenden Sie die Module "Sharing" im Programm "Server-Admin" und "Netzwerk" im Programm "Arbeitsgruppenmanager".

    Weitere Information hierzu finden Sie in den Dokumenten "Dateiserver-Dienste - Überblick" und "Benutzerverwaltung - Überblick".

  • Definieren Sie Benutzer- und Gruppendatensätze und konfigurieren Sie deren Einstellungen.

    Verwenden Sie dazu das Modul "Accounts" im Arbeitsgruppenmanager.

    Weitere Informationen finden Sie im Dokument "Benutzerverwaltung - Überblick".

  • Definieren Sie Listen von Computern, für die dieselben Einstellungen gelten und die denselben Benutzern und Gruppen zur Verfügung stehen.

    Verwenden Sie dazu das Modul "Computer" im Arbeitsgruppenmanager.

    Weitere Informationen finden Sie im Dokument "Benutzerverwaltung - Überblick".

Klicken Sie in allen Fällen auf das kleine Globussymbol über der Benutzerliste und wählen Sie die gewünschte Option aus dem Einblendmenü im Arbeitsgruppenmanager aus, um die LDAP-Verzeichnis-Domain zu öffnen. Wenn das LDAP-Verzeichnis nicht im Einblendmenü aufgeführt ist, wählen Sie "Andere" aus diesem Menü, um das LDAP-Verzeichnis auszuwählen.

Hinweis:  Zum Hinzufügen von Datensätzen und Daten zu einem schreibgeschützten LDAP-Verzeichnis müssen Sie Dienstprogramme auf dem Server verwenden, auf dem sich das schreibgeschützte LDAP-Verzeichnis befindet.
