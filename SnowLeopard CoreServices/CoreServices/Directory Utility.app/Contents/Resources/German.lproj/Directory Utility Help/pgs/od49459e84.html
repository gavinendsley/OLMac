Einrichten des Programms "Verzeichnisdienste" auf einem entfernten Server

Sie können das Programm "Verzeichnisdienste" auf Ihrem Computer verwenden, um festzulegen und zu steuern, wie Mac OS X Server auf einem entfernten Server auf die Verzeichnisdienste zugreift.

Gehen Sie wie folgt vor, um den Zugriff auf die Verzeichnisdienste auf einem entfernten Server zu konfigurieren:

  1. Öffnen Sie die Systemeinstellungen auf Ihrem Computer und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

    Wird die Taste "Bearbeiten" angezeigt, verfügt Ihr Computer über mindestens eine Verbindung zu einem Verzeichnisserver.

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Wählen Sie "Ablage" > "Verbinden".

  6. Geben Sie die folgenden Verbindungs- und Identifizierungsinformationen für den Server ein, den Sie konfigurieren möchten.

    • Adresse: Geben Sie den DNS-Namen oder die IP-Adresse des Servers ein, den Sie konfigurieren wollen.

    • Benutzername: Geben Sie den Benutzernamen eines Serveradministrators ein.

    • Kennwort: Geben Sie das Kennwort für den eingegebenen Benutzernamen ein.

  7. Klicken Sie auf "Verbinden".

  8. Klicken Sie auf die Titel "Dienste" und "Suchpfad" und ändern Sie die Einstellungen nach Bedarf.

    Vorgenommene Änderungen wirken sich auf den entfernten Server aus, zu dem Sie in den vorangegangenen Schritten eine Verbindung hergestellt haben.

  9. Wählen Sie "Trennen" aus dem Menü "Ablage" auf Ihrem Computer.
