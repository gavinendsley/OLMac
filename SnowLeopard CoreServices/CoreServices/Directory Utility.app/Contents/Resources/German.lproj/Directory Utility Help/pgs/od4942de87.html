Festlegen der NIS-Einstellungen

Mithilfe des Programms "Verzeichnisdienste" können Sie eine Konfiguration erstellen, die angibt, wie Mac OS X auf eine NIS-Domain (Network Information Service) zugreift.

Gehen Sie wie folgt vor, um eine Konfiguration für den Zugriff auf eine NIS-Domain zu erstellen:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Wählen Sie in der Liste der Dienste "Lokale BSD-Konfigurationsdateien und NIS" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  8. Geben Sie den Namen der NIS-Domain bzw. den DNS-Namen oder die IP-Adresse des Servers ein, auf dem sich die NIS-Domain befindet.

    Geben Sie den Hostnamen oder die IP-Adresse des NIS-Servers an, wenn diese Angabe aus Sicherheitsgründen benötigt wird oder wenn sich der Server nicht in demselben Teilnetzwerk wie der von Ihnen konfigurierte Computer befindet.

    Wenn Sie keinen Server angeben, verwendet NIS ein Broadcast-Protokoll, um einen NIS-Server im Teilnetzwerk zu erkennen.

  9. Wählen Sie "NIS-Domain zur Identifizierung verwenden" und klicken Sie auf "OK".

    Die NIS-Domain wird dem Suchpfad für die Identifizierung des Computers in folgender Form hinzugefügt: /BSD/Domain. Dabei steht Domain für den Namen, den Sie in Schritt 4 eingegeben haben.
