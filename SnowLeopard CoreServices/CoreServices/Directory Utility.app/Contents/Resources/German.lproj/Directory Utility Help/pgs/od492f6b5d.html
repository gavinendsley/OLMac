Aktivieren des root-Accounts

Sie können das Programm "Verzeichnisdienste" dazu verwenden, den root-Account zu aktivieren. Wenn Sie sich dafür entscheiden, den root-Account zu aktivieren, sollten Sie ein komplexes Kennwort festlegen, das auch alphanumerische Zeichen und Sonderzeichen enthält, um das Entschlüsseln des Kennworts nach Möglichkeit zu verhindern.

ACHTUNG: Der root-Account ist ein Account mit uneingeschränkten Administratorberechtigungen, der auch Änderungen an wichtigen Systemdateien erlaubt. Auch wenn Sie als Administrator angemeldet sind, müssen Sie, um wichtige Systemaufgaben durchzuführen, den root-Account bzw. den Befehl sudo verwenden.

Verwenden Sie den root-Account nie für die Anmeldung an einem Computer (egal ob per Fernzugriff oder lokal). Führen Sie root-Aufgaben stattdessen mit dem Befehl sudo aus. Sie können den Zugriff auf den Befehl sudo einschränken, indem Sie Benutzer zur Datei /etc/sudoers/ hinzufügen.

Weitere Informationen zum root-Account finden Sie in der Hilfe zum Programm "Server-Admin".

Gehen Sie wie folgt vor, um den root-Account zu aktivieren:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

    Wird die Taste "Bearbeiten" angezeigt, verfügt Ihr Computer über mindestens eine Verbindung zu einem Verzeichnisserver.

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Wählen Sie "Bearbeiten > "root-Benutzer aktivieren".
