Aktivieren oder Deaktivieren von LDAP-Verzeichnisdiensten

Sie können im Programm "Verzeichnisdienste" den Zugriff auf Verzeichnisdienste aktivieren und deaktivieren, die LDAPv2 oder LDAPv3 verwenden. Der Zugriff auf LDAPv2 und LDAPv3 erfolgt über das Plug-In "LDAPv3" des Programms "Verzeichnisdienste".

Die von Mac OS X Server bereitgestellten Verzeichnisdienste verwenden LDAPv3, das auch von vielen anderen Servern eingesetzt wird. LDAPv3 ist ein offener Standard, der häufig in heterogenen Netzwerken aus Macintosh-, UNIX- und Windows-Systemen verwendet wird. Einige Server verwenden jedoch die ältere Version LDAPv2 für Verzeichnisdienste.

Wenn LDAP-Verzeichnisse in einen eigenen Suchpfad eingebunden sind und Sie die LDAP-Verzeichnisdienste deaktivieren, werden die betreffenden Verzeichnisdienste in den Bereichen "Identifizierung" und "Kontakte" rot angezeigt.

Gehen Sie wie folgt vor, um die LDAP-Verzeichnisdienste zu aktivieren oder zu deaktivieren:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Aktivieren bzw. deaktivieren Sie das Markierungsfeld "LDAPv3" und klicken Sie auf "Anwenden".
