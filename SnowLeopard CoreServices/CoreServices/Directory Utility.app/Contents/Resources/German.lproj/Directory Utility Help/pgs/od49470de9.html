Definieren automatischer Suchpfade

Im Programm "Verzeichnisdienste" können Sie veranlassen, dass die Suchpfade, die für einen Mac OS X-Computer bei der Benutzeridentifizierung und der Suche nach Kontakten verwendet werden, automatisch definiert werden.

Ein automatisch definierter Suchpfad enthält die lokale Verzeichnis-Domain. Er kann auch einen LDAP-Verzeichnisserver enthalten, der vom DHCP-Dienst definiert wird.

Dies ist die Standardkonfiguration für den Suchpfad für die Benutzeridentifizierung sowie für die Kontakte.

Hinweis:   Die Mac OS X-Programme "Mail" und "Adressbuch" sowie einige andere Programme sind in der Lage, unter Umgehung von Open Directory direkt auf LDAP-Verzeichnisse zuzugreifen. Wenn Sie eines dieser Programme für den direkten Zugriff auf LDAP-Verzeichnisse konfigurieren wollen, müssen Sie das betreffende Programm und die entsprechende Voreinstellung festlegen.
Wichtig:  Wenn Sie Mac OS X so konfigurieren, dass für die Benutzeridentifizierung ein automatischer Suchpfad mit einem über DHCP bereitgestellten LDAP-Server oder eine über DHCP als gemeinsame Ressource bereitgestellte Verzeichnis-Domain verwendet wird, steigt das Risiko, dass ein unbefugter Benutzer die Kontrolle über Ihren Computer erlangen kann. Dieses Risiko ist noch höher, wenn Ihr Computer für den Zugang zu einem drahtlosen Netzwerk konfiguriert ist. Weitere Informationen finden Sie unter Schutz von Computern vor Angriffen durch fingierte DHCP-Server.
Gehen Sie wie folgt vor, wenn der Suchpfad automatisch definiert werden soll:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Suchpfad" und wählen Sie einen Suchpfad aus:

    • Identifizierung: In diesem Bereich wird der Suchpfad angezeigt, der für die Benutzeridentifizierung und die meisten administrativen Aufgaben verwendet wird.

    • Kontakte: In diesem Bereich wird der Suchpfad angezeigt, der für Kontaktinformationen im Programm "Adressbuch" und anderen Programmen verwendet wird.

  7. Wählen Sie "Automatisch" aus dem Einblendmenü "Suchen" aus. Klicken Sie danach auf "Anwenden".

  8. Vergewissern Sie sich, dass für Ihren Computer in der Systemeinstellung "Netzwerk" die Verwendung von DHCP bzw. von DHCP in Verbindung mit einer manuellen IP-Adresse konfiguriert ist.
