Konfigurieren des Zugriffs auf eine Active Directory-Domain

Mithilfe der im Programm "Verzeichnisdienste" aufgeführten ADC-Komponente (Active Directory Connector) können Sie Mac OS X konfigurieren, auf grundlegende Benutzer-Account-Informationen in einer Active Directory-Domain auf einem Windows-Server zuzugreifen.

Die ADC-Komponente generiert alle für die Mac OS X-Identifizierung benötigten Attribute. Es sind keine Änderungen am Active Directory-Schema erforderlich.

Außerdem erkennt und verwendet die ADC-Komponente standardmäßige Mac OS X-Datensatztypen und -Attribute (wie die für die Mac OS X-Client-Verwaltung benötigten Attribute), wenn das Active Directory-Schema um diese erweitert wurde.

ACHTUNG: Die erweiterten Optionen der ADC-Komponente bieten Ihnen die Möglichkeit, die eindeutige Mac OS X-Benutzer-ID (UID), die primäre Gruppen-ID (GID) und die Gruppen-GID den richtigen Attributen zuzuordnen, die dem Active Directory-Schema hinzugefügt wurden. Wenn Sie die Einstellung dieser Zuordnungsoptionen zu einem späteren Zeitpunkt ändern, können Benutzer den Zugriff auf zuvor erstellte Dateien verlieren.
Wichtig: Falls der Name eines Computers einen Bindestrich (-) enthält, besteht die Gefahr, dass der Versuch scheitert, einer Verzeichnis-Domain wie LDAP oder Active Directory beizutreten und die Verbindung herzustellen. Für den Beitritt müssen Sie gegebenenfalls den Bindestrich aus dem Namen des betreffenden Computers entfernen.
Gehen Sie wie folgt vor, um den Zugriff auf eine Active Directory-Domain zu konfigurieren:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Wählen Sie in der Liste der Dienste "Active Directory" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  8. Geben Sie den DNS-Namen der Active Directory-Domain ein, an die der Computer, den Sie gerade konfigurieren, angebunden werden soll.

    Den DNS-Namen, den Sie eingeben müssen, können Sie beim Administrator der Active Directory-Domain erfahren.

  9. Bearbeiten Sie bei Bedarf die Computer-ID.

    Die Computer-ID ist der Name, unter dem der Computer in der Active Directory-Domain bekannt ist. Als Voreinstellung wird dafür der Name des Computers verwendet. Unter Umständen müssen Sie diesen Namen ändern, z. B. damit er den Namenskonventionen Ihres Unternehmens für die Active Directory-Domain entspricht. Bei Unsicherheiten wenden Sie sich an den Administrator der Active Directory-Domain.

  10. (Optional) Legen Sie die erweiterten Optionen fest.

    Wenn die erweiterten Optionen ausgeblendet sind, klicken Sie auf "Erweiterte Optionen einblenden" und legen Sie die Optionen in den Bereichen "Benutzereinstellungen", "Pfade" und "Administration" fest. Sie können die Einstellungen für erweiterte Optionen auch zu einem späteren Zeitpunkt ändern.

    Weitere Informationen über die erweiterten Optionen finden Sie in den folgenden Themen:

    • Einrichten mobiler Benutzer-Accounts in Active Directory

    • Einrichten von Benutzerordnern für Active Directory-Benutzer-Accounts

    • Einrichten einer UNIX-Shell für Active Directory-Benutzer-Accounts

    • Zuordnen der UID zu einem Active Directory-Attribut

    • Zuordnen der primären Gruppen-ID zu einem Active Directory-Attribut

    • Zuordnen der Gruppen-ID in Gruppen-Accounts zu einem Active Directory-Attribut

    • Angeben eines bevorzugten Active Directory-Servers

    • Ändern der Active Directory-Gruppen mit Verwaltungsfunktion für den Computer

    • Steuern der Identifizierung aus allen Domains in der Active Directory-Gesamtstruktur

  11. Klicken Sie auf "Einbinden" und identifizieren Sie sich mithilfe der folgenden Angaben als ein Benutzer, der berechtigt ist, einen Computer mit einer Active Directory-Domain zu verbinden. Wählen Sie die Suchpfade aus, denen Active Directory hinzugefügt werden soll (siehe unten), und klicken Sie auf "OK":

    • Benutzername und Kennwort: Möglicherweise können Sie sich durch Eingabe des Namens und des Kennworts Ihres Active Directory-Benutzer-Accounts identifizieren. Andernfalls muss Ihnen der Administrator der Active Directory-Domain einen Namen und ein Kennwort mitteilen.

    • Computer-OU: Geben Sie die zutreffende Organisationseinheit (Organizational Unit bzw. OU) für den Computer ein, den Sie konfigurieren.

    • Für Identifizierung verwenden: Hier können Sie festlegen, ob Active Directory dem Suchpfad Ihres Computers für die Identitätsüberprüfung hinzugefügt wird.

    • Für Kontakte verwenden: Hier können Sie festlegen, ob Active Directory dem Suchpfad Ihres Computers für Kontakte hinzugefügt wird.

    Wenn Sie auf "OK" klicken, konfiguriert das Programm "Verzeichnisdienste" eine vertrauenswürdige Verbindung zwischen dem Computer, den Sie gerade konfigurieren, und dem Active Directory-Server. Die Suchpfade des Computers werden gemäß den Optionen konfiguriert, die Sie ausgewählt haben, als Sie sich identifiziert haben. Außerdem wird "Active Directory" im Bereich "Dienste" des Programms "Verzeichnisdienste" aktiviert.

    Mit den Standardeinstellungen für die erweiterten Optionen von Active Directory wird die Active Directory-Gesamtstruktur dem Suchpfad des Computers für die Identifizierung und für Kontakte hinzugefügt, wenn Sie die Option "Für Identifizierung verwenden" oder "Für Kontakte verwenden" ausgewählt haben.

    Wenn Sie jedoch die Option "Identifizierung aus jeder Domain in der Gesamtstruktur ermöglichen" im Abschnitt mit den erweiterten Optionen des Bereichs "Administration" deaktivieren, bevor Sie auf "Einbinden" klicken, wird anstelle der Gesamtstruktur die nächstgelegene Active Directory-Domain hinzugefügt.

    Sie können die Suchpfade zu einem späteren Zeitpunkt ändern, indem Sie die Active Directory-Gesamtstruktur oder einzelne Domains hinzufügen oder löschen. Weitere Informationen finden Sie unter Definieren eigener Suchpfade.

  12. (Optional) Binden Sie den Server an den Active Directory-Kerberos-Realm an:

    • Öffnen Sie auf dem Server oder einem Administratorcomputer, der die Verbindung zum Server herstellen kann, das Programm "Server-Admin" und wählen Sie "Open Directory" für den Server aus.

    • Klicken Sie auf "Einstellungen" und anschließend auf "Allgemein".

    • Klicken Sie auf "Kerberos beitreten". Wählen Sie dann den Active Directory-Kerberos-Realm aus dem Einblendmenü aus und geben Sie die Anmeldedaten eines lokalen Administrators auf diesem Server ein.
