Ändern der Active Directory-Gruppen mit Verwaltungsfunktion für den Computer

Auf einem Computer, der so konfiguriert ist, dass er die ADC-Komponente (Active Directory Connector) des Programms "Verzeichnisdienste" verwendet, können Sie Active Directory-Gruppen-Accounts angeben, deren Mitglieder Administratorrechte für den Computer erhalten sollen.

Benutzer, die Mitglieder dieser Active Directory-Benutzer-Accounts sind, können Verwaltungsaufgaben ausführen wie z. B. das Installieren von Software auf dem von Ihnen konfigurierten Mac OS X-Computer.

Gehen Sie wie folgt vor, um Active Directory-Gruppen-Accounts hinzuzufügen bzw. zu löschen, deren Mitglieder Administratorrechte haben:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Wählen Sie in der Liste der Dienste "Active Directory" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  8. Wenn die erweiterten Optionen ausgeblendet sind, klicken Sie auf "Erweiterte Optionen einblenden".

  9. Klicken Sie auf "Administration".

  10. Wählen Sie "Verwaltung ermöglichen durch" aus und ändern Sie die Liste der Active Directory-Gruppen-Accounts, deren Mitglieder Administratorrechte erhalten sollen:

    • Fügen Sie eine Gruppe hinzu, indem Sie auf die Taste "Hinzufügen" (+) klicken, den Active Directory-Domain-Namen eingeben, einen umgekehrten Schrägstrich (\) einfügen und den Namen des Gruppen-Accounts eingeben (Beispiel: ADS\Domain Admins,IL2\Domain Admins).

    • Löschen Sie eine Gruppe, indem Sie sie in der Liste auswählen und anschließend auf die Taste "Löschen" (-) klicken.

  11. Klicken Sie auf "OK".
