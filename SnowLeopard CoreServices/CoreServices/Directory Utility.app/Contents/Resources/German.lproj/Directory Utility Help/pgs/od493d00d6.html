Zuordnen der Primärgruppen-ID zu einem Active Directory-Attribut

Auf einem Computer, der so konfiguriert ist, dass die ADC-Komponente (Active Directory Connector) des Programms "Verzeichnisdienste" verwendet wird, können Sie ein Active Directory-Attribut definieren, das der primären Gruppen-ID (GID) in Mac OS X-Benutzer-Accounts zugeordnet werden soll.

Das Active Directory-Schema muss im Allgemeinen erweitert werden, um ein Attribut zu enthalten, das für die Zuordnung zur primären GID geeignet ist:

  • Wenn der Active Directory-Administrator das Active Directory-Schema erweitert, indem er die Microsoft-Dienste für UNIX installiert, können Sie die primäre GID dem Attribut "msSFU-30-Gid-Number" zuordnen.

  • Wenn der Active Directory-Administrator das Active Directory-Schema manuell so erweitert, dass es das Attribut "RFC 2307" enthält, können Sie die primäre GID dem Attribut "gidNumber" zuordnen.

  • Wenn der Active Directory-Administrator das Active Directory-Schema manuell so erweitert, dass es das Mac OS X-Attribut "PrimaryGroupID" enthält, können Sie die primäre GID diesem Attribut zuordnen.

Wenn die Zuordnung der primären GID deaktiviert ist, generiert die ADC-Komponente eine primäre GID auf der Basis des standardmäßigen Active Directory-GUID-Attributs.

ACHTUNG: Wenn Sie die Zuordnung der primären GID zu einem späteren Zeitpunkt ändern, können Benutzer den Zugriff auf zuvor erstellte Dateien verlieren.
Gehen Sie wie folgt vor, um die primäre GID einem Attribut in einem erweiterten Active Directory-Schema zuzuordnen:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Wählen Sie in der Liste der Dienste "Active Directory" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  8. Wenn die erweiterten Optionen ausgeblendet sind, klicken Sie auf "Erweiterte Optionen einblenden".

  9. Klicken Sie auf "Pfade".

  10. Wählen Sie dann "Eindeutige Benutzerkennung (GID) auf Attribut zuordnen"" und geben Sie den Namen des Active Directory-Attributs ein, das der primären Gruppen-ID in Benutzer-Accounts zugeordnet werden soll.

  11. Klicken Sie auf "OK".
