Festlegen von Einstellungen für die BSD-Konfigurationsdatei

Auf UNIX-Computern wurden administrative Daten seit jeher in Konfigurationsdateien wie "/etc/master.passwd", "/etc/group" und "/etc/hosts" gespeichert. Mac OS X basiert auf einer BSD-Version von UNIX, bezieht administrative Daten aber normalerweise aus Verzeichnissystemen.

Mac OS X Server unterstützt eine bestimmte Gruppe von BSD-Konfigurationsdateien. Sie können weder angeben, welche Konfigurationsdateien verwendet werden sollen, noch können Sie deren Inhalt Mac OS X-Datensatztypen und Attributen zuordnen.

Unter Mac OS X 10.2 oder neuer (einschließlich Mac OS X Server 10.2 oder neuer) kann Open Directory administrative Daten aus BSD-Konfigurationsdateien abrufen. Diese Fähigkeit ermöglicht Unternehmen, die BSD-Konfigurationsdateien haben, Kopien der vorhandenen Dateien auf Mac OS X-Computern zu verwenden. BSD-Konfigurationsdateien können allein oder mit anderen Verzeichnis-Domains verwendet werden.

Gehen Sie wie folgt vor, um BSD-Konfigurationsdateien zu verwenden:

  1. Vergewissern Sie sich, dass die BSD-Konfigurationsdateien die Daten enthalten, die für Mac OS X-Verzeichnisdienste erforderlich sind.

    Weitere Informationen finden Sie unter Konfigurieren von Daten in BSD-Konfigurationsdateien.

  2. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  3. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  4. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

  5. Klicken Sie auf "Verzeichnisdienste öffnen".

  6. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  7. Klicken Sie auf "Dienste".

  8. Wählen Sie in der Liste der Dienste "Lokale BSD-Konfigurationsdateien und NIS" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  9. Wählen Sie "Benutzer- und Gruppeneinträge im lokalen BSD-Knoten verwenden" aus und klicken Sie auf "OK".

    Die Domain für BSD-Konfigurationsdateien wird dem Suchpfad für die Identifizierung des Computers in folgender Form hinzugefügt: /BSD/local.
