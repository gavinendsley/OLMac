Einrichten einer vertrauenswürdigen Bindung für ein LDAP-Verzeichnis

Sie können mithilfe des Programms "Verzeichnisdienste" eine vertrauenswürdige Bindung zwischen dem Computer und einem LDAP-Verzeichnis einrichten, das eine solche Bindung unterstützt. Die Bindung wird auf der Basis eines identifizierten Computereintrags wechselseitig identifiziert. Dieser Eintrag wird beim Einrichten einer vertrauenswürdigen Bindung im Verzeichnis erstellt.

Es ist nicht möglich, einen Computer so zu konfigurieren, dass gleichzeitig eine vertrauenswürdige LDAP-Bindung und ein über DHCP bereitgestelltes LDAP-Verzeichnis verwendet werden. Bei einer vertrauenswürdigen LDAP-Bindung handelt es sich um eine inhärent statische Bindung, während bei einem über DHCP bereitgestellten LDAP-Verzeichnis eine dynamische Bindung vorliegt.

Gehen Sie wie folgt vor, um eine vertrauenswürdige Bindung zu einem LDAP-Verzeichnis einzurichten:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  6. Klicken Sie auf "Dienste".

  7. Wählen Sie in der Liste der Dienste "LDAPv3" aus und klicken Sie auf die Taste "Bearbeiten" (/).

  8. Wenn die Liste der Serverkonfigurationen ausgeblendet ist, klicken Sie auf "Optionen einblenden".

  9. Wählen Sie aus der Liste die gewünschte Serverkonfiguration aus und klicken Sie auf "Bearbeiten".

  10. Klicken Sie auf "Einbinden" und geben Sie dann die folgenden Anmeldedaten ein. Klicken Sie anschließend auf "OK".

    Geben Sie den Namen des Computers sowie den Namen und das Kennwort eines Administrators der LDAP-Verzeichnis-Domain ein. Der Computername, den Sie hier eingeben, darf nicht von einem anderen Computer für eine vertrauenswürdige Bindung oder für andere Netzwerkdienste verwendet werden.

    Unterstützt das ausgewählte LDAP-Verzeichnis keine vertrauenswürdigen Bindungen, wird die Taste "Einbinden" nicht angezeigt.

  11. Vergewissern Sie sich, dass der von Ihnen eingegebene Computername richtig ist.

    Wenn Sie informiert werden, dass bereits ein Eintrag für den eingegebenen Computernamen existiert, können Sie auf "Abbrechen" klicken, wenn Sie den eingegebenen Computernamen ändern wollen. Klicken Sie auf "Überschreiben", wenn der vorhandene Computereintrag überschrieben werden soll.

    Bei einem vorhandenen Computereintrag besteht die Möglichkeit, dass er nicht mehr verwendet wird; es kann aber auch sein, dass er Bezug auf einen anderen Computer nimmt. Sie sollten daher stets den Administrator des LDAP-Verzeichnisses entsprechend informieren, wenn Sie einen vorhandenen Computereintrag ersetzen, da durch das Ersetzen möglicherweise ein anderer Computer deaktiviert wird.

    In einer solchen Situation muss der für das LDAP-Verzeichnis zuständige Administrator dem deaktivierten Computer einen neuen Namen zuordnen und den Computer unter Verwendung dieses neu zugeordneten Namens wieder der Computergruppe hinzufügen, der er ursprünglich angehörte.

  12. Klicken Sie auf "OK", um das Einrichten der vertrauenswürdigen Bindung abzuschließen.
