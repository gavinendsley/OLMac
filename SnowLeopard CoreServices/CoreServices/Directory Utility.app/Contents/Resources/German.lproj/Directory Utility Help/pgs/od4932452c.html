Hinzufügen einer Verbindung zu einem Active Directory-Server

Damit Sie die Verbindung zu einem Active Directory-Server herstellen können, müssen Ihnen der Servername oder die IP-Adresse des Servers sowie der Benutzername und das Kennwort bekannt sein, die den Zugriff als Administrator auf den Active Directory-Server erlauben.

Gehen Sie wie folgt vor, um einen Active Directory-Server hinzuzufügen:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

    Wird die Taste "Bearbeiten" angezeigt, verfügt Ihr Computer über mindestens eine Verbindung zu einem Verzeichnisserver.

  4. Klicken Sie auf die Taste "Hinzufügen" (+).

  5. Wählen Sie aus dem Einblendmenü "Ein neues Verzeichnis hinzufügen - vom Typ" die Option "Active Directory" aus und geben Sie folgende Informationen ein:

    • Active Directory-Domain: Hierbei handelt es sich um den DNS-Namen der Active Directory-Domain (z. B. ads.company.com.)

    • Computer-ID:  Bearbeiten Sie bei Bedarf die ID, die Active Directory für Ihren Server verwenden soll. Dies ist der NetBIOS-Name des Servers. Der Name sollte nicht mehr als 15 Zeichen enthalten, wobei keine Sonderzeichen und Interpunktionszeichen verwendet werden dürfen. Wenn möglich verwenden Sie den nicht qualifizierten DNS-Hostnamen als Servernamen. Wenn Ihr Server auf dem DNS-Server beispielsweise mit "server.beispiel.de" eingetragen ist, benennen Sie Ihren Server "server".

    • AD-Administrator-Benutzername und -Kennwort: Geben Sie den Benutzernamen und das Kennwort des Active Directory-Administrators ein.

  6. Klicken Sie auf "OK".
