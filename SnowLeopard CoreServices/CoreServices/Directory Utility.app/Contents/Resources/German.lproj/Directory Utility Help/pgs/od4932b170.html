Verbindungen zu Verzeichnisservern

Mit der Systemeinstellung "Benutzer" können Sie Verbindungen zwischen Computern und Verzeichnisservern herstellen. Indem Sie auf die Taste "Bearbeiten" im Bereich "Anmeldeoptionen" der Systemeinstellung "Benutzer" klicken, können Sie eine Liste der Verzeichnisserver anzeigen, mit denen der jeweilige Computer verbunden ist. Ihr Mac OS X-Computer greift auf diese aufgelisteten Server zu, um Benutzerinformationen und andere administrative Daten abzurufen, die auf diesen Verzeichnisservern in den jeweiligen Verzeichnis-Domains gespeichert sind.

Wenn Sie der Liste "Verzeichnisserver" einen weiteren Server hinzufügen oder einen vorhandenen Server aus dieser Liste entfernen, werden automatisch auch die entsprechenden Einträge in den Bereichen "Dienste", "Identifizierung" und "Kontakte" hinzugefügt bzw. entfernt. Wenn Sie umgekehrt die vorhandenen Einträge für einen bestimmten Server im Bereich "Dienste", "Identifikation" und "Kontakte" entfernen, wird nicht automatisch auch der übergeordnete Verzeichnisserver aus der Liste "Verzeichnisserver" entfernt.

Für Computer unter Mac OS X 10.6 können Sie Verbindungen zu Open Directory-, Active Directory- oder LDP-Verzeichnisservern herstellen. Bitten Sie Ihren Netzwerkadministrator um Rat, wenn Sie sich nicht sicher sind, zu welchem Server Sie eine Verbindung herstellen sollen.

Wichtig: Falls der Name eines Computers einen Bindestrich (-) enthält, besteht die Gefahr, dass der Versuch scheitert, einer Verzeichnis-Domain wie LDAP oder Active Directory beizutreten und die Verbindung herzustellen. Für den Beitritt müssen Sie gegebenenfalls den Bindestrich aus dem Namen des betreffenden Computers entfernen.
