Ändern des Kennworts für den root-Account

Sie können das Programm "Verzeichnisdienste" verwenden (über die Systemeinstellung "Benutzer" verfügbar), um das Kennwort des root-Accounts zu ändern. Wenn Sie das Kennwort für den root-Account ändern, sollten Sie ein komplexes Kennwort festlegen, das alphanumerische Zeichen und Sonderzeichen enthält, um das Entschlüsseln des Kennworts nach Möglichkeit zu verhindern.

ACHTUNG: Der root-Account ist ein Account mit uneingeschränkten Administratorberechtigungen, der auch Änderungen an wichtigen Systemdateien erlaubt. Auch wenn Sie als Administrator angemeldet sind, müssen Sie, um wichtigen Systemaufgaben durchzuführen, den root-Account bzw. den Befehl sudo verwenden.

Verwenden Sie den root-Account nie für die Anmeldung an einem Computer (egal ob per Fernzugriff oder lokal). Führen Sie root-Aufgaben stattdessen mit dem Befehl sudo aus. Sie können den Zugriff auf den Befehl sudo einschränken, indem Sie Benutzer zur Datei "/etc/sudoers/" hinzufügen.

Weitere Informationen zum root-Account finden Sie in der Hilfe zum Programm "Server-Admin". Weitere Informationen zum Erstellen eines Kennworts finden Sie unter Erstellen eines Kennworts.

Gehen Sie wie folgt vor, um das Kennwort für den root-Account zu ändern:

  1. Öffnen Sie die Systemeinstellungen und klicken Sie auf "Benutzer".

  2. Klicken Sie auf das Schlosssymbol, falls es geschlossen ist, und geben Sie den Namen und das Kennwort eines Administrators ein.

  3. Klicken Sie auf "Anmeldeoptionen" und dann auf "Beitreten" oder "Bearbeiten".

    Wird die Taste "Bearbeiten" angezeigt, verfügt Ihr Computer über mindestens eine Verbindung zu einem Verzeichnisserver.

  4. Klicken Sie auf "Verzeichnisdienste öffnen".

  5. Wählen Sie "Bearbeiten" > "root-Kennwort ändern".

  6. Geben Sie das neue Kennwort in die Felder "Kennwort" und "Bestätigen" ein.

  7. Klicken Sie auf "OK".
