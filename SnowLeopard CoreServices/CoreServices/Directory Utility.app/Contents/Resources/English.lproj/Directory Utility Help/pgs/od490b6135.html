Learn About Directory Utility

Directory Utility determines which directory services a Mac OS X computer uses and how it connects to specific directory domains. Directory Utility also defines search policies for finding authentication and contacts information in specific directory domains.

Setting Up Directory Utility

You can use Directory Utility on your computer to set up and manage how your computer accesses directory services. You can also use Directory Utility to remotely configure how your computer accesses directory services. Directory Utility can also be used to enable the root account or change the root account password.

Configuring Access to Services

You can use Directory Utility to enable and disable different kinds of services that Mac OS X can access on a network. These include directory services that provide access to user accounts and other administrative data stored in directory domains.

Configuring Access to Directories

Using Directory Utility, you can configure the computer to access user accounts and other administrative information in specific LDAP directories, an Active Directory domain, and NIS domain.

Configuring Access to Directories
Setting Up Search Policies

With Directory Utility, you can define the authentication search policy that Mac OS X uses to locate and retrieve user authentication information and other administrative data from directory domains. Directory Utility can also define the contacts search policy that Mac OS X uses to locate and retrieve name, address, and other contact information from directory domains.
