Hakukäytäntöjen lisäasetusten käyttäminen

Hakemistotyökalu määrittelee seuraava hakukäytännöt:

  • Todentaminen: Mac OS X käyttää todentamisen hakukäytäntöä paikantaakseen ja hakeakseen hakemistodomaineista käyttäjien todentamistietoja ja muita ylläpitoon liittyviä tietoja.

  • Yhteystiedot: Mac OS X käyttää yhteystietojen hakukäytäntöä paikantaakseen ja hakeakseen nimi-, osoite- ja muita yhteystietoja hakemistodomaineista. Mac OS X:n Osoitekirja käyttää näitä yhteystietoja. Muut ohjelmat voidaan ohjelmoida käyttämään niitä.

    Jokainen hakukäytäntö koostuu luettelosta hakemistodomaineja. Luettelossa olevien hakemistodomainien järjestys määrittelee hakukäytännön. Mac OS X aloittaa etsinnän luettelon alusta ja etsii jokaisesta luetellusta hakemistodomainista, kunnes löytää etsimänsä tiedon tai saavuttaa luettelon lopun tietoa löytämättä.

    Todentamisen ja yhteystietojen hakukäytännöillä voi olla jokin seuraavista asetuksista:

  • Automaattinen: alkaa paikallisella hakemistodomainilla ja voi sisältää DHCP:n ja hakemistodomainien tarjoaman LDAP-hakemiston, johon tietokone on yhteydessä. Tämä on Mac OS X 10.2:n ja uudempien oletusasetus ja tarjoaa eniten joustavuutta kannettaville tietokoneille.

  • Paikallinen hakemisto: sisällyttää ainoastaan paikallisen hakemistodomainin.

  • Muokattu polku: aloittaa paikallisesta hakemistodomainista ja sisällyttää valintasi mukaan LDAP-hakemistoja, Active Directory -domainin, jaettuja hakemistodomaineja, BSD-määrittelytiedostoja ja NIS-domainin.

    /BSD/paikallinen kansio sisällytetään aina hakupolkuun ja se näkyy aina harmaana.

Tärkeää:  Mac OS X:n määritteleminen käyttämään automaattista todentamisen hakukäytäntöä ja DHCP:n tarjoamaa LDAP-palvelinta lisää riskiä, että pahantahtoinen käyttäjä voi saada tietokoneen hallintaansa. Riski on vielä suurempi, jos tietokone on määritelty yhdistämään langattomaan verkkoon. Lisätietoja löytyy kohdasta Tietokoneiden suojaaminen vahingolliselta DHCP-palvelimelta.

Jos haluat nähdä tehtävien kuvaukset ja ohjeet, katso:

  • Automaattisten hakukäytäntöjen määritteleminen

  • Muokattujen hakukäytäntöjen määritteleminen

  • Paikallisten hakemistonhakukäytäntöjen määritteleminen

  • Etsintätavan muutosten voimaantulo
