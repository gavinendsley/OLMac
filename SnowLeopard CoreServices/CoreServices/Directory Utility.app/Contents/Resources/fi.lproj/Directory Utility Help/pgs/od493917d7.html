Vain luku -LDAP-hakemiston valmisteleminen Mac OS X:ää varten

Jos haluat Mac OS X -tietokoneen hakevan ylläpitotietoja vain luku -LDAP-hakemistosta, tietojen on oltava Mac OS X:n vaatimassa muodossa. Vain luku -LDAP-hakemistoon on ehkä lisättävä tietoja tai siinä olevia tietoja on ehkä muutettava tai järjesteltävä uudelleen.

Koska Mac OS X ei voi kirjoittaa vain luku -hakemistoon, muutokset on tehtävä muita työkaluja käyttäen. Työkalujen on oltava palvelimella, jossa LDAP-hakemisto sijaitsee.

Vain luku -LDAP-hakemiston valmisteleminen Mac OS X:ää varten:

  1. Mene palvelimelle, jossa vain luku -LDAP-hakemisto sijaitsee, ja määrittele se tukemaan LDAP-pohjaista todentamista ja salasanan tarkistusta.

  2. Muuta tarvittaessa LDAP-hakemiston objektiluokat ja attribuutit tarjoamaan Mac OS X:n tarvitsemat tiedot.

Tarkempia tietoja Mac OS X -hakemistopalveluiden tarvitsemista tiedoista löytyy Open Directory Administration -dokumentista.
