Active Directory -palvelimen sidoksen purkaminen

Jos tietokone käyttää Hakemistotyökalun Active Directory -liitintä Active Directory -palvelimeen sitoutumiseen, voit poistaa tietokoneen ja Active Directory -palvelimen välisen sidoksen.

Voit pakottaa sidoksen purkamisen, jos et saa yhteyttä palvelimeen tai jos tietokonetietue on poistettu palvelimelta.

Tietokoneen ja Active Directory -palvelimen sidoksen purkaminen:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Palvelut.

  7. Valitse palveluluettelosta Active Directory ja osoita muokkauspainiketta (/).

  8. Osoita Pura, todentaudu käyttäjänä, jolla on oikeudet Active Directory -domainin yhteyden lopettamiseen, ja osoita OK.

    Jos näkyviin tulee varoitus, jossa sanotaan, että valtakirjaa ei hyväksytä tai että tietokone ei saa yhteyttä Active Directoryyn, pakota yhteys päättymään osoittamalla Pakota poistamaan sidos.

    Jos pakotat sidoksen purkamisen, Active Directory sisältää edelleen tietokoneen tietueen. Ilmoita asiasta Active Directoryn ylläpitäjälle, jotta ylläpitäjä tietää poistaa tietokoneen tietueen.

  9. Poista Palvelut-paneelissa Active Directoryn Käytössä-asetuksen valinta ja osoita Käytä.
