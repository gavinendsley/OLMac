Asiakkaan automaattinen määritteleminen

Jos Mac OS X 10.6 -tietokoneesi on yhteydessä verkkoon, mutta ei ole yhteydessä hakemistopalvelimeen, Mac OS X 10.6 -tietokone etsii Mac OS X 10.6 -palvelimia Bonjourilla.

Jos se löytää Mac OS X 10.6 -palvelimen, jossa toimii Open Directory master, jossa on vähintään yksi automaattisesti määriteltävä palvelu (kuten AFP, SMB, VPN, Mail, iCal, iChat, ilmoitukset) ja jos hakemistodomainilla on käyttäjätili, joka vastaa parhaillaan sisäänkirjautuneen Mac OS X 10.6 -tietokoneen käyttäjän käyttäjätunnusta tai pitkää nimeä, Mac OS X 10.6 -tietokoneeseen tulee näkyviin kutsu.

Kutsussa tarjotaan apua hakemistopalvelinyhteyden käyttöönottoon. Voit hyväksyä tai hylätä yhteydenmuodostuksen palvelimeen tai muodostaa yhteyden myöhemmin.

Jos tietokoneesi löytää useamman Mac OS X 10.6 -hakemistopalvelimen, johon se voi muodostaa yhteyden, palvelimet näkyvät luettelossa kutsuponnahdusvalikossa. Jos päätät liittyä palvelimelle, apuri avautuu ja muodostaa yhteydet ohjelmista ja määrittelee ne käyttämään hakemistopalvelimella toimivia palveluita.

Yhteyden muodostaminen hakemistodomainiin automaattista apuria käyttäen:

  1. Kun apuri tulee näkyviin, osoita ‌Ota käyttöön palveluja.

  2. Valitse Palvelin-ponnahdusvalikosta hakemistodomain, johon haluat liittyä.

    Verkossa saattaa olla tarjolla useampia hakemistodomaineja.

    Jos palvelin, johon haluat liittyä, ei ole luettelossa, kirjoita palvelimen nimi tai IP-osoite Palvelin-kenttään ja osoita OK.

  3. Syötä käyttäjätunnus ja salasana, jotka kuuluvat tilille, jolla on oikeus liittää tietokone siihen hakemistodomainiin, johon olet liittymässä.

    Jos sinulla ei ole näitä tietoja, ota yhteyttä hakemiston ylläpitäjään.

  4. Syötä salasana käyttäjätunnukselle, joka näkyy kohdassa Syötä salasana tämän tietokoneen tilille käyttäjätunnus.

    Apuri vaihtaa käyttäjän hakemistotilin salasanan palvelimella vastaamaan paikallisen tietokoneen tilin salasanaa.

  5. Osoita Jatka.

  6. Kun UpgradeUser-työkalu on valmis, osoita Jatka.

  7. Kun näkyviin tulee viesti, jossa selitetään käyttöönotetut palvelut ja kehotetaan kirjautumaan ulos, osoita Kirjaudu ulos.

    Kun kirjaudut sisään, voit aloittaa uusien palveluiden käyttämisen.
