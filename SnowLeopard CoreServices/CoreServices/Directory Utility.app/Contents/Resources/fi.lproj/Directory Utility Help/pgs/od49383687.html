Siirrettävien käyttäjätunnusten asettaminen Active Directoryyn

Voit sallia tai estää liikkuvien Active Directory -käyttäjätilien käytön tietokoneella, joka on määritelty käyttämään Hakemistotyökalun Active Directory -liitintä. Käyttäjät, joilla on liikkuva tunnus, voivat kirjautua käyttäen Active Directory -valtakirjoja silloin, kun tietokone ei ole yhteydessä Active Directory -palvelimeen.

Active Directory -liitin tallentaa välimuistiin valtakirjat käyttäjän siirrettävää tiliä varten, kun käyttäjä kirjautuu sisään tietokoneen ollessa yhteydessä Active Directory -domainiin. Valtakirjojen tallentaminen välimuistiin ei edellytä Active Directory -mallin vaihtamista.

Jos Active Directory -malli on laajennettu sisältämään Mac OS X:n hallitsemat asiakasattribuutit, kyseisiä liikkuvien tilien asetuksia käytetään Active Directory -liittimen liikkuvan tilin asetuksen sijaan.

Liikkuvat tilit voidaan luoda automaattisesti tai voit vaatia Active Directory -käyttäjiä vahvistamaan liikkuvan tilin luomisen.

Liikkuvien tunnusten käyttöön ottaminen ja käytöstä poistaminen Active Directory -domainissa:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Liity tai Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Palvelut.

  7. Valitse palveluluettelosta Active Directory ja osoita muokkauspainiketta (/).

  8. Jos lisäasetukset on kätketty, osoita Näytä lisäasetukset.

  9. Osoita Käyttökokemus, ”Luo liikkuva tili sisäänkirjauduttaessa” ja valinnaisesti ”Vaadi vahvistus ennen liikkuvan tilin luomista”.

    • Jos molemmat vaihtoehdot ovat valittuina, käyttäjä päättää, luodaanko siirrettävä tunnus sisäänkirjautumisen aikana. Kun käyttäjä kirjautuu Mac OS X:ään Active Directory -käyttäjätunnuksella tai verkkokäyttäjänä, käyttäjä näkee valintataulun, jossa on säätimet liikkuvan tunnuksen luomiseen välittömästi.

    • Jos ensimmäinen vaihtoehto valitaan, mutta toista ei, liikkuvat tunnukset luodaan, kun käyttäjät kirjautuvat sisään.

    • Jos ensimmäistä vaihtoehtoa ei valita, toinen vaihtoehto on pois käytöstä.

  10. Osoita OK.
