root-tilin hallinta

Voit hallita root-käyttäjätilejä sallimalla ja estämällä root-käyttäjän Hakemistotyökalulla (sijaitsee Tilit-asetuksissa). Jos sallit root-tilin käytön, voit myös muuttaa root-tilin salasanan Hakemistotyökalulla.

Lisätietoja root-tilin hallinnasta Hakemistotyökalulla löytyy seuraavista ohjeista:

  • root-tilin käyttöönotto

  • root-tilin salasanan vaihtaminen
