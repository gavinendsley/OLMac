Muokattujen hakukäytäntöjen määritteleminen

Hakemistotyökalulla voit määritellä Mac OS X -tietokoneen todentamisen ja yhteystietojen hakukäytännöt käyttämään muokattua hakemistodomainien luetteloa.

Muokattu luettelo alkaa tietokoneen paikallisella hakemistodomainilla ja voi sisältää Open Directory -domaineja (ja muita LDAP-hakemistodomaineja), Active Directory -domainin, jaettuja hakemistodomaineja, BSD-määrittelytiedostoja ja NIS-domainin.

Jos tietokoneen muokatussa hakukäytännössä määritelty hakemistodomain ei ole käytettävissä, tietokoneen käynnistyessä esiintyy viive.

Muokatun hakemistodomainluettelon määritteleminen hakukäytäntöä varten:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Liity tai Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Hakukäytäntö ja valitse hakukäytäntö:

    • Todentaminen: näyttää hakukäytännön, jota käytetään todentamiselle ja useimmille muille ylläpitoon liittyville tiedoille.

    • Yhteystiedot: näyttää hakukäytännön, jota käytetään yhteystiedoille Osoitekirjan kaltaisissa ohjelmissa.

  7. Valitse Etsi-ponnahdusvalikosta Muokattu polku.

  8. Voit tavittaessa lisätä hakemistodomaineja osoittamalla Lisää, valitsemalla yhden tai useamman hakemiston ja osoittamalla uudelleen Lisää.

  9. Voit tarvittaessa muuttaa luetteloitujen hakemistodomainien järjestystä vetämällä niitä ylös tai alas luettelossa.

  10. Poista luettelosta hakemistodomainit, joita et halua hakukäytäntöön, valitsemalla ne ja osoittamalla poistopainiketta (–).

  11. Vahvista poistaminen osoittamalla OK ja osoita Käytä.

Jos haluat lisätä hakemiston, jota ei ole luetteloitu käytettävissä olevien hakemistojen joukossa, varmista, että tietokone on määritelty käyttämään hakemistoa. Lisätietoja löytyy kohdista:

  • Hakemistopalveluiden lisäasetusten käyttäminen

  • LDAP-palvelun lisäasetusten käyttäminen

  • Active Directory -palveluiden lisäasetusten käyttäminen

  • NIS-asetusten määritteleminen

  • BSD-määrittelytiedostoasetusten määritteleminen
