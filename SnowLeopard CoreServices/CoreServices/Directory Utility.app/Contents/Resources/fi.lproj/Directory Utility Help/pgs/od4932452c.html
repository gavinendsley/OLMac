Active Directory -palvelinyhteyden lisääminen

Kun yhdistät Active Directory -palvelimeen, tarvitset palvelimen nimen tai IP-osoitteen ja Active Directoryn ylläpitäjän käyttäjätunnuksen ja salasanan.

Active Directory -palvelimen lisääminen:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Liity tai Muokkaa.

    Jos Muokkaa-painike on näkyvissä, tietokoneella on ainakin yksi yhteys hakemistopalvelimelle.

  4. Osoita lisäyspainiketta (+).

  5. Valitse Lisää uusi hakemisto tyyppiä -ponnahdusvalikosta Active Directory ja syötä seuraavat tiedot:

    • Active Directory -domain: Tämä on Active Directory -domainin DNS-nimi (esimerkiksi ads.yritys.fi.)

    • Tietokoneen tunnus: Muokkaa halutessasi tunnusta, jota Active Directory käyttää palvelimelle. Tämä on palvelimen NetBIOS-nimi. Nimessä saa olla enintään 15 merkkiä eikä siinä saa olla erikoismerkkejä tai välimerkkejä. Jos se on käytännöllistä, aseta palvelimen nimi vastaamaan DNS-palvelinnimeä. Esimerkiksi jos DNS-palvelimen tietue palvelimellesi on ”palvelin.esimerkki.fi, anna palvelimen nimeksi ”palvelin”.

    • AD-ylläpitäjän käyttäjätunnus ja salasana: Syötä Active Directoryn ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita OK.
