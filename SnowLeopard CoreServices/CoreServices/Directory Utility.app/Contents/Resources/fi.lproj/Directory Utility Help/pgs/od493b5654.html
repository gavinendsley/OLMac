LDAP-hakemistojen täyttäminen tiedoilla Mac OS X:ää varten

Kun olet määritellyt pääsyn LDAP-hakemistodomaineihin ja ottanut käyttöön tietojen kuvauksen, voit täyttää ne tietueilla ja tiedoilla Mac OS X:ää varten. LDAP-hakemistoissa, jotka sallivat etähallinnan (luku- ja kirjoituspääsy), voit käyttää Mac OS X Serverin mukana tulevaa Workgroup Manageria seuraavasti:

  • Nimeä jakopisteet ja jaetut domainit, jotka haluat tuoda näkyviin automaattisesti käyttäjien verkkoselaimissa (mitä käyttäjät näkevät, kun osoittavat Finderin sivupalkissa kohtaa Verkko).

    Käytä Server Adminin Sharing-toimintoa ja Workgroup Managerin Network-moduuleita.

    Lisätietoja löytyy File Services Overview- ja User Management Overview -kohdista.

  • Määrittele käyttäjä- ja ryhmätietueet ja niiden asetukset.

    Käytä Workgroup Managerin Accounts-moduulia.

    Lisätietoja löytyy User Management Overview -kohdasta.

  • Määrittele luettelo tietokoneita, joilla on samat asetukset ja jotka ovat samojen käyttäjien ja ryhmien käytettävissä.

    Käytä Workgroup Managerin Computers-moduulia.

    Lisätietoja löytyy User Management Overview -kohdasta.

Kaikissa tapauksissa, osoita käyttäjäluettelon yläpuolella olevaa pientä maapallosymbolia ja avaa LDAP-hakemistodomain Workgroup Managerin ponnahdusvalikosta. Jos LDAP-hakemisto ei ole luetteloitu ponnahdusvalikossa, valitse LDAP-hakemisto valitsemalla valikosta Muu.

Huomaa:  Jos haluat lisätä tietueita ja tietoa vain luku -LDAP-hakemistoon, sinun on käytettävä LDAP-hakemiston sisältävällä palvelimella olevia työkaluja.
