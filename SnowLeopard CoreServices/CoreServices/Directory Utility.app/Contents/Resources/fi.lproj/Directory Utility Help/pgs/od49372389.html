Kaikilta Active Directory -metsän domaineilta tehtävän todentamisen hallitseminen

Jos tietokone on määritelty käyttämään Hakemistotyökalun Active Directory -liitintä, voit sallia Active Directory -metsän käyttäjien suorittaa todentamisen kaikilta domaineilta tai voit rajoittaa todentamisen yksittäisten domainien käyttäjiin.

Sen säätäminen, voivatko käyttäjät todentautua kaikilta metsän domaineilta:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Palvelut.

  7. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  8. Valitse palveluluettelosta Active Directory ja osoita muokkauspainiketta (/).

  9. Jos lisäasetukset on kätketty, osoita Näytä lisäasetukset.

  10. Osoita Ylläpidollinen.

  11. Valitse Salli todentaminen miltä tahansa metsän domainilta.

    Kun valitset ”Salli todentaminen miltä tahansa metsän domainilta”, voit lisätä Active Directory -metsän tietokoneen muokattuihin todentamisen ja yhteystietojen etsintämenetelmiin.

    Kun lisäät Active Directory -metsän muokattuun hakukäytäntöön, metsä näkyy käytettävien hakemistodomainien luettelossa muodossa ”/Active Directory/All Domains”. (Tämä on oletusasetus.)

    Kun poistat ”Salli todentaminen miltä tahansa metsän domainilta”, voit lisätä yksittäisiä Active Directory -domaineja tietokoneen muokattuihin todentamisen ja yhteystietojen etsintämenetelmiin.

    Kun lisäät Active Directory -domaineja muokattuun hakukäytäntöön, jokainen Active Directory -domain näkyy erikseen käytettävien hakemistodomainien luettelossa.

  12. Osoita OK.

  13. Kun olet valinnut Salli todentaminen miltä tahansa metsän domainilta -vaihtoehdon, sisällytä Active Directory -metsä tai valitut domainit muokattuun hakukäytäntöön Todentaminen- ja Yhteystiedot-osioissa.

    Lisätietoja muokatun hakukäytännön vaihtamisesta löytyy kohdasta Muokattujen hakukäytäntöjen määritteleminen.
