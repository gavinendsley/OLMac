Kotikansioiden asettaminen Active Directory -käyttäjätunnuksille

Voit sallia tai estää verkkokotikansioiden tai paikallisten kotikansioiden käytön Active Directory -käyttäjätileillä tietokoneella, joka on määritelty käyttämään Hakemistotyökalun Active Directory -liitintä.

Verkkokotikansioita käytettäessä käyttäjän Windows-verkon kotikansio tuodaan näkyviin Mac OS X -kotikansiona, kun käyttäjä kirjautuu sisään.

Voit määritellä, tuleeko verkkokotikansion sijainti Active Directoryn normaalista homeDirectory-attribuutista vai Mac OS X:n homeDirectory-attribuutista, jos Active Directory -malli on laajennettu sisältämään sen.

Paikallisia kotikansioita käytettäessä kullakin sisäänkirjautuvalla Active Directory -käyttäjällä on kotikansio Mac OS X -käynnistyslevyllä. Lisäksi käyttäjän verkkokotikansio otetaan käyttöön verkkotaltiona jakopisteen tavoin. Käyttäjä voi kopioida tiedostoja tämän verkkotaltion ja paikallisen kotikansion välillä.

Kotikansioiden asettaminen Active Directory -käyttäjätunnuksille:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Liity tai Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Palvelut.

  7. Valitse palveluluettelosta Active Directory ja osoita muokkauspainiketta (/).

  8. Jos lisäasetukset on kätketty, osoita Näytä lisäasetukset.

  9. Osoita Käyttökokemus.

  10. Jos haluat Active Directory -käyttäjätileille paikalliset kotikansiot tietokoneen /Käyttäjät-kansioon, osoita ”Pakota paikallinen kotikansio käynnistyslevylle”.

    Tämä vaihtoehto ei ole käytettävissä, jos ”Luo siirrettävä tunnus sisäänkirjauduttaessa” on valittuna.

  11. Jos haluat käyttää Active Directoryn normaalia attribuuttia kotikansion sijainnille, valitse ”Käytä Active Directoryn UNC-polkua verkon kotisijainnin selvittämiseen” ja valitse kotikansion käyttöön käytettävä protokolla seuraavista:

    • Jos haluat käyttää normaalia Windows-protokolla SMB:tä, valitse Käytettävä verkkoprotokolla -ponnahdusvalikosta smb.

    • Jos haluat käyttää normaalia Macintosh-protokollaa AFP:tä, valitse Käytettävä verkkoprotokolla -ponnahdusvalikosta afp.

  12. Jos haluat käyttää Mac OS X:n attribuuttia kotikansion sijainnille, poista Käytä Active Directoryn UNC-polkua verkon kotisijainnin selvittämiseen -vaihtoehdon valinta.

    Kun Mac OS X -attribuuttia halutaan käyttää, Active Directory -malli on laajennettava sisältämään sen.

  13. Osoita OK.

Jos muutat käyttäjätunnuksen nimeä Active Directory -domainissa, palvelin luo käyttäjätunnukselle kotikansion (ja alikansioita) seuraavan kerran, kun tunnusta käytetään Mac OS X -tietokoneelle kirjautumiseen. Käyttäjä voi edelleen selata vanhaan kotikansioon ja tarkastella sen sisältöä Finderissa.

Voit estää uuden kotikansion luomisen nimeämällä vanhan uudelleen ennen kuin käyttäjä seuraavan kerran kirjautuu sisään.
