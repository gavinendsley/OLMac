Tietokonetta ylläpitävien Active Directory -ryhmien muuttaminen

Jos tietokone on määritelty käyttämään Hakemistotyökalun Active Directory -liitintä, voit nimetä Active Directory -ryhmätunnukset, joiden jäsenille haluat antaa tietokoneen ylläpito-oikeudet.

Näiden Active Directory -ryhmätunnusten jäsenet voivat suorittaa ylläpitotehtäviä, kuten asentaa ohjelmia määrittelemääsi Mac OS X -tietokoneseen.

Sellaisten Active Directory -ryhmätunnusten lisääminen ja poistaminen, joiden jäsenillä on ylläpitäjän oikeudet:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Palvelut.

  7. Valitse palveluluettelosta Active Directory ja osoita muokkauspainiketta (/).

  8. Jos lisäasetukset on kätketty, osoita Näytä lisäasetukset.

  9. Osoita Ylläpidollinen

  10. Valitse ”Salli hallinnoida” ja muuta luetteloa Active Directory -ryhmätileistä, joiden jäsenille haluat antaa ylläpito-oikeudet:

    • Lisää ryhmä osoittamalla lisäyspainiketta (+) ja syöttämällä Active Directory -domainin nimi, kenoviiva ja ryhmätunnuksen nimi (esimerkiksi, ADS\Domain Admins, IL2\Domain Admins).

    • Poista ryhmä valitsemalla se luettelosta ja osoittamalla poistopainiketta (-).

  11. Osoita OK.
