Open Directory -palvelinyhteyden lisääminen

Kun lisäät Open Directory -palvelinta, tarvitset palvelimen nimen tai IP-osoitteen ja tiedon siitä, käyttääkö palvelin SSL:ää.

Open Directory -palvelimen lisääminen:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Liity tai Muokkaa.

    Jos Muokkaa-painike on näkyvissä, tietokoneella on ainakin yksi yhteys hakemistopalvelimelle.

  4. Osoita lisäyspainiketta (+).

  5. Valitse Lisää uusi hakemisto tyyppiä -ponnahdusvalikosta Open Directory.

  6. Syötä Palvelimen nimi tai IP-osoite -kenttään palvelimen nimi tai IP-osoite.

  7. (Valinnainen) Ennen kuin valitset Salaa käyttäen SSL:ää -valintaneliön, kysy Open Directory -ylläpitäjältä tarvitaanko SSL:ää.

Tärkeää:  Jos muutat Mac OS X -palvelimen IP-osoitetta ja tietokoneen nimeä changeip:llä, kun olet yhteydessä hakemistopalvelimeen, yhteys palvelimeen on katkaistava ja luotava uudelleen, jotta hakemisto päivitetään tietokoneen uudella nimellä ja IP-osoitteella. Jos yhteyttä hakemistopalvelimeen ei katkaista ja luoda uudelleen, hakemistoa ei päivitetä ja se jatkaa tietokoneen vanhan nimen ja IP-osoitteen käyttöä.
