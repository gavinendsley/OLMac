Pääsyn määritteleminen hakemistoihin

Hakemistotyökalulla (sijaitsee Tilit-asetuksissa) voit ottaa käyttöön Mac OS X -tietokoneen tai Mac OS X Server -palvelimen pääsyn hakemistodomaineihin ja hallita pääsytapaa.

Yhteyden ottaminen LDAP-hakemistoihin

Hakemistotyökalulla voit määritellä Mac OS X -tietokoneen tai Mac OS X Server -palvelimen käyttämään tiettyjä LDAP-hakemistoja, mukaan lukien Mac OS X Serverin Open Directory masterin LDAP-hakemisto. Kun haluat lisätietoja, osoita alla olevia aiheita.

Yhteyden muodostaminen Active Directory -domainiin

NIS-domainin käyttäminen

Hakemistotyökalulla voit luoda määrityksen, joka kertoo miten Mac OS X käyttää NIS-domainia. Lisätietoja saat osoittamalla alla olevaa aihetta.

BSD-määrittelytiedostojen käyttäminen

Open Directoryllä voit hakea ylläpitoon liittyviä tietoja BSD-määrittelytiedostoista (kuten /etc/master.passwd), jotka tunnetaan myös nimellä BSD flat file. Hakemistotyökalulla voit myös määritellä pääsyn BSD-määrittelytiedostoihin. Kun haluat lisätietoja, osoita alla olevia aiheita.
