Paikallisten hakemistonhakukäytäntöjen määritteleminen

Hakemistotyökalulla voit määritellä Mac OS X -tietokoneen todentamisen ja yhteystietojen hakukäytännöt käyttämään vain tietokoneen paikallista hakemistoa.

Vain paikallista hakemistoa käyttävä hakukäytäntö rajoittaa tietokoneen pääsyä todentamis- ja muihin ylläpidollisiin tietoihin.

Jos rajoitat tietokoneen todentamisen hakukäytännön käyttämään vain paikallista hakemistoa, ainoastaan käyttäjät, joilla on paikallinen tunnus voivat kirjautua sisään.

Hakukäytännön asettaminen käyttämään vain paikallista hakemistodomainia (paikallista hakemistoa):

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Liity tai Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Hakukäytäntö ja valitse hakukäytäntö:

    • Todentaminen: näyttää hakukäytännön, jota käytetään todentamiselle ja useimmille muille ylläpitoon liittyville tiedoille.

    • Yhteystiedot: näyttää hakukäytännön, jota käytetään yhteystiedoille Osoitekirjan kaltaisissa ohjelmissa.

  7. Valitse Etsi-ponnahdusvalikosta ”Paikallinen hakemisto” ja osoita Käytä.
