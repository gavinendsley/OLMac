Active Directory -palvelun salliminen tai estäminen

Hakemistotyökalulla voit sallia tai estää Windows-palvelimen tarjoamat Active Directory -palvelut. Active Directory on Windows 2000- ja uudempien palvelimien hakemistopalvelu.

Jos estät Active Directory -palvelut ja Active Directory -domainit ovat osa muokattua hakukäytäntöä, ne on luetteloidaan punaisella Hakemistotyökalun Hakukäytännön Todentaminen- tai Yhteystiedot-osassa.

Pääsyn salliminen ja estäminen Active Directoryyn:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Liity tai Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Palvelut.

  7. Valitse Active Directory -kohdan vieressä oleva valintaneliö tai poista sen valinta ja osoita Käytä.
