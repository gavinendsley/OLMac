Active Directoryssa olevien käyttäjätunnusten ja muiden tietueiden muokkaaminen

Voit käyttää Workgroup Manageria muutosten tekemiseen käyttäjätunnuksiin, ryhmätunnuksiin, tietokoneluetteloihin ja muihin Active Directory -domainin tietueisiin. Voit käyttää Workgroup Manageria myös tietueiden poistamiseen Active Directory -domainista.

Jos Active Directory -malli on laajennettu sisältämään normaaleja Mac OS X -tietuetyyppejä (objektiluokkia) ja attribuutteja, voit käyttää Workgroup Manageria tietokoneryhmien luomiseen ja muokkaamiseen Active Directory -domainissa.

Lisätietoja löytyy User Management Overview -kohdasta.

Jos haluat luoda käyttäjä- tai ryhmätunnuksia Active Directory -domainiin, käytä Microsoftin Active Directory -ylläpitotyökaluja Windows Server -ylläpitotietokoneella.
