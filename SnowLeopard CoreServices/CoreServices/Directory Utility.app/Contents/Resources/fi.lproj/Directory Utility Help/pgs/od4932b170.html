Tietoja hakemistopalvelinyhteyksistä

Voit yhdistää tietokoneita hakemistopalvelimiin Tilit-asetuksissa. Voit katsoa luettelon hakemistopalvelimista, joihin tietokone on yhteydessä, osoittamalla Tilit-asetusten Kirjautumisvalinnat-osiossa Muokkaa. Mac OS X -tietokone hakee luettelossa olevien hakemistopalvelimien hakemistodomaineihin tallennettuja käyttäjä- ja muita ylläpitoon liittyviä tietoja.

Kun lisäät tai poistat palvelimen Hakemistopalvelimet-luettelosta, hakemistopalvelimeen liittyvät tietueet lisätään tai poistetaan Palvelut-, Todentaminen- ja Yhteystiedot-luetteloista. Mutta jos poistat hakemistopalvelimeen liittyvät tietueet Palvelut-, Todentaminen- ja Yhteystiedot-luetteloista, hakemistopalvelinta ei poisteta Hakemistopalvelimet-luettelosta.

Mac OS X 10.6 -tietokoneet voivat olla yhteydessä Open Directory-, Active Directory- ja LDAP-hakemistopalvelimiin. Jos et tiedä mihin hakemistopalvelimeen yhdistää, kysy verkon ylläpitäjältä.

Tärkeää:  Jos tietokoneen nimi sisältää väliviivan, et ehkä voi liittyä tai muodostaa sidosta esimerkiksi LDAP- tai Active Directory -hakemistodomainiin. Jos haluat muodostaa sidoksen, käytä tietokoneen nimeä, joka ei sisällä väliviivaa.
