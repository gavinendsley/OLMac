NIS-asetusten määritteleminen

Hakemistotyökalulla voit luoda määrittelyn, joka määrittelee miten Mac OS X pääsee NIS-domainiin.

Määrittelyn luominen NIS-domainin käyttämiseen:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Liity tai Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Palvelut.

  7. Valitse palveluluettelosta BSD Flat File js NIS ja osoita muokkauspainiketta (/).

  8. Syötä NIS-domainnimi tai sen palvelimen DNS-nimi tai IP-osoite, jossa NIS-domain sijaitsee.

    Syötä NIS-palvelimen palvelinnimi tai IP-osoite, jos se vaaditaan turvallisuussyistä tai jos palvelin ei ole samassa aliverkossa kuin määriteltävä tietokone.

    Jos et määrittele palvelinta, NIS käyttää broadcast-protokollaa NIS-palvelimen etsimiseen aliverkosta.

  9. Valitse ”Käytä NIS-domainia todentamiseen” ja osoita OK.

    NIS-domain lisätään tietokoneen todentamisen hakukäytäntöön nimellä /BSD/domain, jossa domain on vaiheessa 4 syötetty nimi.
