BSD-määrittelytiedostojen tietojen määritteleminen

Jos haluat Mac OS X -tietokoneen hakevan ylläpitotiedot BSD-määrittelytiedostoista, tietojen on oltava tiedostoissa ja Mac OS X:n vaatimassa muodossa.

Tiedostoissa olevia tietoja on ehkä lisättävä, muutettava tai järjesteltävä uudelleen. Workgroup Manager ei voi tehdä muutoksia BSD-määrittelytiedostojen tietoihin, joten tarvittavat muutokset on tehtävä tekstieditorilla tai muilla työkaluilla.

Seuraavassa taulukossa on lueteltu tiedostojen nimet ja kuvailtu niiden sisältö.

BSD-määrittelytiedosto
Sisältää

/etc/master.passwd

Käyttäjätunnukset, salasanat, tunnukset, ensisijaisen ryhmän tunnukset ja niin edelleen

/etc/group

Ryhmien nimet, tunnukset ja jäsenet

/etc/fstab

NFS-liitokset

/etc/hosts

Tietokoneiden nimet ja osoitteet

/etc/networks

Verkkojen nimet ja osoitteet

/etc/services

Palveluiden nimet, portit ja protokollat

/etc/protocols

IP-protokollien nimet ja numerot

/etc/rpcs

Open Network Computing RPC-palvelimet

/etc/printcap

Tulostimien nimet ja ominaisuudet

/etc/bootparams

Bootparam-asetukset

/etc/bootp

Bootp-asetukset

/etc/aliases

Sähköpostialiakset ja jakeluluettelot

/etc/netgroup

Verkonlaajuisten ryhmien nimet ja jäsenet
