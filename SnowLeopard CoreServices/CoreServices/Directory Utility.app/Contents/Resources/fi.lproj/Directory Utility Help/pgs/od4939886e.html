BSD-määrittelytiedostoasetusten määritteleminen

UNIX-tietokoneet ovat perinteisesti tallentaneet ylläpitotiedot määrittelytiedostoihin kuten /etc/master.passwd, /etc/group ja /etc/hosts. Mac OS X perustuu UNIXin BSD-versioon, mutta hakee ylläpitotiedot yleensä kuitenkin hakemistojärjestelmistä.

Mac OS X Server tukee tiettyjä BSD-määrittelytiedostoja. Et voi määritellä mitä määrittelytiedostoja käytetään tai kuvata niiden sisältöä Mac OS X:n tietuetyyppeihin tai attribuutteihin.

Mac OS X 10.2:ssa ja uudemmissa (mukaan lukien Mac OS X Server 10.2 ja uudemmat) Open Directory osaa hakea ylläpitoon liittyvät tiedot BSD-määrittelytiedostoista. Tämä ominaisuus sallii organisaatioiden, joilla on BSD-määrittelytiedostoja, käyttää olemassa olevia tiedostoja Mac OS X -tietokoneissa. BSD-määrittelytiedostoja voidaan käyttää yksinään tai muiden hakemistodomainien kanssa.

BSD-määrittelytiedostojen käyttäminen:

  1. Varmista, että BSD-määrittelytiedostot sisältävät Mac OS X:n hakemistopalveluiden vaatimat tiedot.

    Lisätietoja löytyy kohdasta BSD-määrittelytiedostojen tietojen määritteleminen.

  2. Avaa Järjestelmäasetukset ja osoita Tilit.

  3. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  4. Osoita Kirjautumisvalinnat ja osoita sitten Liity tai Muokkaa.

  5. Osoita Avaa Hakemistotyökalu.

  6. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  7. Osoita Palvelut.

  8. Valitse palveluluettelosta BSD Flat File js NIS ja osoita muokkauspainiketta (/).

  9. Valitse Käytä paikallisen BSD-solmun käyttäjä- ja ryhmätietoja ja osoita OK.

    BSD-määrittelytiedostojen domain lisätään tietokoneen todentamisen hakukäytäntöön muodossa /BSD/local.
