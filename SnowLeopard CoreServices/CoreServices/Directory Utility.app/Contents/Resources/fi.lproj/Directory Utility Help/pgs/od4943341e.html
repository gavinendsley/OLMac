Suosittavan Active Directory -palvelimen määritteleminen

Jos tietokone on määritelty käyttämään Hakemistotyökalun Active Directory -liitintä, voit määritellä DNS-nimen palvelimelle, jonka Active Directory -domainia haluat tietokoneen oletusarvoisesti käyttävän.

Jos palvelin lakkaa olemasta käytettävissä, Active Directory -liitin palaa käyttämään metsän toista lähellä olevaa palvelinta.

Jos tämä vaihtoehto ei ole valittuna, Active Directory -liitin selvittää metsän lähimmän Active Directory -domainin.

Active Directory -liittimen ensisijaisesti käyttämän palvelimen määritteleminen:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Osoita Palvelut.

  6. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  7. Valitse palveluluettelosta Active Directory ja osoita muokkauspainiketta (/).

  8. Jos lisäasetukset on kätketty, osoita Näytä lisäasetukset.

  9. Osoita Ylläpidollinen.

  10. Valitse ”Suosi tätä domainpalvelinta” ja syötä Active Directory -palvelimen DNS-nimi.

  11. Osoita OK.
