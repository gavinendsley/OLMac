Active Directory -palveluiden lisäasetusten käyttäminen

Voit määritellä Mac OS X Server -palvelimen tai Mac OS X -tietokoneen käyttämään Windows 2000- tai Windows 2003 -palvelimella olevaa Active Directory -domainia.

Jos haluat nähdä tehtävien kuvaukset ja ohjeet, katso:

  • Tietoja Active Directoryn käytöstä

  • Pääsyn määritteleminen Active Directory -domainille

  • Siirrettävien käyttäjätunnusten asettaminen Active Directoryyn

  • Kotikansioiden asettaminen Active Directory -käyttäjätunnuksille

  • UNIX-komentotulkin asettaminen Active Directory -käyttäjätunnuksille

  • UID:n kuvaaminen Active Directory -attribuuttiin

  • Ensisijaisen ryhmänumeron kuvaaminen Active Directory -attribuuttiin

  • Ryhmätunnuksien ryhmänumeron kuvaaminen Active Directory -attribuuttiin

  • Suosittavan Active Directory -palvelimen määritteleminen

  • Tietokonetta ylläpitävien Active Directory -ryhmien muuttaminen

  • Kaikilta Active Directory -metsän domaineilta tehtävän todentamisen hallitseminen

  • Active Directory -palvelimen sidoksen purkaminen

  • Active Directoryssa olevien käyttäjätunnusten ja muiden tietueiden muokkaaminen

Joissakin verkoissa tarvitaan vaihtoehtoisia tapoja Active Directory -domainiin pääsemiseksi. Katso LDAP-pääsyn käyttöönotto Active Directory -domaineilla.
