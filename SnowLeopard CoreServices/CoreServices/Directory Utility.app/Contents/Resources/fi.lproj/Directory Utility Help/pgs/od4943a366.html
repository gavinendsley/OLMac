UID:n kuvaaminen Active Directory -attribuuttiin

Jos tietokone on määritelty käyttämään Hakemistotyökalun Active Directory -liitintä, voit määritellä Active Directory -attribuutin, jonka haluat kuvattavan Mac OS X:n yksilölliseen käyttäjätunnusattribuuttiin (UID).

Yleensä Active Directory -malli täytyy laajentaa sisältämään UID:n kuvaamiseen sopivan attribuutin:

  • Jos Active Directory -ylläpitäjä laajentaa Active Directory -mallia asentamalla Microsoftin Services for UNIX -ohjelmiston, voit kuvata UID:n msSFU-30-Uid-Number-attribuuttiin.

  • Jos Active Directory -ylläpitäjä laajentaa Active Directory -mallia manuaalisesti sisältämään RFC 2307 -attribuutit, voit kuvata UID:n uidNumberiin.

  • Jos Active Directory -ylläpitäjä laajentaa Active Directory -mallia manuaalisesti sisältämään Mac OS X:n UniqueID-attribuutin, voit kuvata UID:n siihen.

Jos UID-kuvaus on estetty, Active Directory -liitin luo Active Directoryn normaaliin GUID-attribuuttiin perustuvan UID:n.

VAROITUS: Jos muutat UID:n kuvausta myöhemmin, käyttäjät saattavat menettää pääsyn aiemmin luotuihin tiedostoihin.
UID:n kuvaaminen laajennetun Active Directory -mallin attribuuttiin:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Palvelut.

  7. Valitse palveluluettelosta Active Directory ja osoita muokkauspainiketta (/).

  8. Jos lisäasetukset on kätketty, osoita Näytä lisäasetukset.

  9. Osoita Kuvaukset.

  10. Valitse ”Kuvaa UID attribuuttiin” ja syötä UID:hen kuvattavan Active Directory -attribuutin nimi.

  11. Osoita OK.
