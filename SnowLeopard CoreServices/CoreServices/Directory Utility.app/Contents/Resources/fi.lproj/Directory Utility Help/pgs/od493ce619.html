LDAP-palvelun lisäasetusten käyttäminen

Voit määritellä Mac OS X Server -palvelimen tai Mac OS X -tietokoneen käyttämään tiettyjä LDAP-hakemistoja, mukaan lukien Mac OS X Serverin Open Directory masterin LDAP-hakemisto.

Jos haluat nähdä tehtävien kuvaukset ja ohjeet, katso:

  • Yhteyden ottaminen LDAP-hakemistoihin Mailissa ja Osoitekirjassa

  • LDAP-palvelimien käyttöön tarkoitettujen määrittelyiden näyttäminen tai kätkeminen

  • LDAP-hakemistoon pääsyn määrittely

  • Pääsyn määrittäminen LDAP-hakemistoon manuaalisesti

  • LDAP-hakemiston käyttöön käytetyn määrityksen vaihtaminen

  • LDAP-hakemiston käyttöön käytetyn määrityksen monistaminen

  • LDAP-hakemiston käyttöön käytetyn määrityksen poistaminen

  • LDAP-hakemiston yhteysasetusten muuttaminen

  • LDAP-yhteyden turvallisuuskäytännön muuttaminen

  • LDAP-palveluiden ja -kuvausten määrittely

  • Luotetun sidonnan käyttöönotto LDAP-hakemistolle

  • LDAP-hakemiston luotetun sidonnan keskeyttäminen

  • Avaa/sulje aikakatkaisun muuttaminen LDAP-yhteyttä varten

  • Kyselyn aikakatkaisun muuttaminen LDAP-yhteyttä varten

  • Uudelleensidonnan viiveen muuttaminen LDAP-yhteyttä varten

  • Käyttämättömän yhteyden aikakatkaisun muuttaminen LDAP-yhteyttä varten

  • LDAP-palvelinviittausten huomiotta jättäminen

  • LDAP-yhteyden todentaminen

  • LDAP-yhteyden todentamiseen käytetyn salasanan muuttaminen

  • Config Record -attribuuttien kuvaaminen LDAP-hakemistoja varten

  • RFC 2307 -kuvausten muokkaaminen käyttäjien luomisen sallimiseksi

  • Vain luku -LDAP-hakemiston valmisteleminen Mac OS X:ää varten

  • LDAP-hakemistojen täyttäminen tiedoilla Mac OS X:ää varten
