Ryhmätunnusten ryhmänumeron kuvaaminen Active Directory -attribuuttiin

Jos tietokone on määritelty käyttämään Hakemistotyökalun Active Directory -liitintä, voit määritellä Active Directory -attribuutin, jonka haluat kuvata Mac OS X:n ryhmätilien ryhmätunnusattribuuttiin (GID).

Yleensä Active Directory -malli täytyy laajentaa sisältämään GID:n kuvaamiseen sopivan attribuutin:

  • Jos Active Directory -ylläpitäjä laajentaa Active Directory -mallia asentamalla Microsoftin Services for UNIX -ohjelmiston, voit kuvata GID:n msSFU-30-Gid-Number-attribuuttiin.

  • Jos Active Directory -ylläpitäjä laajentaa käsin Active Directory -mallin sisältämään RFC 2307 -attribuutteja, voit kuvata GID:n gidNumberiin.

  • Jos Active Directory -ylläpitäjä laajentaa käsin Active Directory -mallin sisältämään Mac OS X:n gidNumber-attribuutin, voit kuvata GID:n siihen.

Jos GID:n kuvaus on estetty, Active Directory -liitin luo GID:n Active Directoryn normaalin GUID-attribuutin perusteella.

VAROITUS: Jos muutat GID:n kuvausta myöhemmin, käyttäjät saattavat menettää pääsyn aiemmin luotuihin tiedostoihin.
GID:n kuvaaminen laajennetun Active Directory -mallin attribuuttiin:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Palvelut.

  7. Valitse palveluluettelosta Active Directory ja osoita muokkauspainiketta (/).

  8. Jos lisäasetukset on kätketty, osoita Näytä lisäasetukset.

  9. Osoita Kuvaukset.

  10. Valitse ”Kuvaa ryhmän GID attribuuttiin” ja syötä sen Active Directory -attribuutin nimi, johon haluat kuvata ryhmätilien GID:n.

  11. Osoita OK.
