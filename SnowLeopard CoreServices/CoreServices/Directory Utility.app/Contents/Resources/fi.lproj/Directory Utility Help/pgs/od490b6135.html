Tietoja Hakemistotyökalusta

Hakemistotyökalu määrittelee, mitä hakemistopalveluita Mac OS X -tietokone käyttää ja miten se kytkeytyy tiettyihin hakemistodomaineihin. Hakemistotyökalu määrittelee myös hakukäytännöt, joilla etsitään todentamis- ja yhteystietoja tietyissä hakemistodomaineissa.

Hakemistotyökalun käyttöönotto

Voit käyttää Hakemistotyökalua tietokoneella ja määritellä ja hallita miten tietokone käyttää hakemistopalveluita. Hakemistotyökalun avulla voit määritellä myös etänä miten tietokone käyttää hakemistopalveluita. Hakemistotyökalua voidaan käyttää myös root-tilin käyttöönottoon ja sen salasanan muuttamiseen.

Pääsyn määrittäminen palveluihin

Hakemistotyökalulla voit sallia ja estää eri palveluita, joita Mac OS X voi käyttää verkossa. Nämä sisältävät hakemistopalvelut, jotka tarjoavat pääsyn käyttäjätunnuksiin ja muihin hakemistodomaineihin tallennettuihin ylläpidollisiin tietoihin.

Pääsyn määritteleminen hakemistoihin

Hakemistotyökalulla voit määritellä tietokoneen käyttämään käyttäjätunnuksia ja muita ylläpitoon liittyviä tietoja, jotka sijaitsevat tietyssä LDAP-hakemistoissa tai Active Directory- tai NIS-domainissa.

Hakukäytäntöjen asettaminen

Hakemistotyökalulla voit määritellä todentamisen hakukäytännöt, joita Mac OS X käyttää käyttäjien todentamis- ja muiden ylläpitoon liittyvien tietojen etsimiseen ja hakemiseen hakemistodomaineista. Hakemistotyökalulla voidaan myös määritellä yhteystietojen hakukäytännöt, joita Mac OS X käyttää nimi-, osoite- ja muiden yhteystietojen etsimiseen ja hakemiseen hakemistodomaineista.
