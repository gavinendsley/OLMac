UNIX-komentotulkin asettaminen Active Directory -käyttäjätunnuksille

Jos tietokone on määritelty käyttämään Hakemistotyökalun Active Directory -liitintä, voit asettaa komentotulkin, jota Active Directory -tilin omaavat käyttäjät käyttävät oletusarvoisesti Mac OS X:n Pääte-ohjelman kanssa.

Oletuskomentotulkkia käytetään myös etäyhteyksissä SSH:n tai Telnetin kautta. Käyttäjät voivat korvata oletusarvoisen komentotulkin vaihtamalla Pääte-asetuksen.

UNIX-komentotulkin asettaminen Active Directory -käyttäjätunnuksille:

  1. Avaa Järjestelmäasetukset ja osoita Tilit.

  2. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  3. Osoita Kirjautumisvalinnat ja osoita sitten Liity tai Muokkaa.

  4. Osoita Avaa Hakemistotyökalu.

  5. Jos lukkosymboli on lukittu, avaa se osoittamalla sitä ja kirjoittamalla ylläpitäjän käyttäjätunnus ja salasana.

  6. Osoita Palvelut.

  7. Valitse palveluluettelosta Active Directory ja osoita muokkauspainiketta (/).

  8. Jos lisäasetukset on kätketty, osoita Näytä lisäasetukset.

  9. Osoita Käyttökokemus.

  10. Valitse oletusarvoinen komentotulkki ja syötä käyttäjän oletuskomentotulkin polku.

  11. Osoita OK.
