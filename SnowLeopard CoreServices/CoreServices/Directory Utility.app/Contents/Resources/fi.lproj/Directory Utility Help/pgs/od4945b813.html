Config Record -attribuuttien kuvaaminen LDAP-hakemistoja varten

Jos haluat tallentaa hallittujen Mac OS X -käyttäjien tietoja muuhun kuin Applen LDAP-hakemistoon, kuvaa Config-tietuetyypin attribuutit RealName ja DataStamp.

Jos et kuvaa näitä attribuutteja, näkyviin tulee seuraava virheilmoitus, kun yrität muuttaa LDAP-hakemiston käyttäjätietuetta Workgroup Managerilla:

The attribute with name ”dsRecTypeStandard:Config” is not mapped.

Voit jättää ilmoituksen huomioimatta, jollet käytä Mac OS X:n asiakkaan hallintaa, joka vaatii Config -tietuetyypit RealName ja DataStamp välimuistia varten.
