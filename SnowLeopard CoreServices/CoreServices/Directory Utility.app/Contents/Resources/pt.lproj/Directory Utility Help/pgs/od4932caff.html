Como alterar a senha da conta do usuário raiz

Você pode usar o Utilitário de Diretório (localizado no painel de preferências Conta) para alterar a senha da conta do usuário raiz. Ao alterar a senha raiz, use uma senha complexa que contenha caracteres alfanuméricos e especiais, para evitar que a senha seja comprometida.

AVISO:  a conta raiz é uma conta de administrador sem restrições que você usa para efetuar alterações em arquivos críticos do sistema. Mesmo se você estiver conectado como um administrador, ainda é necessário usar a conta raiz, ou sudo, para efetuar tarefas críticas do sistema.

Nunca use a conta raiz para fazer login em um computador (remota ou localmente). Em vez disso, use o comando sudo para efetuar tarefas raiz. Você pode restringir o acesso ao comando sudo adicionando usuários ao arquivo /etc/sudoers/.

Para obter mais informações sobre a conta raiz, consulte Ajuda Server Admin. Para obter mais informações sobre como criar uma senha, consulte Como criar uma senha.

Para alterar a senha da conta do usuário raiz:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Conectar ou Editar.

    Se estiver vendo o botão Editar, isso significa que seu computador possui pelo menos uma conexão a um servidor de diretório.

  4. Clique em Abrir Utilitário de Diretório.

  5. Escolha Editar > Alterar Senha Raiz.

  6. Quando estiver pronto, digite a nova senha raiz nos campos Senha e Verificar.

  7. Clique em OK.
