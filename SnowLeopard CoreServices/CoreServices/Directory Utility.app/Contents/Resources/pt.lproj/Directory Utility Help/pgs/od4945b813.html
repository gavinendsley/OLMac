Como mapear os atributos de registro de configuração dos diretórios LDAP

Para armazenar informações para usuários do Mac OS X gerenciados em um diretório LDAP não-Apple, você deve mapear os seguintes atributos do tipo de registro Config: RealName e DataStamp.

Se você não mapear esses atributos, a seguinte mensagem de erro será exibida quando você usar o Workgroup Manager para alterar um registro de usuário que esteja no diretório LDAP:

O atributo com o nome “dsRecTypeStandard:Config” não está mapeado.

Você pode ignorar esta mensagem se você não estiver usando o gerenciador de clientes do Mac OS X, que depende dos atributos RealName e DataStamp dos tipos de registro de Config para usar uma cache.
