Como editar uma conexão do servidor de diretório

Você pode usar as preferências Conta para editar os servidores de diretório aos quais está conectado.

Para editar uma conexão do servidor de diretório:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Editar.

  4. Na lista de servidores de diretório, selecione um servidor de diretório que deseja editar.

  5. Clique no botão Editar (/).

  6. Altere os ajustes do servidor de diretório.

  7. Clique em OK.
