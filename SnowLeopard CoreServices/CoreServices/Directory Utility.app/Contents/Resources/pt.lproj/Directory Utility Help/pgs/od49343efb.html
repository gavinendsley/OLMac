Como configurar o acesso a diretórios

Você pode usar o Utilitário de Diretório (localizado no painel de preferências Conta) para configurar e gerenciar como um computador com Mac OS X ou um servidor com Mac OS X Server acessam os domínios de diretório.

Como acessar diretórios LDAP

Usando o Utilitário de Diretório, você pode configurar um computador com o Mac OS X ou um servidor com o Mac OS X Server para acessar diretórios específicos LDAP, incluindo o diretório LDAP de um Open Directory principal do Mac OS X Server. Para obter mais informações, clique nos temas abaixo.
