Como desvincular-se do servidor do Active Directory

Se o computador está usando o conector do Active Directory do Utilitário de Diretório para vincular-se a um servidor de Active Directory, você pode desvincular o computador do servidor do Active Directory.

Você pode forçar a desvinculação se o computador não consegue entrar em contato com o servidor ou se o registro do computador tiver sido removido dele.

Para desvincular o computador do servidor do Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Editar.

  4. Clique em Abrir Utilitário de Diretório.

  5. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, selecione Active Directory e clique no botão Editar (/).

  8. Clique em Desvincular, autentique como um usuário que possui direitos para terminar a conexão ao domínio do Active Directory e clique em OK.

    Se você visualizar um alerta dizendo que as credenciais não foram aceitas ou que o computador não pode contatar o Active Directory, clique em Forçar Desvinculação para forçar o término da conexão.

    Se você forçar a desvinculação, o Active Directory ainda conterá um registro de computador para este computador. Notifique o administrador do Active Directory para que o administrador saiba que deve remover o registro do computador.

  9. No painel Serviços, desmarque o ajuste de ativação do Active Directory e clique em Aplicar.
