Como remover uma conexão do servidor de diretório

Antes de remover um servidor de diretório do painel de preferências Conta, certifique-se de que você não está usando os serviços do mesmo para outros aplicativos.

Por exemplo, se o Mail estiver configurado para usar o servidor de diretório para procurar pessoas e você apaga o servidor de diretório, você não poderá procurar ninguém que estava no servidor de diretório.

Para apagar um servidor de diretório:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Editar.

  4. Na lista de servidores de diretório, selecione o servidor de diretório que deseja apagar.

  5. Clique no botão Apagar (—).

  6. Se você tiver certeza de que selecionou o servidor de diretório correto, clique em Interromper o Uso do Servidor.

Se um usuário desconectar a si próprio de um servidor de diretório, o administrador de diretório deverá remover manualmente o registro do computador do servidor de diretório. Os usuários também podem solicitar que o administrador de rede desconecte seus computadores do servidor de diretório usando o Utilitário de Diretório e uma conta que tenha permissão de administrador de diretório, que remove o registro do computador do servidor de diretório.
