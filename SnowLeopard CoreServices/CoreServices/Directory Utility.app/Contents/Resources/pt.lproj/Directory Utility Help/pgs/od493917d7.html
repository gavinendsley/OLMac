Como preparar um diretório LDAP somente leitura para o Mac OS X

Se você desejar que um computador Mac OS X obtenha dados administrativos de um diretório LDAP somente leitura, os dados deverão existir no formato exigido pelo Mac OS X. Talvez seja necessário adicionar, alterar ou reorganizar os dados no diretório LDAP somente leitura.

Como o Mac OS X não pode gravar dados em um diretório somente de leitura, você deve usar outras ferramentas para fazer as alterações. As ferramentas devem residir no servidor que hospeda o diretório LDAP somente leitura.

Para preparar um diretório LDAP somente leitura para Mac OS X:

  1. Vá ao servidor que hospeda o diretório LDAP somente leitura e configure o para ser compatível com a autenticação e a verificação de senha baseadas em LDAP.

  2. Altere as classes de objetos e os atributos do diretório LDAP conforme a necessidade para fornecer os dados necessários para o Mac OS X.

Para obter as especificações dos dados exigidos pelos serviços de diretório do Mac OS X, consulte a Administração do Open Directory.
