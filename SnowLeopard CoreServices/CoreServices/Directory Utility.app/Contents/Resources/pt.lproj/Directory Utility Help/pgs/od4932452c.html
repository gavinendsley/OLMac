Como adicionar uma conexão de servidor do Active Directory

Para conectar-se a um servidor Active Directory, você precisa saber o nome ou o endereço IP do servidor e a senha e nome de usuário do administrador do Active Directory.

Para adicionar um servidor do Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Conectar ou Editar.

    Se estiver vendo o botão Editar, isso significa que seu computador possui pelo menos uma conexão a um servidor de diretório.

  4. Clique no botão Adicionar (+).

  5. No menu local “Adicionar um novo diretório de tipo”, escolha Active Directory e, em seguida, digite as seguintes informações:

    • Domínio do Active Directory: Este é o nome DNS do domínio do Active Directory (por exemplo, ads.company.com.)

    • ID do Computador: como opção, você pode editar a ID que quer que o Active Directory use para o seu servidor. Este é o nome do NetBIOS do servidor. O nome não deve conter mais de 15 caracteres, caracteres especiais ou pontuação. Se achar mais prático, faça com que o nome do servidor combine com o nome de seu host DNS não qualificado. Por exemplo, se o servidor DNS tem uma entrada para seu servidor como “server.example.com,” nomeie seu servidor “server.”

    • Nome de usuário e senha do Administrador do AD: digite o nome de usuário e a senha do administrador do Active Directory.

  6. Clique em OK.
