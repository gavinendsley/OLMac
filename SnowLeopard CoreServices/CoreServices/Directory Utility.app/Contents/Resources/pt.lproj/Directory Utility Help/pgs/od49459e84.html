Como configurar o Utilitário de Diretório em um servidor remoto

Você pode usar o Utilitário de Diretório em seu computador para configurar e gerenciar como o Mac OS X Server que está em um servidor remoto acessa os serviços de diretório.

Para configurar o acesso de diretório em um servidor remoto:

  1. Abra as Preferências do Sistema no seu computador e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Conectar ou Editar.

    Se estiver vendo o botão Editar, isso significa que seu computador possui pelo menos uma conexão a um servidor de diretório.

  4. Clique em Abrir Utilitário de Diretório.

  5. No menu Arquivo, escolha Conectar.

  6. Digite as seguintes informações de conexão e autenticação para o servidor que deseja configurar.

    • Endereço: digite o nome DNS ou endereço IP do servidor que deseja configurar.

    • Nome de Usuário: digite o nome de usuário de um administrador no servidor.

    • Senha: digite a senha para o nome de usuário que você inseriu.

  7. Clique em Conectar.

  8. Clique nas abas Serviços e Política de Busca e altere os ajustes de acordo com a necessidade.

    Todas as alterações feitas afetam o servidor remoto ao qual você se conectou nas etapas anteriores.

  9. Escolha Desconectar no menu Arquivo de seu computador.
