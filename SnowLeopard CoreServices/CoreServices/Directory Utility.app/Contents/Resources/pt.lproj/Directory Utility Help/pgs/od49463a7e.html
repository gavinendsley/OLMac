Como ajustar os dados dos arquivos de configuração BSD

Se você deseja que um computador Mac OS X obtenha dados administrativos de arquivos de configuração BSD, os dados devem existir dentro dos arquivos e devem estar no formato requerido pelo Mac OS X.

Você pode precisar adicionar, alterar ou reorganizar dados nos arquivos. O Workgroup Manager não pode fazer alterações aos dados nos arquivos de configuração BSD, assim você precisa fazer as modificações necessárias usando um editor de texto ou outra ferramenta.

A tabela a seguir lista os nomes dos arquivos e descreve o conteúdo de cada um deles.

Arquivo de configuração BSD
Contém

/etc/master.passwd

Nomes de usuário, senhas, IDs, IDs de grupos principais e assim por diante

/etc/group

Nomes de grupos, IDs e membros

/etc/fstab

Montagens de NFS

/etc/hosts

Nomes e endereços de computadores

/etc/networks

Nomes e endereços de rede

/etc/services

Nomes de serviços, portas e protocolos

/etc/protocols

Nomes e números de protocolos IP

/etc/rpcs

Servidores RPC de computação de rede aberta

/etc/printcap

Nomes e recursos de impressoras

/etc/bootparams

Ajustes de bootparam

/etc/bootp

Ajustes de bootp

/etc/aliases

Atalhos de e-mail e listas de distribuição

/etc/netgroup

Nomes de grupos e membros de rede
