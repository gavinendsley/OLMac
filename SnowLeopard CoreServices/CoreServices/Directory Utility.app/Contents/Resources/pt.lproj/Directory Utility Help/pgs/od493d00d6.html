Como mapear o ID do grupo primário para um atributo do Active Directory

Em um computador configurado para usar o conector do Active Directory do Utilitário de Diretório, você pode especificar um atributo do Active Directory que deseja que seja mapeado para o atributo de ID de grupo principal (GID) em contas de usuário do Mac OS X.

Geralmente, o esquema do Active Directory deve ser ampliado para incluir um atributo que seja adequado para mapear no GID principal:

  • Se o administrador do Active Directory ampliar o esquema do Active Directory instalando os Serviços da Microsoft para UNIX, você poderá mapear o GID principal ao atributo msSFU-30-Gid-Number.

  • Se o administrador Active Directory ampliar manualmente o esquema do Active Directory para incluir os atributos RFC 2307, você pode mapear o GID primário ao gidNumber.

  • Se o administrador do Active Directory ampliar manualmente o esquema do Active Directory para incluir o atributo PrimaryGroupID do Mac OS X, você pode mapear o GID primário a ele.

Se o mapeamento do GID principal estiver desativado, o conector do Active Directory gerará um GID principal baseado no atributo GUID padrão do Active Directory.

AVISO:  se você alterar o mapeamento do GID principal para uma data posterior, os usuários poderão perder o acesso aos arquivos criados anteriormente.
Para mapear o GID principal para um atributo em um esquema do Active Directory estendido:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Editar.

  4. Clique em Abrir Utilitário de Diretório.

  5. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, selecione Active Directory e clique no botão Editar (/).

  8. Se as opções avançadas estiverem ocultas, clique em "Mostrar Opções Avançadas".

  9. Clique em Mapeamentos.

  10. Selecione “Mapear ID de Grupo do usuário para atributo” e digite o nome do atributo do Active Directory que você deseja mapear para o grupo de ID principal nas contas de usuário.

  11. Clique em OK.
