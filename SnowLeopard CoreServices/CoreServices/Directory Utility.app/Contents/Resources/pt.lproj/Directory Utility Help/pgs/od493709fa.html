Como mostrar ou ocultar configurações de servidores LDAP

Você pode mostrar ou ocultar uma lista de configurações disponíveis para acessar diretórios LDAP. Cada configuração especifica como o Open Directory acessa um diretório LDAP. Quando a lista está visível, você pode alterar os ajustes para cada configuração LDAP que não esteja acinzentada.

Para mostrar ou ocultar as configurações do diretório LDAP:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Conectar ou Editar.

  4. Clique em Abrir Utilitário de Diretório.

  5. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, selecione LDAPv3 e clique no botão Editar (/).

  8. Clique no controle Mostrar Opções ou no controle Ocultar Opções, dependendo de qual estiver disponível.
