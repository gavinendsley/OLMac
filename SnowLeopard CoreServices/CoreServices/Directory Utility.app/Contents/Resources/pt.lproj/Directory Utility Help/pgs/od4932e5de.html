Configuração automática de clientes

Se o seu computador Mac OS X v10.6 está conectado a uma rede e não a um domínio de diretório, seu computador Mac OS X v10.6 usará o Bonjour para descobrir os servidores Mac OS X v10.6.

Se ele encontrar um servidor Mac OS X v10.6 executando uma sessão principal do Open Directory que possua, pelo menos, um serviço configurado automaticamente em execução (como um AFP, SMB, VPN, Mail, iCal, iChat, notificações) e uma conta de usuário no domínio de diretório que coincida nomes abreviados ou por extenso como atualmente conectado ao usuário do computador Mac OS X v10.6, um convite aparecerá no seu computador Mac OS X v10.6.

O convite oferece assistentes para configurar a sua conexão a um servidor de diretório. Você pode aceitar ou recusar conectar-se ao servidor, ou decidir conectar-se mais tarde.

Se o seu computador descobrir mais de um servidor de diretório Mac OS X v10.6 ao qual você pode se conectar, os servidores são listados no menu local do convite. Se você decidir conectar um servidor, o assistente se abre, conecta e configura os seus aplicativos com os serviços que o servidor de diretório está executando.

Para conectar-se a um domínio de diretório usando o assistente automatizado:

  1. Quando aparecer o assistente, clique em Configurar Serviços.

  2. No menu local Servidor, escolha o domínio de diretório ao qual deseja se conectar.

    Talvez exista mais de um domínio de diretório disponível na sua rede.

    Se o servidor ao qual deseja se conectar não estiver na lista, digite o nome do servidor ou o endereço IP no campo Servidor e clique em OK.

  3. Digite o nome de usuário e senha de uma conta que possui privilégios para conectar-se a um computador para o domínio de diretório ao qual você está se conectando.

    Se você não tiver estas informações entre em contato com o seu administrador de diretório.

  4. Digite a senha para a conta de usuário que aparece em “Digite a senha para a conta nome de usuário neste computador”.

    O assistente altera a senha da conta do diretório de usuários no servidor para coincidir com a senha da conta do computador local.

  5. Clique em Continuar.

  6. Quando a ferramenta UpgradeUser estiver concluída, clique em Continuar.

  7. Quando aparecer uma mensagem explicando os serviços que foram configurados e solicitando que você finalize a sessão, clique em Finalizar Sessão.

    Ao iniciar a sessão, você poderá começar a usar os novos serviços.
