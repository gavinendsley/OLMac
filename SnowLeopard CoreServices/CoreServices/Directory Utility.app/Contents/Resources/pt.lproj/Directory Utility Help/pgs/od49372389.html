Como controlar a autenticação de todos os domínios na floresta do Active Directory

Em um computador configurado para usar o conector do Active Directory do Utilitário de Diretório, você pode permitir que os usuários da floresta do Active Directory autentiquem em todos os domínios, ou você pode restringir a autenticação a usuários de domínios individuais.

Para controlar se os usuários podem fazer a autenticação de todos os domínios da floresta:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Editar.

  4. Clique em Abrir Utilitário de Diretório.

  5. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  6. Clique em Serviços.

  7. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  8. Na lista de serviços, selecione Active Directory e clique no botão Editar (/).

  9. Se as opções avançadas estiverem ocultas, clique em "Mostrar Opções Avançadas".

  10. Clique em Administrativo.

  11. Selecione “Permitir autenticação de qualquer domínio da floresta”.

    Se você selecionar “Permitir autenticação de qualquer domínio na floresta”, você pode adicionar a floresta do Active Directory às políticas de busca personalizadas do computador de autenticação e contatos.

    Quando você adiciona uma floresta do Active Directory a uma política de busca personalizada, a floresta aparece na lista de domínios de diretórios disponíveis como “/Active Directory/All Domains”. Esse é o ajuste padrão.

    Se você desmarcar “Permitir autenticação de qualquer domínio da floresta”, você pode adicionar domínios do Active Directory individualmente às políticas de busca personalizadas do computador de autenticação e contatos.

    Quando você adiciona domínios do Active Directory a uma política de busca personalizada, cada domínio do Active Directory aparece separadamente na lista dos domínios de diretório disponíveis.

  12. Clique em OK.

  13. Depois de selecionar “Permitir autenticação de qualquer domínio da floresta”, altere a política de busca personalizada no painel Autenticação e no painel Contatos para incluir a floresta do Active Directory ou domínios selecionados.

    Para obter mais informações sobre a alteração de uma política de busca personalizada, consulte Como definir políticas de busca personalizadas.
