Como alterar os grupos do Active Directory que podem administrar o computador

Em um computador configurado para usar o conector do Active Directory do Utilitário de Diretório, você pode identificar as contas de grupos do Active Directory cujos membros deseja que tenham privilégios de administrador no computador.

Os usuários que sejam membros destas contas de grupo do Active Directory podem executar tarefas administrativas, tais como instalar software no computador Mac OS X que você está configurando.

Para adicionar ou remover contas de grupo do Active Directory cujos membros tenham privilégios de administrador:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Editar.

  4. Clique em Abrir Utilitário de Diretório.

  5. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, selecione Active Directory e clique no botão Editar (/).

  8. Se as opções avançadas estiverem ocultas, clique em "Mostrar Opções Avançadas".

  9. Clique em Administrativo.

  10. Selecione “Permitir administração por” e altere a lista de contas de grupo do Active Directory cujos membros deseja que tenham privilégios administrativos:

    • Adicione um grupo clicando no botão Adicionar (+) e digitando o nome de domínio do Active Directory, uma barra invertida e o nome da conta de grupo (por exemplo, ADS\Domain Admins,IL2\Domain Admins).

    • Apague um grupo selecionando-o na lista e, em seguida, clicando no botão Apagar (–).

  11. Clique em OK.
