Como editar contas de usuário e outros registros no Active Directory

Você pode usar o Workgroup Manager para fazer alterações a contas de usuário, contas de grupo, grupos de computadores e outros registros em um domínio do Active Directory. Você também pode usar o Workgroup Manager para excluir registros em um domínio do Active Directory.

Se o esquema do Active Directory tiver sido estendido para incluir os tipos de registros (classes de objeto) e atributos padrão do Mac OS X, você pode usar o Workgroup Manager para criar e editar grupos de computadores no domínio do Active Directory.

Para obter informações adicionais, navegue por Visão Geral do Gerenciamento de Usuário.

Para criar contas de usuário ou de grupo em um domínio do Active Directory, use as ferramentas administrativas do Active Directory da Microsoft em um computador de administração de servidor do Windows.
