Como usar as configurações avançadas dos Serviços de Diretório

O Utilitário de Diretório lista os serviços de diretório que o Mac OS X pode acessar. A lista inclui os serviços de diretório que dão ao Mac OS X acesso às informações de usuário e outros dados administrativos armazenados nos domínios de diretório.

Você pode ativar ou desativar o acesso a cada serviço de diretório. Se você desativar um serviço no Utilitário de Diretório, o Mac OS X não voltará a acessar aquele serviço de diretório.
