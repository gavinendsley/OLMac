Informações sobre o Utilitário de Diretório

O Utilitário de Diretório determina quais serviços de diretório são usados por um computador Mac OS X e como este se conecta a domínios de diretório específicos. O Utilitário de Diretório também define as políticas de busca para encontrar informações sobre autenticação e contatos em domínios de diretório específicos.

Como configurar o Utilitário de Diretório

Você pode usar o Utilitário de Diretório em seu computador para configurar e gerenciar como seu computador acessará serviços de diretório. Você também pode usar o Utilitário de Diretório para configurar remotamente a forma como seu computador acessa serviços de diretório. O Utilitário de Diretório também pode ser usado para ativar a conta raiz ou alterar a senha dessa conta.

Como configurar o acesso a serviços

Você pode usar o Utilitário de Diretório para ativar e desativar diferentes tipos de serviços que o Mac OS X pode acessar em uma rede. Entre eles, se incluem os serviços de diretório que oferecem acesso a contas de usuários e a outros dados administrativos armazenados nos domínios de diretório.

Como configurar o acesso a diretórios

Com o Utilitário de Diretório, você pode configurar o computador para que acesse contas de usuários e outras informações administrativas em diretórios LDAP específicos, em um domínio Active Directory, em um domínio NIS e em domínios NetInfo legados.

Como configurar políticas de busca

Com o Utilitário de Diretório, você pode definir a política de busca de autenticação que o Mac OS X usa para localizar e recuperar informações de autenticação do usuário e outros dados administrativos dos domínios de diretório. Ele também define a política de busca de contatos que o Mac OS X usa para localizar e recuperar o nome, o endereço e outras informações de contato dos domínios de diretório.
