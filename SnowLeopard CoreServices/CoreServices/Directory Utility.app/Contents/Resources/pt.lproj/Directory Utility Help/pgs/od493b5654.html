Como preencher os Diretórios LDAP com dados para Mac OS X

Depois de configurar o acesso a domínios de diretórios LDAP e configurar o gerenciamento de dados, você pode preenchê-los com registros e dados para Mac OS X. Para diretórios LDAP que permitem a administração remota (acesso de leitura/gravação), você pode usar o Workgroup Manager (incluso no Mac OS X Server) da seguinte forma:

  • Identifique os pontos e os domínios compartilhados que deseja montar automaticamente nos navegadores da rede dos usuários (aquilo que os usuários vêem quando clicam em Rede na barra lateral da janela do Finder).

    Use o Compartilhamento no Server Admin e módulos de Rede do Workgroup Manager.

    Para obter informações adicionais, navegue por Visão Geral dos Serviços de Arquivos e Visão Geral do Gerenciamento de Usuário.

  • Defina os registros de usuário e de grupo e configure seus ajustes.

    Use o módulo Contas do Workgroup Manager.

    Para obter informações adicionais, navegue por Visão Geral do Gerenciamento de Usuário.

  • Defina listas de computadores que possuam os mesmos ajustes de preferências e que estejam disponíveis para os mesmos usuários e grupos.

    Use o módulo Computadores do Workgroup Manager.

    Para obter informações adicionais, navegue por Visão Geral do Gerenciamento de Usuário.

Em todos os casos, clique no ícone do pequeno globo acima da lista de usuários e escolha, do menu local no Workgroup Manager, para abrir o domínio de diretório LDAP. Se o diretório LDAP não estiver listado no menu local, escolha Outro nesse menu para selecionar o diretório LDAP.

Nota:  para adicionar registros e dados a um diretório LDAP somente leitura, você precisa usar as ferramentas do servidor que hospeda o diretório LDAP.
