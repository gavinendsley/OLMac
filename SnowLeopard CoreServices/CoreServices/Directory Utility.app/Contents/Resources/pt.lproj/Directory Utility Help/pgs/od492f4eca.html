Como adicionar uma conexão ao servidor de Open Directory

Ao adicionar um servidor de Open Directory, você precisa saber o nome ou o endereço IP do servidor e se o servidor usa SSL (camada de soquete segura, na sigla em inglês).

Para adicionar um servidor de Open Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Conectar ou Editar.

    Se estiver vendo o botão Editar, isso significa que seu computador possui pelo menos uma conexão a um servidor de diretório.

  4. Clique no botão Adicionar (+).

  5. No menu local “Adicionar um novo diretório de tipo”, escolha Open Directory.

  6. No campo “Nome ou Endereço IP do Servidor”, digite o nome ou o endereço IP do servidor.

  7. (Condicional) Antes de selecionar a opção “Criptografar usando SSL”, pergunte ao seu administrador do Open Directory se é necessário usar SSL.

Importante:  se você alterar o endereço IP e o nome do computador do seu servidor Mac OS X usando changeip enquanto estiver conectado a um servidor de diretório, você deve desconectar e conectar novamente o servidor de diretório para atualizar o diretório com o nome e endereço IP do novo computador. Se você não desconectar e conectar novamente o servidor de diretório, o diretório não atualizará e continuará a usar o nome e o endereço IP do antigo computador.
