Como ajustar contas de usuários móveis no Active Directory

Você pode habilitar ou desabilitar contas de usuário móveis do Active Directory em um computador configurado para usar o conector do Active Directory do Utilitário de Diretório. Usuários com contas móveis podem iniciar sessão usando suas credenciais Active Directory enquanto o computador está desconectado do servidor Active Directory.

O conector do Active Directory guarda credenciais para uma conta móvel de usuário quando o usuário inicia a sessão com o computador conectado ao domínio do Active Directory. Este armazenamento de credenciais em cache não requer a alteração do esquema do Active Directory.

Se o esquema do Active Directory foi ampliado para incluir os atributos de clientes gerenciados do Mac OS X, serão usadas as suas configurações de conta móvel em vez do ajuste de conta móvel do conector do Active Directory.

Você pode criar contas móveis automaticamente ou pode pedir ao usuários do Active Directory que confirmem a criação de uma conta móvel.

Para ativar ou desativar contas móveis em um domínio do Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Conectar ou Editar.

  4. Clique em Abrir Utilitário de Diretório.

  5. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, selecione Active Directory e clique no botão Editar (/).

  8. Se as opções avançadas estiverem ocultas, clique em "Mostrar Opções Avançadas".

  9. Clique em Experiência do Usuário, depois clique em “Criar uma conta móvel ao iniciar sessão” e, opcionalmente, clique em “Necessita confirmação antes de criar uma conta móvel”.

    • Se ambas opções estiverem selecionadas, cada usuário decidirá se deseja criar uma conta móvel durante o login. Quando um usuário inicia a sessão no Mac OS X usando uma conta de usuário do Active Directory, ou quando o faz como um usuário de rede, o usuário vê uma caixa de diálogo com controles para a criação imediata de uma conta móvel.

    • Se a primeira opção for selecionada e a segunda opção, desmarcada, as contas móveis serão criadas quando os usuários iniciam a sessão.

    • Se a primeira opção não estiver selecionada, a segunda opção será desmarcada.

  10. Clique em OK.
