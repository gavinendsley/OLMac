Como ajustar as pastas Início das contas de usuários do Active Directory

Em um computador configurado para usar um conector do Active Directory do Utilitário de Diretório, você pode habilitar ou desabilitar as pastas de início de rede ou pastas de início locais de contas de usuário do Active Directory.

Com a pasta de início de rede, a pasta de início de rede do Windows é montado como a pasta de início do Mac OS X quando o usuário inicia a sessão.

Você determina se a localização da pasta de início de rede será obtida do atributo homeDirectory padrão do Active Directory ou do atributo homeDirectory do Mac OS X, se o esquema do Active Directory tiver sido estendido para incluí-lo.

Com as pastas de início locais, cada usuário do Active Directory que iniciou a sessão possui uma pasta de início no disco de inicialização do Mac OS X. Além disso, a pasta de início de rede do usuário está montada como um volume de rede, como um ponto de compartilhamento. O usuário pode copiar arquivos entre este volume de rede e a pasta de início local.

Para configurar pastas de início para contas de usuário do Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Conectar ou Editar.

  4. Clique em Abrir Utilitário de Diretório.

  5. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, selecione Active Directory e clique no botão Editar (/).

  8. Se as opções avançadas estiverem ocultas, clique em "Mostrar Opções Avançadas".

  9. Clique em "Experiência do Usuário".

  10. Se desejar que as contas de usuário do Active Directory tenham pastas de início locais na pasta /Usuários do computador, clique em “Forçar a pasta de início local no disco de inicialização”.

    Esta opção não está disponível se a opção “Criar uma conta móvel ao iniciar sessão” estiver selecionada.

  11. Para usar o atributo padrão do Active Directory para a localização da pasta de início, selecione “Usar caminho UNC do Active Directory para obter localização de início de rede” e, em seguida, escolha um dos seguintes protocolos para acessar a pasta de início:

    • Para usar o protocolo SMB padrão do Windows, escolha smb no menu local “Protocolo de rede que será usado”.

    • Para usar o protocolo AFP padrão do Macintosh, escolha afp no menu local “Protocolo de rede que será usado”.

  12. Para usar o atributo do Mac OS X para a localização da pasta de início, desmarque “Usar caminho UNC do Active Directory para obter início da rede”.

    Para usar o atributo do Mac OS X, o esquema do Active Directory deve ser ampliado para incluí-lo.

  13. Clique em OK.

Se você alterar o nome de uma conta de usuário no domínio do Active Directory, o servidor cria uma pasta de início (e subpastas) para a conta de usuário da próxima vez que for usado para fazer login em um computador Mac OS X. O usuário ainda pode navegar para a pasta de início antiga e ver seu conteúdo no Finder.

Você pode impedir a criação de uma nova pasta de início renomeando a pasta antiga antes que o usuário faça login.
