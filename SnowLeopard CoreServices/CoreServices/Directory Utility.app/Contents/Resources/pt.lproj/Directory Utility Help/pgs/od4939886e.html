Como especificar os ajustes do arquivo de configuração BSD

Historicamente, os computadores UNIX armazenaram os dados administrativos em arquivos de configuração, tais como /etc/master.passwd, /etc/group e /etc/hosts. O Mac OS X está baseado em uma versão BSD do UNIX, mas normalmente contém dados administrativos de sistemas de diretório.

O Mac OS X Server é compatível com um conjunto fixo de arquivos de configuração BSD. Você não pode especificar quais arquivos de configuração usar nem mapear seu conteúdo para os atributos e tipos de registros do Mac OS X.

No Mac OS X v10.2 ou versões posteriores (incluindo o Mac OS X Server v10.2 ou posterior), o Open Directory pode recuperar os dados administrativos dos arquivos de configuração BSD. Esse recurso permite que as empresas que possuem arquivos de configuração BSD usem cópias dos arquivos existentes em computadores Mac OS X. Os arquivos de configuração BSD podem ser usados sozinhos ou com outros domínios de diretório.

Para usar arquivos de configuração BSD:

  1. Certifique-se de que os arquivos de configuração BSD contenham os dados requeridos pelos serviços de diretório do Mac OS X.

    Para obter mais informações, consulte Como ajustar os dados dos arquivos de configuração BSD.

  2. Abra as Preferências do Sistema e clique em Contas.

  3. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  4. Clique em Opções de Início de Sessão, depois clique em Conectar ou Editar.

  5. Clique em Abrir Utilitário de Diretório.

  6. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  7. Clique em Serviços.

  8. Na lista de serviços, selecione “Arquivo Plano BSD e NIS” e clique no botão Editar (/).

  9. Selecione “Use os registros de Usuário e Grupo no nó local BSD,” depois clique em OK.

    O domínio dos arquivos de configuração BSD é adicionado à política de busca de autenticação do computador como /BSD/local.
