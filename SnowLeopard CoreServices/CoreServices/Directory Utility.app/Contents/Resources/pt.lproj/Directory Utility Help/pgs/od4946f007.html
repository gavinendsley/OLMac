Como alterar a senha usada para a autenticação de uma conexão LDAP

Usando o Utilitário de Diretório, você pode atualizar uma conexão de autenticação LDAP para usar uma senha que tenha sido alterada no servidor LDAP. Todos os computadores que possuam uma conexão de autenticação a um servidor LDAP devem ser atualizadas se a senha usada para autenticar a conexão LDAP for alterada no servidor.

Para alterar a senha de uma conexão LDAP:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Editar.

  4. Clique em Abrir Utilitário de Diretório.

  5. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, selecione LDAPv3 e clique no botão Editar (/).

  8. Se a lista de configurações de servidor estiver oculta, clique em Mostrar Opções.

  9. Selecione uma configuração de servidor na lista e clique em Editar.

  10. Clique em Segurança e altere o ajuste Senha:

    • Se a configuração de Senha estiver acinzentada porque a opção “Usar autenticação ao conectar” não estiver selecionada, consulte Como autenticar uma conexão LDAP.

    • Se o ajuste Senha estiver acinzentado porque a opção “Vincular ao diretório como” estiver selecionada (embora acinzentada), isso significa que a conexão não está autenticada com uma senha de usuário. Em vez disso, a conexão usa o registro de um computador autenticado para a vinculação confiável.
