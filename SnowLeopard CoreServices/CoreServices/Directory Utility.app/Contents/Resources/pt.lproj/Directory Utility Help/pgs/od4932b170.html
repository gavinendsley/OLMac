Sobre as conexões do servidor de diretório

Você pode usar o painel de preferências Conta para conectar os computadores aos servidores de diretório. Você pode visualizar listas de servidores de diretório que o seu computador está conectado clicando em Editar no painel Opções de Início das preferências Conta. O seu computador Mac OS X acessa os servidores na lista para obter informações do usuário e outros dados administrativos armazenados no domínio do diretório dos servidores de diretório.

Quando você adiciona ou apaga um servidor na lista Servidores de Diretório, as entradas associadas a esse servidor de diretório são adicionadas ou apagadas da lista Serviços, Autenticação e Contatos. Entretanto, se você remover as entradas associadas dentro das listas Serviços, Autenticação e Contatos, o servidor de diretório não será removido da lista de Servidores de Diretório.

Os computadores com Mac OS X v10.6 podem conectar-se a um servidor de diretório Open Directory, Active Directory ou LDAP. Se você não sabe a qual servidor se conectar, pergunte a seu administrador de rede.

Importante:  se o nome de seu computador contiver um hífen, talvez não seja possível unir-se ou vincular-se a um domínio de diretório como LDAP ou Active Directory. Para estabelecer o vínculo, use um nome de computador que não contenha um hífen.
