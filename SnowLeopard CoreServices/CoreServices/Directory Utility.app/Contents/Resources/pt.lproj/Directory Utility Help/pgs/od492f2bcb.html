Como gerenciar a conta do usuário raiz

Você pode usar o Utilitário de Diretório (localizado no painel de preferências Contas) para gerenciar a conta do usuário raiz ativando ou desativando o usuário raiz. Se você decidiu ativar a conta do usuário raiz, você também pode usar o Utilitário de Diretório para alterar a senha da conta raiz.

Para obter mais informações sobre o gerenciamento da conta do usuário raiz usando o Utilitário de Diretório, consulte:

  • Como ativar a conta do usuário raiz

  • Como alterar a senha da conta do usuário raiz
