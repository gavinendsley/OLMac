Como especificar um servidor preferido do Active Directory

Em um computador configurado para usar o conector do Active Directory do Utilitário de Diretório, você pode especificar o nome DNS do servidor cujo domínio do Active Directory deseja que o computador acesse por padrão.

Se o servidor ficar indisponível no futuro, o conector do Active Directory reverterá para um outro servidor mais próximo na floresta.

Se esta opção estiver desmarcada, o conector do Active Directory determinará o domínio do Active Directory mais próximo na floresta.

Para especificar um servidor de preferência para que o conector do Active Directory acesse:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  3. Clique em Opções de Início de Sessão, depois clique em Editar.

  4. Clique em Abrir Utilitário de Diretório.

  5. Clique em Serviços.

  6. Se o ícone de bloqueio estiver ativado, desative-o clicando sobre ele e digitando o nome e a senha de um administrador.

  7. Na lista de serviços, selecione Active Directory e clique no botão Editar (/).

  8. Se as opções avançadas estiverem ocultas, clique em "Mostrar Opções Avançadas".

  9. Clique em Administrativo.

  10. Selecione “Servidor de domínios preferencial” e digite o nome DNS do servidor do Active Directory.

  11. Clique em OK.
