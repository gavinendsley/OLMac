在远程服务器上设置目录实用工具

您可以使用电脑上的“目录实用工具”来设置和管理远程服务器上的 Mac OS X Server 如何访问目录服务。

要在远程服务器上配置目录访问：

  1. 打开您的电脑上的“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

    如果您看到“编辑”按钮，则您的电脑至少有一个至目录服务器的连接。

  4. 点按“打开目录实用工具”。

  5. 从“文件”菜单中选取“连接”。

  6. 为您想要配置的服务器输入以下连接和鉴定信息。

    • 地址：输入您想要配置的服务器的 DNS 名称或 IP 地址。

    • 用户名称：输入服务器上的管理员的用户名称。

    • 密码：输入您已输入的用户名称的相应密码。

  7. 点按“连接”。

  8. 点按“服务”和“搜索策略”标签，根据需要更改设置。

    您做的更改会影响您在前面步骤中连接到的远程服务器。

  9. 从您的电脑上的“文件”菜单中选取“断开连接”。
