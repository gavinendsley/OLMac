编辑 Active Directory 中的用户帐户和其他记录

您可以使用 Workgroup Manager 更改 Active Directory 域中的用户帐户、组别帐户、电脑列表和其他记录。您还可以使用 Workgroup Manager 删除 Active Directory 域中的记录。

如果 Active Directory 模式已经扩展为包含标准 Mac OS X 记录类型（对象类）和属性，您可以使用 Workgroup Manager 创建和编辑 Active Directory 域中的电脑组别。

有关附加信息，请浏览“用户管理概述”。

要在 Active Directory 域中创建用户或组别帐户，请在 Windows 服务器管理电脑上使用 Microsoft Active Directory 管理工具。
