添加 Open Directory 服务器连接

在添加 Open Directory 服务器时，您必须知道服务器名称或 IP 地址以及服务器是否使用安全套接字层 (SSL)。

要添加 Open Directory 服务器：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

    如果您看到“编辑”按钮，则您的电脑至少有一个至目录服务器的连接。

  4. 点按添加按钮 (+)。

  5. 从“添加新的目录类型”弹出式菜单中选取“Open Directory”。

  6. 在“服务器名称或 IP 地址”栏中，输入服务器名称或 IP 地址。

  7. （有条件地）在选择“使用 SSL 加密”注记格之前，请询问 Open Directory 管理员是否需要 SSL。

重要事项：在您连接到目录服务器时如果要使用 changeip 更改 IP 地址和 Mac OS X 服务器的电脑名称，必须断开连接后重新连接目录服务器，使用新的电脑名称和 IP 地址更新目录。如果您不断开连接并重新连接目录服务器，则目录将不更新，并继续使用旧的电脑名称和 IP 地址。
