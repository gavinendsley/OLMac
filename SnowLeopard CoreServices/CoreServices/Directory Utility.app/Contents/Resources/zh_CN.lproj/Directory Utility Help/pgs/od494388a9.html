启用或停用 Active Directory 服务

您可以使用“目录实用工具”启用或停用由 Windows 服务器提供的 Active Directory 服务。Active Directory 是 Windows 2000 和更高版本服务器的目录服务。

如果您停用 Active Directory 服务并且 Active Directory 域包含在自定搜索策略中，则在“目录实用工具”中“搜索策略”的“鉴定”或“联系”面板中这些域会显示为红色。

要启用或停用对 Active Directory 的访问：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在 Active Directory 旁边，选择或取消选择注记格，然后点按“应用”。
