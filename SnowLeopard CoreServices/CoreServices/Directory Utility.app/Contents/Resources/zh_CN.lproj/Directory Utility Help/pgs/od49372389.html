控制在 Active Directory 网域树系中所有域的鉴定

在配置为使用“目录实用工具”的 Active Directory 连接器的电脑上，您可以允许 Active Directory 网域树系的用户在所有域中进行鉴定，也可以限制用户只能在单个域中进行鉴定。

要控制用户是否能够从网域树系的所有域进行鉴定：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  8. 在服务列表中选择“Active Directory”，然后点按编辑按钮 (/)。

  9. 如果高级选项处于隐藏状态，请点按“显示高级选项”。

  10. 点按“管理”。

  11. 选择“允许来自网域树系中任何域的鉴定”。

    如果您选择了“允许来自网域树系中任何域的鉴定”，则可以将 Active Directory 网域树系添加到电脑的鉴定和联系自定搜索策略中。

    在将 Active Directory 网域树系添加到自定搜索策略时，网域树系将出现在可用目录域（如“/Active Directory/All Domains”）的列表中。（此为默认设置。）

    如果您取消选择“允许来自网域树系中任何域的鉴定”，则可以将 Active Directory 域单独添加到电脑的鉴定和联系自定搜索策略中。

    在将 Active Directory 域添加到自定搜索策略时，每个 Active Directory 域将分别出现在可用目录域的列表中。

  12. 点按“好”。

  13. 选择“允许来自网域树系中任何域的鉴定”选项后，在“鉴定”面板和“联系”面板中更改自定搜索策略，使其包括 Active Directory 网域树系或选定域。

    有关更改自定搜索策略的更多信息，请参阅“定义自定搜索策略”。
