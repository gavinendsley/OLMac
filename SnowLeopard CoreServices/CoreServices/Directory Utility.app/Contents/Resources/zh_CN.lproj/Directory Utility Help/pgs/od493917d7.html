准备只读 LDAP 目录以用于 Mac OS X

如果您想要 Mac OS X 电脑从只读 LDAP 目录获得管理数据，则该数据必须采用 Mac OS X 所要求的格式。您可能需要在只读 LDAP 目录中添加、更改或重新组织数据。

由于 Mac OS X 不能向只读目录写入数据，因此您必须使用其他工具进行更改。这些工具必须位于托管只读 LDAP 目录的服务器上。

要准备只读 LDAP 目录以用于 Mac OS X：

  1. 前往托管只读 LDAP 目录的服务器，将其配置为支持基于 LDAP 的鉴定和密码检查。

  2. 根据需要更改 LDAP 目录的对象类和属性，以提供 Mac OS X 所需的数据。

有关 Mac OS X 目录服务所需数据的规格，请参阅“Open Directory Administration”。
