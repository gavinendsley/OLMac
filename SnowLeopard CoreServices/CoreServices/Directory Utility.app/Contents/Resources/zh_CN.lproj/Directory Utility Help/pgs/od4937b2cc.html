更改 LDAP 连接的查询超时时间

使用“目录实用工具”，您可以指定在 Open Directory 等待多长时间后取消已发送到 LDAP 目录的查询。

要设置 LDAP 连接的查询超时时间：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“LDAPv3”，然后点按编辑按钮 (/)。

  8. 如果服务器配置列表处于隐藏状态，请点按“显示选项”。

  9. 在列表中选择服务器配置，然后点按“编辑”。

  10. 点按“连接”，然后在“查询__秒后超时”中输入值。

    默认值为 120 秒。
