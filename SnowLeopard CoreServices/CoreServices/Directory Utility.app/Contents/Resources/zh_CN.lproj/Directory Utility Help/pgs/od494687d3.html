定义自定搜索策略

使用“目录实用工具”，您可以将 Mac OS X 电脑的鉴定和联系搜索策略配置为使用自定目录域列表。

自定列表从电脑的本地目录域开始，包括 Open Directory（以及其他 LDAP 目录域）、Active Directory 域、共享的目录域、BSD 配置文件和 NIS 域。

如果在电脑的自定搜索策略中指定的某个目录域不可用，则电脑启动时将会发生延迟。

要为搜索策略指定一个自定目录域列表：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“搜索策略”并选取一个搜索策略。

    • 鉴定：显示用于鉴定和大多数其他管理数据的搜索策略。

    • 联系：显示用于应用程序（如“地址簿”）中的联系信息的搜索策略。

  7. 从“搜索”弹出式菜单中选取“自定路径”。

  8. 根据需要添加目录域，方法是点按“添加”，选择目录，然后再次点按“添加”。

  9. 根据需要更改所列目录域的顺序，方法是在列表中上下拖移它们。

  10. 对于您不想包括在搜索策略中的目录域，可以在列表中选择它们，然后点按删除按钮 (-)，即可从列表中删除这些目录域。

  11. 点按“好”确认删除，然后点按“应用”。

要添加一个在可用目录中没有列出的目录，请确定电脑已配置为访问该目录。有关更多信息，请参阅：

  • 使用高级目录服务设置

  • 使用高级 LDAP 服务设置

  • 使用高级 Active Directory 服务设置

  • 指定 NIS 设置

  • 指定 BSD 配置文件设置
