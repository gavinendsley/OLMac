关于目录服务器连接

您可以使用“帐户”偏好设置将电脑连接到目录服务器。您可以查看电脑所连接的目录服务器的列表，方法是点按“帐户”偏好设置中的“登录选项”面板上的“编辑”。您的 Mac OS X 电脑访问列表中的服务器，从而获取目录服务器的目录域中存储的用户信息和其他管理数据。

在“目录服务器”列表中添加或删除服务器时，也会在“服务”、“鉴定”和“联系”列表中添加或删除与该目录服务器关联的条目。但是，如果您在“服务”、“鉴定”和“联系”列表中删除关联的条目，目录服务器并不会从“目录服务器”列表中删除。

Mac OS X v10.6 电脑可以连接到 Open Directory、Active Directory 或 LDAP 目录服务器。如果不知道连接到哪个服务器，请询问网络管理员。

重要事项：如果电脑名称包含连字符，您可能无法加入或捆绑到 LDAP 或 Active Directory 等目录域。要建立捆绑，请使用不包含连字符的电脑名称。
