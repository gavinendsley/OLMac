自动客户端配置

如果您的 Mac OS X v10.6 电脑连接着网络，但没有连接目录域，则您的 Mac OS X v10.6 电脑将会使用 Bonjour 来发现 Mac OS X v10.6 服务器。

如果找到了运行有 Open Directory 主目录的 Mac OS X v10.6 服务器（该服务器正在运行至少一项可自动配置的服务，如 AFP、SMB、VPN、Mail、iCal、iChat、通知），并且目录域上的某个用户帐户与当前登录在 Mac OS X v10.6 电脑上的用户的短名称或长名称相匹配，则您的 Mac OS X v10.6 电脑上会出现一则邀请。

该邀请可以辅助您设定至目录服务器的连接。您可以接受或拒绝连接服务器，或在以后决定是否连接。

如果您的电脑发现多个可以连接的 Mac OS X v10.6 目录服务器，则那些服务器将在邀请弹出式菜单中列出。如果您决定连接一个服务器，助理将会根据目录服务器上运行的服务打开、连接并配置您的应用程序。

要使用自动化助理连接到目录域：

  1. 当助理出现时点按“设置服务”。

  2. 从“服务器”弹出式菜单中选取想要加入的目录域。

    您的网络中可能有多个可用的目录域。

    如果想要加入的服务器不在列表中，请在“服务器”栏输入服务器名称或 IP 地址并点按“好”。

  3. 输入帐户的用户名称和密码，该帐户必须拥有将电脑加入您想要加入的目录域的权限。

    如果您不清楚此信息，请联系您的目录管理员。

  4. 输入用户帐户（出现在“请为这台电脑上的‘用户名’帐户输入密码”中）的密码。

    助理会更改服务器上的用户目录帐户的密码，以便匹配本地电脑帐户的密码。

  5. 点按“继续”。

  6. 当 UpgradeUser 工具完成后，点按“继续”。

  7. 当出现信息说明服务已设置好并请求您注销时，请点按“注销”。

    当您登录后，就可以开始使用新服务了。
