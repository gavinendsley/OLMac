启用 Root 用户帐户

您可以使用“目录实用工具”启用 root 用户帐户。如果您启用 root 用户帐户，请使用包含字母数字和特殊字符的复杂密码，防止密码被泄露。

警告：root 帐户是不受限制的管理员帐户，用于对关键的系统文件执行更改。即使您以管理员身份登录，您仍然必须使用 root 帐户，或使用 sudo 执行关键的系统任务。

不要使用 root 帐户登录电脑（远程或本地）。而应该使用 sudo 来执行 root 任务。您可以通过向 /etc/sudoers/ 文件添加用户来限制对 sudo 的访问。

有关 root 帐户的更多信息，请参阅“Server Admin 帮助”。

要启用 root 用户帐户：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

    如果您看到“编辑”按钮，则您的电脑至少有一个至目录服务器的连接。

  4. 点按“打开目录实用工具”。

  5. 选取“编辑”>“启用 Root 用户”。
