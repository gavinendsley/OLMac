指定 NIS 设置

使用“目录实用工具”，您可以创建指定 Mac OS X 如何访问网络信息服务 (NIS) 域的配置。

要创建用于访问 NIS 域的配置：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中，选择“BSD Flat 文件和 NIS”，然后点按编辑按钮 (/)。

  8. 输入 NIS 域所在服务器的 NIS 域名、DNS 名称或 IP 地址。

    如果有安全性要求，或者服务器与您正在配置的电脑不在同一个子网上，则输入 NIS 服务器的主机名或 IP 地址。

    如果您未指定服务器，NIS 则使用广播协议来寻找子网上的 NIS 服务器。

  9. 选择“将 NIS 域用于鉴定”，然后点按“好”。

    NIS 域将作为 /BSD/domain 添加到电脑的鉴定搜索侧栏，其中 domain 是您在步骤 4 中输入的名称。
