更改用于访问 LDAP 目录的配置

您可以使用“目录实用工具”更改 LDAP 目录配置的设置。配置设置指定 Open Directory 如何访问 LDAPv3 或 LDAPv2 目录。

如果 LDAP 配置由 DHCP 提供，则不能更改，因此该类型的配置在 LDAP 配置列表中显示为暗淡。

要编辑用于访问 LDAP 目录的配置：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“LDAPv3”，然后点按编辑按钮 (/)。

  8. 如果服务器配置列表处于隐藏状态，请点按“显示选项”。

  9. 根据需要对以下设置进行更改：

    • 启用：点按注记格以启用或停用对 LDAP 目录服务器的访问。

    • 配置名称：连按某个配置名称可编辑它。

    • 服务器名称或 IP 地址：连按某个服务器名称或 IP 地址可更改它。

    • LDAP 映射：从弹出式菜单中选取一个模板，输入 LDAP 的搜索基准后缀，然后点按“好”。

      如果选取了一个模板，则必须输入搜索基准后缀，否则电脑无法在 LDAP 目录中查找信息。 通常，搜索基准后缀源自服务器的 DNS 名称。例如，如果服务器的 DNS 名称是“ods.example.com”，搜索基准后缀就为“dc=ods,dc=example,dc=com”。

      如果您选取“来自服务器”而不是选取模板，则不需要搜索基准后缀。在此情况下，Open Directory 假设搜索基准后缀是 LDAP 目录的第一级。

      如果您选取“自定”，则需要在 Mac OS X 记录类型和属性与您要连接到的 LDAP 目录的类和属性之间设置映射。有关更多信息，请参阅“配置 LDAP 搜索和映射”。

    • SSL：点按注记格以启用或停用使用 SSL 协议的加密通信。在选择 SSL 注记格之前，请询问 Open Directory 管理员是否需要 SSL。

  10. 要更改此项 LDAP 配置的以下默认设置，点按“编辑”可显示出所选 LDAP 配置的选项，进行更改，在完成编辑 LDAP 配置选项后点按“好”：

    • 点按“连接”以设置超时选项、指定自定端口、忽略服务器推荐或者强制使用 LDAPv2（只读）协议。有关更多信息，请参阅“更改 LDAP 目录的连接设置”。

    • 点按“搜索与映射”设置 LDAP 服务器的搜索和映射。有关更多信息，请参阅“建立与 LDAP 目录的信任捆绑”。

    • 点按“安全”以设置鉴定连接（不是信任捆绑）和其他安全策略选项。有关更多信息，请参阅“更改 LDAP 连接的安全策略”。

    • 点按“捆绑”设置信任捆绑，或点按“解开捆绑”停止信任捆绑。（如果 LDAP 目录不允许信任捆绑，您可能看不到这些按钮。）有关更多信息，请参阅“建立与 LDAP 目录的信任捆绑”。

  11. 要完成更改访问 LDAP 目录的配置，请点按“好”。
