显示或隐藏 LDAP 服务器的配置

您可以显示或隐藏用于访问 LDAP 目录的可用配置的列表。每个配置都指定 Open Directory 如何访问 LDAP 目录。在列表可见时，您可以更改每个不显示为暗淡的 LDAP 配置的设置。

要显示或隐藏可用的 LDAP 目录配置：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“LDAPv3”，然后点按编辑按钮 (/)。

  8. 点按“显示选项”控制或“隐藏选项”控制（哪一个控制出现就点按哪一个）。
