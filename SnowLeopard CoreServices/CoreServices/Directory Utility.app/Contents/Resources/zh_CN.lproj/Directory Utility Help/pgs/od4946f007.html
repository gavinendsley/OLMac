更改用于鉴定 LDAP 连接的密码

使用“目录实用工具”，您可以更新已鉴定的 LDAP 连接，以便使用已经在 LDAP 服务器上更改的密码。（如果在服务器上更改了用于鉴定 LDAP 连接的密码，则所有带有已鉴定的 LDAP 服务器连接的电脑都必须更新。）

要更改 LDAP 连接的密码：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“LDAPv3”，然后点按编辑按钮 (/)。

  8. 如果服务器配置列表处于隐藏状态，请点按“显示选项”。

  9. 在列表中选择服务器配置，然后点按“编辑”。

  10. 点按“安全”，然后更改“密码”设置：

    • 如果由于没有选择“连接时使用鉴定”，“密码”设置显示为暗淡，请参阅“鉴定 LDAP 连接”。

    • 如果“密码”设置因为选择了“捆绑到目录，身份为”而显示为暗淡，则不使用用户密码鉴定连接。相反，连接会将已鉴定的电脑记录用于信任捆绑。
