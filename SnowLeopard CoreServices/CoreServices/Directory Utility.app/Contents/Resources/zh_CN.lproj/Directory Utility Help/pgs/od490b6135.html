了解目录实用工具

“目录实用工具”用于确定 Mac OS X 电脑可以使用的目录服务以及它连接到特定目录域的方式。 “目录实用工具”也为在特定目录域中查找鉴定和联系信息定义搜索策略。

设置目录实用工具

您可以在电脑上使用“目录实用工具”设置和管理电脑如何访问目录服务。您也可以使用“目录实用工具”远程配置您的电脑如何访问目录服务。“目录实用工具”也可以用于启用 root 帐户或更改 root 帐户密码。

配置服务访问

您可以使用“目录实用工具”来启用和停用 Mac OS X 可以在网络上访问的不同种类的服务。这些服务包括目录服务，它提供对存储在目录域中的用户帐户和其他管理数据的访问。

配置目录访问

使用“目录实用工具”，您可以将电脑配置为访问特定 LDAP 目录、Active Directory 域和 NIS 域中的用户帐户和其他管理信息。

配置目录访问
设置搜索策略

使用“目录实用工具”，您可以定义鉴定搜索策略，Mac OS X 使用它在目录域中查找并获取用户鉴定信息以及其他管理数据。“目录实用工具”还定义联系搜索策略，Mac OS X 使用它查找并获取目录域中的名称、地址和其他联系信息。
