配置 Active Directory 域访问

使用“目录实用工具”中列出的 Active Directory 连接器，您可以将 Mac OS X 配置为访问 Windows 服务器上 Active Directory 域中的基本用户帐户信息。

Active Directory 连接器生成 Mac OS X 鉴定所需的所有属性。Active Directory 模式无需更改。

如果 Active Directory 模式已经扩展为包含了标准 Mac OS X 记录类型和属性（例如 Mac OS X 客户端管理所需的属性），则 Active Directory 连接器将会检测和访问这些记录类型和属性。

警告：使用 Active Directory 连接器的高级选项，您可以将 Mac OS X 的唯一用户 ID (UID)、首选组别 ID (GID) 和组别 GID 属性映射至已添加到 Active Directory 模式中的正确属性。如果您今后更改这些映射选项的设置，用户可能会无法访问以前创建的文件。
重要事项：如果电脑名称包含连字符，您可能无法加入或捆绑到 LDAP 或 Active Directory 等目录域。要建立捆绑，请使用不包含连字符的电脑名称。
要手动配置 Active Directory 域访问：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“Active Directory”，然后点按编辑按钮 (/)。

  8. 输入您想要捆绑到您正在配置的电脑的 Active Directory 域 DNS 名称。

    Active Directory 域管理员会告诉您要输入的 DNS 名称。

  9. 如有必要，请编辑“电脑 ID”。

    “电脑 ID”就是在 Active Directory 域中识别电脑的依据，它被预先设置为电脑的名称。 您可能需要对此进行更改，以符合您的公司制定的有关在 Active Directory 域中命名电脑的模式。 如果您不确定，请询问 Active Directory 域管理员。

  10. （可选）设置高级选项。

    如果高级选项处于隐藏状态，请点按“显示高级选项”，然后设置“用户体验”、“映射”和“管理”面板中的选项。以后也可以更改这些高级选项设置。

    有关高级选项的更多信息，请参阅：

    • 在 Active Directory 中设置移动用户帐户

    • 设置 Active Directory 用户帐户的个人文件夹

    • 设置 Active Directory 用户帐户的 UNIX Shell

    • 将 UID 映射至 Active Directory 属性

    • 将首选组别 ID 映射至 Active Directory 属性

    • 将组别帐户中的组别 ID 映射至 Active Directory 属性

    • 指定首选 Active Directory 服务器

    • 更改可以管理电脑的 Active Directory 组别

    • 控制在 Active Directory 网域树系中所有域的鉴定

  11. 点按“捆绑”，使用以下选项以有权将电脑捆绑到 Active Directory 域的用户身份进行鉴定，选择您想要 Active Directory 添加到的搜索策略（如下所述），然后点按“好”：

    • 用户名称和密码：您也许能够通过输入您的 Active Directory 用户帐户的名称和密码进行鉴定；如果不能，则需要由 Active Directory 域管理员提供名称和密码。

    • 电脑 OU：为您正在配置的电脑输入组织单元 (OU)。

    • 用于鉴定：用于确定 Active Directory 是否添加到电脑的鉴定搜索策略。

    • 用于联系：用于确定 Active Directory 是否添加到电脑的联系搜索策略。

    点按“好”后，“目录实用工具”在正在配置的电脑与 Active Directory 服务器之间建立信任捆绑。电脑的搜索策略根据鉴定时选定的选项进行设置，并且 Active Directory 在“目录实用工具”的“服务”面板中启用。

    通过 Active Directory 高级选项的默认设置，在选择了“用于鉴定”或“用于联系”的情况下，Active Directory 网域树系将添加到电脑的鉴定搜索策略和联系搜索策略中。

    但是，如果在点按“捆绑”之前在“管理”高级选项面板中取消选择“允许来自网域树系中任何域的鉴定”选项，系统将添加最近的 Active Directory 域，而不添加网域树系。

    您可以在以后通过添加或删除 Active Directory 网域树系或单个域来更改搜索策略。有关更多信息，请参阅“定义自定搜索策略”。

  12. （可选）将服务器加入 Active Directory Kerberos 领域：

    • 在可以连接到服务器的服务器或管理员电脑上，打开“服务器管理”，然后为服务器选择“Open Directory”。

    • 点按“设置”，然后点按“通用”。

    • 点按“加入 Kerberos”，然后从弹出式菜单中选取 Active Directory Kerberos 领域，输入本地管理员在此服务器上的凭证。
