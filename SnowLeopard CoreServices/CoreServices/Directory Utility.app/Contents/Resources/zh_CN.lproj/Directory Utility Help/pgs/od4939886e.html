指定 BSD 配置文件设置

过去，UNIX 电脑将管理数据存储在 /etc/master.passwd、/etc/group 和 /etc/hosts 等配置文件中。虽然 Mac OS X 建立于 UNIX 的 BSD 版本基础之上，但它通常要从目录系统中获得管理数据。

Mac OS X Server 版本支持固定的一组的 BSD 配置文件。您不能指定使用哪些配置文件，也不能将其内容映射至 Mac OS X 的记录类型和属性。

在 Mac OS X v10.2 或更高版本（包括 Mac OS X Server v10.2 或更高版本）中，Open Directory 可以从 BSD 配置文件获取管理数据。此功能使拥有 BSD 配置文件的机构可以在 Mac OS X 电脑上使用现有文件的副本。BSD 配置文件可以单独使用，也可与其他目录域配合使用。

要使用 BSD 配置文件：

  1. 确定 BSD 配置文件包含 Mac OS X 目录服务所需的数据。

    有关更多信息，请参阅“设置 BSD 配置文件中的数据”。

  2. 打开“系统偏好设置”并点按“帐户”。

  3. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  4. 点按“登陆选项”，然后点按“加入”或“编辑”。

  5. 点按“打开目录实用工具”。

  6. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  7. 点按“服务”。

  8. 在服务列表中，选择“BSD Flat 文件和 NIS”，然后点按编辑按钮 (/)。

  9. 选择“使用 BSD 本地节点中的用户和组别记录”，然后点按“好”。

    此时，BSD 配置文件域将作为 /BSD/local 添加到电脑的鉴定搜索策略。
