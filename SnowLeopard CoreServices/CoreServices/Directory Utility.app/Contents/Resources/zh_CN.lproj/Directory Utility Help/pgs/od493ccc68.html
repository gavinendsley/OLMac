解开与 Active Directory 服务器的捆绑

如果电脑使用“目录实用工具”的 Active Directory 连接器捆绑到 Active Directory 服务器，您可以解开电脑与 Active Directory 服务器的捆绑。

如果电脑无法联系服务器或者电脑记录已从此服务器删除，您可以强制解开捆绑。

要解开电脑与 Active Directory 服务器的捆绑：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“Active Directory”，然后点按编辑按钮 (/)。

  8. 点按“解开捆绑”，以有权终止 Active Directory 域连接的用户身份进行鉴定，然后点按“好”。

    如果您看到一则警告，提示不接受凭证或者电脑无法联系 Active Directory，请点按“强制解开捆绑”以强制断开此连接。

    如果您强制解开捆绑，Active Directory 仍将包含此电脑的电脑记录。通知 Active Directory 管理员，让管理员知道要删除此电脑的记录。

  9. 在“服务”面板中，取消选择 Active Directory 的“启用”设置，然后点按“应用”。
