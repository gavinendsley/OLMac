将首选组别 ID 映射至 Active Directory 属性

在配置为使用“目录实用工具”的 Active Directory 连接器的电脑上，您可以在用户帐户中指定一个要映射至 Mac OS X 的首选组别 ID (GID) 属性的 Active Directory 属性。

通常，必须扩展 Active Directory 模式，使其包含一个适合映射至首选 GID 的属性：

  • 如果 Active Directory 管理员通过安装 Microsoft 的 UNIX 服务来扩展 Active Directory 模式，您就可以将首选 GID 映射至 msSFU-30-Gid-Number 属性。

  • 如果 Active Directory 管理员手动将 Active Directory 模式扩展为包含 RFC 2307 属性，您就可以将首选 GID 映射至 gidNumber。

  • 如果 Active Directory 管理员手动将 Active Directory 模式扩展为包含 Mac OS X PrimaryGroupID 属性，您就可以将首选 GID 映射至该属性。

如果停用了首选 GID 映射，Active Directory 连接器会根据 Active Directory 的标准 GUID 属性生成一个首选 GID。

警告：如果您今后更改首选 UID 的映射，用户可能会无法访问以前创建的文件。
要在扩展的 Active Directory 模式中将首选 UID 映射至属性：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“Active Directory”，然后点按编辑按钮 (/)。

  8. 如果高级选项处于隐藏状态，请点按“显示高级选项”。

  9. 点按“映射”。

  10. 选择“将用户 GID 映射至属性”，并输入用户帐户中要映射至首选组别 ID 的 Active Directory 属性的名称。

  11. 点按“好”。
