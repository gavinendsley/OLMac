更改 Root 用户帐户密码

您可以使用“目录实用工具”（位于“帐户”偏好设置中）来更改 root 帐户的密码。在更改 root 密码时，请使用包含字母数字和特殊字符的复杂密码，防止密码泄露。

警告：  root 帐户是不受限制的管理员帐户，用于对关键的系统文件执行更改。即使您以管理员身份登录，您仍然必须使用 root 帐户，或使用 sudo 执行关键的系统任务。

不要使用 root 帐户登录电脑（远程或本地）。而应该使用 sudo 来执行 root 任务。您可以向 /etc/sudoers/ 文件添加用户，限制对 sudo 的访问。

有关 Root 帐户的更多信息，请参阅“Server Admin 帮助”。有关创建密码的更多信息，请参阅“编写密码”。

要更改 root 用户帐户密码：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

    如果您看到“编辑”按钮，则您的电脑至少有一个至目录服务器的连接。

  4. 点按“打开目录实用工具”。

  5. 选取“编辑”>“更改 Root 密码”。

  6. 出现提示时，在“密码”和“验证”栏中键入新的 root 密码。

  7. 点按“好”。
