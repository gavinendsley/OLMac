设置 BSD 配置文件中的数据

如果您想要 Mac OS X 电脑从 BSD 配置文件获得管理数据，则该数据必须在文件中，并且必须采用 Mac OS X 要求的格式。

您可能需要在文件中添加、更改或重新组织数据。Workgroup Manager 不能更改 BSD 配置文件中的数据，因此您必须使用文本编辑器或其他工具进行必要的修改。

下面的表格列出了文件名称并描述了文件的内容。

BSD 配置文件
包含

/etc/master.passwd

用户名、密码、ID、首选组别 ID 等

/etc/group

组别名称、ID 和成员

/etc/fstab

NFS 装载

/etc/hosts

电脑名称和地址

/etc/networks

网络名称和地址

/etc/services

服务名称、端口和协议

/etc/protocols

IP 协议名称和编号

/etc/rpcs

开放网络计算 RPC 服务器

/etc/printcap

打印机名称和功能

/etc/bootparams

bootparams 设置

/etc/bootp

Bootp 设置

/etc/aliases

邮件别名和分发列表

/etc/netgroup

网络内的组别名称和成员
