在 Active Directory 中设置移动用户帐户

您可以在配置为使用“目录实用工具”的 Active Directory 连接器的电脑上启用或停用 Active Directory 移动用户帐户。具有移动帐户的用户可以在电脑未连接到 Active Directory 服务器的情况下使用他们的 Active Directory 凭证登录。

用户在其电脑连接到 Active Directory 域的情况下登录时，Active Directory 连接器会将用户的移动帐户凭证置于高速缓存中。这种凭证高速缓存并不需要更改 Active Directory 模式。

如果 Active Directory 模式已经扩展为包括了 Mac OS X 管理的客户端属性，则系统将使用这些属性的移动帐户设置，而不使用 Active Directory 连接器的移动帐户设置。

您可以让系统自动创建移动帐户，也可以要求 Active Directory 用户确认是否要创建移动帐户。

要启用或停用 Active Directory 域中的移动帐户：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“Active Directory”，然后点按编辑按钮 (/)。

  8. 如果高级选项处于隐藏状态，请点按“显示高级选项”。

  9. 点按“用户体验”，然后点按“登录时创建移动帐户”，并且还可以点按“创建移动帐户前要求确认”。

    • 如果选择了这两个选项，每个用户就可以在登录时确定是否要创建移动帐户。当用户使用 Active Directory 用户帐户登录到 Mac OS X 或者以网络用户身份登录时，用户可以看到一个对话框，其中显示了用于立即创建移动帐户的控制。

    • 如果选择了第一个选项，但未选择第二个选项，则移动帐户会在用户登录时创建。

    • 如果未选择第一个选项，则第二个选项将被停用。

  10. 点按“好”。
