使用高级搜索策略设置

“目录实用工具”定义以下搜索策略：

  • 鉴定：Mac OS X 使用鉴定搜索策略从目录域查找和获取用户鉴定信息和其他管理数据。

  • 联系：Mac OS X 使用联系搜索策略从目录域查找和获取名称、地址和其他联系信息。Mac OS X 地址簿使用此联系信息。其他应用程序也可以通过编程方式使用该信息。

    每个搜索策略包括一个目录域列表。列表中的目录域的顺序定义搜索策略。Mac OS X 从列表顶部开始搜索所列出的每个目录域，直到搜索到所需信息或者达到列表结尾而没有找到信息。

    鉴定和联系搜索策略可能采用以下设置之一：

  • 自动：从本地目录域开始，并且可以包括 DHCP 自动提供的 LDAP 目录和电脑连接到的目录域。这是 Mac OS X v10.2 或更高版本的默认设置，为移动电脑提供最大的灵活性。

  • 本地目录：仅包括本地目录域。

  • 自定路径：从本地目录域开始，并包括您选择的 LDAP 目录、Active Directory 域、共享目录域、BSD 配置文件和 NIS 域。

    “/BSD/local”文件夹始终包括在搜索路径中，并且始终是灰色的。

重要事项：如果您将 Mac OS X 配置为使用自动鉴定搜索策略和 DHCP 提供的 LDAP 服务器，恶意用户获得您电脑的控制权的风险将会提高。如果将电脑配置为连接到无线网络，这种风险会更高。有关更多信息，请参阅“保护电脑免受来自恶意 DHCP 服务器的攻击”。
