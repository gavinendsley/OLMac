去掉目录服务器连接

从“帐户”偏好设置中去掉目录服务器之前，请确定没有其他应用程序使用它的服务。

例如，如果 Mail 配置为使用目录服务器搜索联系人，而您删除了该目录服务器，则无法搜索该目录服务器上的任何人。

要删除目录服务器：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 从目录服务器列表中选择您要删除的目录服务器。

  5. 点按删除按钮 (–)。

  6. 如果您确定选择了正确的目录服务器，请点按“停止使用服务器”。

如果用户自己从目录服务器断开连接，则目录管理员必须手动从目录服务器删除电脑记录。用户也可以请求网络管理员使用“目录实用工具”和拥有目录管理权限的帐户从目录服务器断开他们的电脑，这样会从目录服务器中删除电脑记录。
