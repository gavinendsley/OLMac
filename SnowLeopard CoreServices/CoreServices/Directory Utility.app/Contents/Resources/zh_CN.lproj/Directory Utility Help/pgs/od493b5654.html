使用数据填充 LDAP 目录以用于 Mac OS X

在配置对 LDAP 目录域的访问和设置数据映射之后，您可以使用记录和数据填充目录以用于 Mac OS X。对于允许远程管理（读/写访问）的 LDAP 目录，您可以使用 Mac OS X Server 附带的 Workgroup Manager，如下所述：

  • 标识您想要在用户的网络浏览器中自动装载的共享点和共享域（用户在 Finder 窗口边栏中点按“网络”时可以看到）。

    使用 Workgroup Manager 的“在服务器管理中共享”和“网络”模块。

    有关附加信息，请浏览“文件服务概述”和“用户管理概述”。

  • 定义用户记录和组别记录，并配置这些记录的设置。

    使用 Workgroup Manager 的“帐户”模块。

    有关附加信息，请浏览“用户管理概述”。

  • 定义拥有相同偏好设置并可用于相同用户和组别的电脑列表。

    使用 Workgroup Manager 的“电脑”模块。

    有关附加信息，请浏览“用户管理概述”。

在所有情况下，在 Workgroup Manager 中点按用户列表上面的球形小图标，然后从弹出式菜单中选取并打开 LDAP 目录域。如果 LDAP 目录没有在弹出式菜单中列出，请从此菜单中选取“其他”，以选择该 LDAP 目录。

注：要向只读 LDAP 目录添加记录和数据，您必须使用托管 LDAP 目录的服务器上的工具。
