定义自动搜索策略

使用“目录实用工具”，您可以将 Mac OS X 电脑的鉴定和联系搜索策略配置为自动定义。

自动定义的搜索策略包含本地目录域。它还可以包含由 DHCP 服务指定的 LDAP 目录服务器。

以下是鉴定和联系搜索策略的默认配置。

注：  有些应用程序（如 Mac OS X Mail 和“地址簿”）可以直接访问 LDAP 目录，而不必使用 Open Directory。 要将这些应用程序之一设置为直接访问 LDAP 目录，请打开应用程序，然后设置正确的偏好设置。
重要事项：如果您将 Mac OS X 配置为使用自动鉴定搜索策略和 DHCP 提供的 LDAP 服务器或 DHCP 提供的共享目录域，恶意用户获得您电脑的控制权的风险将会提高。如果将电脑配置为连接到无线网络，这种风险会更高。有关更多信息，请参阅“保护电脑免受来自恶意 DHCP 服务器的攻击”。
要将搜索策略配置为自动定义：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“搜索策略”并选取一个搜索策略：

    • 鉴定：显示用于鉴定和大多数其他管理数据的搜索策略。

    • 联系：显示用于应用程序（如“地址簿”）中的联系信息的搜索策略。

  7. 从“搜索”弹出式菜单中选择“自动”，然后点按“应用”。

  8. 在“系统偏好设置”中，请确定将电脑的“网络”偏好设置配置为使用 DHCP 或使用 DHCP 并手动设定 IP 地址。
