停止 LDAP 目录的信任捆绑

您可以使用“目录实用工具”来停止电脑和 LDAP 目录之间的信任捆绑，LDAP 目录允许但不要求信任捆绑。

要停止 LDAP 目录的信任捆绑：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择 LDAPv3，然后点按编辑按钮 (/)。

  8. 如果服务器配置列表处于隐藏状态，请点按“显示选项”。

  9. 选择您想要编辑的服务器配置，然后点按“编辑”。

  10. 点按“解开捆绑”，输入下面的凭证，然后点按“好”。

    输入 LDAP 目录的管理员（并非本地电脑管理员）的名称和密码。

    如果该电脑没有设置信任捆绑，“解开捆绑”按钮将不会显示。

    如果您看到一则警告，提示电脑无法联系 LDAP 服务器，而您想要强制停止信任捆绑，请点按“好”。

    如果您强制停止信任捆绑，则 LDAP 目录中仍保留该电脑的电脑记录。请通知 LDAP 目录管理员，让管理员将该电脑从电脑组别中删除。

  11. 要完成停止信任捆绑，请点按“好”。
