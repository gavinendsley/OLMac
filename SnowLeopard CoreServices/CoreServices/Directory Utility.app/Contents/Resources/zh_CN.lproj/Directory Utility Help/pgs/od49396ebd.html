定义本地目录搜索策略

使用“目录实用工具”，您可以将 Mac OS X 电脑的鉴定和联系搜索策略配置为仅使用电脑的本地目录。

仅使用本地目录的搜索策略会限制电脑对鉴定信息和其他管理数据的访问。

如果您将电脑的鉴定搜索策略限制为只使用本地目录，则只有具有本地帐户的用户才能登录。

要将搜索策略配置为只使用本地目录域（本地目录）：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“搜索策略”并选取一个搜索策略：

    • 鉴定：显示用于鉴定和大多数其他管理数据的搜索策略。

    • 联系：显示用于应用程序（如“地址簿”）中的联系信息的搜索策略。

  7. 从“搜索”弹出式菜单中选取“本地目录”，然后点按“应用”。
