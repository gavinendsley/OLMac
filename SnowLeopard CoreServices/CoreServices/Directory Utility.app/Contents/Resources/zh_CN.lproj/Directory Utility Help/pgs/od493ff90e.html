删除用于访问 LDAP 目录的配置

您可以使用“目录实用工具”删除指定电脑如何访问 LDAPv3 或 LDAPv2 目录的配置。

如果 LDAP 配置由 DHCP 提供，则不能更改，因此该配置选项在 LDAP 配置列表中显示为暗淡。

要删除用于访问 LDAP 目录的配置：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“LDAPv3”，然后点按编辑按钮 (/)。

  8. 如果服务器配置列表处于隐藏状态，请点按“显示选项”。

  9. 在列表中选择服务器配置，点按“删除”，然后点按“好”。

  10. 从以下项目中选取：

    • 如果您看到一则警告，提示电脑已绑定到 LDAP 目录，而您想要停止信任捆绑，请点按“好”，然后输入 LDAP 目录管理员（而不是本地电脑管理员）的名称和密码。

    • 如果您看到一则警告，提示电脑无法联系 LDAP 服务器，则可以点按“好”以强制停止信任捆绑。

      如果您强制停止信任捆绑，则 LDAP 目录中仍保留该电脑的电脑记录。请通知 LDAP 目录管理员，让管理员将该电脑从电脑组别中删除。

已删除的配置会从鉴定和联系的自定搜索策略中删除。
