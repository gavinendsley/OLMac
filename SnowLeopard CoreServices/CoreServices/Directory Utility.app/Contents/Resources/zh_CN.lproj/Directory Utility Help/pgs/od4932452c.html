添加 Active Directory 服务器连接

在连接 Active Directory 服务器时，您必须知道服务器名称或 IP 地址，还必须知道 Active Directory 管理员用户名称和密码。

要添加 Active Directory 服务器：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

    如果您看到“编辑”按钮，则您的电脑至少有一个至目录服务器的连接。

  4. 点按添加按钮 (+)。

  5. 从“添加新的目录类型”弹出式菜单中，选取“Active Directory”，然后输入以下信息：

    • Active Directory 域：这是 Active Directory 域的 DNS 名称（例如，ads.company.com）。

    • 电脑 ID：（可选操作）编辑想要 Active Directory 为您的服务器使用的 ID。这是服务器的 NetBIOS 名称。名称应包含不超过 15 个字符，无特殊字符，无标点符号。如果可以实现，可让服务器的名称匹配其无限定的 DNS 主机名。例如，如果您的 DNS 服务器中您的服务器条目为“server.example.com”，则为您的服务器命名为“server”。

    • AD 管理员用户名称和密码：输入 Active Directory 管理员的用户名称和密码。

  6. 点按“好”。
