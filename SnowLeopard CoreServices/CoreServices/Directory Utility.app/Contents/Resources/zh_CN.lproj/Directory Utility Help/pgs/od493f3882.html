更改可以管理电脑的 Active Directory 组别

在配置为使用“目录实用工具”的 Active Directory 连接器的电脑上，您可以确定想要其成员拥有电脑管理员权限的 Active Directory 组别帐户。

作为这些 Active Directory 组别帐户成员的用户可以在正在配置的 Mac OS X 电脑上执行管理任务，如安装软件。

要添加或删除其成员拥有管理员权限的 Active Directory 组别帐户：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“Active Directory”，然后点按编辑按钮 (/)。

  8. 如果高级选项处于隐藏状态，请点按“显示高级选项”。

  9. 点按“管理”

  10. 选择“将管理权限赋予”，然后更改您想要其成员拥有管理员权限的 Active Directory 组别帐户的列表：

    • 通过点按添加按钮 (+) 添加组别，然后输入 Active Directory 域名、反斜杠符号和组别帐户名称（例如，ADS\Domain Admins、IL2\Domain Admins）。

    • 通过在列表中选择一个组别，然后点按删除按钮 (–)，可以删除该组别。

  11. 点按“好”。
