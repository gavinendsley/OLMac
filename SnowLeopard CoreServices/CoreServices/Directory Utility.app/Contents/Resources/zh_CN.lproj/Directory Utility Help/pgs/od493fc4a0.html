忽略 LDAP 服务器推荐

使用“目录实用工具”，您可以指定电脑忽略还是接受 LDAP 服务器关于在其他 LDAP 服务器或复制服务器中查找信息的推荐。

服务器推荐可以帮助电脑查找信息，但如果电脑必须验证其他 LDAP 服务器的推荐，则也会导致登录延迟或其他延迟。

要指定是否忽略 LDAP 服务器推荐：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“LDAPv3”，然后点按编辑按钮 (/)。

  8. 如果服务器配置列表处于隐藏状态，请点按“显示选项”。

  9. 在列表中选择服务器配置，然后点按“编辑”。

  10. 点按“连接”，然后选择“忽略服务器推荐”。
