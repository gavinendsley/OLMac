设置 Active Directory 用户帐户的 UNIX Shell

在配置为使用“目录实用工具”的 Active Directory 连接器的电脑上，您可以设置默认情况下具有 Active Directory 帐户的用户与 Mac OS X 交互时在“终端”中使用的命令行 Shell。

默认的 shell 还可用于通过 SSH（安全 Shell）或 Telnet 进行远程交互。每个用户均可以通过更改“终端”偏好设置来覆盖此默认 Shell。

要设置 Active Directory 用户帐户的 UNIX shell：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“Active Directory”，然后点按编辑按钮 (/)。

  8. 如果高级选项处于隐藏状态，请点按“显示高级选项”。

  9. 点按“用户体验”。

  10. 选择“默认用户 shell”，然后输入默认用户 shell 的路径。

  11. 点按“好”。
