鉴定 LDAP 连接

使用“目录实用工具”，您可以建立到 LDAP 目录的已鉴定连接。该鉴定是单向的。 电脑向 LDAP 目录证明其身份，但 LDAP 目录不会向电脑证明其真实性。有关相互鉴定的信息，请参阅“建立与 LDAP 目录的信任捆绑”。

注：如果电脑和 LDAP 目录之间已经建立信任捆绑，已鉴定的连接将会是多余的，您不能设置该连接。
要建立经鉴定的 LDAPv3 连接：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“LDAPv3”，然后点按编辑按钮 (/)。

  8. 如果服务器配置列表处于隐藏状态，请点按“显示选项”。

  9. 在列表中选择服务器配置，然后点按“编辑”。

  10. 点按“安全”。

  11. 选择“连接时使用鉴定”，然后输入用户的识别名和密码。

    识别名称可以指明有权限查看目录数据的任何用户帐户。例如，LDAP 服务器上短名称为“authenticator”、地址为“ods.example.com”的用户帐户，它的识别名称为“uid=authenticator,cn=users,dc=ods,dc=example,dc=com”。

    重要事项：如果识别名称或密码不正确，则不能使用 LDAP 目录中的用户帐户登录电脑。
