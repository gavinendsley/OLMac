指定首选 Active Directory 服务器

在配置为使用“目录实用工具”的 Active Directory 连接器的电脑上，您可以指定默认情况下电脑要访问的 Active Directory 域所在服务器的 DNS 名称。

如果该服务器在将来变得不可用，Active Directory 连接器会自动回退到网域树系中附近的另一个服务器。

如果取消选择该选项，Active Directory 连接器会确定网域树系中最近的一个 Active Directory 域。

要指定 Active Directory 连接器访问的首选服务器：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 点按“服务”。

  6. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  7. 在服务列表中选择“Active Directory”，然后点按编辑按钮 (/)。

  8. 如果高级选项处于隐藏状态，请点按“显示高级选项”。

  9. 点按“管理”。

  10. 选择“优先使用此域服务器”，并输入 Active Directory 服务器的 DNS 名称。

  11. 点按“好”。
