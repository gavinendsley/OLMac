更改 LDAP 目录的连接设置

您可以使用“目录实用工具”更改指定电脑如何访问 LDAPv3 或 LDAPv2 目录的配置的连接设置。

要更改用于访问 LDAP 目录的连接设置：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“LDAPv3”，然后点按编辑按钮 (/)。

  8. 如果服务器配置列表处于隐藏状态，请点按“显示选项”。

  9. 在列表中选择服务器配置，然后点按“编辑”。

  10. 点按“连接”，更改以下设置：

    • 配置名称：在 LDAP 目录配置的列表中标识此配置。（您也可以在 LDAP 目录配置列表中更改此名称。）

    • 服务器名称或 IP 地址：指定服务器的 DNS 名称或它的 IP 地址。（您也可以在 LDAP 目录配置列表中更改服务器名称或 IP 地址。）

    • 打开/关闭超时时间：指定一个连接尝试在被取消之前可以持续的最长时间。

    • 查询超时时间：指定查询在被取消之前可以持续的最长时间。

    • 尝试重新绑定的时间：指定在 LDAP 服务器无法响应时尝试重新连接之前要等待的秒数。要阻止连续的重新连接尝试，请提高此值。

    • 中断连接闲置时间：指定允许闲置或无响应连接保持打开的分钟数。

    • 使用 SSL 加密：确定是否使用 SSL 连接加密与 LDAP 目录的通信。（您也可以在 LDAP 目录配置列表中更改此设置。）在选择 SSL 注记格之前，请询问 Open Directory 管理员是否需要 SSL。

    • 使用自定端口：为 LDAP 连接指定标准端口（不使用 SSL 时为 389，使用 SSL 时为 636）以外的端口号。

    • 忽略服务器推荐：决定是忽略还是接受 LDAP 服务器有关在其他 LDAP 服务器或复制服务器上查找信息的推荐。 服务器推荐可以帮助电脑查找信息，但如果电脑必须验证其他 LDAP 服务器的推荐，则也会导致登录延迟或其他延迟。

    • 使用 LDAPv2（只读）：决定是否对 LDAP 目录的只读访问使用较早的 LDAPv2 协议。
