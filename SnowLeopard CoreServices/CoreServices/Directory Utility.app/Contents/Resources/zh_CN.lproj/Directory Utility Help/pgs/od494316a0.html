配置 LDAP 搜索和映射

使用“目录实用工具”，您可以编辑映射、搜索基准以及用于指定 Mac OS X 如何在 LDAP 目录中查找特定数据项的搜索范围。对于“目录实用工具”中列出的每个 LDAP 目录配置，您可以分别编辑它们的这些设置。每个 LDAP 目录配置都会指定 Mac OS X 如何访问 LDAPv3 或 LDAPv2 目录中的数据。

您可以编辑以下内容：

  • 每个 Mac OS X 记录类型到 LDAP 对象类的映射

  • Mac OS X 数据类型或属性到每个记录类型的 LDAP 属性的映射。

  • LDAP 搜索基准和搜索范围，它们确定 Mac OS X 在 LDAP 目录中查找特定 Mac OS X 记录类型的位置。

将 Mac OS X 用户属性映射至读/写 LDAP 目录域（并非为只读的 LDAP 域）时，映射至 RealName 的 LDAP 属性一定不能与映射至 RecordName 的 LDAP 属性的列表中的第一个属性相同。

例如，如果 cn 也被映射至 RealName，则属性 cn 一定不能是映射至 RecordName 的第一个属性。

如果映射至 RealName 的 LDAP 属性与映射至 RecordName 的第一个属性相同，则当您在 Workgroup Manager 中尝试编辑全（长）名称或第一个短名称时会出现问题。

有关 Mac OS X 记录类型和属性的更多信息，请参阅“Open Directory Administration”（可以从 www.apple.com.cn/server/resources/ 获得）。

要编辑 LDAP 服务器的搜索基准和映射：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在服务列表中选择“LDAPv3”，然后点按编辑按钮 (/)。

  8. 如果服务器配置列表处于隐藏状态，请点按“显示选项”。

  9. 在列表中选择服务器配置，然后点按“编辑”。

  10. 点按“搜索和映射”。

  11. 选择您想要用作起点的映射，否则请选取“自定”从没有预定义的映射开始。

    如果您选取其中一个 LDAP 映射模板，就会出现一个您可以更改的搜索基准后缀，或者您也可以通过点按“好”接受默认的搜索基准后缀。

    点按“访问此 LDAPv3 服务器使用”弹出式菜单，选取一个映射模板，以使用其映射作为起点。

  12. 根据需要添加记录类型并更改其搜索基准：

    • 要添加记录类型，请点按“添加”（位于“记录类型和属性”列表下方），然后在出现的表单中，选择“记录类型”，从列表中选择记录类型，然后点按“好”。

    • 要更改一个记录类型的搜索基准和搜索范围，请在“记录类型和属性”列表中选择该记录类型，然后编辑“搜索基准”栏。从搜索基准下拉列表中选择“所有子树”设定搜索范围包括 LDAP 目录层次，或选择“仅第一级”设定搜索范围仅包括该搜索基准以及在 LDAP 目录层次中较之低一级的层。

    • 要删除记录类型，请在“记录类型和属性”列表中选择该类型，然后点按“删除”。

    • 要添加一个记录类型的映射，请在“记录类型和属性”列表中选择该记录类型，点按“添加”（在“映射至列表中的__项”下），然后根据 LDAP 目录输入对象类的名称。

      要添加另一个 LDAP 对象类，请按下 Return 键，输入该对象类的名称，然后使用列表上方的弹出式菜单指定是否使用列出的 LDAP 对象类。

    • 要更改记录类型的映射，请在“记录类型和属性”列表中选择该记录类型，在“映射至列表中的__项”中连按您想要更改的 LDAP 对象类，然后编辑它。使用列表上方的弹出式菜单指定是否使用列出的 LDAP 对象类。

    • 要删除一个记录类型的映射，请在“记录类型和属性”列表中选择该记录类型，选择您想要从“映射至列表中的__项”中删除的 LDAP 对象类，然后点按“删除”（在“映射至列表中的__项”下面）。

  13. 根据需要添加属性并更改它们的映射：

    • 要向记录类型添加属性，请在“记录类型和属性”列表中选择该记录类型，点按“添加”（在“记录类型和属性”列表下面）；然后在出现的表单中，选择“属性类型”，选择属性类型，然后点按“好”。

    • 要添加一个属性的映射，请在“记录类型和属性”列表中选择该属性，点按“添加”（在“映射至列表中的__项”下），然后根据 LDAP 目录输入属性的名称。要添加另一个 LDAP 属性，请按下 Return 键并输入属性的名称。

    • 要更改一个属性的映射，请在“记录类型和属性”列表中选择该属性，在“映射至列表中的__项”中连按您想要更改的项，然后编辑项名称。

    • 要删除一个属性的映射，请在“记录类型和属性”列表中选择该属性，选择您想要从“映射至列表中的__项”中删除的项，然后点按“删除”（在“映射至列表中的__项”下面）。

    • 要更改在右边的列表中显示的属性的顺序，请在列表中向上或向下拖移属性。

  14. 要将您的映射保存为模板，请点按“存储模板”。

    当您下一次打开“目录实用工具”时，存储在默认位置的模板将在 LDAP 映射模板的弹出式菜单中列出。已存储模板的默认位置位于您的个人文件夹，路径如下：

    ~/资源库/应用程序支持/目录访问/LDAPv3/Templates

  15. 要在 LDAP 目录中储存映射，使 LDAP 目录能够自动向客户端提供这些映射，请点按“写入服务器”，然后输入用于储存映射的搜索基准、对搜索基准拥有写权限的管理员或其他用户的识别名称（例如“uid=diradmin,cn=users,dc=ods,dc=example,dc=com”）和密码。

    如果要将映射写入到 Open Directory LDAP 服务器，则正确的搜索基准为“cn=config,suffix”（其中 suffix 是服务器的搜索基准后缀，例如“dc=ods,dc=example,dc=com”）。

如果 Mac OS X 客户端的自定搜索策略包含了被配置为从 LDAP 服务器获取映射的连接，LDAP 目录将为该客户端提供其映射。

LDAP 目录还会将其映射提供给所有使用自动搜索策略的 Mac OS X 客户端。有关更多信息，请参阅“配置 LDAP 目录访问”和“使用高级搜索策略设置”。
