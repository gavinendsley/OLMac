启用或停用 LDAP 目录服务

您可以使用“目录实用工具”来启用或停用对使用 LDAPv2 和 LDAPv3 的目录服务的访问。单个名为 LDAPv3 的“目录实用工具”插件提供对 LDAP2 和 LDAPv3 的访问。

与其他许多服务器一样，Mac OS X Server 提供的目录服务也采用 LDAPv3。LDAPv3 是 Macintosh、UNIX 和 Windows 系统组成的混合网络中常见的开放标准。某些服务器还使用旧版本的 LDAPv2 来提供目录服务。

如果您停用 LDAP 目录服务并且 LDAP 目录包含在自定搜索策略中，则在“目录实用工具”中“搜索策略”的“鉴定”或“联系”面板中这些目录会显示为红色。

要启用或停用 LDAP 目录服务：

  1. 打开“系统偏好设置”并点按“帐户”。

  2. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  3. 点按“登陆选项”，然后点按“加入”或“编辑”。

  4. 点按“打开目录实用工具”。

  5. 如果锁图标已锁定，点按该图标然后输入管理员的名称和密码，即可将其解锁。

  6. 点按“服务”。

  7. 在 LDAPv3 旁边，选择或取消选择注记格，然后点按“应用”。
