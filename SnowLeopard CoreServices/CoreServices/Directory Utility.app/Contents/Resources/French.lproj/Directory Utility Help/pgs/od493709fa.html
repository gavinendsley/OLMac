Affichage ou masquage de configurations pour serveurs LDAP

Vous pouvez afficher ou masquer la liste des configurations disponibles pour l’accès aux annuaires LDAP. Chaque configuration spécifie le mode d’accès à un annuaire LDAP pour Open Directory. Lorsque la liste est visible, vous pouvez modifier les réglages de chaque configuration LDAP non estompée.

Pour afficher ou masquer les configurations d’annuaire LDAP disponibles :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste des services, sélectionnez LDAPv3 et cliquez sur le bouton de modification (/).

  8. Cliquez sur l’option Afficher les options ou Masquer les options, selon celle qui figure à l’écran.
