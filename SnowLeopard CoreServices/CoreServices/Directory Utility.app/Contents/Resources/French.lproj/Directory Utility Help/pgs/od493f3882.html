Modification des groupes Active Directory autorisés à administrer l’ordinateur

Sur un ordinateur configuré pour utiliser le connecteur Active Directory d’Utilitaire d’annuaire, vous pouvez identifier les comptes de groupe Active Directory dont les membres doivent recevoir des autorisations administrateur sur l’ordinateur.

Les utilisateurs membres de ces comptes de groupe Active Directory peuvent réaliser des tâches d’administration telles que l’installation de logiciels sur l’ordinateur Mac OS X que vous êtes en train de configurer.

Pour ajouter ou supprimer des comptes de groupe Active Directory dont les membres possèdent des autorisations administrateur :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste de services, sélectionnez Active Directory, puis cliquez sur le bouton de modification (/).

  8. Si la liste des options avancées est masquée, cliquez sur "Afficher les options avancées".

  9. Cliquez sur Administratif.

  10. Sélectionnez « Permettre l’administration par » et modifiez la liste des comptes de groupe Active Directory dont les membres doivent recevoir des autorisations administrateur :

    • Ajoutez un groupe en cliquant sur le bouton Ajouter (+) et en saisissant le nom du domaine Active Directory, une barre oblique inverse et le nom du compte de groupe (par exemple, ADS\Admins Domaine, IL2\Admins Domaine).

    • Supprimez un groupe de la liste en le sélectionnant et en cliquant sur le bouton Supprimer (–).

  11. Cliquez sur OK.
