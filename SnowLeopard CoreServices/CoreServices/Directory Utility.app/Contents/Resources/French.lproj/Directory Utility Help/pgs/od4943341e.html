Choix d’un serveur Active Directory préféré

Sur un ordinateur configuré pour utiliser le connecteur Active Directory d’Utilitaire d’annuaire, vous pouvez spécifier le nom DNS du serveur dont le domaine Active Directory doit être celui auquel l’ordinateur accède par défaut.

Si le serveur devient indisponible, le connecteur Active Directory revient à un autre serveur proche dans la forêt.

Si cette option est désélectionnée, le connecteur Active Directory identifie le domaine Active Directory le plus proche dans la forêt.

Pour préciser un serveur préféré auquel le connecteur Active Directory doit accéder :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Cliquez sur Services.

  6. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  7. Dans la liste de services, sélectionnez Active Directory, puis cliquez sur le bouton de modification (/).

  8. Si la liste des options avancées est masquée, cliquez sur "Afficher les options avancées".

  9. Cliquez sur Administratif.

  10. Sélectionnez « Préférer ce serveur de domaine » et saisissez le nom DNS du serveur Active Directory.

  11. Cliquez sur OK.
