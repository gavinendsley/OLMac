Modification d’une connexion à un serveur d’annuaire

Vous pouvez utiliser les Préférences du compte pour modifier les serveurs d’annuaire auxquels vous êtes connecté.

Pour modifier une connexion à un serveur d’annuaire :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Dans la liste des serveurs d’annuaire, sélectionnez celui que vous souhaitez modifier.

  5. Cliquez sur le bouton de modification (/).

  6. Modifiez les réglages du serveur d’annuaire.

  7. Cliquez sur OK.
