Activation du compte utilisateur root

Utilitaire d’annuaire permet d’activer le compte utilisateur root. Dans ce cas, veillez à choisir un mot de passe complexe, contenant des caractères alphanumériques et spéciaux, afin de ne pas compromettre sa sécurité.

ATTENTION :   le compte root est un compte administrateur sans restriction, utilisé pour effectuer des modifications sur des fichiers système critiques. Même si vous avez ouvert une session en tant qu’administrateur, vous devez utiliser le compte root ou sudo pour effectuer des tâches système critiques.

N’utilisez jamais le compte root pour ouvrir une session sur un ordinateur (que ce soit à distance ou au niveau local). Utilisez plutôt sudo pour effectuer des tâches root. Vous pouvez restreindre l’accès à sudo en ajoutant des utilisateurs au fichier /etc/sudoers.

Pour en savoir plus sur le compte root, consultez l’Aide Admin Serveur.

Pour activer le compte utilisateur root :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

    Si un bouton Modifier s’affiche, votre ordinateur dispose d’au moins une connexion au serveur d’annuaire.

  4. Cliquez sur Utilitaire Open Directory.

  5. Sélectionnez Édition > Activer l’utilisateur root.
