Contrôle des connexions à un serveur d’annuaire

Vous pouvez utiliser Préférences du compte pour surveiller l’état des serveurs d’annuaire auxquels votre ordinateur est connecté. Ces informations peuvent vous être utiles lorsque vous essayez de savoir pourquoi vous ne parvenez pas à vous connecter à un serveur d’annuaire donné.

Pour contrôler l’état d’un serveur d’annuaire :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Notez que le point d’état situé à gauche du serveur d’annuaire peut être de différentes couleurs :

    • Vert  :   le serveur d’annuaire répond à l’ordinateur.

    • Jaune  :   l’ordinateur attend une réponse du serveur d’annuaire.

    • Rouge  :   le serveur d’annuaire ne répond pas à l’ordinateur.
