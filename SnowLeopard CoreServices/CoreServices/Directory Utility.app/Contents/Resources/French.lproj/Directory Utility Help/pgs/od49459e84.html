Configuration d’Utilitaire d’annuaire sur un serveur distant

Utilitaire d’annuaire permet de configurer et de définir la façon dont Mac OS X Server accède à des services d’annuaire sur un serveur distant.

Pour configurer l’accès à des annuaires sur un serveur distant :

  1. Ouvrez les Préférences Système sur votre ordinateur, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

    Si un bouton Modifier s’affiche, votre ordinateur dispose d’au moins une connexion au serveur d’annuaire.

  4. Cliquez sur Utilitaire Open Directory.

  5. Sélectionnez Connecter à partir du menu Fichier.

  6. Tapez les informations suivantes de connexion et d’authentification du serveur que vous souhaitez configurer.

    • Adresse  :   saisissez le nom DNS ou l’adresse IP du serveur à configurer.

    • Nom d’utilisateur  :   saisissez le nom d’utilisateur d’un administrateur sur le serveur.

    • Mot de passe  :   saisissez le mot de passe correspondant au nom d’utilisateur.

  7. Cliquez sur Se connecter.

  8. À partir des onglets Services et Règles de recherche, modifiez les réglages en fonction de vos besoins.

    Toutes les modifications apportées s’appliquent au serveur distant auquel vous vous êtes connecté lors des étapes précédentes.

  9. Dans le menu Fichier de votre ordinateur, choisissez Déconnecter.
