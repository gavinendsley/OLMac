Utilisation des réglages avancés de Services d’annuaire

Utilitaire d’annuaire répertorie les services d’annuaire auxquels Mac OS X peut accéder. Cette liste inclut les services d’annuaire qui permettent à Mac OS X d’accéder aux informations sur les utilisateurs et à d’autres données d’administration stockées dans les domaines de répertoire.

Vous avez la possibilité d’activer ou de désactiver l’accès à chaque service d’annuaire. Si vous désactivez un service dans Utilitaire d’annuaire, Mac OS X n’y accédera plus.

  • Activation ou désactivation du service Active Directory

  • Activation ou désactivation des services d’annuaire LDAP
