Arrêt d’une liaison fiable avec un annuaire LDAP

Utilitaire d’annuaire permet d’arrêter une liaison entre un ordinateur et un annuaire LDAP qui autorise mais n’impose pas une liaison fiable.

Pour arrêter une liaison fiable avec un annuaire LDAP :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste des services, sélectionnez LDAPv3, puis cliquez sur le bouton de modification (/).

  8. Si la liste des configurations serveur est masquée, cliquez sur Afficher les options.

  9. Sélectionnez la configuration serveur souhaitée, puis cliquez sur Édition.

  10. Cliquez sur Rompre la liaison, puis saisissez les références suivantes et cliquez sur OK.

    Saisissez le nom et le mot de passe d’un administrateur d’annuaire LDAP (pas celui de l’administrateur d’un ordinateur local).

    Si une liaison fiable n’a pas été définie sur cet ordinateur, le bouton Rompre la liaison n’apparaît pas.

    Si un message vous informe que l’ordinateur ne parvient pas à entrer en contact avec le serveur LDAP, cliquez sur OK pour interrompre la liaison fiable de force.

    Si vous forcez l’arrêt de la liaison, cet ordinateur possède toujours un enregistrement dans l’annuaire LDAP. Informez-en l’administrateur d’annuaire LDAP pour qu’il puisse supprimer l’ordinateur du groupe correspondant.

  11. Pour terminer l’opération de rupture de la liaison fiable, cliquez sur OK.
