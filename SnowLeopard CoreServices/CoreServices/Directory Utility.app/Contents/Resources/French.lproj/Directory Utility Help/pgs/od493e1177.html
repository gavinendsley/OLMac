Modification de comptes utilisateur et d’autres enregistrements dans Active Directory

Gestionnaire de groupe de travail permet de modifier des comptes utilisateur, des comptes de groupe, des listes d’ordinateurs et d’autres enregistrements d’un domaine Active Directory. Vous pouvez également l’utiliser pour supprimer des enregistrements d’un domaine Active Directory.

Si le schéma Active Directory a été étendu pour recevoir les attributs et types d’enregistrements (classes d’objets) Mac OS X, vous pouvez vous servir de Gestionnaire de groupe de travail pour créer et modifier des groupes d’ordinateurs du domaine Active Directory.

Pour plus d’informations, reportez-vous à la vue d’ensemble de la gestion des utilisateurs.

Pour créer des comptes d’utilisateurs ou de groupe dans un domaine Active Directory, utilisez les outils d’administration Active Directory Microsoft sur un ordinateur d’administration de serveur Windows.
