Rupture de la liaison avec le serveur Active Directory

Si l’ordinateur utilise le connecteur Active Directory d’Utilitaire d’annuaire pour établir une liaison avec un serveur Active Directory, vous pouvez rompre cette liaison entre l’ordinateur et le serveur Active Directory.

Vous pouvez forcer la rupture de la liaison si l’ordinateur ne peut entrer en contact avec le serveur ou si l’enregistrement de l’ordinateur a été supprimé du serveur.

Pour rompre la liaison entre l’ordinateur et le serveur Active Directory :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste de services, sélectionnez Active Directory, puis cliquez sur le bouton de modification (/).

  8. Cliquez sur Rompre la liaison, authentifiez-vous en tant qu’utilisateur autorisé à mettre fin à une connexion au domaine Active Directory, puis cliquez sur OK.

    Si un message vous informe que les informations d’authentification ont été refusées ou que l’ordinateur ne peut contacter Active Directory, cliquez sur "Forcer à rompre le lien" pour arrêter la connexion.

    Si vous forcez l’opération de rupture, Active Directory possède encore l’enregistrement de l’ordinateur. Demandez à l’administrateur Active Directory de supprimer l’enregistrement de cet ordinateur.

  9. Dans la sous-fenêtre Services, désélectionnez le réglage Activer d’Active Directory, puis cliquez sur Appliquer.
