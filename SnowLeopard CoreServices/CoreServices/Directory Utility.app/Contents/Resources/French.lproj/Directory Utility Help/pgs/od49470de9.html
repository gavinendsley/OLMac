Définition de règles de recherche automatiques

À l’aide d’Utilitaire d’annuaire, vous pouvez configurer les règles de recherche d’authentification et de contact sur un ordinateur Mac OS X afin qu’elles soient définies automatiquement.

Lorsqu’elle est définie automatiquement, une règle de recherche inclut le domaine de répertoire local. Elle peut également inclure un serveur d’annuaire LDAP spécifié par le service DHCP.

Cela correspond à la configuration par défaut pour les règles de recherche d’authentification et de contact.

Remarque  :   certaines applications, telles que Mac OS X Mail et Carnet d’adresses, peuvent accéder aux annuaires LDAP directement, sans passer par Open Directory. Pour configurer une de ces applications en vue de l’accès direct aux annuaires LDAP, ouvrez l’application et définissez la préférence qui convient.
Important  :   si vous configurez Mac OS X afin qu’il utilise une règle d’authentification automatique et un serveur LDAP fourni par DHCP ou un domaine de répertoire partagé fourni par DHCP, vous augmentez le risque de prise de contrôle de votre ordinateur par un utilisateur malintentionné. Le risque est encore plus important si l’ordinateur est configuré pour se connecter à un réseau sans fil. Pour en savoir plus, consultez la rubrique Protection d’ordinateurs contre un serveur DHCP malveillant.
Pour configurer la définition automatique d’une règle de recherche :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Règles de recherche, puis choisissez une règle de recherche :

    • Authentification  :   affiche la règle de recherche utilisée pour l’authentification et la plupart des autres données administratives.

    • Contacts  :   affiche la règle de recherche utilisée pour les informations de contact dans les applications telles que le Carnet d’adresses.

  7. Dans le menu local Rechercher, sélectionnez Automatique, puis cliquez sur Appliquer.

  8. Dans les Préférences Système, vérifiez que les préférences réseau de l’ordinateur sont bien configurées en vue d’utiliser le protocole DHCP ou le DHCP avec une adresse IP manuelle.
