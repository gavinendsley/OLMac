Configuration d’une liaison sécurisée pour un annuaire LDAP

Utilitaire d’annuaire vous permet de configurer une liaison fiable entre l’ordinateur et un annuaire LDAP reconnaissant ce type de liaison. La liaison est mutuellement authentifiée par un enregistrement d’ordinateur authentifié créé dans l’annuaire lors de la configuration de la liaison fiable.

Il n’est pas possible de configurer l’ordinateur en vue de l’utilisation d’une liaison LDAP fiable et d’un annuaire LDAP fourni par DHCP. La liaison LDAP fiable est statique par nature ; toutefois, la liaison LDAP fournie par DHCP est dynamique.

Pour configurer une liaison fiable avec un annuaire LDAP :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste des services, sélectionnez LDAPv3 et cliquez sur le bouton de modification (/).

  8. Si la liste des configurations serveur est masquée, cliquez sur Afficher les options.

  9. Sélectionnez la configuration serveur souhaitée et cliquez sur Modifier.

  10. Cliquez sur Liaison, puis saisissez les références suivantes et cliquez sur OK.

    Tapez le nom de l’ordinateur ainsi que le nom et le mot de passe d’un administrateur de domaine de répertoire LDAP. Le nom de l’ordinateur ne peut pas être utilisé par un autre ordinateur pour la liaison fiable ou d’autres services réseau.

    Si le bouton de liaison n’apparaît pas, cela signifie que l’annuaire LDAP ne prend pas en charge la liaison fiable.

  11. Vérifiez que le nom d’ordinateur que vous avez indiqué est correct.

    Si un message d’avertissement vous indique qu’il existe un enregistrement de l’ordinateur, cliquez sur Annuler pour revenir en arrière et modifier le nom de l’ordinateur ou cliquez sur Écraser pour remplacer l’enregistrement existant.

    Il est possible que l’enregistrement de l’ordinateur ne soit plus d’usage ou qu’il appartienne à un autre ordinateur. Si vous remplacez un enregistrement d’ordinateur existant, informez-en l’administrateur d’annuaire LDAP au cas où ce remplacement désactiverait un autre ordinateur.

    Dans ce genre de situation, l’administrateur d’annuaire LDAP doit renommer l’ordinateur désactivé et l’ajouter au groupe auquel il appartenait, en utilisant un autre nom pour cet ordinateur.

  12. Pour terminer la configuration de la liaison fiable, cliquez sur OK.
