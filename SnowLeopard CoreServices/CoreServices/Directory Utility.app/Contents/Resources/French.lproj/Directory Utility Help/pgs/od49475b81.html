Authentification d’une connexion LDAP

Utilitaire d’annuaire permet de configurer une connexion authentifiée auprès d’un annuaire LDAP. Cette authentification n’est pas réciproque. L’ordinateur prouve son identité à un annuaire LDAP, mais celui-ci ne prouve pas son identité à l’ordinateur. Pour une authentification mutuelle, consultez Configuration d’une liaison sécurisée pour un annuaire LDAP.

Remarque  :   si une liaison sécurisée est établie entre l’ordinateur et l’annuaire LDAP, il n’est pas possible de configurer une connexion authentifiée car cela serait redondant.
Pour configurer une connexion LDAPv3 authentifiée :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste des services, sélectionnez LDAPv3 et cliquez sur le bouton de modification (/).

  8. Si la liste des configurations serveur est masquée, cliquez sur Afficher les options.

  9. Dans la liste, sélectionnez une configuration serveur, puis cliquez sur Édition.

  10. Cliquez sur Sécurité.

  11. Sélectionnez “Utiliser l’authentification lors de la connexion”, puis tapez un nom et un mot de passe d’utilisateur.

    Ce nom peut correspondre à n’importe quel compte utilisateur autorisé à consulter des données dans le répertoire. Par exemple, un compte utilisateur dont le nom abrégé est « authentificateur » sur un serveur LDAP et dont l’adresse est ods.example.com a pour nom uid=authenticator,cn=users,dc=ods,dc=example,dc=com.

    Important  :   si le nom absolu ou le mot de passe est incorrect, personne ne peut ouvrir de session sur l’ordinateur à l’aide de comptes utilisateur issus de l’annuaire LDAP.
