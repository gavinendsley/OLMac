À propos des connexions aux serveurs d’annuaire

Vous pouvez utiliser les Préférences du compte pour connecter des ordinateurs aux serveurs d’annuaire. Vous pouvez afficher la liste des serveurs d’annuaire auxquels votre ordinateur est connecté, cliquez sur Modifier dans la sous-fenêtre Options d’ouverture de session des Préférences du compte. Votre ordinateur Mac OS X consulte les serveurs de cette liste afin d’obtenir des informations utilisateur et d’autres données administratives stockées dans le domaine de répertoire des serveurs d’annuaire.

Lorsque vous ajoutez ou supprimez un serveur sur la liste des serveurs d’annuaire, les entrées associées à celui-ci sont également ajoutées ou supprimées sur la liste Services, Authentification et Contacts. Toutefois, si vous supprimez les entrées associées sur la liste Services, Authentification et Contacts, le serveur d’annuaire figurera toujours sur la liste Serveurs d’annuaire.

Les ordinateurs Mac OS X 10.6 peuvent se connecter à un serveur d’annuaire Open Directory, Active Directory ou LDAP. Si vous ne savez pas sur quel serveur vous connecter, adressez-vous à votre administrateur réseau.

Important  :   si le nom de votre ordinateur contient un trait d’union, cela peut vous empêcher de vous connecter à un domaine de répertoire tel que LDAP ou Active Directory ou de créer un lien avec de tels domaines. Pour en établir une, utilisez un nom d’ordinateur ne comportant pas de trait d’union.
