Remplissage d’annuaires LDAP avec des données pour Mac OS X

Après avoir configuré l’accès aux domaines de répertoires LDAP et le mappage de données, vous pouvez leur ajouter des enregistrements et des données pour Mac OS X. Pour les annuaires LDAP autorisant l’administration à distance (accès en lecture-écriture), vous pouvez utiliser Gestionnaire de groupe de travail, inclus dans Mac OS X Server, comme suit :

  • Identifiez les points de partage et les domaines partagés dont vous souhaitez activer le montage automatique dans les navigateurs réseau des utilisateurs (c’est-à-dire dans la fenêtre qui apparaît lorsque les utilisateurs cliquent sur Réseau dans la barre latérale d’une fenêtre du Finder).

    Utilisez Partage dans Admin Serveur, ainsi que les modules Réseau de Gestionnaire de groupe de travail.

    Pour plus d’informations, reportez-vous à la vue d’ensemble des services de fichiers et à celle de la gestion des utilisateurs.

  • Définissez des enregistrements d’utilisateurs et de groupes et configurez leurs réglages.

    Utilisez le module Comptes de Gestionnaire de groupe de travail.

    Pour plus d’informations, reportez-vous à la vue d’ensemble de la gestion des utilisateurs.

  • Définissez des listes d’ordinateurs qui possèdent les mêmes réglages de préférences et qui sont disponibles pour les mêmes utilisateurs et groupes.

    Utilisez le module Ordinateurs de Gestionnaire de groupe de travail.

    Pour plus d’informations, reportez-vous à la vue d’ensemble de la gestion des utilisateurs.

Dans tous les cas, cliquez sur l’icône représentant un petit globe, au-dessus de la liste des utilisateurs, et effectuez votre choix dans le menu local de Gestionnaire de groupe de travail pour ouvrir le domaine de répertoires LDAP. Si l’annuaire LDAP ne figure pas dans le menu local, choisissez Autre dans celui-ci pour sélectionner l’annuaire LDAP.

Remarque  :   pour ajouter des enregistrements et des données à un annuaire LDAP en lecture seule, vous devez employer des outils qui résident sur le serveur hébergeant l’annuaire LDAP.
