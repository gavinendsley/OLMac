Contrôle de l’authentification à partir de tous les domaines d’une forêt Active Directory

Sur un ordinateur configuré pour utiliser le connecteur Active Directory d’Utilitaire d’annuaire, vous pouvez autoriser les utilisateurs de la forêt Active Directory à s’authentifier à partir de tous les domaines ou restreindre l’authentification des utilisateurs à partir de certains domaines seulement.

Pour contrôler si les utilisateurs peuvent ou non s’authentifier à partir de tous les domaines d’une forêt :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  8. Dans la liste de services, sélectionnez Active Directory, puis cliquez sur le bouton de modification (/).

  9. Si la liste des options avancées est masquée, cliquez sur "Afficher les options avancées".

  10. Cliquez sur Administratif.

  11. Sélectionnez « Autoriser l’authentification depuis n’importe quel domaine de la forêt ».

    Si vous sélectionnez l’option « Autoriser l’authentification depuis n’importe quel domaine de la forêt », vous pouvez ajouter la forêt Active Directory aux règles de recherche personnalisées de l’ordinateur pour l’authentification et les contacts.

    Lorsque vous ajoutez une forêt Active Directory à une règle de recherche personnalisée, cette forêt apparaît dans la liste des domaines de répertoires disponibles sous la forme « /Active Directory/Tous les domaines ». (Il s’agit du réglage par défaut.)

    Si vous désélectionnez l’option « Autoriser l’authentification depuis n’importe quel domaine de la forêt », vous pouvez ajouter des domaines Active Directory individuellement aux règles de recherche personnalisées de l’ordinateur pour l’authentification et les contacts.

    Lorsque vous ajoutez des domaines Active Directory à une règle de recherche personnalisée, chacun d’eux apparaît séparément dans la liste des domaines de répertoires disponibles.

  12. Cliquez sur OK.

  13. Après avoir sélectionné « Autoriser l’authentification depuis n’importe quel domaine de la forêt », modifiez la règle de recherche personnalisée dans les sous-fenêtres Authentification et Contacts afin d’inclure la forêt complète ou les domaines Active Directory de votre choix.

    Pour plus d’informations sur la modification d’une règle de recherche personnalisée, reportez-vous à la rubrique Définition de règles de recherche personnalisées.
