Gestion du compte utilisateur root

Vous pouvez utiliser Utilitaire d’annuaire (situé dans préférences Comptes) pour gérer le compte utilisateur root en activant ou désactivant cet utilisateur. Si vous activez ce compte, vous pouvez également utiliser Utilitaire d’annuaire pour changer le mot de passe du compte root.

Pour en savoir plus au sujet de la gestion du compte utilisateur root à l’aide d’Utilitaire d’annuaire, consultez les rubriques suivantes :

  • Activation du compte utilisateur root

  • Modification du mot de passe du compte utilisateur root
