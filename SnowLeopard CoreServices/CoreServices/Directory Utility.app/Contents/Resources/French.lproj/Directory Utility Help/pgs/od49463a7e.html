Configuration de données des fichiers de configuration BSD

Si vous souhaitez qu’un ordinateur Mac OS X récupère des données d’administration dans des fichiers de configuration BSD, celles-ci doivent figurer dans les fichiers et être enregistrées dans le format requis par Mac OS X.

Il peut être nécessaire d’ajouter, de modifier ou de réorganiser des données dans les fichiers. Comme il est impossible de modifier les fichiers de configuration BSD via Gestionnaire de groupe de travail, vous devez apporter les modifications nécessaires à l’aide d’un éditeur de texte ou d’un autre outil.

Le tableau suivant dresse la liste des noms des fichiers et une description de leur contenu.

Fichier de configuration BSD
Contenu

/etc/master.passwd

Noms d’utilisateur, mots de passe, identifiants, identifiants de groupe principal, etc.

/etc/group

Noms de groupe, identifiants et membres

/etc/fstab

Montages NFS

/etc/hosts

Noms et adresses d’ordinateur

/etc/networks

Noms et adresses réseau

/etc/services

Noms de service, ports et protocoles

/etc/protocols

Noms et numéros de protocole IP

/etc/rpcs

Réseau ouvert calculant les serveurs RPC

/etc/printcap

Noms et fonctions d’imprimante

/etc/bootparams

Paramètres Bootparam

/etc/bootp

Paramètres Bootp

/etc/aliases

Alias de courrier et listes de distribution

/etc/netgroup

Noms et membres de groupe réseau
