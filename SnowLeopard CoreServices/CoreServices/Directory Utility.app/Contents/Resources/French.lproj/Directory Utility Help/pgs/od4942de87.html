Définition de réglages NIS

Utilitaire d’annuaire permet de créer une configuration définissant la façon dont Mac OS X accède à un domaine Network Information Service (NIS).

Pour créer une configuration d’accès à un domaine NIS :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste de services, sélectionnez « Fichier plat BSD et NIS » puis cliquez sur le bouton de modification (/).

  8. Saisissez le nom de domaine NIS, le nom DNS ou l’adresse IP du serveur qui héberge le domaine NIS.

    Indiquez le nom d’hôte ou l’adresse IP du serveur NIS si cette information est nécessaire pour la sécurité ou si le serveur ne réside pas sur le même sous-réseau que l’ordinateur en cours de configuration.

    Si vous n’indiquez pas de serveur, le NIS utilise un protocole de diffusion pour détecter un serveur NIS sur le sous-réseau.

  9. Sélectionnez « Utiliser le domaine NIS pour l’authentification », puis cliquez sur OK.

    Le domaine NIS est ajouté à la règle de recherche d’authentification de l’ordinateur en tant que /BSD/domaine, domaine étant le nom que vous avez saisi à l’étape 4.
