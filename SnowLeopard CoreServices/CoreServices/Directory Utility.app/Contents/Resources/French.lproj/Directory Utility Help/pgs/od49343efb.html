Configuration de l’accès aux annuaires

Vous pouvez utiliser Utilitaire d’annuaire (situé dans Préférences Comptes) pour configurer et gérer l’accès aux domaines de répertoire par un ordinateur sous Mac OS X ou par un serveur sous Mac OS X Server.

Accès aux annuaires LDAP

À l’aide d’Utilitaire d’annuaire, vous pouvez configurer un ordinateur Mac OS X ou un serveur Mac OS X Server pour qu’il accède à certains annuaires LDAP en particulier, y compris l’annuaire LDAP d’un objet principal Open Directory de Mac OS X Server. Pour en savoir plus, cliquez sur les rubriques ci-dessous.

Utilisation de fichiers de configuration BSD

Il est possible d’utiliser Open Directory pour récupérer des données d’administration de fichiers de configuration BSD, tels que /etc/master.passwd, aussi appelés fichiers plats BSD. Vous pouvez également configurer l’accès aux fichiers de configuration BSD à l’aide d’Utilitaire d’annuaire. Pour en savoir plus, cliquez sur les rubriques ci-dessous.
