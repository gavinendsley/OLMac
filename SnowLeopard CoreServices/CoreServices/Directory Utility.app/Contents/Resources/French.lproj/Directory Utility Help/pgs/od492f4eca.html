Ajout d’une connexion à un serveur Open Directory

Lorsque vous ajoutez un serveur Open Directory, vous devez connaître le nom ou l’adresse IP du serveur et savoir s’il utilise le protocole SSL (secure socket layer).

Pour ajouter un serveur Open Directory :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

    Si un bouton Modifier s’affiche, votre ordinateur dispose d’au moins une connexion au serveur d’annuaire.

  4. Cliquez sur le bouton Ajouter (+).

  5. Dans le menu local « Ajouter un nouveau répertoire de type », sélectionnez Open Directory.

  6. Dans le champ « Nom du serveur ou adresse IP », saisissez le nom ou l’adresse IP du serveur.

  7. (Conditionnel) Avant de cocher la case Chiffrer via SSL, demandez à votre administrateur Open Directory si SSL est nécessaire.

Important :  si vous modifiez l’adresse IP et le nom d’ordinateur de votre serveur Mac OS X au moyen de la commande changeip alors que vous êtes connecté à un serveur d’annuaire, vous devez vous déconnecter et vous reconnecter à ce serveur d’annuaire afin de mettre à jour l’annuaire avec le nouveau nom de l’ordinateur et sa nouvelle adresse IP. Si vous ne procédez pas ainsi, l’annuaire n’est pas mis à jour et il continue d’utiliser l’ancien nom et l’ancienne adresse IP de l’ordinateur.
