Modification d’une configuration d’accès à un annuaire LDAP

Utilitaire d’annuaire vous permet de modifier les réglages d’une configuration d’annuaire LDAP. Les réglages de configuration servent à spécifier le mode d’accès à un annuaire LDAPv3 ou LDAPv2 pour Open Directory.

Si la configuration LDAP a été fournie par DHCP, elle n’est pas modifiable, raison pour laquelle elle apparaît estompée sur la liste des configurations LDAP.

Pour modifier une configuration d’accès à un annuaire LDAP :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste des services, sélectionnez LDAPv3 et cliquez sur le bouton de modification (/).

  8. Si la liste des configurations serveur est masquée, cliquez sur Afficher les options.

  9. Modifiez les réglages suivants selon vos besoins :

    • Activer  :   cochez ou décochez la case pour activer ou désactiver l’accès au serveur d’annuaire LDAP.

    • Nom de la configuration  :   double-cliquez sur le nom d’une configuration pour le modifier.

    • Nom du serveur ou adresse IP  :   Double-cliquez sur le nom ou l’adresse IP d’un serveur pour les modifier.

    • Mappage LDAP  :   dans le menu local, choisissez un modèle, saisissez l’extension de la base de recherche pour l’annuaire LDAP et cliquez sur OK.

      Si vous avez choisi un modèle, vous devez taper une extension de base de recherche, sinon l’ordinateur ne sera pas en mesure de rechercher des informations dans l’annuaire LDAP. En général, l’extension de la base de recherche provient du nom DNS du serveur. Par exemple, pour un serveur dont le nom DNS serait ods.example.com, l’extension de la base de recherche serait « dc=ods,dc=example,dc=com. »

      Si vous choisissez l’option Du serveur plutôt qu’un modèle, aucune extension de base de recherche n’est nécessaire. Dans ce cas, Open Directory considère que l’extension de la base de recherche correspond au premier niveau de l’annuaire LDAP.

      Si vous avez choisi Personnalisé, vous devez configurer des mappages entre les types et attributs d’enregistrement Mac OS X et les catégories et attributs de l’annuaire LDAP auquel vous vous connectez. Pour en savoir plus, consultez la rubrique Configuration des recherches et mappages LDAP.

    • SSL  :   cochez ou décochez cette case pour activer ou désactiver les communications chiffrées utilisant le protocole SSL. Avant de cocher la case SSL, demandez à votre administrateur Open Directory si cette option est nécessaire.

  10. Pour modifier les réglages suivants par défaut de cette configuration LDAP, cliquez sur Modifier afin d’afficher les options pour la configuration LDAP sélectionnée, effectuez vos modifications, puis cliquez sur OK lorsque vous avez terminé :

    • Cliquez sur Connexion pour définir les options de délai, spécifier un port personnalisé, ignorer les références du serveur ou forcer l’utilisation du protocole LDAPv2 (en lecture seule). Pour en savoir plus, consultez la rubrique Modification des réglages de connexion pour un annuaire LDAP.

    • Cliquez sur Recherche et mappages pour configurer des options de recherche et de mappage pour un serveur LDAP. Pour en savoir plus, consultez Configuration d’une liaison sécurisée pour un annuaire LDAP.

    • Cliquez sur Sécurité pour configurer une connexion authentifiée (au lieu d’une liaison fiable) et d’autres options de règlement de sécurité. Pour en savoir plus, consultez la rubrique Modification du règlement de sécurité pour une connexion LDAP.

    • Cliquez sur Liaison pour configurer des liaisons fiables ou sur Rompre la liaison pour interrompre celle-ci. (Ces boutons peuvent être non visibles si l’annuaire LDAP n’autorise pas de liaison sécurisée.) Pour en savoir plus, consultez Configuration d’une liaison sécurisée pour un annuaire LDAP.

  11. Pour terminer la modification de la configuration d’accès à un annuaire LDAP, cliquez sur OK.
