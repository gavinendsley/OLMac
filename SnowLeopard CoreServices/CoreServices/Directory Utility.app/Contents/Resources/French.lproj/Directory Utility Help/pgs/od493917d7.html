Préparation d’un annuaire LDAP en lecture seule pour Mac OS X

Si vous souhaitez qu’un ordinateur Mac OS X récupère des données d’administration issues d’un annuaire LDAP en lecture seule, celles-ci doivent être enregistrées dans le format requis par MAC OS X. Il peut être nécessaire d’ajouter, de modifier ou de réorganiser des données dans l’annuaire LDAP en lecture seule.

Comme Mac OS X ne peut pas écrire de données dans un annuaire en lecture seule, vous devez employer d’autres outils pour effectuer les modifications. Ces outils doivent résider sur le serveur d’hébergement de l’annuaire LDAP en lecture seule.

Pour préparer un annuaire LDAP en lecture seule pour Mac OS X :

  1. Accédez au serveur qui héberge l’annuaire LDAP en lecture seule et configurez-le pour qu’il prenne en charge l’authentification et le contrôle de mots de passe LDAP.

  2. Effectuez les modifications de classes et d’attributs d’objet de l’annuaire LDAP nécessaires pour fournir les données à Mac OS X.

Pour en savoir plus sur les données requises par les services d’annuaire Mac OS X, consultez l’administration d’Open Directory.
