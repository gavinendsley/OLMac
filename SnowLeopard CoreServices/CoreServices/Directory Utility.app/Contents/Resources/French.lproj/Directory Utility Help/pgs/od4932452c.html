Ajout d’une connexion à un serveur Active Directory

Lorsque vous vous connectez à un serveur Active Directory, vous devez connaître le nom ou l’adresse IP du serveur et disposer du nom d’utilisateur et du mot de passe de l’administrateur Active Directory.

Pour ajouter un serveur Active Directory :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

    Si un bouton Modifier s’affiche, votre ordinateur dispose d’au moins une connexion au serveur d’annuaire.

  4. Cliquez sur le bouton Ajouter (+).

  5. Dans le menu local « Ajouter un nouveau répertoire de type », sélectionnez Active Directory, puis saisissez les informations suivantes :

    • Domaine Active Directory  :   il s’agit du nom DNS du domaine Active Directory (par exemple, ads.company.com.)

    • Identifiant de l’ordinateur  :   si vous le souhaitez, vous pouvez modifier l’identifiant qu’Active Directory doit utiliser pour votre serveur. Il s’agit du nom NetBIOS du serveur. Ce nom doit contenir un maximum de 15 caractères (hors caractères spéciaux) et aucun signe de ponctuation. Par souci de commodité, vous pouvez faire coïncider le nom du serveur avec son nom d’hôte DNS non qualifié. Par exemple, si votre serveur DNS dispose pour votre serveur de l’entrée « serveur.exemple.com », donnez à votre serveur le nom « serveur ».

    • Nom d’utilisateur et mot de passe de l’administrateur AD  :   saisissez le nom d’utilisateur et le mot de passe de l’administrateur Active Directory.

  6. Cliquez sur OK.
