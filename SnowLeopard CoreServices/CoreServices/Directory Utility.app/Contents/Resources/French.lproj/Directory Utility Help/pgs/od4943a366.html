Mappage de l’UID avec un attribut Active Directory

Sur un ordinateur configuré pour utiliser le connecteur Active Directory d’Utilitaire d’annuaire, vous pouvez spécifier un attribut Active Directory que vous souhaitez mapper sur l’attribut UID (identifiant d’utilisateur unique) de Mac OS X.

Le schéma Active Directory doit généralement être étendu pour inclure un attribut convenant au mappage avec l’UID :

  • Si l’administrateur Active Directory étend le schéma Active Directory en installant les services de Microsoft pour UNIX, vous pouvez mapper l’UID sur l’attribut msSFU-30-Uid-Number.

  • Si l’administrateur Active Directory étend manuellement le schéma Active Directory pour inclure des attributs RFC 2307, vous pouvez mapper l’UID avec uidNumber.

  • Si l’administrateur Active Directory étend manuellement le schéma Active Directory pour inclure l’attribut UniqueID de Mac OS X, vous pouvez mapper l’UID avec cet attribut.

Si le mappage UID est désactivé, le connecteur Active Directory génère un UID basé sur l’attribut GUID standard d’Active Directory.

ATTENTION :   si vous modifiez le mappage avec l’UID ultérieurement, il se peut que les utilisateurs perdent l’accès aux fichiers déjà créés.
Pour mapper l’UID avec un attribut d’un schéma Active Directory étendu :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste de services, sélectionnez Active Directory, puis cliquez sur le bouton de modification (/).

  8. Si la liste des options avancées est masquée, cliquez sur "Afficher les options avancées".

  9. Cliquez sur Mappages.

  10. Sélectionnez « Mappage UID à attribuer », puis saisissez le nom de l’attribut Active Directory à mapper avec l’UID.

  11. Cliquez sur OK.
