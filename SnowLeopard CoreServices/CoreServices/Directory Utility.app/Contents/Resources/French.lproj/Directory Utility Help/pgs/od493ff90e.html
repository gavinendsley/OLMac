Suppression d’une configuration d’accès à un annuaire LDAP

Utilitaire d’annuaire vous permet de supprimer une configuration spécifiant le mode d’accès à un annuaire LDAPv3 ou LDAPv2 pour l’ordinateur.

Si la configuration LDAP a été fournie par DHCP, elle n’est pas modifiable, raison pour laquelle cette option apparaît estompée sur la liste des configurations LDAP.

Pour supprimer une configuration d’accès à un annuaire LDAP :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste des services, sélectionnez LDAPv3 et cliquez sur le bouton de modification (/).

  8. Si la liste des configurations serveur est masquée, cliquez sur Afficher les options.

  9. Dans la liste, sélectionnez une configuration serveur et cliquez sur Supprimer, puis sur OK.

  10. Choisissez parmi les éléments suivants :

    • Si vous voyez un avertissement signalant que l’ordinateur est relié à l’annuaire LDAP et si vous souhaitez interrompre la liaison sécurisée, cliquez sur OK et saisissez ensuite le nom et mot de passe d’un administrateur d’annuaire LDAP (non pas d’un administrateur d’ordinateur local).

    • Si un message vous informe que l’ordinateur ne parvient pas à entrer en contact avec le serveur LDAP, vous pouvez cliquer sur OK pour interrompre la liaison fiable de force.

      Si vous forcez l’arrêt de la liaison, cet ordinateur possède toujours un enregistrement dans l’annuaire LDAP. Informez-en l’administrateur d’annuaire LDAP pour qu’il puisse supprimer l’ordinateur du groupe correspondant.

La configuration supprimée est éliminée des règles de recherche personnalisées pour l’authentification et les contacts.
