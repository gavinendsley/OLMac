Activation ou désactivation du service Active Directory

Utilitaire d’annuaire vous permet d’activer ou de désactiver l’utilisation des services Active Directory fournis par un serveur Windows. Active Directory est le service d’annuaire de Windows 2000 et serveurs ultérieurs.

Si vous désactivez les services Active Directory et que les domaines Active Directory font partie d’une règle de recherche personnalisée, ils apparaissent en rouge dans la sous-fenêtre Authentification ou Contacts de la règle de recherche dans Utilitaire d’annuaire.

Pour activer ou désactiver l’accès à Active Directory :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. À côté de Active Directory, cochez ou désélectionnez la case et cliquez sur Appliquer.
