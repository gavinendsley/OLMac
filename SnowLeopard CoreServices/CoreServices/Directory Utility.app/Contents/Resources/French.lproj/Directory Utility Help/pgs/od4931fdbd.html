Suppression d’une connexion à un serveur d’annuaire

Avant de supprimer un serveur d’annuaire dans Préférences du compte, vérifiez que vous n’utilisez pas ses services pour d’autres applications.

Par exemple, si la configuration de Mail prévoit l’utilisation du serveur d’annuaire pour rechercher des personnes et que vous supprimez celui-ci, vous ne pourrez retrouver personne figurant sur ce serveur d’annuaire.

Pour supprimer un serveur d’annuaire :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Dans la liste des serveurs d’annuaire, sélectionnez le serveur d’annuaire à supprimer.

  5. Cliquez sur le bouton Supprimer (—).

  6. Si vous êtes sûr(e) d’avoir sélectionné le bon serveur d’annuaire, cliquez sur "Arrêter l’utilisation du serveur".

Si un utilisateur se déconnecte d’un serveur d’annuaire, l’administrateur de l’annuaire doit supprimer manuellement l’enregistrement de l’ordinateur dans le serveur d’annuaire. Les utilisateurs peuvent également demander que l’administrateur réseau déconnecte leur ordinateur du serveur d’annuaire au moyen d’Utilitaire d’annuaire et d’un compte bénéficiant d’une autorisation d’administration d’annuaire, ce qui supprime l’enregistrement de l’ordinateur dans le serveur d’annuaire.
