Configuration automatique de client

Si votre ordinateur Mac OS X 10.6 est connecté à un réseau sans être connecté à un domaine de répertoire, il utilise Bonjour pour détecter les serveurs Mac OS X 10.6.

S’il détecte un serveur Mac OS X 10.6 exécutant un maître Open Directory qui comporte au moins un service auto-configurable en cours d’exécution (tel qu’AFP, SMB, VPN, Mail, iCal, iChat, notifications) et un compte utilisateur sur le domaine de répertoire dont le nom complet ou abrégé correspond à celui de l’utilisateur de l’ordinateur Mac OS X 10.6 en cours de session, une invitation apparaît sur votre ordinateur Mac OS X 10.6.

Cette invitation propose des assistants pour configurer votre connexion à un serveur d’annuaire. Vous pouvez accepter ou refuser de vous connecter au serveur ; vous pouvez également décider de vous connecter ultérieurement.

Si votre ordinateur détecte plusieurs serveurs d’annuaire Mac OS X 10.6 auxquels vous pouvez vous connecter, ceux-ci sont affichés dans un menu local d’invitation. Si vous décidez de vous connecter à un serveur, l’assistant ouvre vos applications, les connecte aux services exécutés par le serveur d’annuaire et les configure automatiquement.

Pour vous connecter à un domaine de répertoire au moyen de l’assistant automatisé :

  1. Lorsque l’assistant apparaît, cliquez sur Configurer les services.

  2. Dans le menu local Serveur, choisissez le domaine de répertoire auquel vous souhaitez vous connecter.

    Plusieurs domaines de répertoires peuvent être disponibles sur votre réseau.

    Si le serveur auquel vous souhaitez vous connecter n’est pas présent dans la liste, saisissez le nom ou l’adresse IP du serveur dans le champ Serveur, puis cliquez sur OK.

  3. Saisissez le nom d’utilisateur et le mot de passe d’un compte bénéficiant des privilèges nécessaires pour se connecter à un ordinateur du domaine de répertoire auquel vous vous connectez.

    Si vous ne disposez pas de ces informations, contactez votre administrateur d’annuaire.

  4. Saisissez le mot de passe du compte utilisateur qui apparaît dans « Saisissez le mot de passe du compte nom d’utilisateur sur cet ordinateur ».

    L’assistant modifie le mot de passe du compte d’annuaire de l’utilisateur sur le serveur afin qu’il corresponde au mot de passe du compte sur l’ordinateur local.

  5. Cliquez sur Continuer.

  6. Lorsque l’outil UpgradeUser a terminé, cliquez sur Continuer.

  7. Lorsqu’un message apparaît, décrivant les services configurés et vous demandant de fermer votre session, cliquez sur Fermer la session.

    Vous pouvez commencer à utiliser les nouveaux services dès que vous ouvrez une nouvelle session.
