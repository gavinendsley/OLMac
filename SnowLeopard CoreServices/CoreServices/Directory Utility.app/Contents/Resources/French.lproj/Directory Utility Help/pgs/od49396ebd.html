Définition de règles de recherche de répertoire local

Utilitaire d’annuaire permet de configurer des règles de recherche d’authentification et de contact sur un ordinateur Mac OS X en vue de l’utilisation exclusive du répertoire local de l’ordinateur.

Lorsqu’une règle de recherche utilise uniquement le répertoire local, l’accès de l’ordinateur aux informations d’authentification et autres données administratives en est limité.

Si vous limitez la règle de recherche d’authentification d’un ordinateur en vue de l’utilisation exclusive du répertoire local, seuls les utilisateurs dotés de comptes locaux peuvent ouvrir une session.

Pour configurer une règle de recherche afin qu’elle utilise uniquement le domaine de répertoire local (annuaire local) :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis cliquez sur Se connecter ou sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Règles de recherche, puis choisissez une règle de recherche :

    • Authentification  :   affiche la règle de recherche utilisée pour l’authentification et la plupart des autres données administratives.

    • Contacts :  affiche la règle de recherche utilisée pour les informations de contact dans des applications comme le Carnet d’adresses.

  7. Dans le menu local Rechercher, sélectionnez Répertoire local, puis cliquez sur Appliquer.
