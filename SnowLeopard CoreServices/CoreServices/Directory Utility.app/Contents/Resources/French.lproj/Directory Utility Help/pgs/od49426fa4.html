Mappage de l’identifiant de groupe dans les comptes de groupe avec un attribut Active Directory

Sur un ordinateur configuré pour utiliser le connecteur Active Directory d’Utilitaire d’annuaire, vous pouvez spécifier un attribut Active Directory qui doit être mappé sur l’attribut GID (identifiant de groupe) de Mac OS X dans les comptes de groupe.

Le schéma Active Directory doit généralement être étendu pour inclure un attribut convenant au mappage avec le GID :

  • Si l’administrateur Active Directory étend le schéma Active Directory en installant les services de Microsoft pour UNIX, vous pouvez mapper le GID sur l’attribut msSFU-30-Gid-Number.

  • Si l’administrateur Active Directory étend manuellement le schéma Active Directory pour inclure des attributs RFC 2307, vous pouvez mapper le GID avec gidNumber.

  • Si l’administrateur Active Directory étend manuellement le schéma Active Directory pour inclure l’attribut gidNumber de Mac OS X, vous pouvez mapper le GID avec cet attribut.

Si le mappage du GID est désactivé, le connecteur Active Directory génère un GID basé sur l’attribut GUID standard d’Active Directory.

ATTENTION :   si vous modifiez le mappage avec le GID ultérieurement, il se peut que les utilisateurs perdent l’accès aux fichiers déjà créés.
Pour mapper le GID avec un attribut d’un schéma Active Directory étendu :

  1. Ouvrez les Préférences Système, puis cliquez sur Comptes.

  2. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  3. Cliquez sur Options, puis sur Modifier.

  4. Cliquez sur Utilitaire Open Directory.

  5. Si l’icône de verrouillage est activée, déverrouillez-la en cliquant dessus puis en saisissant le nom et le mot de passe d’un administrateur.

  6. Cliquez sur Services.

  7. Dans la liste de services, sélectionnez Active Directory, puis cliquez sur le bouton de modification (/).

  8. Si la liste des options avancées est masquée, cliquez sur "Afficher les options avancées".

  9. Cliquez sur Mappages.

  10. Sélectionnez « Mappage du GID de groupe à attribuer », puis saisissez le nom de l’attribut Active Directory à mapper sur le GID dans les comptes de groupe.

  11. Cliquez sur OK.
