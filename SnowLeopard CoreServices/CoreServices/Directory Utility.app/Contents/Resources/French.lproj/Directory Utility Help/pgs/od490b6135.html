À propos d’Utilitaire d’annuaire

Utilitaire d’annuaire détermine quels sont les services d’annuaire qui sont utilisés par un ordinateur Mac OS X, et la manière dont celui-ci se connecte à des domaines de répertoire spécifiques. Il permet également de définir des règles de recherche afin de retrouver des informations d’authentification et de contact dans des domaines de répertoire spécifiques.

Installation d’Utilitaire d’annuaire

Utilitaire d’annuaire vous permet notamment de configurer et de gérer le mode d’accès aux services d’annuaire pour votre ordinateur. Vous pouvez également l’utiliser pour configurer à distance l’accès de votre ordinateur aux services d’annuaire. Utilitaire d’annuaire peut aussi servir à activer le compte root ou à changer de mot de passe pour celui-ci.

Configuration de l’accès aux services

Utilitaire d’annuaire vous permet d’activer et de désactiver différents types de services auxquels Mac OS X peut accéder en réseau. Ils comprennent notamment les services d’annuaire qui donnent accès aux comptes utilisateur et autres données administratives stockées dans des domaines de répertoire.

Configuration de l’accès aux annuaires

À l’aide d’Utilitaire d’annuaire, vous pouvez configurer l’ordinateur afin qu’il accède à des comptes utilisateur et autres informations administratives dans des annuaires LDAP donnés, dans un domaine Active Directory et dans un domaine NIS.

Configuration de règles de recherche

Utilitaire d’annuaire vous permet de définir la règle de recherche d’authentification utilisée par Mac OS X pour localiser et récupérer des informations d’authentification utilisateur et autres données administratives à partir de domaines de répertoire. Il vous permet également de définir la règle de recherche de contacts utilisée par Mac OS X pour localiser et récupérer des informations de contact (nom, adresse, etc.) à partir de domaines de répertoire.
