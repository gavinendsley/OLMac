컴퓨터를 관리할 수 있는 Active Directory 그룹 변경하기

디렉토리 유틸리티의 Active Directory 커넥터를 사용하도록 구성된 컴퓨터에서, 컴퓨터에 대해 관리자 권한을 부여하려는 구성원의 Active Directory 그룹 계정을 식별할 수 있습니다.

이 Active Directory 그룹 계정의 구성원인 사용자들은 구성하는 Mac OS X 컴퓨터에서 소프트웨어를 설치하는 것과 같은 관리자 작업을 실행할 수 있습니다.

구성원이 관리자 권한을 가진 Active Directory 그룹 계정을 추가 또는 제거하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 서비스를 클릭하십시오.

  7. 서비스 목록에서 Active Directory를 선택하고 편집(/) 단추를 클릭하십시오.

  8. 고급 옵션이 가려져 있으면, 고급 옵션 보기를 클릭하십시오.

  9. 관리를 클릭하십시오.

  10. “관리 허용”을 선택하고 관리자 권한을 가지려는 구성원이 있는 Active Directory 그룹 계정 목록을 변경하십시오.

    • 추가(+) 단추를 클릭하고 Active Directory 도메인 이름, 백슬래시 및 그룹 계정 이름(예: ADS\Domain Admins,IL2\Domain Admins)을 입력하여 그룹을 추가하십시오.

    • 목록에서 그룹을 선택하고 삭제(-) 단추를 클릭하여 그룹을 제거하십시오.

  11. 승인을 클릭하십시오.
