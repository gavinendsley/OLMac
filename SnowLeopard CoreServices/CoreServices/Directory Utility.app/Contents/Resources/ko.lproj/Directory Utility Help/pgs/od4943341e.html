권장 Active Directory 서버 지정하기

디렉토리 유틸리티의 Active Directory 커넥터를 사용하도록 구성된 컴퓨터에서, 컴퓨터를 기본적으로 접근시키려는 Active Directory 도메인을 가진 서버의 DNS 이름을 지정하십시오.

나중에 이 서버를 사용할 수 없게 되면, Active Directory 커넥터가 자동으로 포리스트의 다른 인접 서버로 돌아갑니다.

이 옵션이 선택되지 않으면, Active Directory 커넥터는 가장 가까운 Active Directory 도메인을 결정합니다.

접근하려는 Active Directory 커넥터에 대한 권장 서버를 지정하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 서비스를 클릭하십시오.

  6. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  7. 서비스 목록에서 Active Directory를 선택하고 편집(/) 단추를 클릭하십시오.

  8. 고급 옵션이 가려져 있으면, 고급 옵션 보기를 클릭하십시오.

  9. 관리를 클릭하십시오.

  10. “이 도메인 서버 권장”을 선택하고 Active Directory 서버의 DNS 이름을 입력하십시오.

  11. 승인을 클릭하십시오.
