원격 서버에 디렉토리 유틸리티 설정하기

컴퓨터의 디렉토리 유틸리티를 사용하여 원격 서버상의 Mac OS X Server가 어떻게 디렉토리 서비스에 접근할 지를 설정 및 관리할 수 있습니다.

원격 서버상에 디렉토리 접근을 구성하려면,

  1. 컴퓨터에서 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

    편집 단추가 보인다면, 컴퓨터가 최소 하나의 디렉토리 서버에 연결되어 있는 것입니다.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 파일 메뉴에서 연결을 선택하십시오.

  6. 구성하려는 서버에 대해 다음 연결 및 인증 정보를 입력하십시오.

    • 주소: 설정하고자 하는 서버의 DNS 이름 또는 IP 주소를 입력합니다.

    • 사용자 이름: 서버의 관리자 이름을 입력합니다.

    • 암호: 입력한 사용자 이름에 해당하는 암호를 입력합니다.

  7. 연결을 클릭하십시오.

  8. 서비스 및 검색 정책 탭을 클릭하고 필요한 대로 설정을 변경하십시오.

    변경사항은 이전 단계에서 연결될 원격 서버에 영향을 미칩니다.

  9. 컴퓨터의 파일 메뉴에서 연결 해제를 선택하십시오.
