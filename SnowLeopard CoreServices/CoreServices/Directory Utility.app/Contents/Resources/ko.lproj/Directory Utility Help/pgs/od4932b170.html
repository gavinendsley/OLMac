디렉토리 서버 연결에 관하여

계정 환경설정을 사용하여 컴퓨터를 디렉토리 서버에 연결할 수 있습니다. 컴퓨터에 연결된 디렉토리 서버 목록은 계정 환경설정의 로그인 옵션에 있는 편집을 클릭하면 볼 수 있습니다. Mac OS X 컴퓨터는 디렉토리 서버의 디렉토리 도메인에 저장되어 있는 사용자 정보 및 기타 관리 데이터 목록에 있는 서버에 접근합니다.

디렉토리 서버 목록에서 서버를 추가하거나 삭제하면 해당 디렉토리에 연관된 항목들이 서비스, 인증 및 연락처 목록에서 추가 또는 삭제됩니다. 하지만 서비스, 인증 및 연락처 목록 안에 있는 관련 항목을 제거하면 디렉토리 서버가 디렉토리 서버 목록에서 제거되지 않습니다.

Mac OS X v10.6 컴퓨터는 Open Directory, Active Directory 또는 LDAP 디렉토리 서버에 연결할 수 있습니다. 어떤 서버에 연결해야 할지 모르겠다면 네트워크 관리자에게 문의하십시오.

중요: 컴퓨터 이름에 하이픈이 포함되어 있다면 LDAP 또는 Active Directory와 같은 디렉토리 도메인에 접속하거나 바인딩할 수 없습니다. 바인딩을 구축하려면 하이픈이 포함되어 있지 않은 컴퓨터 이름을 사용하십시오.
