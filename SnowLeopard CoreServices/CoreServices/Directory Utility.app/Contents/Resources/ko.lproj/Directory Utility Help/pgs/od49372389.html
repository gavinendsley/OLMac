홈
Active Directory 포리스트의 모든 도메인으로부터 인증 제어하기

디렉토리 유틸리티의 Active Directory 커넥터를 사용하도록 구성된 컴퓨터에서, Active Directory 포리스트의 모든 도메인으로부터의 사용자를 인증하도록 허용하거나 아니면 개별 도메인으로부터의 사용자만 인증하도록 제한할 수 있습니다.

사용자가 포리스트의 모든 도메인으로부터 인증할 수 있는지 여부를 제어하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 서비스를 클릭하십시오.

  7. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  8. 서비스 목록에서 Active Directory를 선택하고 편집(/) 단추를 클릭하십시오.

  9. 고급 옵션이 가려져 있으면, 고급 옵션 보기를 클릭하십시오.

  10. 관리를 클릭하십시오.

  11. “포리스트의 모든 도메인으로부터 인증 허용”을 선택하십시오.

    "포리스트의 모든 도메인으로부터 인증 허용"을 선택하면, Active Directory 포리스트를 컴퓨터의 인증 및 연락처용 사용자 설정 검색 정책에 추가할 수 있습니다.

    사용자 설정 검색 정책에 Active Directory 포리스트를 추가하면, 포리스트가 사용 가능한 디렉토리 목록에 "/Active Directory/All Domains"로 나타납니다. (이것이 기본 설정입니다.)

    "포리스트의 모든 도메인으로부터 인증 허용"을 선택 해제하면, Active Directory 도메인을 컴퓨터의 인증 및 연락처용 사용자 설정 검색 정책에 개별적으로 추가할 수 있습니다.

    사용자 설정 검색 정책에 Active Directory 도메인을 추가하면, 각 Active Directory 도메인이 사용 가능한 디렉토리 목록에 별도로 나타납니다.

  12. 승인을 클릭하십시오.

  13. “포리스트의 모든 도메인으로부터 인증 허용”을 선택한 다음, 인증 패널 및 연락처 패널에 있는 사용자 설정 검색 정책을 변경하여 Active Directory 포리스트 또는 선택한 도메인을 포함하십시오.

    사용자 설정 검색 정책 변경에 관한 추가 정보를 보려면 사용자 설정 검색 정책 지정하기를 참조하십시오.
