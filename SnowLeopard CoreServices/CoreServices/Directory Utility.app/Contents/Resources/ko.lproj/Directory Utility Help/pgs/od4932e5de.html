자동화된 클라이언트 구성

사용자의 Mac OS X v10.6 컴퓨터가 네트워크에 연결되어 있고 디렉토리 도메인에는 연결되어 있지 않을 경우, Mac OS X v10.6 컴퓨터에서 Bonjour를 사용하여 Mac OS X v10.6 서버를 검색합니다.

Bonjour가 최소 한 개의 자동 구성이 가능한 서비스(AFP, SMB, VPN, Mail, iCal, iChat 및 알림과 같은)가 실행되고 있는 Open Directory 마스터가 실행 중인 Mac OS X v10.6 서버 및 현재 로그인된 Mac OS X v10.6 컴퓨터 사용자와 짧은 또는 긴 이름이 일치하는 디렉토리 도메인에 있는 사용자 계정을 발견했다면, 사용자의 Mac OS X v10.6 컴퓨터에 초대가 나타납니다.

이 초대는 디렉토리 서버로의 연결을 설정하기 위한 지원을 제공합니다. 서버로의 연결을 허용 또는 거절하거나 나중에 연결할 수도 있습니다.

컴퓨터가 연결 가능한 하나 이상의 Mac OS X v10.6 디렉토리 서버를 발견하면 초대 팝업 메뉴에 해당 서버가 나열됩니다. 서버에 연결하면, 해당 지원이 디렉토리 서버가 실행 중인 서비스를 열어서 연결한 다음 응용 프로그램을 구성합니다.

자동 지원을 사용하여 디렉토리 도메인에 연결하려면,

  1. 해당 지원이 나타나면 서비스 설정을 클릭하십시오.

  2. 서버 팝업 메뉴에서 연결하려는 디렉토리 도메인을 선택하십시오.

    네트워크에 사용 가능한 디렉토리 도메인이 하나 이상 있을 수도 있습니다.

    연결하려는 서버가 목록에 없으면, 서버 필드에 서버 이름 또는 IP 주소를 입력하고 승인을 클릭하십시오.

  3. 권한이 있는 계정의 사용자 이름 및 암호를 입력하여 연결 중인 디렉토리 도메인 컴퓨터에 연결하십시오.

    이 정보를 모른다면, 디렉토리 관리자에게 문의하십시오.

  4. “이 컴퓨터에 사용자 이름 계정에 대한 암호를 입력하십시오.”에 나타나는 사용자 계정의 암호를 입력하십시오.

    해당 지원이 로컬 컴퓨터 계정의 암호와 일치시키기 위해 사용자 디렉토리 계정 암호를 변경합니다.

  5. 계속을 클릭하십시오.

  6. UpgradeUser 도구가 완료되면, 계속을 클릭하십시오.

  7. 설정된 서비스를 설명하는 메시지 나타나고 로그 아웃을 요청하면, 로그 아웃을 클릭하십시오.

    로그인하면 새로운 서비스를 사용하여 시작할 수 있습니다.
