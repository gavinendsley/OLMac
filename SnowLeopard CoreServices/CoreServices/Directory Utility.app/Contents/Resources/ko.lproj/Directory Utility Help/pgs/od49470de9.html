자동 검색 정책 지정하기

디렉토리 유틸리티를 사용하여 Mac OS X 컴퓨터의 인증 및 연락처 검색 정책이 자동으로 지정되도록 구성할 수 있습니다.

자동으로 설정된 검색 정책에는 로컬 디렉토리 도메인이 포함됩니다. 또한 DHCP 서비스에 의해 지정된 LDAP 디렉토리 서버도 포함될 수 있습니다.

자동 검색은 인증 및 연결 검색 정책에서 기본 구성입니다.

참고: Mac OS X Mail 및 주소록과 같은 일부 응용 프로그램은 Open Directory를 사용하지 않고 LDAP 디렉토리에 직접 접근할 수 있습니다. 이러한 응용 프로그램에서 LDAP 디렉토리에 직접 접근하도록 설정하려면, 해당 응용 프로그램을 열고 올바른 환경설정을 설정해 주면 됩니다.
중요: 자동 인증 검색 정책 및 DHCP 공급 LDAP 서버 또는 DHCP 공급 공유 디렉토리 도메인을 사용하도록 Mac OS X을 구성하였다면, 악의를 가진 사용자가 여러분의 컴퓨터에 침입할 수 있는 위험이 높아집니다. 또한, 컴퓨터가 무선 네트워크에 연결되도록 구성되어 있다면, 이 위험은 더욱 높아집니다. 추가 정보를 보려면 악의적인 DHCP 서버로부터 컴퓨터 보호하기 부분을 참조하십시오.
검색 정책을 자동으로 지정하도록 설정하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 검색 정책을 클릭하여 다음 검색 정책을 선택하십시오.

    • 인증: 인증에 사용되는 검색 정책과 기타 관리 데이터를 보여줍니다.

    • 연락처: 주소록과 같은 응용 프로그램에 있는 연락처 정보에 사용되는 검색 정책을 보여줍니다.

  7. 검색 팝업 메뉴에서 자동을 선택한 다음 적용을 클릭하십시오.

  8. 시스템 환경설정에서, 컴퓨터의 네트워크 환경설정이 DHCP 사용이나, 수동 IP 주소로 DHCP 사용이 설정되어 있어야 합니다.
