홈
LDAP 디렉토리에 대한 구성 레코드 속성 매핑하기

비 Apple LDAP 디렉토리에 관리되는 Mac OS X 사용자에 대한 정보를 저장하려면, RealName 및 DataStamp Config 레코드 유형 속성을 매핑해야 합니다.

이 속성들을 매핑하지 않으면, Workgroup Manager를 사용하여 LDAP 디렉토리에 있는 사용자 레코드를 변경할 때 다음과 같은 오류 메시지가 나타납니다:

"dsRecTypeStandard:Config"라는 이름의 속성이 매핑되지 않았습니다.

캐시를 위해 구성 레코드 유형의 RealName과 DataStamp 속성을 따르는 Mac OS X 클라이언트 관리를 사용하고 있지 않으면, 이 메시지를 무시할 수 있습니다.
