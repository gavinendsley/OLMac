Open Directory 서버 연결 추가하기

Open Directory 서버를 추가할 때 서버 이름이나 IP 주소, 서버가 SSL(Secure Socker Layer)을 사용하는지 여부를 알아야 합니다.

Open Directory 서버를 추가하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

    편집 단추가 보인다면, 컴퓨터가 최소 하나의 디렉토리 서버에 연결되어 있는 것입니다.

  4. 추가(+) 단추를 클릭하십시오.

  5. “새로운 유형의 디렉토리 추가” 팝업 메뉴에서 Open Directory를 선택합니다.

  6. “서버 이름 또는 IP 주소” 필드에 해당 서버 이름이나 IP 주소를 입력하십시오.

  7. (조건 )“SSL을 사용하여 암호화” 체크상자를 선택하기 전에 Open Directory 관리자에게 SSL이 필요한지 문의하십시오.

중요:  디렉토리 서버에 연결되어 있는 동안 changeip를 사용하여 IP 주소 및 Mac OS X Server의 컴퓨터 이름을 변경하면, 디렉토리 서버에 연결 해제하고 다시 연결하여 새로운 컴퓨터 이름 및 IP 주소로 디렉토리를 업데이트해야 합니다. 디렉토리 서버에 연결 해제 및 다시 연결하지 않으면 디렉토리가 업데이트되지 않으며 기존 컴퓨터 이름 및 IP 주소를 계속 사용하게 됩니다.
