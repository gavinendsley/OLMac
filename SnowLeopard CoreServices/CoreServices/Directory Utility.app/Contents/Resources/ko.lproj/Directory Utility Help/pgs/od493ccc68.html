Active Directory 서버로부터 바인드 해제하기

컴퓨터가 디렉토리 유틸리티의 Active Directory 커넥터를 사용하여 Active Directory 서버에 바인드한다면, Active Directory 서버로부터 컴퓨터를 바인드 해제할 수 있습니다.

컴퓨터가 서버에 접속할 수 없거나 컴퓨터 레코드가 서버에서 제거되었다면, 강제로 바인드 해제할 수 있습니다.

Active Directory 서버로부터 컴퓨터를 바인드 해제하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 서비스를 클릭하십시오.

  7. 서비스 목록에서 Active Directory를 선택하고 편집(/) 단추를 클릭하십시오.

  8. 바인드 해제를 클릭하고 Active Directory 도메인에 대한 연결을 종료할 수 있는 권한을 가진 사용자로 인증한 다음, 승인을 클릭하십시오.

    인증이 수락되지 않았거나 컴퓨터에서 Active Directory에 접속할 수 없다는 경고 메시지가 나타나면, 강제로 바인드 해제를 클릭하여 연결을 강제로 종료할 수 있습니다.

    강제로 바인드를 해제하면, Active Directory에 여전히 이 컴퓨터에 대한 컴퓨터 레코드가 포함됩니다. Active Directory 관리자에게 통보하여 관리자가 컴퓨터 레코드를 제거하도록 해야 합니다.

  9. 서비스 패널에서 Active Directory의 활성화 설정을 선택 해제한 다음, 적용을 클릭하십시오.
