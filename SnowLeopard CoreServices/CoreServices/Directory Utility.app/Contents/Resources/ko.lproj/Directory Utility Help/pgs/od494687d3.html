사용자 설정 검색 정책 지정하기

디렉토리 유틸리티를 사용하여 Mac OS X 컴퓨터의 인증 및 연락처 검색 정책에서 사용자 설정 디렉토리 도메인 목록을 사용하도록 설정할 수 있습니다.

사용자 설정 목록은 컴퓨터의 로컬 디렉토리 도메인에서 시작하고, Open Directory(그리고 기타 LDAP 디렉토리 도메인), Active Directory 도메인, 공유 디렉토리 도메인, BSD 구성 파일, NIS 도메인을 포함시킬 수 있습니다.

컴퓨터의 사용자 설정 검색 정책에 지정된 디렉토리 도메인을 사용할 수 없다면, 컴퓨터가 시동되면 지연이 발생합니다.

검색 정책에 대한 디렉토리 도메인의 사용자 설정 목록을 지정하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 검색 정책을 클릭하여 다음 검색 정책을 선택하십시오.

    • 인증: 인증에 사용되는 검색 정책과 기타 관리 데이터를 보여줍니다.

    • 연락처: 주소록과 같은 응용 프로그램에 있는 연락처 정보에 사용되는 검색 정책을 보여줍니다.

  7. 검색 팝업 메뉴에서 “사용자 설정 경로”를 선택하십시오.

  8. 추가를 클릭하고 디렉토리를 선택한 후 다시 추가를 클릭하면 디렉토리 도메인을 필요한 만큼 추가할 수 있습니다.

  9. 디렉토리 도메인을 목록의 위/아래로 드래그하여 나열된 순서를 필요한 대로 변경하십시오.

  10. 검색 정책에 넣지 않으려는 디렉토리 도메인을 선택하고 삭제(–) 단추를 클릭하여 나열된 디렉토리 도메인을 제거하십시오.

  11. 승인을 클릭하여 제거를 확인하고 적용을 클릭하십시오.

사용 가능한 디렉토리에 있지 않은 디렉토리를 추가하고자 한다면, 컴퓨터가 해당 디렉토리에 접근하도록 구성되어 있어야 합니다. 추가 정보는 다음을 참조하십시오.

  • 고급 LDAP 서비스 설정 사용하기

  • 고급 Active Directory 서비스 설정 사용하기

  • NIS 설정 지정하기
