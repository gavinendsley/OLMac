BSD 구성 파일에서 데이터 설정하기

Mac OS X 컴퓨터에서 BSD 구성 파일로부터 관리 데이터를 얻게 하려면 데이터가 파일 안에 존재하고 Mac OS X에서 필요로 하는 포맷이어야 합니다.

파일에 데이터를 추가, 변경 또는 재구성해야 할 수 있습니다. Workgroup Manager는 BSD 구성 파일에 있는 데이터를 변경할 수 없으므로 텍스트 편집기나 기타 도구를 사용하여 필요한 수정을 해야 합니다.

다음 테이블에는 파일 이름이 나열되어 있으며 이들 안에 포함되어 있는 내용을 설명합니다.

BSD 구성 파일
다음을 포함

/etc/master.passwd

사용자 이름, 암호, ID, 주요 그룹 ID 등

/etc/group

그룹 이름, ID 및 구성원

/etc/fstab

NFS 마운트

/etc/hosts

컴퓨터 이름과 주소

/etc/networks

네트워크 이름과 주소

/etc/services

서비스 이름, 포트 및 프로토콜

/etc/protocols

IP 프로토콜 이름과 번호

/etc/rpcs

Open Network Computing RPC 서버

/etc/printcap

프린터 이름 및 기능

/etc/bootparams

Bootparam 설정

/etc/bootp

Bootp 설정

/etc/aliases

메일 가상본 및 배포 목록

/etc/netgroup

네트워크에 걸친 그룹 이름 및 구성원
