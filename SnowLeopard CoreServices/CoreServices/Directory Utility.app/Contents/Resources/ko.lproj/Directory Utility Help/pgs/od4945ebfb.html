Mac OS X Server가 설치된 서버 또는 Mac OS X이 설치된 컴퓨터를 Windows 2000 또는 Windows 2003 서버상의 Active Directory 도메인에 접근하도록 구성할 수 있습니다.

작업 설명 및 지침을 보려면, 다음을 참조하십시오.

  • Active Directory 접근에 관하여

  • Active Directory 도메인으로의 접근 구성하기

  • Active Directory에서 모바일 사용자 계정 설정하기

  • Active Directory 사용자 계정의 홈 폴더 설정하기

  • Active Directory 사용자 계정의 UNIX 셸 설정하기

  • UID를 Active Directory 속성으로 매핑하기

  • 주 그룹 ID를 Active Directory 속성으로 매핑하기

  • 그룹 계정의 그룹 ID를 Active Directory 속성으로 매핑하기

  • 권장 Active Directory 서버 지정하기

  • 컴퓨터를 관리할 수 있는 Active Directory 그룹 변경하기

  • Active Directory 포리스트의 모든 도메인으로부터 인증 제어하기

  • Active Directory 서버로부터 바인드 해제하기

  • Active Directory에서 다른 레코드 및 사용자 계정 편집하기

Active Directory 도메인에 접근할 수 있는 다른 방법은 일부 네트워크와 관련이 있습니다. Active Directory 도메인으로의 LDAP 접근 설정하기를 참조하십시오.
