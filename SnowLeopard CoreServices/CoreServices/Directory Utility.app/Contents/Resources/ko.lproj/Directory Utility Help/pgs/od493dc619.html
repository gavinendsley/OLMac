디렉토리 유틸리티는 Mac OS X에서 사용할 수 있는 서비스 목록을 보여줍니다. 이 서비스 목록에는 Mac OS X에서 디렉토리 도메인에 저장된 사용자 정보 및 기타 관리 데이터를 사용할 수 있도록 해 주는 디렉토리 서비스가 포함됩니다.

각 디렉토리 서비스에 대한 접근을 활성 또는 비활성시킬 수 있습니다. 디렉토리 유틸리티에서 서비스를 비활성화하면 Mac OS X에서 더 이상 해당 디렉토리 서비스에 접근할 수 없습니다.

  • Active Directory 서비스 활성 또는 비활성화시키기
