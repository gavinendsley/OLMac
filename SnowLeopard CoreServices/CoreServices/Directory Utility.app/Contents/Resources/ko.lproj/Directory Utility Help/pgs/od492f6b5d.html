root 사용자 계정 활성화하기

디렉토리 유틸리티를 사용하여 root 사용자 계정을 활성화할 수 있습니다. root 사용자 계정을 활성화면 위험으로부터 암호를 보호하기 위해 알파벳과 특수 문자가 포함된 적절하게 암호화된 암호를 사용하십시오.

경고:  root 계정은 중요한 시스템 파일에 변경을 수행하는 데 사용되는 제한이 없는 관리 계정입니다. 관리자로 로그인했다고 해도, 여전히 root 계정을 사용해야 하거나 sudo를 사용하여 중요한 시스템 작업을 수행해야 합니다

root 계정을 사용하여 컴퓨터에 로그인하지 마십시오(원격 또는 로컬로). 대신, sudo를 사용하여 작업을 수행하십시오. /etc/sudoers/ 파일에 사용자를 추가하여 sudo 명령에 대한 접근을 제한할 수 있습니다.

root 계정에 대한 추가 정보를 보려면, 서버 관리자(Server Admin) 도움말을 참조하십시오.

root 사용자 계정을 활성화하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

    편집 단추가 보인다면, 컴퓨터가 최소 하나의 디렉토리 서버에 연결되어 있는 것입니다.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 편집 > Root 사용자 활성화를 선택하십시오.
