LDAP 연결 인증에 사용하는 암호 변경하기

디렉토리 유틸리티를 사용하여, LDAP 서버에서 변경된 암호를 사용하도록 인증된 LDAP 연결을 업데이트할 수 있습니다. (LDAP 연결 인증에 사용된 암호가 서버에서 변경되었다면, LDAP 서버에 대해 인증된 연결을 가지고 있는 모든 컴퓨터를 업데이트해야 합니다.)

LDAP 연결에 대한 암호를 변경하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 서비스를 클릭하십시오.

  7. 서비스 목록에서 LDAPv3를 선택하고 편집(/) 단추를 클릭하십시오.

  8. 서버 구성 목록이 가려져 있다면, 옵션 보기를 클릭하십시오.

  9. 목록에서 서버 구성 하나를 선택한 다음, 편집을 클릭하십시오.

  10. 보안을 클릭하고 암호 설정을 변경하십시오.

    • “연결 시 인증 사용”이 선택 되어 있지 않아서 암호 설정이 흐리게 보인다면, LDAP 연결 인증하기를 참조하십시오.

    • “디렉토리에 다음으로 바인드됨”이 선택되어 있어서 암호 설정이 흐리게 표시되어 있다면 사용자 암호로 연결이 인증되어 있지 않은 것입니다. 대신에 연결이 신뢰 바인딩을 위해 인증 컴퓨터 레코드를 사용하는 것입니다.
