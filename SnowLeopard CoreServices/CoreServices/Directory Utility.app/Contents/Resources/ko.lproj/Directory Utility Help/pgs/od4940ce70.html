홈
Active Directory 도메인으로의 LDAP 접근 설정하기

디렉토리 유틸리티를 사용하여, Windows 서버의 Active Directory 도메인에 접근할 수 있도록 LDAPv3 구성을 설정할 수 있습니다. LDAPv3 설정은 Mac OS X 레코드 유형과 속성을 Active Directory 객체 클래스, 검색 기반, 속성에 매핑하여 조정할 수 있도록 해줍니다.

고유한 UID(User ID)와 같은 Mac OS X 레코드 유형 및 속성을 매핑하려면 확장된 Active Directory 체계가 필요합니다.

LDAPv3 구성에는 디렉토리 유틸리티에 있는 Active Directory 커넥터의 다음과 같은 기능들이 포함되지 않습니다.

  • 고유한 사용자 ID 및 주요 그룹 ID의 동적 생성

  • 로컬 Mac OS X 홈 폴더의 생성

  • Windows 홈 폴더의 자동 마운트

  • 캐시화된 인증 증명서가 있는 모바일 사용자 계정

  • Active Directory 포리스트에서 모든 도메인 발견

  • Active Directory 복제 및 장애 지원

추가 정보를 보려면 Active Directory 접근에 관하여를 참조하십시오.

Active Directory 서버 구성을 생성하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 서비스를 클릭하십시오.

  7. 서비스 목록에서 LDAPv3를 선택하고 편집(/) 단추를 클릭하십시오.

  8. 신규를 클릭하고 Active Directory 서버의 DNS 이름 또는 IP 주소를 입력하십시오.

  9. 디렉토리 유틸리티를 위한 다음 옵션을 선택한 다음, 계속을 클릭하여 디렉토리 접근이 Active Directory 서버로부터 정보를 받도록 설정하십시오.

    • Open Directory에서 Active Directory 서버로의 연결에 SSL을 사용하게 하려면 “SSL을 사용하여 암호화”를 선택하십시오. SSL 체크상자를 선택하기 전에 Open Directory 관리자에게 문의하여 SSL이 필요한 지 확인하십시오.

    • 이 디렉토리에 누군가가 서비스 인증 또는 로그인에 사용할 사용자 계정을 가지고 있다면, “인증 시 사용”을 선택하십시오.

    • 이 디렉토리에 주소록에서 사용하고자 하는 메일 주소 및 기타 정보가 있다면, “연결에 사용”을 선택하십시오.

    디렉토리 유틸리티에서 Active Directory 서버에 접속할 수 없다는 메시지가 나타나면 접근을 수동으로 구성하거나 설정 과정을 취소해야 합니다. 추가 정보를 보려면 LDAP 디렉토리에 대한 접근 수동으로 구성하기를 참조하십시오.

    "인증 시 사용" 또는 "연락처에 사용"을 선택했다면, Active Directory 도메인에 대한 LDAP 연결이 디렉토리 유틸리티의 인증 또는 연락처 패널에 있는 사용자 설정 검색 정책에 추가됩니다.

    서비스 패널에 있는 LDAPv3이 활성화되어 있어야 컴퓨터에서 방금 설정한 연결을 사용할 수 있습니다. 추가 정보를 보려면, LDAP 디렉토리 서비스 활성 또는 비활성화시키기를 참조하십시오.

  10. 대화상자가 매핑 옵션을 표시하도록 확장되면, 팝업 메뉴에서 Active Directory를 선택하고 검색 기반을 입력한 다음, 계속을 클릭하십시오.

    LDAPv3 구성을 위한 Active Directory 매핑 템플릿은 일부 Mac OS X 레코드 유형 및 속성을 표준 Active Directory 체계의 일부가 아닌 객체 클래스 및 속성과 매핑시켜 줍니다. 템플릿에서 정의된 매핑을 변경하거나 Active Directory 체계를 확장할 수 있습니다.

    (또는, LDAPv3 대신에 Active Directory 커넥터를 통해 Active Directory 도메인에 접근할 수도 있습니다. 추가 정보를 보려면, Active Directory 도메인으로의 접근 구성하기를 참조하십시오.

  11. 대화상자가 확장되어 연결 옵션이 나타나면, Active Directory 사용자 계정의 식별 이름과 암호를 입력하십시오.

  12. 승인을 클릭하여 LDAP 연결 생성을 완료하십시오.

  13. 승인을 클릭하여 LDAPv3 옵션 구성을 완료하십시오.
