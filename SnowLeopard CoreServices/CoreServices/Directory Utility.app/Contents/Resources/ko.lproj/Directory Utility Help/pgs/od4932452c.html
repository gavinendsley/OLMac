Acitive Directory 서버 연결 추가하기

Active Directory 서버에 연결하려면 서버 이름 또는 IP 주소와 Active Directory 관리자 이름 및 암호를 알아야 합니다.

Active Directory 서버를 추가하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

    편집 단추가 보인다면, 컴퓨터가 최소 하나의 디렉토리 서버에 연결되어 있는 것입니다.

  4. 추가(+) 단추를 클릭하십시오.

  5. “새로운 유형의 디렉토리 추가” 팝업 메뉴에서 Active Directory를 선택하고 다음 정보를 입력하십시오.

    • Active Directory 도메인: Active Directory 도메인의 DNS 이름입니다(예: ads.company.com).

    • 컴퓨터 ID: 선택적으로 Active Directory에서 해당 서버에 대해 사용하려는 ID를 편집할 수 있습니다. 이것은 서버의 NetBIOS 이름입니다. 이름에는 15개 이상의 문자가 포함되어 있어야 하고 특수 문자 및 구두점은 없어야 합니다. 필요하다면 서버 이름을 해당 DNS 호스트 이름과 일치시키십시오. 예를 들어, DNS 서버에 해당 서버에 대한 엔트리가 “server.example.com”이라면 서버 이름을 “server”로 하십시오.

    • AD 관리자 이름 및 암호: Active Directory 관리자의 사용자 이름 및 암호를 입력하십시오.

  6. 승인을 클릭하십시오.
