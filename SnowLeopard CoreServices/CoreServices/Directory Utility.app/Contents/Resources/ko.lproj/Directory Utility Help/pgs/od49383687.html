홈
Active Directory에서 모바일 사용자 계정 설정하기

디렉토리 유틸리티의 Active Directory 커넥터를 사용하도록 구성된 컴퓨터에서 모바일 Active Directory 사용자 계정의 사용을 활성화 또는 비활성화시킬 수 있습니다. 모바일 계정이 있는 사용자는 컴퓨터가 Active Directory 서버에 연결되어 있지 않아도 자신의 Active Directory 인증을 사용하여 로그인할 수 있습니다.

Active Directory 커넥터는 컴퓨터가 Active Directory 도메인에 연결되어 있을 때 사용자가 로그인하면, 사용자의 모바일 계정에 대한 인증을 캐시화합니다. 이러한 인증 캐시는 Active Directory 체계를 변경할 필요가 없습니다.

Active Directory 체계가 확장되어 Mac OS X 관리 클라이언트 속성까지 포함한다면, Active Directory 커넥터의 모바일 계정 설정 대신에 이들의 모바일 계정 설정이 사용됩니다.

자동으로 모바일 계정을 생성하도록 하거나 Active Directory 사용자가 모바일 계정 생성을 확인하도록 요청할 수 있습니다.

Active Directory 도메인으로부터 모바일 계정을 활성 또는 비활성화하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 서비스를 클릭하십시오.

  7. 서비스 목록에서 Active Directory를 선택하고 편집(/) 단추를 클릭하십시오.

  8. 고급 옵션이 가려져 있으면, 고급 옵션 보기를 클릭하십시오.

  9. 사용자 환경을 클릭하고 “로그인 시 모바일 계정 생성”을 클릭한 다음, (옵션으로) “모바일 계정을 생성하기 전에 확인 필요”를 클릭하십시오.

    • 두 옵션이 모두 선택되었다면, 각 사용자가 로그인시에 모바일 계정을 생성할지 여부를 결정합니다. 사용자가 Active Directory 사용자 계정을 사용하여 Mac OS X에 로그인하거나 네트워크 사용자로 로그인하면, 즉시 모바일 계정을 생성해주는 조절기가 있는 대화상자가 나타납니다.

    • 첫번째 옵션이 선택되어 있고 두번째 옵션이 선택되어 있지 않다면, 사용자가 로그인할 때 모바일 계정이 생성됩니다.

    • 첫번째 옵션이 선택되어 있지 않으면, 두번째 옵션이 비활성화됩니다.

  10. 승인을 클릭하십시오.
