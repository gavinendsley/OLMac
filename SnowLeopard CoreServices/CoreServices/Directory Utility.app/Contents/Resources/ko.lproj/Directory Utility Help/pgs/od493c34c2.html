LDAP 디렉토리와의 신뢰 바인딩 중단하기

컴퓨터와 신뢰 바인딩을 필요로 하지는 않지만 허용은 하는 LDAP 디렉토리 사이에 구축된 신뢰 바인딩을 중단하기 위해 디렉토리 유틸리티를 사용할 수 있습니다.

LDAP 디렉토리와의 신뢰 바인딩을 중단하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 서비스를 클릭하십시오.

  7. 서비스 목록에서 LDAPv3을 선택하고 편집(/) 단추를 클릭하십시오.

  8. 서버 구성 목록이 가려져 있다면, 옵션 보기를 클릭하십시오.

  9. 원하는 서버 구성 하나를 선택한 다음, 편집을 클릭하십시오.

  10. 바인드 해제를 클릭하고, 다음 인증을 입력하고 승인을 클릭하십시오.

    LDAP 디렉토리 관리자의 이름 및 암호를 입력하십시오(로컬 컴퓨터 관리자가 아님).

    이 컴퓨터에 대해 신뢰 바인딩이 설정되지 않았다면, 바인드 해제 단추가 나타나지 않습니다.

    컴퓨터에서 LDAP 서버에 접근할 수 없다는 경고 메시지가 나타난다면, 승인을 클릭하여 신뢰 바인딩을 강제로 중단하십시오.

    신뢰 바인딩을 강제로 중단하면, 이 컴퓨터의 LDAP 디렉토리에 컴퓨터 레코드가 여전히 남게 됩니다. LDAP 디렉토리 관리자에게 통보하여 관리자가 컴퓨터 그룹에서 컴퓨터를 제거하도록 해야 합니다.

  11. 신뢰 바인딩의 중단을 완료하려면 승인을 클릭하십시오.
