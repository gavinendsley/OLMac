Mac OS X용 데이터로 LDAP 디렉토리 나타내기

LDAP 디렉토리 도메인에 대한 접근을 구성하고 데이터 매핑을 설정한 다음 Mac OS X용 레코드 및 데이터로 이들을 나타나게 할 수 있습니다. 원격 관리(읽기/쓰기 접근)를 허용하는 LDAP 디렉토리의 경우, 다음과 같이 Mac OS X Server에 포함되어 있는 Workgroup Manager을 사용할 수 있습니다.

  • 사용자의 네트워크 브라우저(Finder 윈도우 사이드바에서 네트워크를 클릭할 때 사용자들이 볼 수 있는 것)에서 자동으로 마운트시키려는 공유 지점 및 공유 도메인을 식별합니다.

    Workgroup Manager의 Server Admin 및 Network 모듈에서 Sharing을 사용합니다.

    추가 정보를 보려면 File Services Overview(파일 서비스 개요) 및 User Management Overview(사용자 관리 개요)를 탐색하십시오.

  • 사용자 및 그룹 레코드를 지정하고 이들의 설정을 구성하십시오.

    Workgroup Manager의 Accounts 모듈을 사용하십시오.

    추가 정보를 보려면 User Management Overview(사용자 관리 개요)를 탐색하십시오.

  • 동일한 환경설정을 가지고 있고 동일한 사용자와 그룹이 사용할 수 있는 컴퓨터 목록을 지정하십시오.

    Workgroup Manager의 Computers 모듈을 사용하십시오.

    추가 정보를 보려면 User Management Overview(사용자 관리 개요)를 탐색하십시오.

모든 경우 사용자 목록 위에 있는 작은 지구 모양 아이콘을 클릭하고 Workgroup Manager에 있는 팝업 메뉴 중에서 선택하여 LDAP 디렉토리 도메인을 여십시오. LDAP 디렉토리가 팝업 메뉴에 없다면 이 메뉴에서 Other를 선택하여 LDAP 디렉토리를 선택하십시오.

참고: 읽기 전용 LDAP 디렉토리에 레코드 및 데이터를 추가하려면 해당 LDAP 디렉토리를 호스팅하는 서버에서 도구를 사용해야 합니다.
