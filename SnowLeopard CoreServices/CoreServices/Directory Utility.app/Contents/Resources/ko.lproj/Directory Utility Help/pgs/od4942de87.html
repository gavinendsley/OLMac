NIS 설정 지정하기

디렉토리 유틸리티를 사용하여, Mac OS X이 NIS(Network Information Servivce) 도메인에 접근하는 방법을 설정할 수 있습니다.

NIS 도메인에 접근하기 위한 구성을 생성하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 서비스를 클릭하십시오.

  7. 서비스 목록에서 “BSD 플랫 파일과 NIS”를 선택하고 편집(/) 단추를 클릭하십시오.

  8. NIS 도메인이 있는 서버들의 IP 주소 또는 DNS 이름 또는 NIS 도메인 이름을 입력할 수도 있습니다.

    보안을 위해 필요한 경우, 또는 서버가 구성 중인 컴퓨터와 같은 서브넷에 있지 않을 경우, NIS 서버의 호스트이름 또는 IP 주소를 포함시키십시오.

    서버를 지정하지 않으면, NIS는 브로드캐스트 프로토콜을 사용하여 서브넷에서 NIS 서버를 검색합니다.

  9. “인증을 위해 NIS 도메인 사용”을 선택한 다음, 승인을 클릭하십시오.

    NIS 도메인이 컴퓨터의 인증 검색 정책에 /BSD/도메인(4단계에서 입력한 도메인)으로 추가됩니다.
