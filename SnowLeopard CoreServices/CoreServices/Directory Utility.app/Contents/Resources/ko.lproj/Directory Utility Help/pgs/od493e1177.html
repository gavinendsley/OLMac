홈
Active Directory에서 다른 레코드 및 사용자 계정 편집하기

Workgroup Manager를 사용하여 사용자 계정, 그룹 계정, 컴퓨터 목록 및 Active Directory 도메인의 기타 레코드를 변경할 수 있습니다. 또한 Workgroup Manager를 사용하여 Active Directory 도메인 내의 레코드를 삭제할 수도 있습니다.

Active Directory 범위가 확장되어 표준 Mac OS X 레코드 유형(오브젝트 클래스) 및 속성을 포함한다면 Workgroup Manager를 사용하여 Active Directory 도메인에 컴퓨터 그룹을 생성 및 편집할 수 있습니다.

추가 정보를 보려면 User Management Overview(사용자 관리 개요)를 탐색하십시오.

Active Directory 도메인에서 사용자 또는 그룹 계정을 생성하려면 Windows 서버 관리 컴퓨터에 있는 Microsoft Active Directory 관리 도구를 사용하십시오.
