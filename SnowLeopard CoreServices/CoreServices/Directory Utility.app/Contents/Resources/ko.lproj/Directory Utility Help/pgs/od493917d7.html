Mac OS X용 읽기 전용 LDAP 디렉토리 준비하기

Mac OS X 컴퓨터에서 읽기 전용 LDAP 디렉토리로부터 관리 데이터를 가져오게 하려면 해당 데이터가 Mac OS X에서 필요로 하는 포맷으로 존재해야 합니다. 읽기 전용 LDAP 디렉토리에서 데이터를 추가, 변경 또는 재구성해야 할 수 있습니다.

Mac OS X은 읽기 전용 디렉토리에 데이터를 쓸 수 없기 때문에 다른 도구를 사용하여 변경해야 합니다. 이 도구는 읽기 전용 LDAP 디렉토리를 호스팅하는 서버상에 있어야 합니다.

Mac OS X용 읽기 전용 LDAP 디렉토리를 준비하려면,

  1. 읽기 전용 LDAP 디렉토리를 호스팅하는 서버로 이동하여 LDAP 기반 인증 및 암호 확인을 지원하도록 구성하십시오.

  2. LDAP 디렉토리 오브젝트 클래스 및 속성을 필요한 대로 변경하여 Mac OS X에서 필요로 하는 데이터를 제공하십시오.

Mac OS X 디렉토리 서비스에서 필요로 하는 데이터의 사양을 보려면, Open Directory Administration(Open Directory 관리)을 참조하십시오.
