BSD 구성 파일 설정 지정하기

역사적으로 UNIX 컴퓨터는 /etc/master.passwd, /etc/group 및 /etc/hosts와 같은 구성 파일에 관리 데이터를 저장합니다. Mac OS X은 UNIX의 BSD 버전에 기반하고 있지만, 보통 디렉토리 시스템으로부터 관리 데이터를 받습니다.

Mac OS X Server는 BSD 구성 파일의 고정 세트를 지원합니다. 어떤 구성 파일을 사용할 것인지 지정하거나, 그 파일의 내용을 Mac OS X 레코드 유형 및 속성과 매핑시킬 수 없습니다.

Mac OS X v10.2 이상(Mac OS X Server v10.2 이상 포함)에서, Open Directory는 BSD 구성 파일에서 관리 데이터를 가져올 수 있습니다. 이와 같은 기능은 BSD 구성 파일을 가지고 있는 기관에서도 Mac OS X 컴퓨터에 있는 기존 파일의 복사본을 사용할 수 있도록 해 줍니다. BSD 구성 파일은 독자적으로 사용되거나 다른 디렉토리 도메인과 결합되어 사용할 수 있습니다.

BSD 구성 파일을 사용하려면,

  1. BSD 구성 파일에 Mac OS X 디렉토리 서비스에서 필요로 하는 데이터가 포함되어 있어야 합니다.

    추가 정보를 보려면 BSD 구성 파일에서 데이터 설정하기를 참조하십시오.

  2. 시스템 환경설정을 열고 계정을 클릭하십시오.

  3. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  4. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

  5. 디렉토리 유틸리티 열기를 클릭하십시오.

  6. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  7. 서비스를 클릭하십시오.

  8. 서비스 목록에서 “BSD 플랫 파일과 NIS”를 선택하고 편집(/) 단추를 클릭하십시오.

  9. “BSD 로컬 노드에서 사용자 및 그룹 레코드 사용”을 선택하고 승인을 클릭하십시오.

    BSD 구성 파일 도메인이 컴퓨터의 인증 검색 정책에 /BSD/local로 추가됩니다.
