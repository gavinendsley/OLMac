디렉토리 서버 연결 편집하기

계정 환경설정을 사용하여 연결된 디렉토리 서버를 편집할 수 있습니다.

디렉토리 서버 연결을 편집하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 편집을 클릭하십시오.

  4. 디렉토리 서버 목록에서 편집하려는 디렉토리 서버를 선택하십시오.

  5. 편집(/) 단추를 클릭하십시오.

  6. 디렉토리 서버 설정을 변경하십시오.

  7. 승인을 클릭하십시오.
