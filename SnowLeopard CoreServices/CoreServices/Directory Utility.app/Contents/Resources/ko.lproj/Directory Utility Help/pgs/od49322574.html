디렉토리 서버 연결 모니터링하기

계정 환경설정을 사용하여 컴퓨에 연결된 디렉토리 서버 상태를 모니터링할 수 있습니다. 이 정보는 특정 디렉토리 서버에 연결할 수 없는 이유를 결정하고자 할 때 도움이 될 수 있습니다.

디렉토리 서버의 상태를 모니터링하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 편집을 클릭하십시오.

  4. 디렉토리 서버 왼쪽에 있는 상태 점의 색상을 참고하십시오.

    • 녹색: 디렉토리 서버가 컴퓨터에 응답하고 있음을 나타냅니다.

    • 노란색: 컴퓨터에서 디렉토리 서버의 응답을 기다리고 있음을 나타냅니다.

    • 빨간색: 디렉토리 서버가 컴퓨터에 응답하고 있지 않음을 나타냅니다.
