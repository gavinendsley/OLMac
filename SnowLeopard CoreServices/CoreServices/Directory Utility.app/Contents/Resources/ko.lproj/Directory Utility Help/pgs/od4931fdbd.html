디렉토리 서버 연결 제거하기

계정 환경설정에서 디렉토리 서버를 제거하기 전에 다른 응용 프로그램에서 해당 서비스를 사용하지 않는지 확인하십시오.

예를 들어, Mail이 디렉토리 서버를 사용하여 사람들을 검색하도록 구성되어 있는데 해당 디렉토리 서버를 삭제하면 이 디렉토리 서버에 있던 사람을 검색할 수 없습니다.

디렉토리 서버를 삭제하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 편집을 클릭하십시오.

  4. 디렉토리 서버 목록에서 삭제하려는 디렉토리 서버 목록을 선택하십시오.

  5. 삭제(—) 단추를 클릭하십시오.

  6. 올바른 디렉토리 서버를 선택했는지 확인하고 서버 사용 중단을 클릭하십시오.

사용자가 스스로 디렉토리 서버에서 연결 해제하면, 디렉토리 관리자가 디렉토리 서버에서 컴퓨터 레코드를 수동으로 제거해야 합니다. 네트워크 관리자에게 디렉토리 유틸리티를 사용하여 디렉토리 서버에서 컴퓨터 및 계정(디렉토리 서버에서 컴퓨터 기록을 제거할 수 있는 디렉토리 관리자 권한이 있는 계정)을 연결 해제해 줄 것을 요청할 수도 있습니다.
