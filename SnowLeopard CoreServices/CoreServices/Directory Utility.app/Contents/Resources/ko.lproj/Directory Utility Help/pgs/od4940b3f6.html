Active Directory 사용자 계정의 UNIX 셸 설정하기

디렉토리 유틸리티의 Active Directory 커넥터를 사용하도록 구성된 컴퓨터에서, Active Directory 계정이 있는 사용자가 터미널에서 Mac OS X과 대화식으로 작업할 때 기본적으로 사용할 명령어 라인 셸을 설정할 수 있습니다.

기본 셸은 SSH(Secure Shell) 또는 텔넷을 통한 원격 대화식 작업에 사용됩니다. 각 사용자는 터미널의 환경설정을 변경하여 기본 셸을 무시할 수 있습니다.

Active Directory 사용자 계정의 UNIX 셸을 설정하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 서비스를 클릭하십시오.

  7. 서비스 목록에서 Active Directory를 선택하고 편집(/) 단추를 클릭하십시오.

  8. 고급 옵션이 가려져 있으면, 고급 옵션 보기를 클릭하십시오.

  9. 사용자 환경을 클릭하십시오.

  10. 기본 사용자 셸을 선택하고 기본 사용자 셸의 경로를 입력하십시오.

  11. 승인을 클릭하십시오.
