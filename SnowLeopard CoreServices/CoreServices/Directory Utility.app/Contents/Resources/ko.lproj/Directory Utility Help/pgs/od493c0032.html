홈
LDAP 디렉토리 서비스 활성 또는 비활성화시키기

디렉토리 유틸리티를 사용하여 LDAPv2 및 LDAPv3을 사용하는 디렉토리 서비스에 대한 접근을 활성화 또는 비활성화할 수 있습니다. LDAPv3이라는 단일 디렉토리 유틸리티 플러그인은 LDAPv2 및 LDAPv3에 대한 접근을 제공합니다.

Mac OS X Server에서 제공하는 디렉토리 서비스는 다른 서버들처럼 LDAPv3을 사용합니다. LDAPv3은 Macintosh, UNIX 및 Windows 시스템의 혼합 네트워크에서 일반적인 개방형 표준입니다. 일부 서버는 이전 버전인 LDAPv2를 사용하여 디렉토리 서비스를 제공합니다.

LDAP 디랙토리 서비스를 비활성화하고 LDAP 디렉토리가 사용자 설정 검색 정책의 일부라면, 이들이 디렉토리 유틸리티 내 검색 정책의 인증 또는 연락처 패널에 빨간색으로 나열됩니다.

LDAP 디렉토리 서비스를 활성화 또는 비활성화하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 서비스를 클릭하십시오.

  7. LDAPv3 옆에 있는 체크상자를 선택 또는 선택 해제하고 적용을 클릭하십시오.
