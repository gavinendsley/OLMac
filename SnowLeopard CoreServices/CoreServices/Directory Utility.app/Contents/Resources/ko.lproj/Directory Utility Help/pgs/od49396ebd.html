로컬 디렉토리 검색 정책 지정하기

디렉토리 유틸리티를 사용하여 Mac OS X 컴퓨터의 인증 및 연락처 검색 정책에서 컴퓨터의 로컬 디렉토리만을 사용하도록 설정할 수 있습니다.

로컬 디렉토리만을 사용하는 검색 정책은 컴퓨터의 인증 정보 및 기타 관리 데이터에 대한 접근을 제한합니다.

컴퓨터의 인증 검색 정책에서 로컬 디렉토리만을 사용하도록 제한하면, 로컬 계정을 가진 사용자만이 로그인할 수 있습니다.

로컬 디렉토리 도메인만(로컬 디렉토리)을 사용하도록 검색 정책을 지정하려면,

  1. 시스템 환경설정을 열고 계정을 클릭하십시오.

  2. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  3. 로그인 옵션을 클릭한 다음 연결 또는 편집을 클릭하십시오.

  4. 디렉토리 유틸리티 열기를 클릭하십시오.

  5. 자물쇠 아이콘이 잠겨 있으면, 해당 아이콘을 클릭하고 관리자의 이름과 암호를 입력하여 잠금 해제하십시오.

  6. 검색 정책을 클릭하여 다음 검색 정책을 선택하십시오.

    • 인증: 인증에 사용되는 검색 정책과 기타 관리 데이터를 보여줍니다.

    • 연락처: 주소록과 같은 응용 프로그램에 있는 연락처 정보에 사용되는 검색 정책을 보여줍니다.

  7. 검색 팝업 메뉴에서 “로컬 디렉토리”를 선택한 다음 적용을 클릭하십시오.
