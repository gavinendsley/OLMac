Como especificar um servidor do Active Directory preferido

Num computador configurado para utilizar o conector do Active Directory do Utilitário de Directório, é possível especificar o nome DNS do servidor a cujo domínio do Active Directory pretende que o seu computador aceda por predefinição.

Se o servidor se tornar indisponível no futuro, o conector do Active Directory reverte para outro servidor nas proximidades.

Se esta opção não estiver seleccionada, o conector do Active Directory determina qual o domínio do Active Directory mais próximo.

Para especificar um servidor preferido para o conector do Active Directory aceder:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Clique em Serviços.

  6. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  7. Na lista de serviços, seleccione “Active Directory” e clique no botão de edição (/).

  8. Se as opções avançadas estiverem ocultas, clique em “Mostrar opções avançadas”.

  9. Clique em Administração.

  10. Seleccione “Preferir este servidor” e introduza o nome DNS do servidor do Active Directory.

  11. Clique em OK.
