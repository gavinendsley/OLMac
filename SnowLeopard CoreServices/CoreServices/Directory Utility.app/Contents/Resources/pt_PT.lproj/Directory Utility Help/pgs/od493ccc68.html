Como desligar do servidor do Active Directory

Se o computador estiver a utilizar o conector do Active Directory do Utilitário de Directório para estabelecer ligação a um servidor do Active Directory, é possível desligar o computador do servidor do Active Directory.

Poderá forçar a anulação da ligação se o computador não conseguir contactar o servidor ou se o registo do computador tiver sido removido do servidor.

Para desligar o computador do servidor do Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, seleccione “Active Directory” e clique no botão de edição (/).

  8. Clique em Desligar, autentique-se como um utilizador com direitos para terminar uma ligação ao domínio do Active Directory e clique em OK.

    Se vir um aviso a informar que as credenciais não foram aceites ou que o computador não consegue contactar o Active Directory, force a interrupção da ligação.

    Se forçar a interrupção da ligação, o Active Directory continuará a ter um registo de computador relativo a este computador. Avise o administrador do Active Directory para que este remova o registo do computador.

  9. No painel Serviços, anule a selecção da opção Activar do Active Directory e clique em Aplicar.
