Como associar o ID do grupo em contas de grupo a um atributo do Active Directory

Num computador configurado para utilizar o conector do Active Directory do Utilitário de Directório, é possível especificar um atributo do Active Directory que pretenda associar ao atributo de ID do grupo (GID) do Mac OS X em contas de grupo.

Normalmente, o esquema do Active Directory tem de ser ampliado para incluir um atributo adequado para associação ao GID:

  • Se o administrador do Active Directory ampliar o esquema do Active Directory, instalando os Services for UNIX da Microsoft, poderá associar o GID ao atributo msSFU-30-Gid-Number.

  • Se o administrador do Active Directory ampliar manualmente o esquema do Active Directory para incluir atributos RFC 2307, poderá associar o GID a gidNumber.

  • Se o administrador do Active Directory ampliar manualmente o esquema do Active Directory para incluir o atributo gidNumber do Mac OS X, pode associar ao mesmo o GID.

Se o mapeamento do GID estiver desactivado, o conector do Active Directory gera um GID com base no atributo GUID padrão do Active Directory.

AVISO:  se alterar o mapeamento do GID mais tarde, os utilizadores poderão perder o acesso a ficheiros anteriormente criados.
Para associar o GID a um atributo num esquema ampliado do Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, seleccione “Active Directory” e clique no botão de edição (/).

  8. Se as opções avançadas estiverem ocultas, clique em “Mostrar opções avançadas”.

  9. Clique em Mapeamentos.

  10. Seleccione “Associar GID do grupo ao atributo” e introduza o nome do atributo do Active Directory que pretende associar ao GID em contas de grupo.

  11. Clique em OK.
