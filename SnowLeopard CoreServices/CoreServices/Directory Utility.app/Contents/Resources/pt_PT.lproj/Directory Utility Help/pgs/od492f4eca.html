Como adicionar uma ligação a um servidor do Open Directory

Quando adiciona um servidor do Open Directory, tem de saber o nome ou o endereço IP do servidor, bem como se o servidor utiliza SSL (Secure Socket Layer).

Para adicionar um servidor Open Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Aceder ou Editar.

    Se vir um botão Editar, o computador estabeleceu pelo menos uma ligação a um servidor de directório.

  4. Clique no botão Adicionar (+).

  5. A partir do menu instantâneo “Adicionar um novo directório do tipo”, seleccione “Open Directory”.

  6. No campo “Servidor ou endereço IP”, introduza o nome do servidor ou o endereço IP.

  7. (Condicional) Antes de seleccionar a opção “Encriptar usando SSL”, consulte o administrador do Open Directory para determinar se é necessário SSL.

Importante:  se alterar o endereço IP e o nome do computador do seu Mac OS X Server utilizando o changeip enquanto está ligado a um servidor de directórios, é necessário desligar e voltar a ligar ao servidor de directórios para actualizar o directório com o nome e o endereço IP novos do computador. Se não desligar e voltar a ligar ao servidor de directórios, o directório não será actualizado e continuará a utilizar o nome e endereço IP antigos.
