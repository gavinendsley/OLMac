Como definir uma shell UNIX para contas de utilizador do Active Directory

Num computador configurado para utilizar o conector do Active Directory do Utilitário de Directório, é possível especificar a shell da linha de comandos que os utilizadores com contas do Active Directory irão utilizar, por predefinição, ao interagir com o Mac OS X no Terminal.

A shell predefinida também é utilizada para interacção remota através de secure shell (SSH) ou Telnet. Cada utilizador pode substituir a shell predefinida, alterando uma preferência do Terminal.

Para especificar uma shell UNIX para contas de utilizador do Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Aceder ou Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, seleccione “Active Directory” e clique no botão de edição (/).

  8. Se as opções avançadas estiverem ocultas, clique em “Mostrar opções avançadas”.

  9. Clique em Utilizador.

  10. Seleccione “Shell predefinida do utilizador” e introduza o respectivo percurso.

  11. Clique em OK.
