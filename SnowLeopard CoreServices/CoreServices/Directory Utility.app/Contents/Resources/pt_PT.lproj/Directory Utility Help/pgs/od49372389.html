Como controlar a autenticação a partir de todos os domínios na floresta do Active Directory

Num computador configurado para utilizar o conector do Active Directory do Utilitário de Directório, é possível permitir aos utilizadores na floresta do Active Directory autenticarem-se a partir de todos os domínios, ou pode restringir a autenticação a utilizadores de domínios específicos.

Para controlar se os utilizadores se podem autenticar a partir de todos os domínios na floresta:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  8. Na lista de serviços, seleccione “Active Directory” e clique no botão de edição (/).

  9. Se as opções avançadas estiverem ocultas, clique em “Mostrar opções avançadas”.

  10. Clique em Administração.

  11. Seleccione “Permitir autenticação de cada domínio na raiz”.

    Se seleccionar “Permitir autenticação de cada domínio na raiz”, é possível adicionar a floresta do Active Directory às políticas de pesquisa personalizadas do computador para autenticação e contactos.

    Quando adiciona uma floresta do Active Directory a uma política de pesquisa personalizada, a floresta aparece na lista de domínios de directórios disponíveis como “/Active Directory/All Domains” (esta é a predefinição).

    Se anular a selecção de “Permitir autenticação de cada domínio na raiz”, é possível adicionar individualmente domínios do Active Directory às políticas de pesquisa personalizadas do computador para autenticação e contactos.

    Quando adiciona domínios do Active Directory a uma política de pesquisa personalizada, cada domínio do Active Directory aparece separadamente na lista de domínios de directórios disponíveis.

  12. Clique em OK.

  13. Depois de seleccionar “Permitir autenticação de cada domínio na raiz”, altere a política de pesquisa personalizada no painel Autenticação e no painel Contactos para incluir a floresta do Active Directory ou os domínios seleccionados.

    Para mais informações sobre como alterar uma política de pesquisa personalizada, consulte Como definir políticas de pesquisa personalizadas.
