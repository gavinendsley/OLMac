Como adicionar uma ligação a um servidor do Active Directory

Quando se liga a um servidor do Active Directory, tem de saber o nome ou o endereço IP do servidor, bem como o nome de utilizador e a palavra-passe do administrador do Active Directory.

Para adicionar um servidor do Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Aceder ou Editar.

    Se vir um botão Editar, o computador estabeleceu pelo menos uma ligação a um servidor de directório.

  4. Clique no botão Adicionar (+).

  5. A partir do menu instantâneo “Adicionar um novo directório do tipo”, seleccione “Active Directory” e introduza as seguintes informações:

    • Domínio do Active Directory: isto é o nome DNS do domínio do Active Directory (por exemplo, ads.empresa.com.)

    • ID do computador: opcionalmente, pode editar o ID que pretende que o Active Directory utilize para o seu servidor. Este é o nome NetBIOS do servidor. O nome não pode conter mais de 15 caracteres, caracteres especiais ou pontuação. Se for prático, atribua ao servidor um nome igual ao seu nome de host DNS não qualificado. Por exemplo, se o seu servidor de DNS tiver uma entrada para o seu servidor como “servidor.exemplo.com”, atribua ao seu servidor o nome “servidor”.

    • Nome de utilizador e palavra-passe do administrador AD: digite o nome de utilizador e a palavra-passe do administrador Active Directory.

  6. Clique em OK.
