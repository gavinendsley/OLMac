Informações sobre o Utilitário de Directório

O Utilitário de Directório determina quais os serviços de directório utilizados por um computador Mac OS X e como se liga a domínios de directórios específicos. O Utilitário de Directório também define políticas de pesquisa para procurar informações de autenticação e contactos em domínios de directórios específicos.

Como configurar o Utilitário de Directório

Pode utilizar o Utilitário de Directório no seu computador para configurar e gerir a forma como o mesmo acede a serviços de directório. Pode também utilizar o Utilitário de Directório para configurar remotamente a forma como o seu computador acede a serviços de directório. O Utilitário de Directório pode igualmente ser utilizado para activar a conta da raiz ou alterar a palavra-passe da conta da raiz.

Como configurar o acesso a serviços

Pode utilizar o Utilitário de Directório para activar e desactivar diferentes tipos de serviços aos quais o Mac OS X pode aceder numa rede. Estes incluem serviços de directório que proporcionam acesso a contas de utilizador e outros dados administrativos guardados em domínios de directórios.

Como configurar o acesso a directórios

Ao utilizar o Utilitário de Directório, é possível configurar o computador para aceder a contas de utilizador e outras informações administrativas em directórios LDAP específicos, num domínio Active Directory e num domínio NIS.

Como configurar as políticas de pesquisa

Com o Utilitário de Directório, é possível especificar a política de pesquisa de autenticação utilizada pelo Mac OS X para localizar e obter informações de autenticação de utilizador e outros dados administrativos em domínios de directórios. O Utilitário de Directório também pode especificar a política de pesquisa de contactos utilizada pelo Mac OS X para localizar e obter o nome, endereço e outras informações de contacto em domínios de directórios.
