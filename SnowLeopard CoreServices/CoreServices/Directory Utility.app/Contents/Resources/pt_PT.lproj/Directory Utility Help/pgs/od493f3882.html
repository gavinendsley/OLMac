Como alterar os grupos do Active Directory que podem administrar o computador

Num computador configurado para utilizar o conector do Active Directory do Utilitário de Directório, é possível identificar as contas de grupos do Active Directory a cujos membros pretende atribuir privilégios de administrador do computador.

Os utilizadores que são membros destas contas de grupos do Active Directory podem realizar tarefas administrativas, tais como instalar software no computador Mac OS X que está a configurar.

Para adicionar ou remover contas de grupos do Active Directory cujos membros possuem privilégios de administrador:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, seleccione “Active Directory” e clique no botão de edição (/).

  8. Se as opções avançadas estiverem ocultas, clique em “Mostrar opções avançadas”.

  9. Clique em Administração.

  10. Seleccione “Permitir administração por” e altere a lista de contas de grupos do Active Directory a cujos membros pretende atribuir privilégios de administrador:

    • Adicione um grupo clicando no botão Adicionar (+) e introduzindo o nome do domínio do Active Directory, uma barra invertida e o nome de conta do grupo (por exemplo, ADS\Domain Admins, IL2\Domain Admins).

    • Apague um grupo seleccionando-o na lista e clicando no botão Apagar (–).

  11. Clique em OK.
