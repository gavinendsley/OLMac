Como mostrar ou ocultar configurações para servidores LDAP

É possível mostrar ou ocultar uma lista das configurações disponíveis para aceder a directórios LDAP. Cada configuração especifica a forma como o Open Directory acede a um directório LDAP. Quando a lista está visível, é possível alterar as definições referentes a cada configuração LDAP que não esteja esbatida (indisponível).

Para mostrar ou ocultar as configurações de directório LDAP disponíveis:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Aceder ou Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, seleccione LDAPv3 e clique no botão de edição (/).

  8. Clique no controlo “Mostrar opções” ou no controlo “Ocultar opções”, conforme o que estiver presente.
