Como definir políticas de pesquisa automáticas

Utilizando o Utilitário de Directório, é possível configurar as políticas de pesquisa de autenticação e contactos de um computador Mac OS X para serem definidas automaticamente.

Uma política de pesquisa definida automaticamente inclui o domínio de directórios local. Pode incluir também um servidor de directórios LDAP especificado pelo serviço DHCP.

Esta é a configuração predefinida para as políticas de pesquisa de autenticação e contactos.

Nota:  algumas aplicações, tais como o Mail e a Agenda do Mac OS X, podem aceder directamente a directórios LDAP, sem utilizar o Open Directory. Para configurar uma destas aplicações para aceder directamente a directórios LDAP, abra a aplicação e especifique a preferência correcta.
Importante:  se configurar o Mac OS X para utilizar uma política de pesquisa de autenticação automática e um servidor LDAP obtido por DHCP ou um domínio de directórios partilhado obtido por DHCP, aumenta o risco de um utilizador com intuitos maliciosos controlar o seu computador. O risco ainda é maior se o seu computador estiver configurado para se ligar a uma rede sem fios. Para mais informações, consulte Como proteger os computadores de um servidor DHCP malicioso.
Para ter uma política de pesquisa definida automaticamente:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Aceder ou Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em “Política de pesquisa” e seleccione uma política de pesquisa.

    • Autenticação: mostra a política de pesquisa utilizada para a autenticação e para a maior parte dos restantes dados administrativos.

    • Contactos: mostra a política de pesquisa utilizada para informações de contacto em aplicações como a Agenda.

  7. No menu instantâneo Pesquisar, seleccione Automaticamente e clique em Aplicar.

  8. Nas Preferências do Sistema, certifique-se de que as preferências da rede do computador estão configuradas para utilizar DHCP ou DHCP com um endereço IP manual.
