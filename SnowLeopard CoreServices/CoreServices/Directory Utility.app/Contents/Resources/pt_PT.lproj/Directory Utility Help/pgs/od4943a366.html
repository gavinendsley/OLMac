Como associar o UID a um atributo do Active Directory

Num computador configurado para utilizar o conector Active Directory do Utilitário de Directório, é possível especificar um atributo do Active Directory que pretenda associar ao atributo de ID de utilizador exclusivo (UID) do Mac OS X.

Normalmente, o esquema do Active Directory tem de ser ampliado para incluir um atributo adequado para associação ao UID:

  • Se o administrador do Active Directory ampliar o esquema do Active Directory, instalando os Services for UNIX da Microsoft, poderá associar o UID ao atributo msSFU-30-Uid-Number.

  • Se o administrador do Active Directory ampliar manualmente o esquema do Active Directory para incluir atributos RFC 2307, pode associar o UID a uidNumber.

  • Se o administrador do Active Directory ampliar manualmente o esquema do Active Directory para incluir o atributo UniqueID do Mac OS X, pode associar ao mesmo o UID.

Se o mapeamento do UID estiver desactivado, o conector do Active Directory gera um UID com base no atributo GUID padrão do Active Directory.

AVISO:  se alterar o mapeamento do UID mais tarde, os utilizadores poderão perder o acesso a ficheiros anteriormente criados.
Para associar o UID a um atributo num esquema ampliado do Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, seleccione “Active Directory” e clique no botão de edição (/).

  8. Se as opções avançadas estiverem ocultas, clique em “Mostrar opções avançadas”.

  9. Clique em Mapeamentos.

  10. Seleccione “Associar UID a atributo” e introduza o nome do atributo do Active Directory que pretende associar ao UID.

  11. Clique em OK.
