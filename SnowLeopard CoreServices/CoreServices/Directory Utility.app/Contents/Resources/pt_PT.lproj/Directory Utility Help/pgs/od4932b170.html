Acerca das ligações a servidores de directórios

Pode utilizar as preferências de contas para ligar computadores a servidores de directórios. É possível ver as listas dos servidores de directórios aos quais o computador está ligado, clicando na opção Editar, localizada no painel Opções das preferências de Contas. O seu computador Mac OS X acede aos servidores na lista para obter informações de utilizador e outros dados administrativos guardados no domínio de directórios dos servidores de directórios.

Quando adiciona ou apaga um servidor da lista Servidores de directório, as entradas associadas a esse servidor de directórios são adicionadas ou apagadas das listas Serviços, Autenticação e Contactos. Contudo, se remover as entradas associadas das listas Serviços, Autenticação e Contactos, o servidor de directórios não é removido da lista Servidores de directório.

Os computadores a executar o Mac OS X v10.6 podem ligar-se a um servidor de directórios Open Directory, Active Directory ou LDAP. Se não souber a que servidor se deverá ligar, contacte o administrador da rede.

Importante:  se o nome do computador incluir um hífen, poderá não conseguir aceder ou ligar-se a um domínio de directórios como o LDAP ou Active Directory. Para estabelecer ligação, utilize um nome de computador sem hífen.
