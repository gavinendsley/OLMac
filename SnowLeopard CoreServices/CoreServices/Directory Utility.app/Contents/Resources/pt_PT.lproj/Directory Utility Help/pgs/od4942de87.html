Como configurar as definições de NIS

Utilizando o Utilitário de Directório, é possível criar uma configuração que especifique a forma como o Mac OS X acede a um domínio NIS (Network Information Service).

Para criar uma configuração para aceder a um domínio NIS:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Aceder ou Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, seleccione “BSD Flat File e NIS” e clique no botão de edição (/).

  8. Digite o nome do domínio NIS, ou o nome DNS ou o endereço IP do servidor onde reside o domínio NIS.

    Inclua o nome de anfitrião ou o endereço IP do servidor NIS, se tal for necessário por motivos de segurança ou se o servidor não estiver na mesma sub-rede do computador que está a configurar.

    Se não especificar um servidor, o NIS utiliza um protocolo de difusão para localizar um servidor NIS na sub-rede.

  9. Seleccione “Usar domínio NIS para autenticação” e clique em OK.

    O domínio NIS é adicionado à política de pesquisa de autenticação do computador como /BSD/domínio, sendo domínio o nome que introduziu no passo 4.
