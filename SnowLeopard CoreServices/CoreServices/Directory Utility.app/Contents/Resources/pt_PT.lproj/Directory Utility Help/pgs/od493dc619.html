Como utilizar as definições avançadas dos serviços de directório

O Utilitário de Directório lista os serviços de directório aos quais o Mac OS X consegue aceder. A lista inclui serviços de directório que proporcionam ao Mac OS X acesso a informações de utilizador e outros dados administrativos guardados em domínios de directórios.

É possível activar ou desactivar o acesso a cada serviço de directório. Se desactivar um serviço no Utilitário de Directório, o Mac OS X deixa de poder aceder a esse serviço de directório.
