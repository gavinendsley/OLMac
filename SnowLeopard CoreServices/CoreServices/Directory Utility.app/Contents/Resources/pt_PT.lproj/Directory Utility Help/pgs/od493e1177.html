Como editar contas de utilizador e outro registos no Active Directory

É possível utilizar o Gestor do Grupo de Trabalho para alterar contas de utilizador, contas de grupo, listas de computadores e outros registos num domínio do Active Directory. Pode também utilizar o Gestor do Grupo de Trabalho para apagar registos num domínio do Active Directory.

Se o esquema do Active Directory tiver sido ampliado para incluir tipos de registos (classes de objectos) e atributos padrão do Mac OS X, pode utilizar o Gestor do Grupo de Trabalho para criar e editar grupos de computadores no domínio do Active Directory.

Para informações adicionais, consulte Descrição geral da gestão de utilizadores.

Para criar contas de utilizador ou de grupo num domínio do Active Directory, utilize as ferramentas administrativas do Microsoft Active Directory num computador administrador de servidor Windows.
