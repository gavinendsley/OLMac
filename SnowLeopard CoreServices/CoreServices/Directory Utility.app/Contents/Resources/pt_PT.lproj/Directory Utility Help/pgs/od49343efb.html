Como configurar o acesso a directórios

É possível utilizar o Utilitário de Directório (localizado nas preferências de Contas) para configurar e gerir a forma como um computador com o Mac OS X ou um servidor com o Mac OS X Server acedem a domínios de directórios.

Como aceder a directórios LDAP

Utilizando o Utilitário de Directório, é possível configurar um computador com o Mac OS X ou um servidor com o Mac OS X Server para acederem a directórios LDAP específicos, incluindo o directório LDAP do servidor principal do Open Directory de um Mac OS X Server. Para mais informações, clique nos tópicos seguintes.

Como utilizar ficheiros de configuração BSD

É possível utilizar o Open Directory para obter dados administrativos a partir de ficheiros de configuração BSD, tais como /etc/master.passwd, também conhecidos como BSD Flat Files. Também poderá configurar o acesso a ficheiros de configuração BSD utilizando o Utilitário de Directório. Para mais informações, clique nos tópicos seguintes.
