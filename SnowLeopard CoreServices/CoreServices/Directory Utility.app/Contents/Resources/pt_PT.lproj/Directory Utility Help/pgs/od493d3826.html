Como alterar as definições de ligação de um directório LDAP

É possível utilizar o Utilitário de Directório para alterar as definições de ligação de uma configuração que especifique a forma como o computador acede a um directório LDAPv3 ou LDAPv2.

Para alterar as definições de ligação para aceder a um directório LDAP:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, seleccione LDAPv3 e clique no botão de edição (/).

  8. Se a lista de configurações do servidor estiver oculta, clique em “Mostrar opções”.

  9. Na lista, seleccione uma configuração de servidor e clique em Editar.

  10. Clique em Ligação e altere qualquer uma das seguintes definições:

    • Configuração: identifica esta configuração na lista de configurações do directório LDAP (também pode alterar o nome na lista de configurações do directório LDAP).

    • Servidor ou endereço IP: especifica o nome DNS do servidor ou o respectivo endereço IP. (também pode alterá-lo na lista de configurações do directório LDAP).

    • A abertura/fecho termina dentro de: especifica a duração máxima de uma tentativa de ligação, antes de ser cancelada.

    • O tempo disponível para a pesquisa termina dentro de: especifica a duração máxima de uma pesquisa, antes de ser cancelada.

    • Tentar restabelecer ligação dentro de: especifica o número de segundos a aguardar antes de tentar efectuar uma nova ligação, caso o servidor LDAP não responda. Para evitar novas tentativas de ligação contínuas, aumente este valor.

    • A ligação ficará indisponível dentro de: especifica o número de minutos durante os quais se permite que uma ligação inactiva ou sem resposta permaneça aberta.

    • Encriptar usando SSL: determina se as comunicações com o directório LDAP serão encriptadas por meio de uma ligação SSL. (também pode alterar esta definição na lista de configurações do directório LDAP). Antes de seleccionar a opção SSL, pergunte ao administrador do Open Directory se é necessário SSL.

    • Usar porta personalizada: especifica um número de porta diferente da porta padrão utilizada nas ligações LDAP (389 sem SSL, 636 com SSL).

    • Ignorar referências ao servidor: determina se a referência de um servidor LDAP para procurar informações noutros servidores LDAP ou réplicas deve ser ignorada ou cumprida. As referências ao servidor podem ajudar um computador a encontrar informações, mas também podem atrasar os acessos ou provocar outros atrasos se o computador tiver de verificar referências a outros servidores LDAP.

    • Usar LDAPv2 (apenas leitura): determina se o protocolo LDAPv2 mais antigo será utilizado para acesso de apenas leitura a um directório LDAP.
