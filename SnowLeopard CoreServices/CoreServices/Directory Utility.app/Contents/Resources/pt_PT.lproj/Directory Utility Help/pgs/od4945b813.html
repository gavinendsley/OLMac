Como associar atributos do registo Config de directórios LDAP

Para guardar informações de clientes Mac OS X geridos num directório LDAP que não seja Apple, terá de associar os seguintes atributos do tipo de registo Config: RealName e DataStamp.

Se não associar estes atributos, a seguinte mensagem de erro irá aparecer quando utilizar o Gestor do Grupo de Trabalho para alterar um registo de utilizador a residir no directório LDAP:

O atributo com o nome “dsRecTypeStandard:Config” não está associado.

Pode ignorar esta mensagem se não estiver a utilizar a gestão de clientes Mac OS X, a qual depende dos atributos RealName e DataStamp do tipo de registo Config para uma cache.
