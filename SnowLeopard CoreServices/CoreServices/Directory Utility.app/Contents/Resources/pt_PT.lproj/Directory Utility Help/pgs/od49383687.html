Como configurar contas de utilizador móveis no Active Directory

Pode activar ou desactivar as contas de utilizador do Active Directory móveis num computador configurado para utilizar o conector do Active Directory do Utilitário de Directório. Os utilizadores com contas móveis podem aceder utilizando as respectivas credenciais do Active Directory quando o computador não está ligado ao servidor do Active Directory.

O conector do Active Directory coloca em cache as credenciais da conta móvel de um utilizador quando este acede com o computador ligado ao domínio do Active Directory. Este processo não requer a alteração do esquema do Active Directory.

Se o esquema do Active Directory tiver sido ampliado para incluir atributos de clientes geridos pelo Mac OS X, essas definições de conta móvel são utilizadas em vez da definição de conta móvel do conector do Active Directory.

Pode fazer com que as contas móveis sejam criadas automaticamente, ou pode pedir que os utilizadores do Active Directory confirmem a criação de uma conta móvel.

Para activar ou desactivar contas móveis num domínio do Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Aceder ou Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Na lista de serviços, seleccione “Active Directory” e clique no botão de edição (/).

  8. Se as opções avançadas estiverem ocultas, clique em “Mostrar opções avançadas”.

  9. Clique em Utilizador e depois clique em “Criar conta móvel ao aceder”; opcionalmente, clique em “Pedir confirmação antes de criar uma conta móvel”.

    • Se ambas as opções estiverem seleccionadas, cada utilizador decide se pretende criar uma conta móvel ao aceder. Quando um utilizador acede ao Mac OS X utilizando uma conta de utilizador do Active Directory ou como utilizador de rede, vê uma caixa de diálogo com controlos para criar imediatamente uma conta móvel.

    • Se a primeira opção estiver seleccionada, mas a segunda não, as contas móveis são criadas quando os utilizadores acedem.

    • Se a primeira opção não estiver seleccionada, a segunda opção é desactivada.

  10. Clique em OK.
