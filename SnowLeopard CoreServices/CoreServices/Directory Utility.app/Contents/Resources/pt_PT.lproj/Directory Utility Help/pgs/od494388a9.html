Como activar ou desactivar o serviço Active Directory

É possível utilizar o Utilitário de Directório para activar ou desactivar a utilização dos serviços Active Directory fornecidos por um servidor Windows. O Active Directory consiste no serviço de directório dos servidores Windows 2000 e posteriores.

Se desactivar os serviços Active Directory e os domínios do Active Directory fizerem parte de uma política de pesquisa personalizada, ficarão listados a vermelho nos painéis Autenticação ou Contactos da “Política de pesquisa” no Utilitário de Directório.

Para activar ou desactivar o acesso ao Active Directory:

  1. Abra as Preferências do Sistema e clique em Contas.

  2. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  3. Clique em Opções e depois clique em Aceder ou Editar.

  4. Clique em “Abrir o Utilitário de Directório”.

  5. Se o ícone de cadeado estiver fechado, abra-o clicando no mesmo e introduzindo um nome e palavra-passe de administrador.

  6. Clique em Serviços.

  7. Junto a Active Directory, seleccione ou anule a selecção da opção e clique em Aplicar.
