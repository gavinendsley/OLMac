Como preencher directórios LDAP com dados para o Mac OS X

Depois de configurar o acesso a domínios de directórios LDAP e de configurar o mapeamento de dados, pode preenchê-los com registos e dados para o Mac OS X. Para directórios LDAP que permitam a administração remota (acesso de leitura/escrita), é possível utilizar o Gestor do Grupo de Trabalho, incluído no Mac OS X Server, da seguinte forma:

  • Identifique os pontos de partilha e os domínios partilhados que pretende montar de forma automática nos navegadores de rede dos utilizadores (o que os utilizadores vêem quando clicam em Rede na barra lateral de uma janela do Finder).

    Utilize a Partilha no Administrador do Servidor e os módulos de Rede do Gestor do Grupo de Trabalho.

    Para informações adicionais, consulte Descrição geral dos serviços de ficheiros e Descrição geral da gestão de utilizadores.

  • Defina registos de utilizador e de grupo, e configure as respectivas definições.

    Utilize o módulo de Contas do Gestor do Grupo de Trabalho.

    Para informações adicionais, consulte Descrição geral da gestão de utilizadores.

  • Defina listas de computadores que possuam as mesmas preferências e estejam disponíveis para os mesmos utilizadores e grupos.

    Utilize o módulo de Computadores do Gestor do Grupo de Trabalho.

    Para informações adicionais, consulte Descrição geral da gestão de utilizadores.

Em todos os casos, clique no ícone do globo pequeno por cima da lista de utilizadores e seleccione no menu instantâneo do Gestor do Grupo de Trabalho para abrir o domínio de directórios LDAP. Se o directório LDAP não estiver listado no menu instantâneo, seleccione Outro a partir deste menu para seleccionar o directório LDAP.

Nota:  para adicionar registos e dados a um directório LDAP de apenas leitura, tem de utilizar as ferramentas no servidor onde o directório LDAP está alojado.
