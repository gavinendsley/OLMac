Como preparar um directório LDAP de apenas leitura para o Mac OS X

Se pretende que um computador Mac OS X obtenha dados administrativos de um directório LDAP de apenas leitura, os dados têm de existir no formato requerido pelo Mac OS X. Poderá ser necessário adicionar, alterar ou reorganizar os dados no directório LDAP de apenas leitura.

Uma vez que o Mac OS X não pode substituir dados num directório de apenas leitura, terá de utilizar outras ferramentas para efectuar as alterações. As ferramentas têm de residir no servidor que aloja o directório LDAP de apenas leitura.

Para preparar um directório LDAP de apenas leitura para o Mac OS X:

  1. Aceda ao servidor que aloja o directório LDAP de apenas leitura e configure-o para suportar autenticação e verificação de palavra-passe com base em LDAP.

  2. Altere as classes de objectos e os atributos do directório LDAP conforme o necessário para fornecer os dados de que o Mac OS X precisa.

Para a especificação dos dados de que os serviços de directório do Mac OS X necessitam, consulte a Administração do Open Directory.
