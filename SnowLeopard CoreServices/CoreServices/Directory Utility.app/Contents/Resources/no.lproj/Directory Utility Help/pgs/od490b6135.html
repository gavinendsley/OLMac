Lær om Katalogverktøy

Katalogverktøy bestemmer hvilke katalogtjenester en Mac OS X-maskin bruker og hvordan maskinen kobler seg til bestemte katalogdomener. I tillegg definerer Katalogverktøy retningslinjer for søk etter godkjennings- og kontaktinformasjon i bestemte katalogdomener.

Konfigurere Katalogverktøy

Du kan bruke Katalogverktøy på maskinen til å konfigurere og definere hvordan maskinen får tilgang til katalogtjenester. Du kan også bruke Katalogverktøy til å konfigurere hvordan maskinen får tilgang til katalogtjenester, fra en annen maskin. Katalogverktøy kan også brukes til å aktivere rotkontoen eller endre passordet for rotkontoen.

Konfigurere tilgang til tjenester

Du kan bruke Katalogverktøy til å aktivere og deaktivere forskjellige typer tjenester som Mac OS X kan gi tilgang til på et nettverk. Slike katalogtjenester inkluderer blant annet tilgang til brukerkontoer og andre administrative data som arkiveres på katalogdomener.

Konfigurere tilgang til kataloger

Med Katalogverktøy kan du konfigurere maskinen slik at den får tilgang til brukerkontoer og annen administrativ informasjon i bestemte LDAP-kataloger, et Active Directory-domene og et NIS-domene.

Sette opp søkekriterier

Med Katalogverktøy kan du definere godkjenningssøkekriteriene som brukes av Mac OS X til å finne og hente brukergodkjenningsinformasjon og andre administrative data fra katalogdomener. Katalogverktøy kan også definere kontaktsøkekriteriene som brukes av Mac OS X til å finne informasjon om navn, adresse og annen kontaktinformasjon i katalogdomener.
