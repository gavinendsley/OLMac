Konfigurere tilgang til kataloger

Du kan bruke Katalogverktøy (i Kontoer-valgpanelet) til å konfigurere og administrer hvordan en maskin med Mac OS X eller en tjener med Mac OS X Server får tilgang til katalogdomener.

Tilgang til LDAP-kataloger

Ved hjelp av Katalogverktøy kan du konfigurere en maskin med Mac OS X eller en tjener med Mac OS X Server for tilgang til LDAP-kataloger, inkludert LDAP-katalogen i en Mac OS X Server Open Directory-master. Hvis du vil vite mer, klikker du på emnene nedenfor:

Bruke BSD-konfigurasjonsfiler
