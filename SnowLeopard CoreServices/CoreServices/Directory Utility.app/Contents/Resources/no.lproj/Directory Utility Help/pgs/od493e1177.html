Redigere brukerkontoer og andre oppføringer i Active Directory

Du kan bruke Workgroup Manager til å endre brukerkontoer, gruppekontoer, maskinlister og andre oppføringer i et Active Directory-domene. Du kan også bruke Workgroup Manager til å slette oppføringer i et Active Directory-domene.

Hvis Active Directory-oppsettet er utvidet slik at det inkluderer standard Mac OS X-oppføringstyper (objektklasser) og attributter, kan du bruke Workgroup Manager til å opprette og redigere maskingrupper i et Active Directory-domene.

Hvis du vil ha ytterligere informasjon, kan du lese i User Management Overview.

Hvis du vil opprette bruker- eller gruppekontoer i et Active Directory-domene, bruker du Microsoft Active Directory-administrasjonsverktøyene på en Windows-tjeneradministrasjonsmaskin.
