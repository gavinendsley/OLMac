Forberede en LDAP-katalog med kun lesetilgang for Mac OS X

Hvis du vil at en maskin med Mac OS X skal motta administrative data fra en LDAP-katalog med kun lesetilgang, må dataene eksistere i formatet som kreves av Mac OS X. Det er mulig at du må legge til, endre eller omorganisere data i LDAP-katalogen med kun lesetilgang.

Ettersom Mac OS X ikke kan skrive data til en katalog med kun lesetilgang, må du bruke andre verktøy for å utføre endringene. Verktøyene må være til stede på tjeneren som er vert for LDAP-katalogen med kun lesetilgang.

Slik forbereder du en LDAP-katalog med kun lesetilgang for Mac OS X:

  1. Gå til tjeneren som er vert for LDAP-katalogen med kun lesetilgang, og konfigurer den slik at den støtter LDAP-basert godkjenning og passordkontroll.

  2. Endre LDAP-katalogens objektklasser og attributter etter behov for å levere dataene som er nødvendige for Mac OS X.

Du finner spesifikasjonene for dataene som kreves av Mac OS X-katalogtjenester i Open Directory Administration.
