Administrere rotbrukerkontoen

Du kan bruke Katalogverktøy (i Kontoer-valgpanelet) til å administrere rotbrukerkontoen ved å aktivere eller deaktivere rotbrukeren. Hvis du aktiverer rotbrukerkontoen, kan du også bruke Katalogverktøy til å endre passordet til rotkontoen.

Hvis du vil vite mer om administrering av rotbrukerkontoen ved hjelp av Katalogverktøy, kan du lese følgende:

  • Aktivere rotbrukerkontoen

  • Endre passordet til rotbrukerkontoen
