Konfigurere data i BSD-konfigurasjonsfiler

Hvis du ønsker at en datamaskin med Mac OS X skal hente administrative data fra BSD-konfigurasjonsfiler, må dataene eksistere i filene og være i formatet som kreves av Mac OS X.

Det er mulig du må legge til, endre eller omorganisere data i filene. Workgroup Manager kan ikke endre dataene i BSD-konfigurasjonsfiler, og du må gjøre de nødvendige endringene ved hjelp av et tekstredigeringsprogram eller andre verktøy.

Følgende tabell viser filnavnene og beskriver innholdet i dem.

BSD-konfigurasjonsfil
Inneholder

/etc/master.passwd

Brukernavn, passord, IDer, primære gruppe-IDer og så videre

/etc/group

Gruppenavn, IDer og medlemmer

/etc/fstab

NFS-aktiveringer

/etc/hosts

Maskinnavn og adresser

/etc/networks

Nettverksnavn og adresser

/etc/services

Tjenestenavn, porter og protokoller

/etc/protocols

IP-protokollnavn og numre

/etc/rpcs

Open Network Computing RPC-tjenere

/etc/printcap

Skrivernavn og -egenskaper

/etc/bootparams

Bootparam-innstillinger

/etc/bootp

Bootp-innstillinger

/etc/aliases

E-postaliaser og distribusjonslister

/etc/netgroup

Nettverksomfattende gruppenavn og medlemmer
