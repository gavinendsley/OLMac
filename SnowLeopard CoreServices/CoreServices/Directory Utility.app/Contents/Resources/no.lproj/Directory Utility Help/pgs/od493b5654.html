Populere LDAP-kataloger med data for Mac OS X

Når tilgang til LDAP-katalogdomener er konfigurert og datatilordning er satt opp, kan du populere dem med oppføringer og data for Mac OS X. For LDAP-kataloger som tillater ekstern administrasjon (lese-/skrivetilgang), kan du bruke Workgroup Manager, som følger med Mac OS X Server, på følgende måte:

  • Identifiser delingspunkter og delte domener som du vil aktivere automatisk i brukernes Nettverk-navigasjonsvinduer (det brukerne ser når de klikker på Nettverk i sidepanelet i et Finder-vindu).

    Bruk Sharing i Server Admin- og Network-modulene i Workgroup Manager.

    Hvis du vil ha ytterligere informasjon, kan du lese i File Services Overview og User Management Overview.

  • Definere bruker- og gruppeoppføringer og konfigurere innstillingene deres.

    Bruk Accounts-modulen i Workgroup Manager.

    Hvis du vil ha ytterligere informasjon, kan du lese i User Management Overview.

  • Definere lister med maskiner som har de samme innstillingene, og som er tilgjengelige for de samme brukerne og gruppene.

    Bruk Computers-modulen i Workgroup Manager.

    Hvis du vil ha ytterligere informasjon, kan du lese i User Management Overview.

I alle tilfeller klikker du på det lille globussymbolet over listen med brukere og velger å åpne LDAP-katalogdomenet fra lokalmenyen i Workgroup Manager. Hvis LDAP-katalogen ikke vises i lokalmenyen, velger du Other i denne menyen for å velge LDAP-katalogen.

Merk: Hvis du vil legge til oppføringer og data til i en LDAP-katalog med kun lesetilgang, må du bruke verktøy på tjeneren som er vert for LDAP-katalogen.
