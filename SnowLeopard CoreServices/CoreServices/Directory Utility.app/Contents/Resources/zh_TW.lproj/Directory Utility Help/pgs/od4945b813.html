LDAP 目錄的對應設定記錄屬性

若要在非 Apple 的 LDAP 目錄中儲存受管理 Mac OS X 使用者的資訊，您必須對應“設定”記錄類型的下列屬性：RealName 和 DataStamp。

若沒有對應這些屬性，在使用 Workgroup Manager（工作群組管理程式）更改 LDAP 目錄中的使用者記錄時，會出現下列錯誤訊息：

The attribute with name“dsRecTypeStandard:Config”is not mapped.（名為“dsRecTypeStandard:Config”的屬性沒有可對應的屬性）。

如果您沒有使用 Mac OS X 用戶端管理功能（其依照“設定”記錄類型的 RealName 與 DataStamp 屬性來進行資料快取），則可以忽略此訊息。
