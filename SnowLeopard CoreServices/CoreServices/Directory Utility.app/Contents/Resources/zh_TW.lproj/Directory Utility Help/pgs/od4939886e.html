指定 BSD 配置檔的設定

傳統上，UNIX 電腦會將管理資料存放在設定檔中，例如 /etc/master.passwd、/etc/group、以及 /etc/hosts 等。Mac OS X 以 UNIX 的 BSD 版本為架構，但可以從目錄系統正常取得管理資料。

Mac OS X Server 支援 BSD 配置檔的修正集。您將無法指定要使用哪一個設定檔，也無法將 Mac OS X 的記錄類型與屬性對應至 BSD 配置檔的內容。

在 Mac OS X v10.2 或以上版本（包括 Mac OS X Server v10.2 或以上版本）的系統中，Open Directory 可以自 BSD 配置檔中取得管理資料。此項功能讓具有 BSD 配置檔的組織，可以在 Mac OS X 電腦上直接使用現有 BSD 配置檔的拷貝。BSD 配置檔可獨立使用，或是與其他的目錄網域搭配使用。

若要使用 BSD 配置檔：

  1. 請確定每個 BSD 配置檔都含有 Mac OS X 目錄服務所需的資料。

    如需更多資訊，請參閱「設定 BSD 配置檔的資料」。

  2. 打開“系統偏好設定”並按“帳號”一下。

  3. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  4. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  5. 按“打開目錄工具程式⋯”一下。

  6. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  7. 按“服務”一下。

  8. 在服務列表中，選擇“BSD 扁平式檔案和 NIS”，然後按“編輯”（/）按鈕一下。

  9. 請選擇“使用 BSD 本地節點的使用者和群組記錄”，然後按“好”一下。

    BSD 配置檔網域將以 /BSD/local 加入到電腦的認證搜尋規則。
