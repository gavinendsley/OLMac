設定 Active Directory 使用者帳號的個人專屬檔案夾

若電腦被設定為使用“目錄工具程式”的 Active Directory 連接器，您可以啟用或停用 Active Directory 使用者帳號的網路個人專屬檔案夾或本機個人專屬檔案夾。

使用網路個人專屬檔案夾，當使用者登入時，使用者的 Windows 網路個人專屬檔案夾會裝載為 Mac OS X 個人專屬檔案夾。

若已延伸 Active Directory 描述語言來包含網路個人專屬檔案夾，您可以決定是要從 Active Directory 標準的 homeDirectory 屬性或從 Mac OS X 的 homeDirectory 屬性，取得網路個人專屬檔案夾位置。

使用本機個人專屬檔案夾，每個登入的 Active Directory 使用者都會在 Mac OS X 啟動磁碟上擁有個人專屬檔案夾。另外，使用者的網路個人專屬檔案夾會裝載為網路卷宗，它的功能如同一個共享處。使用者可以在此網路卷宗和本機個人專屬檔案夾之間拷貝檔案。

若要設定 Active Directory 使用者帳號的個人專屬檔案夾：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 Active Directory 並按“編輯”按鈕（/）一下。

  8. 若進階選項為隱藏狀態，請按“顯示進階選項”一下。

  9. 按“使用者經驗”一下。

  10. 如果您想要 Active Directory 使用者帳號在電腦的“使用者”檔案夾裡擁有本機個人專屬檔案夾，請按“將本機個人專屬檔案夾強制於啟動磁碟上”一下。

    如果您已經選取“登入時建立行動帳號”選項，則該選項無法使用。

  11. 若要對個人專屬檔案夾位置使用 Active Directory 的標準屬性，請選擇“使用 Active Directory 上的 UNC 路徑來取得網路個人專屬位置”，然後從下列通訊協定中選擇取用個人專屬檔案夾的通訊協定：

    • 若要使用標準的 Windows 通訊協定 SMB，請從“要使用的網路通訊協定”彈出式選單中選擇 smb。

    • 若要使用標準的 Macintosh 通訊協定 AFP，請從“要使用的網路通訊協定”彈出式選單中選擇 afp。

  12. 若要將 Mac OS X 的屬性用於個人專屬檔案夾位置，請取消選取“使用 Active Directory 上的 UNC 路徑來取得網路個人專屬位置”。

    若要使用 Mac OS X 的屬性，則必須延伸 Active Directory 描述語言來將其納入。

  13. 按“好”一下。

如果您更改 Active Directory 網域中使用者帳號的名稱，伺服器會為該使用者帳號建立個人專屬檔案夾（和子檔案夾），下次伺服器便會用其來登入 Mac OS X 電腦。使用者仍可以瀏覽舊有的個人專屬檔案夾，並於 Finder 中查看其內容。

透過在使用者下次登入前重新命名舊檔案夾，您可以避製作新的個人專屬檔案夾。
