啟用或停用 LDAP 目錄服務

對於使用 LDAPv2 和 LDAPv3 的目錄服務，您可以使用“目錄工具程式”來啟用或停用對它們的存取權限。僅需透過一個名為 LDAPv3 的“目錄工具程式”外掛模組，即可使用 LDAPv2 與 LDAPv3。

Mac OS X Server 使用 LDAPv3 提供目錄服務，就像其他伺服器一樣。LDAPv3 是一種開放的標準，在 Macintosh、UNIX 和 Windows 系統的混合網路中很普遍。有些伺服器會使用較舊版的 LDAPv2 來提供目錄服務。

若您停用 LDAP 目錄服務，且 LDAP 目錄是自定搜尋規則的一部分，它們將在“目錄工具程式”的“認證”或“聯絡資訊”面板中以紅色列出。

若要啟用或停用存取 LDAP 目錄服務：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 選擇或取消選擇 LDAPv3 旁邊的註記框，然後按“套用”一下。
