更改 LDAP 目錄的連線設定

您可以使用“目錄工具程式”來更改配置的連線設定，此配置指定了電腦存取 LDAPv3 或 LDAPv2 目錄的方式。

若要更改存取 LDAP 目錄的連線設定：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按“編輯”一下。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 LDAPv3 並按“編輯”按鈕（/）一下。

  8. 若伺服器配置列表為隱藏狀態，請按“顯示選項”一下。

  9. 在列表中選擇伺服器配置，並按“編輯⋯”一下。

  10. 按“連線”一下，並更改以下任何設定：

    • 配置名稱：用來辨別 LDAP 目錄配置列表中的配置。（您也可以在 LDAP 目錄配置列表中更改名稱。）

    • 伺服器名稱或 IP 位址：指定伺服器的 DNS 名稱或其 IP 位址。（您也可以在 LDAP 目錄配置列表中更改此項設定。）

    • 打開∕關閉逾時：指定取消連線前，可持續嘗試連線的最長時間。

    • 詢問逾時：指定取消詢問前，可持續進行詢問的最長時間。

    • 重新綁定嘗試登入：若 LDAP 伺服器無回應，指定在嘗試重新連接前要等待的秒數。您可以提高此數值，來避免連續嘗試重新連接。

    • 連線閒置中斷：指定允許閒置或無回應的連線保持開啟的分鐘數。

    • 使用 SSL 加密：決定與 LDAP 目錄進行通訊時是否要使用 SSL 連線進行加密保護。（您也可以在 LDAP 目錄配置列表中更改此項設定。）在您勾選 SSL 註記框前，請詢問您的 Open Directory 管理者是否需要使用 SSL。

    • 使用自定埠：指定使用標準埠之外的埠號進行 LDAP 連線（不使用 SSL 的 389 或使用 SSL 的 636）。

    • 忽略伺服器的指示：決定是否忽略或依照 LDAP 伺服器的指示，以在其他 LDAP 伺服器或複本目錄裡尋找資訊。 伺服器的指示可以幫助電腦尋找資訊，但也會導致登入延遲，或是在電腦必須驗證其他的 LDAP 伺服器的指示時，造成其他延遲。

    • 使用 LDAPv2（唯讀）：決定是否為 LDAP 目錄的唯讀連線使用較舊的 LDAPv2 通訊協定。
