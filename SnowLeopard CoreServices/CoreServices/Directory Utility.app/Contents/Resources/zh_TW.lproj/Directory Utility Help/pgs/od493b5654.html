為 Mac OS X 置入含有資料的 LDAP 目錄

在完成 LDAP 目錄網域連線和資料對應的設定後，您可以為其置入適用於 Mac OS X 的記錄與資料。對於允許遠端管理（讀取∕寫入權限）的 LDAP 目錄，您可以使用 Mac OS X Server 隨附的 Workgroup Manager（工作群組管理程式）來執行以下動作：

  • 識別您想要在使用者的“網路”瀏覽器（即使用者在 Finder 視窗側邊欄中按一下“網路”後所見的畫面）中自動裝載的共享點與共享網域。

    在 Workgroup Manager（工作群組管理程式）的“伺服器管理”與“網路”模組中，使用“共享”功能。

    如需其他資訊，請瀏覽「檔案服務概述」和「使用者管理概述」。

  • 定義使用者與群組記錄並為其進行設定。

    使用 Workgroup Manager（工作群組管理程式）的“帳號”模組。

    如需其他資訊，請瀏覽「使用者管理概述」。

  • 定義偏好設定相同、且可被相同使用者和群組所使用的電腦列表。

    使用 Workgroup Manager（工作群組管理程式）的“電腦”模組。

    如需其他資訊，請瀏覽「使用者管理概述」。

在各種情況下，請按一下使用者上方的球形圖像，並從 Workgroup Manager（工作群組管理程式）的彈出式選單中進行選擇來打開 LDAP 目錄網域。如果 LDAP 目錄並未被列於彈出式選單，請從此選單中選擇“其他”來選擇 LDAP 目錄。

注意：若要為唯讀 LDAP 目錄加入記錄和資料，您必須使用 LDAP 目錄所在主機伺服器上的工具。
