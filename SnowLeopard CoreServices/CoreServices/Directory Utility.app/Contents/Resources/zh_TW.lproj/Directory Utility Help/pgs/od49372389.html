從 Active Directory 樹系的所有網域控制認證

若電腦被設定為使用“目錄工具程式”的 Active Directory 連接器，您可以允許 Active Directory 樹系中的使用者從所有網域進行認證，或者您可以限制使用者從個別網域進行認證。

若要控制使用者是否可從樹系裡的所有網域進行認證：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按“編輯”一下。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  8. 在服務列表中，請選擇 Active Directory 並按“編輯”按鈕（/）一下。

  9. 若進階選項為隱藏狀態，請按“顯示進階選項”一下。

  10. 按“管理”一下。

  11. 選擇“允許從樹系裡的任何網域認證”。

    如果您選擇“允許從樹系裡的任何網域認證”，您可以將 Active Directory 樹系加入到用於認證和聯絡資訊的電腦自定搜尋規則裡。

    當您將 Active Directory 樹系加入自定搜尋規則時，該樹系會在可用目錄網域列表裡顯示為“/Active Directory/All Domains”。（此為預設設定）。

    如果您取消選擇“允許從樹系裡的任何網域認證”，您可以將單個 Active Directory 網域加入到用於認證和聯絡資訊的電腦自定搜尋規則裡。

    當您將 Active Directory 網域加入自定搜尋規則時，各 Active Directory 網域會個別顯示在可用目錄網域的列表中。

  12. 按“好”一下。

  13. 選擇“允許從樹系裡的任何網域認證”後，請更改“認證”面板和“聯絡資訊”面板中的自定搜尋規則，以納入 Active Directory 樹系或所選網域。

    如需更多關於更改自定搜尋規則的資訊，請參閱「定義自定搜尋規則」。
