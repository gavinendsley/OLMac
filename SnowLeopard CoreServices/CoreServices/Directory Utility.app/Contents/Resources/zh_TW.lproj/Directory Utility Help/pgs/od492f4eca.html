加入 Open Directory 伺服器連線

當加入 Open Directory 伺服器時，您必須知道伺服器名稱或 IP 位址，以及伺服器是否使用 SSL（Secure Socket Layer）。

若要加入 Open Directory 伺服器：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

    如果您看到“編輯”按鈕，表示您的電腦至少有一個目錄伺服器的連線。

  4. 按一下“新增”（+）按鈕。

  5. 從“加入新的類型目錄”彈出式選單，選擇 Open Directory。

  6. 在“伺服器名稱或 IP 位址”欄位，輸入伺服器名稱或 IP 位址。

  7. （選擇性）在您勾選“使用 SSL 加密”註記框前，請詢問您的 Open Directory 管理者是否需要使用 SSL。

重要事項：若您在連接到目錄伺服器時使用 changeip 更改 Mac OS X 伺服器的 IP 位址和電腦名稱，您必須先中斷連接並重新連接到目錄伺服器，藉此使用新的電腦名稱和 IP 位址來更新目錄。若您未中斷連接並重新連接到目錄伺服器，目錄將不會進行更新，並會繼續使用舊有電腦名稱和 IP 位址。
