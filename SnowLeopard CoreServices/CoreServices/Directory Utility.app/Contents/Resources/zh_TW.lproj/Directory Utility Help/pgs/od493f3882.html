更改可以管理電腦的 Active Directory 群組

若電腦被設定為使用“目錄工具程式”的 Active Directory 連接器，對於您想讓其擁有電腦管理者權限的成員，您可以識別這些 Active Directory 群組帳號。

這些 Active Directory 群組帳號成員的使用者可以執行您所設定的管理工作，例如在 Mac OS X 電腦上安裝軟體。

若要新增或移除所屬成員具有管理者權限的 Active Directory 群組帳號：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按“編輯”一下。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 Active Directory 並按“編輯”按鈕（/）一下。

  8. 若進階選項為隱藏狀態，請按“顯示進階選項”一下。

  9. 按“管理”一下。

  10. 選擇“賦予管理者權限”，並更改您希望讓所屬成員具有管理者權限的 Active Directory 群組帳號列表。

    • 透過按“加入”（+）按鈕一下來加入群組，並輸入 Active Directory 網域名稱、反斜線，以及群組帳號名稱（例如 ADS\Domain Admins，IL2\Domain Admins)。

    • 透過按“刪除”（-）按鈕一下來刪除列表裡所選的群組。

  11. 按“好”一下。
