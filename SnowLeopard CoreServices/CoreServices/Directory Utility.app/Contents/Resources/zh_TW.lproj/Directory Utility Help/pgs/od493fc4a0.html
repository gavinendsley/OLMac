忽略 LDAP 伺服器指示

有了“目錄工具程式”，您可以指定電腦是否忽略或依照 LDAP 伺服器的指示，以在其他 LDAP 伺服器或複本目錄裡尋找資訊。

伺服器的指示可以幫助電腦尋找資訊，但也會導致登入延遲，或是在電腦必須驗證其他的 LDAP 伺服器指示時，造成其他延遲。

若要指定是否忽略 LDAP 伺服器指示：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按“編輯”一下。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 LDAPv3 並按“編輯”按鈕（/）一下。

  8. 若伺服器配置列表為隱藏狀態，請按“顯示選項”一下。

  9. 在列表中選擇伺服器配置，並按“編輯⋯”一下。

  10. 按“連線”一下然後選擇“忽略服務器的指示”。
