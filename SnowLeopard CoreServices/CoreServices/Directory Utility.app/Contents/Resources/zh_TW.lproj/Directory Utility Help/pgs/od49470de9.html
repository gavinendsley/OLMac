定義自動搜尋規則

透過“目錄工具程式”，您可以將 Mac OS X 電腦的認證與聯絡資訊搜尋規則設定為自動定義。

自動定義的搜尋規則包含本機目錄網域。它還可包含 DHCP 服務所指定的 LDAP 目錄服務器。

認證與聯絡資訊搜尋規則都是以自動定義的方式作為其預設的搜尋規則。

注意：某些應用程式（如 Mac OS X 的 Mail 和“通訊錄”）能直接取用 LDAP 目錄，而不需使用 Open Directory。 若要設定這些應用程式直接存取 LDAP 目錄，請打開應用程式並設定正確的偏好設定。
重要事項：若您設定 Mac OS X 使用自動的認證搜尋規則和 DHCP 提供的 LDAP 伺服器或 DHCP 提供的共享目錄網域，將增加惡意使用者控制您電腦的風險。若您的電腦設定為連接到無線網路，這種風險甚至會更高。如需更多資訊，請參閱「保護電腦不受惡意 DHCP 伺服器的侵害」。
若要自動定義搜尋規則：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“搜尋規則”一下並選擇搜尋規則：

    • 認證：會顯示用於認證和其他大部分的管理資料的搜尋規則。

    • 聯絡資訊：會顯示應用程式（如“通訊錄”）所使用的聯絡資訊。

  7. 從“搜尋”彈出式選單中選擇“自動”，然後按“套用”一下。

  8. 在“系統偏好設定”中，確定電腦的“網路”偏好設定內的設定為“使用 DHCP”或“使用 DHCP 並手動設定 IP 位址”。
