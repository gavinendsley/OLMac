啟用或停用 Active Directory 服務

您可以使用“目錄工具程式”來啟用或停用由 Windows 伺服器提供的 Active Directory 服務。Active Directory 是 Windows 2000 與其更新版本之伺服器的目錄服務。

若您停用 Active Directory 服務，且 Active Directory 網域是自定搜尋規則的一部分，它們將在“目錄工具程式”的“認證”或“聯絡資訊”面板中以紅色列出。

若要啟用或停用存取 Active Directory：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 選擇或取消選擇 Active Directory 旁邊的註記框，然後按“套用”一下。
