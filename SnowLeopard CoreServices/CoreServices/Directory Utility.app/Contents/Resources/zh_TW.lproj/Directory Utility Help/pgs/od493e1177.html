編輯 Active Directory 的使用者帳號和其他記錄

您可以使用 Workgroup Manager（工作群組管理程式）來更改 Active Directory 網域裡的使用者帳號、群組帳號、電腦列表以及其他記錄。您也可以使用 Workgroup 管理程式來刪除 Active Directory 網域裡的記錄。

如果已延伸 Active Directory 描述語言來納入標準的 Mac OS X 記錄類型（物件類別）和屬性，您可以使用 Workgroup Manager（工作群組管理程式）在 Active Directory 網域中建立和編輯電腦群組。

如需其他資訊，請瀏覽「使用者管理概述」。

若要在 Active Directory 網域裡建立使用者或群組帳號，請在一台 Windows 伺服器管理電腦上使用 Microsoft Active Directory 管理工具。
