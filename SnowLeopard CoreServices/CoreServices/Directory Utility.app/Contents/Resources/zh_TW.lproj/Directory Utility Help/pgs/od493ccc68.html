從 Active Directory 伺服器解除綁定

若使用“目錄工具程式”的 Active Directory 連接器將電腦綁定到 Active Directory 伺服器上，您可以解除電腦與 Active Directory 伺服器之間的綁定。

若電腦無法聯絡伺服器或者電腦記錄已從該伺服器上移除，您可以強制解除該電腦的綁定。

若要解除電腦與 Active Directory 伺服器之間的綁定：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按“編輯”一下。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 Active Directory 並按“編輯”按鈕（/）一下。

  8. 按“解除綁定”一下，然後以具有中止 Active Directory 網域連線權限的使用者身份進行認證，並按“好”一下。

    若您看到提示說明憑證無法接受或電腦無法聯絡 Active Directory，請按“強制解除綁定”一下來強制中斷連線。

    若您強制解除綁定，Active Directory 仍會保留該電腦的記錄。請通知 Active Directory 管理者來移除該電腦記錄。

  9. 在“服務”面板裡，取消選取 Active Directory“啟用”設定，然後按“套用”一下。
