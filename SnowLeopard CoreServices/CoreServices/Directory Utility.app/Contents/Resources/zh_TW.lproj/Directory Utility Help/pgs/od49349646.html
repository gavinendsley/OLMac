使用進階搜尋規則設定

“目錄工具程式”會定義下列搜尋規則：

  • 認證：Mac OS X 會使用認證搜尋規則，從目錄網域裡搜尋並取得使用者認證資訊與其他管理資料。

  • 聯絡資訊：Mac OS X 會使用聯絡資訊搜尋規則，從目錄網域中搜尋並取得名稱、位址及其他聯絡資訊。Mac OS X 的“通訊錄”便會使用此聯絡資訊。其他應用程式也會被設計使用此筆資訊。

    每個搜尋規則由目錄網域的列表所組成。列表中的目錄網域順序定義了搜尋規則。從列表最上方開始，Mac OS X 會搜尋每個列出的目錄網域直到找到所需資訊，若沒有找到所需資訊，將會向下尋找資訊直到列表末端為止。

    認證與聯絡資訊的搜尋規則可含有以下任一設定：

  • 自動模式：從本機目錄網域開始，可以包含 DHCP 所提供的 LDAP 目錄，與電腦所連結的目錄網域。這是 Mac OS X v10.2 或以上版本的預設設定，並可為行動電腦提供最大的彈性。

  • 本機目錄：僅包含本機目錄網域。

  • 自定路徑：從本機目錄網域開始，並包含您所選擇的 LDAP 目錄、Active Directory 網域、共用目錄網域、BSD 設定檔案，與 NIS 網域。

    /BSD/local 檔案夾一律包含在搜尋路徑之內，並且一律顯示為灰色。

重要事項：若您將 Mac OS X 設定為使用自動認證搜尋規則和 DHCP 提供的 LDAP 伺服器，將會增加惡意使用者控制您電腦的風險。若您的電腦設定為連接到無線網路，這種風險甚至會更高。如需更多資訊，請參閱「保護電腦不受惡意 DHCP 伺服器的侵害」。
