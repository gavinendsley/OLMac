設定 Active Directory 網域存取

使用“目錄工具程式”中列出的 Active Directory 連接器，您可以在 Windows 伺服器上的 Active Directory 網域中設定 Mac OS X 取用基本的使用者帳號資訊。

Active Directory 連接器會產生 Mac OS X 認證所需的所有屬性。您不需要對 Active Directory 概要做出任何更動。

若 Active Directory 描述語言已經延伸為包含標準的 Mac OS X 記錄類型和屬性（例如：執行 Mac OS X 用戶端管理所需的屬性），Active Directory 連接器便會偵測並取用這些屬性。

警告：透過 Active Directory 連接器的進階選項，您可對應至 Mac OS X 使用者獨有 ID（UID）、主要群組 ID（GID），並將群組 GID 屬性對應至已加入 Active Directory 描述語言的正確屬性。日後如果您修改這些對應選項的設定值，使用者可能會無法存取之前所製作的檔案。
重要事項：若您的電腦名稱包含連字號，您可能無法加入或綁定到“目錄網域”（如 LDAP 或 Active Directory）。若要建立綁定，請使用不含連字號的電腦名稱。
若要設定存取 Active Directory 網域：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 Active Directory 並按“編輯”按鈕（/）一下。

  8. 請將 Active Directory 網域的 DNS 名稱輸入到您正在設定綁定的電腦上。

    Active Directory 網域管理者會告訴您要輸入的 DNS 名稱。

  9. 如有必要，編輯“電腦識別碼”。

    “電腦識別碼”是在 Active Directory 網域中用來識別電腦的名稱，它會預設為電腦的名稱。 您可能需要將它更改為符合您的公司所制定的 Active Directory 網域電腦命名規則的名稱。 如果您不確定，請詢問 Active Directory 網域管理者。

  10. （可留空）設定進階選項。

    如果進階選項隱藏了，按“顯示進階選項”一下並設定“使用者經驗”、“對應”和“管理”面板上的選項。您也可以稍後再更改進階選項設定。

    如需更多進階選項的相關資訊，請參閱：

    • 設定 Active Directory 裡的行動使用者帳號

    • 設定 Active Directory 使用者帳號的個人專屬檔案夾

    • 設定 Active Directory 使用者帳號的 UNIX Shell

    • 將 UID 對應至 Active Directory 屬性

    • 將主要群組 ID 對應至 Active Directory 屬性

    • 將群組帳號裡的群組 ID 對應至 Active Directory 屬性

    • 指定偏好的 Active Directory 伺服器

    • 更改可以管理電腦的 Active Directory 群組

    • 從 Active Directory 樹系的所有網域控制認證

  11. 按“綁定⋯”一下，使用下列內容將您自己認證為有權將電腦綁定到 Active Directory 網域的使用者，選擇您想要讓 Active Directory 加入的搜尋規則（請見下方），然後按“好”一下：

    • 使用者名稱和密碼：請輸入 Active Directory 使用者帳號的名稱和密碼，或是 Active Directory 網域管理者需要提供您名稱與密碼，以便進行認證。

    • 電腦 OU：請輸入您正在設定的電腦所屬的組織單位（OU）。

    • 用於認證：使用此設定來決定是否將 Active Directory 加入電腦的認證搜尋規則裡。

    • 用於聯絡資訊：使用此設定來決定是否將 Active Directory 加入電腦的聯絡資訊搜尋規則。

    當您按“好”一下，“目錄工具程式”會在您設定的電腦和 Active Directory 伺服器之間建立信任綁定。電腦的搜尋規則是依照認證時所選的選項來進行設定，且 Active Directory 已在“目錄工具程式”的“服務”面板中啟用。

    依照 Active Directory 進階選項的預設設定，如果您已選擇“用於認證”或“用於聯絡資訊”，Active Directory 樹系會增加到電腦的認證搜尋規則和聯絡資訊搜尋規則。

    然而，若在按一下“綁定⋯”之前先取消選取“管理”進階選項面板中的“允許從樹系裡的任何網域認證”，則會加入最鄰近的 Active Directory 網域而非樹系。

    您可以稍後再透過增加或移除 Active Directory 樹系或個人網域來更改搜尋規則。如需更多資訊，請查閱「定義自定搜尋規則」。

  12. （可留空）將伺服器加入 Active Directory Kerberos 領域：

    • 在伺服器或可以連接伺服器的管理者電腦上，請打開“伺服器管理”並為伺服器選擇 Open Directory。

    • 請按“設定”一下，然後按“一般”一下。

    • 請按“加入 Kerberos”一下，然後選擇從彈出式選單中選擇“Active Directory Kerberos 領域”，並輸入此伺服器上的本機管理者憑證。
