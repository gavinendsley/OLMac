更改 Root 使用者帳號密碼

您可以使用“目錄工具程式”（位於“帳號”偏好設定）來更改 root 帳號密碼。當您更改 root 密碼時，請使用複雜的密碼（包含字母、數字和特殊字元）以防止密碼洩漏。

警告：root 帳號是不受限制的管理者帳號，可用來對重要的系統檔案進行更動。即使您以管理者身分登入，您還是必須使用 root 帳號或 sudo 來執行重要的系統任務。

請不要使用 root 帳號來登入電腦（遠端或本機）。或者，使用 sudo 來執行 root 任務。您可以限制對 sudo 的取用，方法是將使用者加入到 /etc/sudoers/ 檔案。

如需更多 root 帳號的相關資訊，請參閱“伺服器管理”輔助說明。如需更多製作密碼的相關資訊，請參閱「編寫密碼」。

若要更改 root 使用者帳號密碼：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

    如果您看到“編輯”按鈕，表示您的電腦至少有一個目錄伺服器的連線。

  4. 按“打開目錄工具程式⋯”一下。

  5. 選擇“編輯”>“更改 Root 密碼”。

  6. 收到提示時，請在“密碼”與“驗證”欄位輸入新的 root 密碼。

  7. 按“好”一下。
