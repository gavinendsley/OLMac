設定 Active Directory 使用者帳號的 UNIX Shell

若電腦被設定為使用“目錄工具程式”的 Active Directory 連接器，您可以設定指令列 shell，以便擁有 Active Directory 帳號的使用者在“終端機”中操作 Mac OS X 時，可以在預設情況下進行使用。

預設 shell 也可以透過“安全 Shell”（SSH）或 Telnet 用於遠端操作。每個使用者可以透過更改“終端機”偏好設定來覆蓋預設的 shell。

若要設定 Active Directory 使用者帳號的 UNIX Shell：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 Active Directory 並按“編輯”按鈕（/）一下。

  8. 若進階選項為隱藏狀態，請按“顯示進階選項”一下。

  9. 按“使用者經驗”一下。

  10. 選擇“預設”的使用者 shell，然後輸入預設使用者的 shell 路徑。

  11. 按“好”一下。
