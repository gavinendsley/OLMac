停止與 LDAP 目錄的信任綁定

您可以使用“目錄工具程式”來停止電腦和 LDAP 目錄（允許但不要求信任綁定）之間的信任綁定。

若要停止與 LDAP 目錄的信任綁定：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按“編輯”一下。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 LDAPv3，然後按“編輯”按鈕（/）一下。

  8. 若伺服器配置列表為隱藏狀態，請按“顯示選項”一下。

  9. 請選擇想要的伺服器配置，然後按“編輯”一下。

  10. 按“移除綁定”一下，然後輸入下列憑證，再按“好”一下。

    輸入 LDAP 目錄的管理者（而非本機電腦的管理者）的名稱和密碼。

    如果您尚未在這部電腦上設定信任綁定，“解除綁定”按鈕就不會顯示。

    若您看見一則提示，告訴您電腦無法聯絡 LDAP 伺服器，您可以按“好”一下來強制停止信任綁定。

    如果您強制停止信任綁定，電腦仍會在 LDAP 目錄中保留一份電腦記錄。請通知 LDAP 目錄的管理者，讓他瞭解並從電腦群組裡移除電腦。

  11. 若要結束停止信任綁定，請按“好”一下。
