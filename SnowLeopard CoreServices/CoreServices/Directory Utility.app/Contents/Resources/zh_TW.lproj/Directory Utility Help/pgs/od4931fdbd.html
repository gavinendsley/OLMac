移除目錄伺服器連線

在從“帳號”偏好設定移除目錄伺服器前，請確定您並未在其他應用程式中使用其服務。

例如，若 Mail 被設定為使用目錄伺服器來搜尋聯絡人，而您刪除了目錄伺服器的話，則您不能再搜尋位於目錄伺服器的任何人。

若要刪除目錄伺服器：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按“編輯”一下。

  4. 從目錄伺服器列表中，選擇您想要刪除的目錄伺服器。

  5. 按“刪除”（–）按鈕一下。

  6. 若您確定已選擇正確的目錄伺服器，請按“停止使用伺服器”。

如果使用者中斷與目錄伺服器的連線，目錄管理者必須手動從目錄伺服器移除電腦記錄。使用者還可以請求網路管理者使用“目錄工具程式”和具有目錄管理權限的帳號來中斷其電腦與目錄伺服器的連線，這樣可從目錄伺服器移除電腦記錄。
