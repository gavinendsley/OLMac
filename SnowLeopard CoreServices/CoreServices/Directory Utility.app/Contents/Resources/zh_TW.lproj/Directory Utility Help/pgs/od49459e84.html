設定遠端伺服器上的目錄工具程式

您可以使用電腦上的“目錄工具程式”來設定和管理遠端伺服器上的 Mac OS X Server 如何取用目錄服務。

若要在遠端伺服器上設定目錄服務：

  1. 打開電腦上的“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

    如果您看到“編輯”按鈕，表示您的電腦至少有一個目錄伺服器的連線。

  4. 按“打開目錄工具程式⋯”一下。

  5. 從“檔案”選單中選擇“連接”。

  6. 請為想要設定的伺服器輸入下列連線與認證資訊。

    • 位址：為您要設定的伺服器輸入 DNS 名稱或 IP 位址。

    • 使用者名稱：為伺服器上的管理者輸入使用者名稱。

    • 密碼：為使用者輸入密碼。

  7. 請按“連接”一下。

  8. 按一下“服務”和“搜尋規則”標籤頁並根據需要更改設定。

    您所做的更動會影響您在上述步驟中所連接的遠端伺服器。

  9. 請從電腦上的“檔案”選單中選擇“中斷連線”。
