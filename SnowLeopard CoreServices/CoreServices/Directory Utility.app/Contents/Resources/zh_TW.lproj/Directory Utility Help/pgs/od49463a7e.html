設定 BSD 配置檔的資料

如果您想讓 Mac OS X 電腦從 BSD 配置檔取得管理資料，設定檔中必須含有資料，且須為 Mac OS X 所要求的格式。

您可能需要新增、更改或重新組織設定檔中的資料。Workgroup Manager（工作群組管理程式）無法更改 BSD 配置檔中的資料，因此您必須使用文字編輯器或其他工具來進行必要的修改。

下表列出檔案的名稱並對各檔案的內容進行了描述。

BSD 配置檔
包含

/etc/master.passwd

使用者名稱、密碼、ID、主要群組 ID 等等

/etc/group

群組姓名、ID 和成員

/etc/fstab

NFS 裝載

/etc/hosts

電腦名稱和位址

/etc/networks

網路名稱和位址

/etc/services

服務姓名、傳輸埠和通訊協定

/etc/protocols

IP 通訊協定名稱和編號

/etc/rpcs

Open Network Computing RPC 伺服器

/etc/printcap

印表機名稱和功能

/etc/bootparams

Bootparam 設定

/etc/bootp

Bootp 設定

/etc/aliases

電子郵件別名和發送列表

/etc/netgroup

完整網路的群組名稱和成員
