刪除 LDAP 目錄的存取配置

您可以使用“目錄工具程式”來刪除配置，此配置指定了電腦存取 LDAPv3 或 LDAPv2 目錄的方式。

如果 LDAP 配置是由 DHCP 所提供，則無法進行更改，因此此配置選項在 LDAP 配置列表裡是變暗的。

若要刪除 LDAP 目錄的存取配置：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按“編輯”一下。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 LDAPv3 並按“編輯”按鈕（/）一下。

  8. 若伺服器配置列表為隱藏狀態，請按“顯示選項”一下。

  9. 請選擇列表中的伺服器設定並按“刪除”一下，然後按“好”一下。

  10. 請從以下項目選取：

    • 如果您看到一則提示，告知您電腦已綁定到 LDAP 目錄，而且您想要停止信任綁定，請按“好”一下，然後輸入 LDAP 目錄管理者（而非本機電腦管理者）的名稱和密碼。

    • 若您看見一則提示，告訴您電腦無法聯絡 LDAP 伺服器，您可以按“好”一下來強制停止信任綁定。

      如果您強制停止信任綁定，電腦仍會在 LDAP 目錄中保留一份電腦記錄。請通知 LDAP 目錄的管理者，讓他瞭解並從電腦群組裡移除電腦。

已刪除的配置會從認證和聯絡資訊的自定搜尋規則裡移除。
