指定偏好的 Active Directory 伺服器

若電腦被設定為使用“目錄工具程式”的 Active Directory 連接器，您可以指定讓電腦依照預設存取的 Active Directory 網域伺服器的 DNS 名稱。

如果伺服器之後無法使用，Active Directory 連接器會回復至樹系中鄰近的其他伺服器。

若取消選擇此選項，Active Directory 連接器會選用樹系中最鄰近的 Active Directory 網域。

若要指定讓 Active Directory 連接器優先連接的伺服器：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按“編輯”一下。

  4. 按“打開目錄工具程式⋯”一下。

  5. 按“服務”一下。

  6. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  7. 在服務列表中，請選擇 Active Directory 並按“編輯”按鈕（/）一下。

  8. 若進階選項為隱藏狀態，請按“顯示進階選項”一下。

  9. 按“管理”一下。

  10. 選擇“優先使用此網域伺服器”，並輸入 Active Directory 伺服器的 DNS 名稱。

  11. 按“好”一下。
