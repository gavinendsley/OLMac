設定 Active Directory 裡的行動使用者帳號

在設定為使用“目錄工具程式”之 Active Directory 連接器的電腦上，您可以啟用或停用行動式 Active Directory 使用者帳號。在電腦尚未連接至 Active Directory 伺服器時，擁有行動帳號的使用者可以使用他們的 Active Directory 憑證來進行登入。

若使用者在電腦連接到 Active Directory 網域時登入，Active Directory 連接器會快取使用者的行動帳號憑證。此憑證快取功能無須更改 Active Directory 描述語言。

如果 Active Directory 的描述語言已經延伸為包含 Mac OS X 所管理的用戶端屬性，則將會使用行動帳號設定，而非 Active Directory 連接器的行動帳號設定。

您可以自動製作行動帳號，或者您可以要求 Active Directory 使用者確認行動帳號的製作。

若要在 Active Directory 網域啟用或停用行動帳號：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 Active Directory 並按“編輯”按鈕（/）一下。

  8. 若進階選項為隱藏狀態，請按“顯示進階選項”一下。

  9. 按“使用者經驗”一下，然後按“登入時建立行動帳號”一下並按需要選擇“在建立行動帳號前需要先確認”。

    • 如果這兩個選項您都選了，則每個使用者在登入時都要決定是否建立行動帳號。在使用者以 Active Directory 使用者帳號登入 Mac OS X 或以網路使用者進行登入時，便會看到立即建立行動帳號的控制選項對話框。

    • 如果您選擇第一個選項而沒有選擇第二個選項，在使用者登入時會建立行動帳號。

    • 若沒有選擇第一個選項，則會停用第二個選項。

  10. 按“好”一下。
