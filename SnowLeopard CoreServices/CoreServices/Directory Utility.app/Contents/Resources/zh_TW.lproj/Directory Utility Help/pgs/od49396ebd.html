定義本機目錄搜尋規則

透過“目錄工具程式”，您可以將 Mac OS X 電腦的認證與聯絡資訊設定為僅使用電腦的本機目錄網域。

若搜尋規則僅使用本機目錄，將會限制電腦用來認證資訊與存取其他管理資料的權限。

若您將電腦的認證搜尋規則限制於僅使用本機目錄，則只有擁有本機帳號的使用者才能登入。

若要令搜尋規則僅使用本機目錄網域（本機目錄）：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“搜尋規則”一下並選擇搜尋規則：

    • 認證：會顯示用於認證和其他大部分的管理資料的搜尋規則。

    • 聯絡資訊：會顯示應用程式（如“通訊錄”）所使用的聯絡資訊。

  7. 從“搜尋”彈出式選單中選擇“本機目錄”，然後按“套用”一下。
