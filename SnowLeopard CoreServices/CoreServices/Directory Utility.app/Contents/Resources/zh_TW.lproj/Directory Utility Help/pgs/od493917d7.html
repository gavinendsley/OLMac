為 Mac OS X 準備唯讀 LDAP 目錄

若您想讓 Mac OS X 電腦從唯讀 LDAP 目錄取得管理性資料，資料必須使用 Mac OS X 所要求的格式。您也許需要新增、更改、或重新管理唯讀 LDAP 目錄中的資料。

因為 Mac OS X 無法將資料寫入唯讀目錄，您必須使用其他工具來進行更改。這些工具必須位於管理唯讀 LDAP 目錄的伺服器上。

若要為 Mac OS X 準備唯讀 LDAP 目錄：

  1. 請前往管理唯讀 LDAP 目錄的伺服器，並設定使其支援以 LDAP 為基準的認證與密碼檢查功能。

  2. 請按需要更改 LDAP 目錄的物件類型和屬性，以使其提供 Mac OS X 所需的資料。

如需 Mac OS X 目錄服務所需資料的技術規格，請參閱“Open Directory 管理”。
