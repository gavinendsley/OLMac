指定 NIS 設定

使用“目錄工具程式”，您可以製作配置來指定 Mac OS X 連接“網路資訊服務”（NIS）網域的方式。

若要建立存取 NIS 網域的配置：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，選擇“BSD 扁平式檔案和 NIS”，然後按“編輯”（/）按鈕一下。

  8. 輸入 NIS 網域所在伺服器的 NIS 網域名稱、DNS 名稱或 IP 位址。

    若有安全性的需要，或是若伺服器與您正在設定的電腦不在同一子網路內，請填入 NIS 伺服器的主機名稱或 IP 位址。

    若您未指定伺服器，NIS 會使用廣播通訊協定來搜尋相同子網路下的 NIS 伺服器。

  9. 選擇“使用 NIS 網域來認證”，然後按“好”一下。

    NIS 網域會以 /BSD/domain 加入到電腦的認證搜尋規則，其中 domain 即是您在第四步所輸入的名稱。
