定義自定搜尋規則

透過“目錄工具程式”，您可以將 Mac OS X 電腦的認證與聯絡資訊設定為使用目錄網域的自定列表。

自定列表是從本機目錄網域開始，可以包含 Open Directory（和其他 LDAP 目錄）、Active Directory 網域、共享目錄網域、BSD 配置檔，與 NIS 網域。

若無法使用電腦自定搜尋規則裡的指定的目錄網域，則啟動電腦時會較緩慢。

若要為搜尋規則指定自定的目錄網域列表：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“搜尋規則”一下並選擇搜尋規則。

    • 認證：會顯示用於認證和其他大部分的管理資料的搜尋規則。

    • 聯絡資訊：會顯示應用程式（如“通訊錄”）所使用的聯絡資訊。

  7. 從“搜尋”彈出式選單裡選擇“自定路徑”。

  8. 根據需要按“加入”一下來加入目錄網域，選擇目錄，然後再按“加入”一下。

  9. 按您的需要來透過在列表內上下拖移目錄網域來更改它們的順序。

  10. 若要移除您不想要包含在搜尋規則內所列出的目錄網域，請選擇它們，然後按一下“刪除”（-）按鈕。

  11. 按“好”一下來確認移除動作，然後按“套用”一下。

若您想要加入某個沒有列在可存取目錄的目錄，請確定電腦設定為存取目錄。如需更多資訊，請參閱：

  • 使用進階目錄服務設定

  • 使用進階 LDAP 服務設定

  • 使用進階 Active Directory 服務設定

  • 指定 NIS 設定

  • 指定 BSD 配置檔的設定
