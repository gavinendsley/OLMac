對 LDAP 目錄設定信任綁定

您可以使用“目錄工具程式”，在電腦和支援信任綁定的 LDAP 目錄之間設定信任綁定。 綁定是藉由已授權的電腦記錄（當您設定信任綁定時在目錄裡建立的）來相互授權的。

電腦無法被設定為使用信任的 LDAP 綁定和 DHCP 提供的 LDAP 目錄。信任的 LDAP 綁定在繼承上是靜態的，但是 DHCP 提供的 LDAP 是動態的。

若要對 LDAP 目錄設定信任綁定：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 LDAPv3 並按“編輯”按鈕（/）一下。

  8. 若伺服器配置列表為隱藏狀態，請按“顯示選項”一下。

  9. 請選擇想要的伺服器設定，並按“編輯”一下。

  10. 按“綁定”一下，然後輸入下列憑證，再按“好”一下。

    輸入電腦的名稱，LDAP 目錄網域管理者的名稱和密碼。其他電腦無法使用此電腦名稱來取得信任綁定或其他網路服務。

    若“綁定”按鈕沒有出現，則表示 LDAP 目錄無法支援信任綁定。

  11. 請確認您提供正確的電腦名稱。

    如果您看見警告訊息告知已存在一份電腦記錄，請按“取消”並返回上一步來更改電腦名稱，或是按“覆寫”來取代已存在的電腦記錄。

    現有的電腦記錄可能已經被丟棄，或它是屬於另外一台電腦。若您要取代現有的電腦記錄，請通知 LDAP 目錄管理者以防取代電腦記錄會停用其他電腦。

    在此狀況下，LDAP 目錄管理者必須為停用的電腦提供另一個名稱，並使用那部電腦的另一個名稱將其加入所屬的電腦群組。

  12. 若要完成設定信任綁定，請按“好”一下。
