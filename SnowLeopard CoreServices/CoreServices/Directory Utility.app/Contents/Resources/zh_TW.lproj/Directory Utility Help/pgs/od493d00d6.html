將主要群組 ID 對應至 Active Directory 屬性

若電腦被設定為使用“目錄工具程式”的 Active Directory 連接器，您可以指定 Active Directory 屬性來對應至 Mac OS X 使用者帳號中的主要群組 ID（GID）屬性。

這步驟通常需要延伸 Active Directory 描述語言，以加入對應主要 GID 的適用屬性。

  • 如果 Active Directory 管理者透過安裝 Microsoft 的“UNIX 服務”來延伸 Active Directory 描述語言，您可以將主要 GID 對應至 msSFU-30-Gid-Number 屬性。

  • 如果 Active Directory 管理者手動延伸 Active Directory 描述語言來包含 RFC 2307 屬性，您可以將主要 GID 對應至 gidNumber。

  • 如果 Active Directory 管理者手動延伸 Active Directory 描述語言來納入 Mac OS X PrimaryGroupID 屬性，您可以將主要 UID 和它對應。

若已停用主要 GID 對應的選項，Active Directory 連接器會根據 Active Directory 的標準 GUID 屬性來產生主要 GID 資料。

警告：如果您以後修改主要 GID 對應檔的設定值，使用者可能會無法存取之前所製作的檔案。
若要將主要 GID 對應至手動延伸之 Active Directory 描述語言中的屬性：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按“編輯”一下。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 Active Directory 並按“編輯”按鈕（/）一下。

  8. 若進階選項為隱藏狀態，請按“顯示進階選項”一下。

  9. 按“對應”一下。

  10. 選擇“將使用者 GID 對應至屬性”，並輸入您想要在使用者帳號裡對應至主要群組 ID 的 Active Directory 屬性名稱。

  11. 按“好”一下。
