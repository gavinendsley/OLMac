顯示或隱藏 LDAP 伺服器設定

您可以設定電腦顯示或隱藏存取 LDAP 目錄可用的設定列表。每種設定都會指定 Open Directory 要以何種方式來存取 LDAP 目錄。當您看到列表時，便可以對每一個沒有變暗的 LDAP 配置更改設定。

若要顯示或隱藏可用的 LDAP 目錄設定：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

  4. 按“打開目錄工具程式⋯”一下。

  5. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  6. 按“服務”一下。

  7. 在服務列表中，請選擇 LDAPv3 並按“編輯”按鈕（/）一下。

  8. 按“顯示選項”或“隱藏選項”控制鈕一下來顯示或隱藏列表。
