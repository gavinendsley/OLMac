瞭解目錄工具程式

“目錄工具程式”決定 Mac OS X 電腦會使用哪些目錄服務，以及電腦要如何連接到指定的目錄網域。 “目錄工具程式”也定義了在指定目錄網域裡尋找認證和聯絡資訊的搜尋規則。

設定目錄工具程式

您可以使用電腦的“目錄工具程式”來設定和管理電腦如何存取目錄服務。您還可以使用“目錄工具程式”來遠端設定電腦如何存取目錄服務。“目錄工具程式”還可以用來啟用 root 帳號或更改 root 帳號密碼。

設定存取服務

您可以使用“目錄工具程式”來啟用和停用 Mac OS X 可以在網路上存取的不同種類的服務。其中包括提供存取使用者帳號，以及存取其他儲存在目錄網域裡的管理資料的目錄服務。

設定存取目錄

透過“目錄工具程式”，您可以設定電腦在以下指定的位址存取使用者帳號和其他管理性資訊：LDAP 目錄、Active Directory 網域和 NIS 網域。

設定搜尋規則

透過“目錄工具程式”，您可以定義 Mac OS X 使用的認證搜尋規則，Mac OS X 會使用這些規則來從目錄網域裡尋找和檢索使用者認證資訊和其他管理性資訊。透過“目錄工具程式”，您還可以定義 Mac OS X 使用的聯絡人搜尋規則，Mac OS X 會使用這些規則來從目錄網域裡尋找和檢索使名稱、位址和其他聯絡資訊。
