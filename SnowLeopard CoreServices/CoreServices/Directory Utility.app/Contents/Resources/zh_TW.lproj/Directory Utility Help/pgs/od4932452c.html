加入 Active Directory 伺服器連線

當您連接到 Active Directory 伺服器時，您必須知道伺服器名稱或 IP 位址，以及 Active Directory 管理者的使用者名稱和密碼。

若要加入 Active Directory 伺服器：

  1. 打開“系統偏好設定”並按“帳號”一下。

  2. 若鎖頭圖像是鎖定的，請按圖像一下，然後輸入管理者的名稱和密碼來解除鎖定。

  3. 按“登入選項”一下，然後按一下“加入”或“編輯”。

    如果您看到“編輯”按鈕，表示您的電腦至少有一個目錄伺服器的連線。

  4. 按一下“新增”（+）按鈕。

  5. 從“加入新的類型目錄”彈出式選單，選擇“Active Directory”，然後輸入下列資訊：

    • Active Directory 網域：這是 Active Directory 網域的 DNS 名稱（例如，ads.company.com）。

    • 電腦識別碼：請選擇編輯您要讓 Active Directory 用於您伺服器的 ID。這是伺服器的 NetBIOS 名稱。此名稱不應包含 15 個以上的字元，不可使用特殊字元及標點符號。若要實際使用，請讓伺服器名稱與其不合格的 DNS 主機名稱相符。例如，若 DNS 伺服器針對您的伺服器有一筆“server.example.com”的項目，請將您的伺服器名稱命名為“server”。

    • AD 管理者名稱與密碼：請輸入 Active Directory 管理者的使用者名稱與密碼。

  6. 按“好”一下。
