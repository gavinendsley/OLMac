關於目錄伺服器連線

您可以使用“帳號”偏好設定來將電腦連接到目錄伺服器。您可以按一下“帳號”偏好設定中“登入選項”面板的“編輯”來檢視電腦所連接的目錄伺服器列表。您的 Mac OS X 電腦連接列表中的伺服器的使用者資訊和其他管理性資料，是儲存在目錄伺服器的目錄網域中。

當您在“目錄伺服器”列表中加入或刪除伺服器時，與目錄伺服器相關的項目也會從“服務”、“認證”和“聯絡人”列表中增加或刪除。然而，若您刪除“服務”、“認證”和“聯絡人”列表中的相關項目，目錄伺服器並不會從“目錄伺服器”列表中移除。

Mac OS X v10.6 電腦可以連接到 Open Directory、Active Directory 或 LDAP 目錄伺服器。若您不確定連接的是哪種類型，請詢問您的網路管理者。

重要事項：若您的電腦名稱包含連字號，您可能無法加入或綁定到“目錄網域”（如 LDAP 或 Active Directory）。若要建立綁定，請使用不含連字號的電腦名稱。
