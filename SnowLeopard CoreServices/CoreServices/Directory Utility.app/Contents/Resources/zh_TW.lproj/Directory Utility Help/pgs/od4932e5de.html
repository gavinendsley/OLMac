自動用戶端設定

如果您的 Mac OS X v10.6 電腦已連接到網路但沒有連接到目錄網域，則您的 Mac OS X v10.6 電腦將使用 Bonjour 來搜尋 Mac OS X v10.6 伺服器。

如果它找到 Mac OS X v10.6 伺服器，而其所執行的 Open Directory 主體至少在執行一個可自動設定的服務（例如 AFP、SMB、VPN、Mail、iCal、iChat、通知），並且目錄網域上有使用者帳號與目前登入 Mac OS X v10.6 的電腦使用者簡稱或全名相符，則您的 Mac OS X v10.6 電腦上將出現邀請。

該邀請將提供輔助程式，幫助您設定與目錄伺服器的連線。您可以接受或拒絕連接到伺服器，或者決定稍後再進行連接。

如果您的電腦找到多個 Mac OS X v10.6 目錄伺服器可供您連接，便會在邀請彈出式選單中列出這些伺服器。如果您決定加入伺服器，則輔助程式會開啟並將您的應用程式連接到目錄伺服器正在執行的服務，然後進行設定。

若要使用自動化的輔助程式來連接到目錄網域：

  1. 在出現輔助程式時按“設定服務”一下。

  2. 從“伺服器”彈出式選單中選擇您想要加入的目錄網域。

    您的網路上可能有多個可用的目錄網域。

    如果您要加入的伺服器不在列表內，請在“伺服器”欄位中輸入伺服器名稱或 IP 地址，然後按“好”一下。

  3. 輸入帳號的使用者名稱和密碼，該帳號必須有權將電腦加入您目前正在加入的目錄網域。

    如果您不知道這項資訊，請聯絡您的目錄管理者。

  4. 請輸入使用者帳號的密碼（指出現在“輸入這台電腦上的帳號“使用者名稱”的密碼”）。

    輔助程式會更改伺服器上使用者目錄帳號的密碼，以符合本機電腦帳號的密碼。

  5. 按“繼續”一下。

  6. UpgradeUser 工具完成時，請按“繼續”一下。

  7. 當出現訊息說明服務已設定完成並請求您登出時，請按“登出”一下。

    登入後，即可開始使用新服務。
