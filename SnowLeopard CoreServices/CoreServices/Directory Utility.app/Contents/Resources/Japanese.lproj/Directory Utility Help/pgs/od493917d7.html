Mac OS X のために読み出し専用 LDAP ディレクトリを準備する

Mac OS X コンピュータが読み出し専用 LDAP ディレクトリから管理データを取得するには、それらのデータが Mac OS X の必要とするフォーマットになっていなければなりません。このため、読み出し専用 LDAP ディレクトリ上のデータを追加、変更、または再編成しなければならないことがあります。

Mac OS X では読み出し専用ディレクトリにデータを書き込むことはできないので、変更するために別のツールを使用する必要があります。それらのツールは、読み出し専用 LDAP ディレクトリを運用するサーバ上にある必要があります。

Mac OS X のために読み出し専用 LDAP ディレクトリを準備するには：

  1. 読み出し専用 LDAP ディレクトリを運用するサーバに移動して、LDAP ベースの認証およびパスワード確認に対応するように構成します。

  2. Mac OS X が必要とするデータを渡すことができるように、LDAP ディレクトリのオブジェクトクラスと属性を必要に応じて変更します。

Mac OS X ディレクトリサービスが必要とするデータの仕様については、「Open Directory の管理」を参照してください。
