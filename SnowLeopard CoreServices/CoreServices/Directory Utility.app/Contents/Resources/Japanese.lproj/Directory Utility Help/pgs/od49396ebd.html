ローカルディレクトリ検索方式を定義する

「ディレクトリユーティリティ」を使用すると、Mac OS X コンピュータの認証検索方式とコンタクト検索方式でコンピュータのローカルディレクトリのみが使用されるように構成できます。

ローカルディレクトリしか使用しない検索方式では、認証情報やその他の管理データへのコンピュータによるアクセスが制限されます。

コンピュータの認証検索方式でローカルディレクトリしか使用しないように制限すると、ローカルアカウントのあるユーザしかログインできません。

検索方式がローカルディレクトリドメイン（ローカルディレクトリ）だけを使用するように設定するには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「接続」または「編集」をクリックします。

  4. 「ディレクトリユーティリティを開く」をクリックします。

  5. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  6. 「検索方式」をクリックして、検索方式を選択します：

    • 「認証」：認証およびその他多くの管理データに使用する検索方式が表示されます。

    • 「コンタクト」：「アドレスブック」などのアプリケーションの連絡先情報に使用する検索方式が表示されます。

  7. 「検索」ポップアップメニューから「ローカルディレクトリ」を選択し、「適用」をクリックします。
