Active Directory のユーザアカウントおよびその他のレコードを編集する

「ワークグループマネージャ」を使用して、Active Directory ドメインのユーザアカウント、グループアカウント、コンピュータリスト、およびその他のレコードを変更できます。「ワークグループマネージャ」を使用して、Active Directory ドメインのレコードを削除することもできます。

Mac OS X の標準レコードタイプ（オブジェクトクラス）と属性を含めるように Active Directory スキーマを拡張している場合は、「ワークグループマネージャ」を使用して Active Directory ドメイン内のコンピュータグループを作成および編集できます。

詳しくは、「ユーザ管理の概要」を参照してください。

Active Directory ドメインにユーザアカウントまたはグループアカウントを作成するには、Windows サーバ管理コンピュータ上で Microsoft Active Directory 管理ツールを使用します。
