ホーム
自動化されたクライアント構成

Mac OS X バージョン 10.6 コンピュータがネットワークに接続されていて、ディレクトリドメインに接続されていない場合、Mac OS X バージョン 10.6 コンピュータは Bonjour を使用して Mac OS X バージョン 10.6 サーバを検出します。

1 つ以上の自動構成可能なサービス（AFP、SMB、VPN、Mail、iCal、iChat 通知など）が実行されていて、現在ログインしている Mac OS X バージョン 10.6 コンピュータのユーザのユーザ名またはロングネームと一致するユーザアカウントがディレクトリドメインにある Open Directory のマスターを実行する Mac OS X バージョン 10.6 サーバを、Mac OS X バージョン 10.6 コンピュータが見つけた場合は、案内が Mac OS X バージョン 10.6 コンピュータに表示されます。

案内には、ディレクトリサーバへの接続を設定するアシスタントが用意されています。サーバへの接続を受け入れるか、拒否することができます。または、後で接続することができます。

お使いのコンピュータが、接続可能な複数の Mac OS X バージョン 10.6 ディレクトリサーバを検出した場合は、サーバのリストが案内のポップアップメニューに表示されます。サーバに接続することにした場合は、アシスタントがアプリケーションを開いて、ディレクトリサーバが実行するサービスに接続して構成します。

自動化されたアシスタントを使用してディレクトリドメインに接続するには：

  1. アシスタントが表示されたら、「サービスを設定」をクリックします。

  2. 「サーバ」ポップアップメニューから、接続したいディレクトリドメインを選択します。

    ネットワーク上には複数の使用可能なディレクトリドメインがあることがあります。

    接続したいサーバがリストにない場合は、サーバ名または IP アドレスを「サーバ」フィールドに入力して「OK」をクリックします。

  3. 接続対象のディレクトリドメインにコンピュータを接続する権限のあるアカウントのユーザ名とパスワードを入力します。

    これらの情報が分からない場合は、ディレクトリ管理者に問い合わせてください。

  4. 「このコンピュータ上のアカウント“ユーザ名”のパスワードを入力してください。」と表示されているユーザアカウントのパスワードを入力します。

    アシスタントは、ローカルコンピュータのアカウントのパスワードに合わせてサーバのユーザ・ディレクトリ・アカウントのパスワードを変更します。

  5. 「続ける」をクリックします。

  6. UpgradeUser ツールの処理が完了したら、「続ける」をクリックします。

  7. 設定されたサービスについて説明し、ログアウトするように求めるメッセージが表示されたら、「ログアウト」をクリックします。

    ログインすると、新しいサービスの利用を開始できます。
