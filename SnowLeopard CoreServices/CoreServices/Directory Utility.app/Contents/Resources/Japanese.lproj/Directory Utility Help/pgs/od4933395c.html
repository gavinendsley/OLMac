ディレクトリサーバ接続を編集する

「アカウント」環境設定を使用して、接続しているディレクトリサーバを編集できます。

ディレクトリサーバ接続を編集するには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「編集」をクリックします。

  4. ディレクトリサーバのリストから、編集したいディレクトリサーバを選択します。

  5. 編集（/）ボタンをクリックします。

  6. ディレクトリサーバの設定を変更します。

  7. 「OK」をクリックします。
