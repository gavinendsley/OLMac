Open Directory サーバ接続を追加する

Open Directory サーバを追加するときは、サーバ名または IP アドレスと、サーバが SSL（Secure Sockets Layer）を使用しているかどうかが分かっている必要があります。

Open Directory サーバを追加するには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「接続」または「編集」をクリックします。

    「編集」ボタンが表示されている場合は、コンピュータにディレクトリサーバへの接続が 1 つ以上あります。

  4. 「追加」（＋）ボタンをクリックします。

  5. 「追加する新しいディレクトリタイプ」ポップアップメニューから「Open Directory」を選択します。

  6. 「サーバ名または IP アドレス」フィールドに、サーバ名または IP アドレスを入力します。

  7. （状況に応じて）「SSL を使って暗号化」チェックボックスを選択する前に、SSL が必要かどうかを Open Directory 管理者に確認してください。

重要： ディレクトリサーバに接続しているときに changeip を使って Mac OS X サーバの IP アドレスとコンピュータ名を変更する場合は、新しいコンピュータ名と IP アドレスを使ってディレクトリをアップデートするために、ディレクトリサーバを接続解除してから再接続する必要があります。ディレクトリサーバを接続解除して再接続しない場合は、ディレクトリはアップデートされず、以前のコンピュータ名と IP アドレスが引き続き使用されます。
