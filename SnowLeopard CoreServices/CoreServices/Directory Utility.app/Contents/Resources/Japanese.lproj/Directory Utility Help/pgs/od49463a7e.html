BSD 構成ファイルのデータを設定する

Mac OS X コンピュータが BSD 構成ファイルから管理データを取得するには、Mac OS X の必要とするフォーマットのデータがファイルになければなりません。

ファイルのデータの追加、変更、または再編成が必要な場合があります。「ワークグループマネージャ」を使用して BSD 構成ファイルのデータを変更することはできないので、テキストエディタまたはその他のツールを使って必要な変更を行う必要があります。

次の表に、ファイル名およびそれらの内容をまとめてあります。

BSD 構成ファイル
内容

/etc/master.passwd

ユーザの名前、パスワード、ID、プライマリグループ ID など

/etc/group

グループの名前、ID、およびメンバー

/etc/fstab

NFS マウント

/etc/hosts

コンピュータの名前とアドレス

/etc/networks

ネットワークの名前とアドレス

/etc/services

サービスの名前、ポート、およびプロトコル

/etc/protocols

IP プロトコル名と IP 番号

/etc/rpcs

ONC（Open Network Computing）RPC サーバ

/etc/printcap

プリンタの名前と機能

/etc/bootparams

Bootparam 設定

/etc/bootp

Bootp 設定

/etc/aliases

メールエイリアスと配信リスト

/etc/netgroup

ネットワーク全体のグループの名前とメンバー
