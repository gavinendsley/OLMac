ディレクトリユーティリティについて

「ディレクトリユーティリティ」は、Mac OS X コンピュータが使うディレクトリサービス、および特定のディレクトリドメインに接続する方法を設定します。 また、特定のディレクトリドメインの認証および連絡先情報を検索するための検索方式も定義します。

ディレクトリユーティリティを設定する

お使いのコンピュータで「ディレクトリユーティリティ」を使用して、コンピュータからディレクトリサービスにアクセスする方法を設定および管理できます。「ディレクトリユーティリティ」を使用して、コンピュータからディレクトリサービスにアクセスする方法をリモートから構成することもできます。さらに、ルートアカウントを有効にしたり、ルートアカウントのパスワードを変更したりすることもできます。

サービスへのアクセスを構成する

「ディレクトリユーティリティ」を使用して、Mac OS X でネットワーク上でアクセスできるさまざまな種類のサービスを有効にしたり無効にしたりできます。これらのサービスには、ディレクトリドメインに保管されているユーザアカウントやその他の管理データへのアクセスを提供するディレクトリサービスがあります。

ディレクトリへのアクセスを構成する

「ディレクトリユーティリティ」を使用すれば、特定の LDAP ディレクトリ、Active Directory ドメイン、および NIS ドメインにある、ユーザアカウントやその他の管理情報にアクセスするようにコンピュータを構成できます。

検索方式を設定する

「ディレクトリユーティリティ」を使用して、Mac OS X でディレクトリドメイン内のユーザ認証情報およびその他の管理データを検索および取得するときに使用する、認証検索方式を定義できます。また、Mac OS X では、連絡先情報検索方式を使用して、ディレクトリドメイン内の名前、アドレス、その他の連絡先情報も検索および取得できます。
