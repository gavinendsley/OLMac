リモートサーバのディレクトリユーティリティを設定する

お使いのコンピュータで「ディレクトリユーティリティ」を使用して、リモートサーバ上の Mac OS X Server でのディレクトリサービスへのアクセス方法を設定および管理できます。

リモートサーバのディレクトリアクセスを構成するには：

  1. お使いのコンピュータの「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「接続」または「編集」をクリックします。

    「編集」ボタンが表示されている場合は、コンピュータにディレクトリサーバへの接続が 1 つ以上あります。

  4. 「ディレクトリユーティリティを開く」をクリックします。

  5. 「ファイル」メニューから、「接続」を選択します。

  6. 構成したいサーバの接続情報と認証情報を次のように入力します。

    • 「アドレス」：構成するサーバの DNS 名または IP アドレスを入力します。

    • 「ユーザ名」：サーバ上の管理者のユーザ名を入力します。

    • 「パスワード」：入力したユーザ名のパスワードを入力します。

  7. 「接続」をクリックします。

  8. 「サービス」および「検索方式」タブをクリックして、必要に応じて設定を変更します。

    以上のステップで加えた変更が、接続先のリモートサーバに反映されます。

  9. お使いのコンピュータの「ファイル」メニューから、「接続解除」を選択します。
