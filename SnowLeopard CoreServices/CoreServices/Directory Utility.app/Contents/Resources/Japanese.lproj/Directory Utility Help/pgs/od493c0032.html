LDAP ディレクトリサービスを有効にする／無効にする

「ディレクトリユーティリティ」を使用して、LDAPv2 および LDAPv3 を使用しているディレクトリサービスへのアクセスを有効／無効にできます。「LDAPv3」という「ディレクトリユーティリティ」プラグインは、1 つで LDAP2 と LDAPv3 の両方へのアクセスを提供しています。

Mac OS X Server が提供するディレクトリサービスでは、ほかの多くのサーバと同様に LDAPv3 が使用されます。LDAPv3 は、オープン規格の 1 つで、Macintosh、UNIX、および Windows システムが混在するネットワークでよく使われます。旧バージョンの LDAPv2 を使用してディレクトリサービスを提供するサーバもあります。

LDAP ディレクトリサービスを無効にする場合、LDAP ディレクトリがカスタム検索方式に含まれているときは、「ディレクトリユーティリティ」の「検索方式」の「認証」または「コンタクト」パネルのリストにそれらが赤で表示されます。

LDAP ディレクトリサービスを有効または無効にするには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「接続」または「編集」をクリックします。

  4. 「ディレクトリユーティリティを開く」をクリックします。

  5. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  6. 「サービス」をクリックします。

  7. 「LDAPv3」の横にあるチェックボックスを選択または選択解除して、「適用」をクリックします。
