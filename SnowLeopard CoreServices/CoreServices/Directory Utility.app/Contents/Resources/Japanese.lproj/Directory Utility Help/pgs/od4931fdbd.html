ディレクトリサーバ接続を削除する

「アカウント」環境設定からディレクトリサーバを削除する前に、別のアプリケーションがそのサービスを使用していないことを確認します。

たとえば、ディレクトリサーバを使ってユーザを検索するように「Mail」が構成されているときにディレクトリサーバを削除すると、そのディレクトリサーバ上のユーザを検索できなくなります。

ディレクトリサーバを削除するには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「編集」をクリックします。

  4. ディレクトリサーバのリストから、削除したいディレクトリサーバを選択します。

  5. 削除（－）ボタンをクリックします。

  6. 正しいディレクトリサーバを選択したことを確認してから、「サーバの利用を中止」をクリックします。

ユーザがディレクトリサーバの接続を解除した場合、ディレクトリ管理者はコンピュータレコードをディレクトリサーバから手動で削除する必要があります。また、ネットワーク管理者が「ディレクトリユーティリティ」とディレクトリ管理アクセス権のあるアカウントを使用してユーザのコンピュータをディレクトリサーバから接続解除するようにユーザが要求することもできます。これにより、コンピュータレコードがディレクトリサーバから削除されます。
