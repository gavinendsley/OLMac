LDAP サービスの詳細設定を使用する

Mac OS X Server のサーバや Mac OS X のコンピュータで、Mac OS X Server Open Directory のマスターの LDAP ディレクトリを含む、特定の LDAP ディレクトリにアクセスするように構成できます。

タスクの説明と手順については、以下を参照してください：

  • Mail やアドレスブックで LDAP ディレクトリにアクセスする

  • LDAP サーバの構成を表示する／隠す

  • LDAP ディレクトリへのアクセスを構成する

  • LDAP ディレクトリへのアクセスを手動で構成する

  • LDAP ディレクトリへのアクセスの構成を変更する

  • LDAP ディレクトリへのアクセスの構成を複製する

  • LDAP ディレクトリへのアクセスの構成を削除する

  • LDAP ディレクトリの接続設定を変更する

  • LDAP 接続のセキュリティポリシーを変更する

  • LDAP 検索とマッピングを構成する

  • LDAP ディレクトリの信頼されたバインディングを設定する

  • LDAP ディレクトリとの信頼されたバインディングを終了する

  • LDAP 接続の開く／閉じるのタイムアウトを変更する

  • LDAP 接続のクエリーのタイムアウトを変更する

  • LDAP 接続の再バインド試行の待ち時間を変更する

  • LDAP 接続が待機状態のときに接続を解除するまでの時間を変更する

  • LDAP サーバの紹介を無視する

  • LDAP 接続を認証する

  • LDAP 接続の認証に使用されるパスワードを変更する

  • LDAP ディレクトリの構成レコード属性をマップする

  • RFC 2307 マッピングを編集してユーザの作成を可能にする

  • Mac OS X のために読み出し専用 LDAP ディレクトリを準備する

  • Mac OS X のデータを LDAP ディレクトリに入力する
