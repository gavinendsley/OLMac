Mac OS X のデータを LDAP ディレクトリに入力する

LDAP ディレクトリドメインへのアクセスを構成してデータマッピングを設定した後に、Mac OS X のレコードとデータをそれらに入力することができます。リモート管理（読み出し／書き込みアクセス）を許可している LDAP ディレクトリでは、Mac OS X Server に用意されている「ワークグループマネージャ」を使って、以下のような操作を行うことができます：

  • ユーザのネットワークブラウザ（「Finder」ウインドウのサイドバーの「ネットワーク」をクリックすると表示されます）に自動的にマウントしたい共有ポイントと共有ドメインを指定できます。

    「サーバ管理」の「共有」と「ワークグループマネージャ」の「ネットワーク」モジュールを使用します。

    詳しくは、「ファイルサービスの概要」と「ユーザ管理の概要」を参照してください。

  • ユーザおよびグループレコードを定義して、それらの設定を構成できます。

    「ワークグループマネージャ」の「アカウント」モジュールを使用します。

    詳しくは、「ユーザ管理の概要」を参照してください。

  • 環境設定およびアクセスできるユーザとグループが同じコンピュータのリストを定義できます。

    「ワークグループマネージャ」の「コンピュータ」モジュールを使用します。

    詳しくは、「ユーザ管理の概要」を参照してください。

以上のすべての操作について、ユーザリストの上にある地球のアイコンをクリックし、LDAP ディレクトリドメインを開くために「ワークグループマネージャ」のポップアップメニューから選択します。LDAP ディレクトリがポップアップメニューにない場合は、このメニューから「その他」を選び、LDAP ディレクトリを選択します。

注記： レコードとデータを読み出し LDAP ディレクトリに追加するときは、LDAP ディレクトリを運用するサーバ上のツールを使用する必要があります。
