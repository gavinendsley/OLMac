ディレクトリへのアクセスを構成する

「ディレクトリユーティリティ」（「アカウント」環境設定にあります）を使用すると、Mac OS X のコンピュータや Mac OS X Server のサーバでディレクトリドメインにアクセスする方法を設定したり、管理したりできます。

LDAP ディレクトリにアクセスする

「ディレクトリユーティリティ」を使用すると、Mac OS X のコンピュータや Mac OS X Server のサーバで、Mac OS X Server Open Directory のマスターの LDAP ディレクトリを含む、特定の LDAP ディレクトリにアクセスするように構成できます。詳細については、下のトピックをクリックしてください。
