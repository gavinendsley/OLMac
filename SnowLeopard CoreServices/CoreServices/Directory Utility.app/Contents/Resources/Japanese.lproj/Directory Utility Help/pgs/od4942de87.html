NIS 設定を指定する

「ディレクトリユーティリティ」を使用すると、Mac OS X が NIS（Network Information Service）ドメインにアクセスする方法を指定する構成を作成できます。

NIS ドメインへのアクセスの構成を作成するには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「接続」または「編集」をクリックします。

  4. 「ディレクトリユーティリティを開く」をクリックします。

  5. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  6. 「サービス」をクリックします。

  7. サービスのリストから「BSD フラットファイルおよび NIS」を選択し、編集（/）ボタンをクリックします。

  8. NIS ドメインが常駐するサーバの NIS ドメイン名、DNS 名、または IP アドレスを入力します。

    セキュリティを強化する場合や、構成しているコンピュータと同じサブネットにサーバがない場合は、NIS サーバのホスト名または IP アドレスも含めます。

    サーバを指定しなかった場合は、NIS によりブロードキャストプロトコルを使ってサブネット上の NIS サーバが検出されます。

  9. 「NIS ドメインを認証に使用」を選択して、「OK」をクリックします。

    コンピュータの認証検索方式に、NIS ドメインが「/BSD/ドメイン」として追加されます。「ドメイン」は、手順 4 で入力した名前です。
