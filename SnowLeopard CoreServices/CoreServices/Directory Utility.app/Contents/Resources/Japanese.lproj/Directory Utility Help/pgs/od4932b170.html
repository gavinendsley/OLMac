ディレクトリサーバの接続について

「アカウント」環境設定を使用して、コンピュータをディレクトリサーバに接続できます。 お使いのコンピュータが接続されているディレクトリサーバのリストは、「アカウント」環境設定の「ログインオプション」パネルにある「編集」をクリックして表示できます。Mac OS X コンピュータは、一覧にあるサーバにアクセスして、ディレクトリサーバのディレクトリドメインに保管されているユーザ情報やその他の管理データを取得します。

「ディレクトリサーバ」リストにサーバを追加すると、そのディレクトリサーバに関連する項目が「サービス」、「認証」、および「コンタクト」リストに追加されます。リストからサーバを削除すると、後者のリストから項目が削除されます。ただし、「サービス」、「認証」、および「コンタクト」リストからサーバに関連する項目を削除しても、ディレクトリサーバは「ディレクトリサーバ」リストからは削除されません。

Mac OS X バージョン 10.6 コンピュータは、Open Directory、Active Directory、または LDAP ディレクトリサーバに接続できます。接続するサーバが分からない場合は、ネットワーク管理者にお問い合わせください。

重要： コンピュータ名にハイフンが含まれている場合は、LDAP や Active Directory などのディレクトリドメインに接続およびバインドできないことがあります。バインドを確立するときは、ハイフンを含まないコンピュータ名を使用してください。
