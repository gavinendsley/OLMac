ディレクトリサーバ接続を監視する

「アカウント」環境設定を使って、お使いのコンピュータが接続されているディレクトリサーバの状況を監視できます。この説明は、特定のディレクトリサーバになぜ接続できないかを判断するときに役立ちます。

ディレクトリサーバの状況を監視するには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「編集」をクリックします。

  4. ディレクトリサーバの左側にある状況を示すドットの色を確認します：

    • 緑：ディレクトリサーバがユーティリティに応答しています。

    • 黄：コンピュータがディレクトリサーバからの応答を待機しています。

    • 赤：ディレクトリサーバがユーティリティに応答していません。
