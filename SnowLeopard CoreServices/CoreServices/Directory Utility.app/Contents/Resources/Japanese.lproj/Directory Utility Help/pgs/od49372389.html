Active Directory フォレスト内のすべてのドメインからの認証を制御する

「ディレクトリユーティリティ」の Active Directory コネクタを使用するように構成されたコンピュータで、Active Directory フォレスト内のすべてのドメインのユーザが認証できるようにするか、または認証を個々のドメインのユーザに制限することができます。

フォレスト内のすべてのドメインのユーザに認証を許可するかどうかを制御するには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「編集」をクリックします。

  4. 「ディレクトリユーティリティを開く」をクリックします。

  5. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  6. 「サービス」をクリックします。

  7. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  8. サービスのリストから「Active Directory」を選択し、編集（/）ボタンをクリックします。

  9. 詳細オプションが表示されていない場合は、「詳細オプションを表示」をクリックします。

  10. 「管理」をクリックします。

  11. 「フォレスト内の任意のドメインから認証」を選択します。

    「フォレスト内の任意のドメインから認証」を選択すると、コンピュータの認証および連絡先情報のカスタム検索方式に Active Directory フォレストを追加できます。

    Active Directory フォレストをカスタム検索方式に追加すると、使用可能なディレクトリドメインのリストに、そのフォレストが「/Active Directory/All Domains」として表示されます。（これがデフォルトの設定です。）

    「フォレスト内の任意のドメインから認証」の選択を解除すると、コンピュータの認証および連絡先情報のカスタム検索方式に Active Directory ドメインを個別に追加できます。

    Active Directory ドメインをカスタム検索方式に追加すると、使用可能なディレクトリドメインのリストに、各 Active Directory ドメインが個別に表示されます。

  12. 「OK」をクリックします。

  13. 「フォレスト内の任意のドメインから認証」を選択した後に、Active Directory フォレストまたは選択したドメインを追加するために「認証」パネルと「カスタム」パネルのカスタム検索方式を変更します。

    カスタム検索方式の変更方法について詳しくは、「カスタム検索方式を定義する」を参照してください。
