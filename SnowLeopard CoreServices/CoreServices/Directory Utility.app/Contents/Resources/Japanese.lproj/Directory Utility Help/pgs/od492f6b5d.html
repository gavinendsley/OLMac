ルート・ユーザ・アカウントを有効にする

「ディレクトリユーティリティ」を使用して、ルート・ユーザ・アカウントを有効にできます。ルート・ユーザ・アカウントを有効にする場合は、パスワードが盗用されることがないように、英数字および特殊文字が含まれる複雑なパスワードを使用してください。

警告： ルートアカウントは、重要なシステムファイルを変更するときに使用する、アクセス権に制限のない管理者アカウントです。管理者としてログインしている場合でも、重要なシステムタスクを実行するときは、ルートアカウントまたは sudo を使用する必要があります。

コンピュータにログインするときは、リモートおよびローカルのどちらの場合でも、ルートアカウントを使わないでください。ルートタスクを実行するときは、代わりに sudo を使用します。/etc/sudoers/ ファイルにユーザを追加することで、sudo へのアクセスを制限することができます。

ルートアカウントについて詳しくは、「サーバ管理」のヘルプを参照してください。

ルート・ユーザ・アカウントを有効にするには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「接続」または「編集」をクリックします。

    「編集」ボタンが表示されている場合は、コンピュータにディレクトリサーバへの接続が 1 つ以上あります。

  4. 「ディレクトリユーティリティを開く」をクリックします。

  5. 「編集」＞「ルートユーザを有効」と選択します。
