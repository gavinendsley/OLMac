Active Directory サービスの詳細設定を使用する

Mac OS X Server のサーバや Mac OS X のコンピュータで、Windows 2000 または Windows 2003 サーバ上の Active Directory ドメインにアクセスするように構成できます。

タスクの説明と手順については、以下を参照してください：

  • Active Directory アクセスについて

  • Active Directory ドメインへのアクセスを構成する

  • Active Directory のモバイルユーザのアカウントを設定する

  • Active Directory ユーザアカウントのホームフォルダを設定する

  • Active Directory ユーザアカウントの UNIX シェルを設定する

  • UID を Active Directory 属性にマップする

  • プライマリグループ ID を Active Directory 属性にマップする

  • グループアカウントのグループ ID を Active Directory 属性にマップする

  • 優先する Active Directory サーバを指定する

  • コンピュータを管理できる Active Directory のグループを変更する

  • Active Directory フォレスト内のすべてのドメインからの認証を制御する

  • Active Directory サーバからバインドを解除する

  • Active Directory のユーザアカウントおよびその他のレコードを編集する

ネットワークによっては、Active Directory ドメインにアクセスするための別の方法が重要な場合があります。「Active Directory ドメインへの LDAP アクセスを設定する」を参照してください。
