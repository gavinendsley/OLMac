BSD 構成ファイルの設定を指定する

従来 UNIX コンピュータでは、/etc/master.passwd、/etc/group、/etc/hosts のような構成ファイルに管理データが格納されていました。Mac OS X は、BSD バージョンの UNIX を基にしていますが、通常は、ディレクトリシステムから管理データを取得します。

Mac OS X Server は、固定のセットの BSD 構成ファイルをサポートしています。使用する構成ファイルを指定したり、内容を Mac OS X のレコードタイプや属性にマップすることはできません。

Mac OS X バージョン 10.2 以降（Mac OS X Server バージョン 10.2 以降を含む）では、Open Directory で BSD 構成ファイルから管理データを取り出すことができます。この機能により、BSD 構成ファイルを使用している組織でも、Mac OS X コンピュータで既存のファイルのコピーを使うことができます。BSD 構成ファイルは単独で使用することも、ほかのディレクトリドメインと併用することもできます。

BSD 構成ファイルを使用するには：

  1. BSD 構成ファイルに、Mac OS X ディレクトリサービスに必要なデータが含まれていることを確認します。

    詳しくは、「BSD 構成ファイルのデータを設定する」を参照してください。

  2. 「システム環境設定」を開き、「アカウント」をクリックします。

  3. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  4. 「ログインオプション」をクリックして、「接続」または「編集」をクリックします。

  5. 「ディレクトリユーティリティを開く」をクリックします。

  6. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  7. 「サービス」をクリックします。

  8. サービスのリストから「BSD フラットファイルおよび NIS」を選択し、編集（/）ボタンをクリックします。

  9. 「BSD ローカルノードのユーザおよびグループレコードを使う」を選択して、「OK」をクリックします。

    コンピュータの認証検索方式に、BSD 構成ファイルのドメインが「/BSD/local」として追加されます。
