Active Directory サーバ接続を追加する

Active Directory サーバに接続するときは、サーバ名または IP アドレスと、Active Directory 管理者のユーザ名およびパスワードが分かっている必要があります。

Active Directory サーバを追加するには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「接続」または「編集」をクリックします。

    「編集」ボタンが表示されている場合は、コンピュータにディレクトリサーバへの接続が 1 つ以上あります。

  4. 「追加」（＋）ボタンをクリックします。

  5. 「追加する新しいディレクトリタイプ」ポップアップメニューから「Active Directory」を選択してから、次の情報を入力します：

    • 「Active Directory のドメイン」：Active Directory ドメインの DNS 名です（たとえば、ads.company.com）。

    • 「コンピュータ ID」：必要に応じて、Active Directory でそのサーバのために使用したい ID を編集します。これは、サーバの NetBIOS 名になります。名前は 15 文字以内にする必要があります。特殊文字や句読記号は使用できません。できれば、非修飾の DNS ホスト名と同じ名前にすることをお勧めします。たとえば、サーバのエントリーが DNS サーバに「server.example.com」として登録されている場合は、サーバ名を「server」にします。

    • 「AD 管理者のユーザ名／パスワード」：Active Directory 管理者のユーザ名とパスワードを入力します。

  6. 「OK」をクリックします。
