コンピュータを管理できる Active Directory のグループを変更する

「ディレクトリユーティリティ」の Active Directory コネクタを使用するように構成されたコンピュータで、所属メンバーにコンピュータの管理者権限を与える Active Directory グループアカウントを指定できます。

これらの Active Directory グループアカウントのメンバーであるユーザは、構成している Mac OS X コンピュータにソフトウェアをインストールするなどの管理タスクを実行できます。

管理者権限を割り当てる Active Directory グループアカウントを追加または削除するには：

  1. 「システム環境設定」を開き、「アカウント」をクリックします。

  2. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  3. 「ログインオプション」をクリックして、「編集」をクリックします。

  4. 「ディレクトリユーティリティを開く」をクリックします。

  5. ロックアイコンがロックされている場合は、それをクリックしてロックを解除してから、管理者の名前とパスワードを入力します。

  6. 「サービス」をクリックします。

  7. サービスのリストから「Active Directory」を選択し、編集（/）ボタンをクリックします。

  8. 詳細オプションが表示されていない場合は、「詳細オプションを表示」をクリックします。

  9. 「管理」をクリックします。

  10. 「管理を許可するユーザ」を選択し、管理者権限を割り当てたい Active Directory グループアカウントのリストを変更します：

    • グループを追加するときは、追加ボタン（＋）をクリックし、Active Directory ドメイン名、バックスラッシュ、およびグループアカウント名を入力します（例：ADS\Domain Admins,IL2\Domain Admins）。

    • グループを削除するときは、リストでグループを選択してから、削除ボタン（－）をクリックします。

  11. 「OK」をクリックします。
