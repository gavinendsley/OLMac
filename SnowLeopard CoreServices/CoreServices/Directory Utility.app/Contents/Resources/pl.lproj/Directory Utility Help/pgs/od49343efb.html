Konfigurowanie dostępu do usług katalogowych

Narzędzia katalogowego (dostępnego w preferencjach kont) można użyć do konfigurowania i zarządzania sposobem dostępu do domen katalogowych przez Mac OS X lub serwer z Mac OS X Server.

Dostęp do katalogów LDAP

Używając Narzędzia katalogowego można skonfigurować komputer z Mac OS X lub serwer z Mac OS X Server tak, aby miał dostęp do określonych katalogów LDAP, włączając w to nadrzędny katalog LDAP w Open Directory na Mac OS X Server. Dowiesz się więcej klikając w poniższe tematy.

Dostęp do domeny NIS

Używając Narzędzia katalogowego można utworzyć konfigurację określającą, jak Mac OS X łączy się z domeną NIS. Dowiesz się więcej klikając w poniższy temat.

Używanie plików konfiguracyjnych BSD

Można użyć Open Directory do pobierania danych administracyjnych z plików konfiguracyjnych BSD, takich jak /etc/master.passwd, znanych jako pliki jednorodne BSD. Można też skonfigurować dostęp do plików konfiguracyjnych BSD używając Narzędzia katalogowego. Dowiesz się więcej klikając w poniższe tematy.
