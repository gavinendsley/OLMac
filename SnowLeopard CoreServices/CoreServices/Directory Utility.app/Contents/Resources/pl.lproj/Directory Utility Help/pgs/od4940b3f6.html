Konfigurowanie powłoki UNIX dla kont użytkowników Active Directory

Na komputerze skonfigurowanym do używania wtyczki Active Directory w Narzędziu katalogowym można skonfigurować powłokę linii poleceń, której użytkownicy z kontami Active Directory używać będą jako domyślnej do interakcji z Mac OS X w programie Terminal.

Domyślna powłoka jest również używana do zdalnej interakcji przez bezpieczną powłokę SSH lub Telnet. Każdy użytkownik może zmienić domyślną powłokę zmieniając preferencje Terminalu.

Jak ustawić powłokę UNIX dla kont użytkowników Active Directory?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Przyłącz lub Edycja.

  4. Kliknij w przycisk „Otwórz Narzędzie katalogowe”.

  5. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  6. Kliknij w Usługi.

  7. Na liście usług, zaznacz Active Directory i kliknij w przycisk Edycja (/).

  8. Jeśli ukryte są opcje zaawansowane, kliknij w Pokaż opcje zaawansowane.

  9. Kliknij w Użytkownik.

  10. Zaznacz „Domyślna powłoka użytkownika”, po czym wpisz ścieżkę domyślnej powłoki użytkownika.

  11. Następnie kliknij w przycisk OK.
