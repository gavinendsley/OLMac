Odwzorowanie UID na atrybut Active Directory

Na komputerze skonfigurowanym do używania wtyczki Active Directory w programie „Narzędzie katalogowe” można określić atrybut Active Directory, który ma być odwzorowany na atrybut unikalnego ID użytkownika (UID) Mac OS X.

Zwykle, schemat Active Directory musi być rozszerzony tak, aby zawierał odpowiedni atrybut odwzorowania UID:

  • Jeżeli administrator Active Directory rozszerzył schemat Active Directory, instalując pakiet usług Microsoft dla UNIX, można odwzorować UID na atrybut msSFU-30-Uid-Number.

  • Jeżeli administrator Active Directory ręcznie rozszerzył schemat Active Directory tak, żeby zawierał atrybuty RFC 2307, można odwzorować UID na uidNumber.

  • Jeżeli administrator Active Directory ręcznie rozszerzył schemat Active Directory tak, aby zawierał atrybut UniqueID Mac OS X, może na niego odwzorować UID.

Jeżeli odwzorowywanie UID jest wyłączone, wtyczka Active Directory generuje UID w oparciu o standardowy atrybut GUID z Active Directory.

UWAGA: Jeśli później zmienimy odwzorowanie UID, użytkownicy mogą utracić dostęp do poprzednio utworzonych plików.
Jak odwzorować UID na atrybut w rozszerzonym schemacie Active Directory?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Edycja.

  4. Kliknij w przycisk „Otwórz Narzędzie katalogowe”.

  5. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  6. Kliknij w Usługi.

  7. Na liście usług, zaznacz Active Directory i kliknij w przycisk Edycja (/).

  8. Jeśli ukryte są opcje zaawansowane, kliknij w Pokaż opcje zaawansowane.

  9. Kliknij w Odwzorowania.

  10. Zaznacz „Odwzorowuj UID na atrybut“ i wpisz nazwę atrybutu Active Directory, na który chcesz odwzorować UID.

  11. Następnie kliknij w przycisk OK.
