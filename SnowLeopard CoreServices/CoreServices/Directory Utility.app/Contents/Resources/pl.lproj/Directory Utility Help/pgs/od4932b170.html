Połączenia z serwerem katalogowym

Można użyć preferencji kont do podłączenia komputerów do serwerów katalogowych. Listę serwerów katalogowych, z którymi komputer jest połączony można zobaczyć klikając w przycisk Edycja w panelu „Opcje logowania”, w preferencjach kont. Komputer pracujący z Mac OS X korzysta z serwerów znajdujących się na liście w celu uzyskiwania informacji o użytkownikach i innych danych administracyjnych przechowywanych w domenie katalogowej serwerów katalogowych.

Dodając lub usuwając serwer z listy Serwerów katalogowych, wpisy związane z tym serwerem katalogowym są dodawane lub usuwane z list Usługi, Uwierzytelnienie i Kontakty. Ale po usunięciu takich wpisów z listy Usługi, Uwierzytelnienie i Kontakty, serwer katalogowy nie jest usuwany z listy Serwery katalogowe.

Komputery z Mac OS X v10.6 mogą łączyć się z serwerem katalogowym Open Directory, Active Directory lub serwer katalogowy LDAP. Jeżeli nie wiadomo, z jakim serwerem się połączyć, należy zapytać administratora sieci.

Ważne:  Jeśli nazwa komputera zawiera myślnik, może wystąpić problem z łączeniem lub powiązaniem z domeną katalogową LDAP lub Active Directory. Powiązanie można ustanowić używając nazwy komputera, która nie zawiera myślnika.
