Dodawanie połączenia do serwera Open Directory

Przy dodawaniu serwera Open Directory, trzeba znać nazwę serwera lub adres IP oraz wiedzieć, czy serwer używa protokołu SSL.

Jak dodać serwer Open Directory?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Przyłącz lub Edycja.

    Jeśli widzisz przycisk Edycja, oznacza to, że komputer posiada co najmniej jedno połączenie z serwerem katalogowym.

  4. Następnie kliknij w przycisk Dodaj (+).

  5. Z rozwijanego menu „Dodaj nowy katalog typu“ wybierz Open Directory.

  6. W polu „Nazwa serwera lub adres IP“, wpisz nazwę serwera lub adres IP.

  7. (Warunkowo) Przed zaznaczeniem pola „Szyfruj używając SSL“, zapytaj administratora Open Directory, czy potrzebny jest SSL.

WAŻNE: Jeżeli zmienisz adres IP i nazwę komputera z systemem Mac OS X Server za pomocą changeip, podczas przyłączenia do serwera katalogowego, trzeba odłączyć się i ponownie podłączyć do serwera katalogowego, aby uaktualnić katalog nową nazwą komputera i adresem IP. Jeżeli się nie rozłączysz i ponownie podłączysz do serwera katalogowego, katalog nie zostanie uaktualniony i będzie nadal używał starej nazwy komputera i adresu IP.
