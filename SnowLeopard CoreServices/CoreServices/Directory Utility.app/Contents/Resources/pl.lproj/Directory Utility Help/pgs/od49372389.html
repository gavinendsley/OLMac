Kontrolowanie uwierzytelniania ze wszystkich domen z lasu Active Directory

Na komputerze skonfigurowanym do używania wtyczki Active Directory w Narzędziu katalogowym można pozwolić użytkownikom z lasu Active Directory na uwierzytelniania ze wszystkich domen lub ograniczyć uwierzytelnianie do wybranych domen.

Jak kontrolować, czy użytkownicy mogą uwierzytelniać się ze wszystkich domen lasu?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Edycja.

  4. Kliknij w przycisk „Otwórz Narzędzie katalogowe”.

  5. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  6. Kliknij w Usługi.

  7. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  8. Na liście usług zaznacz Active Directory i kliknij w przycisk Edycja (/).

  9. Jeśli ukryte są opcje zaawansowane, kliknij w Pokaż opcje zaawansowane.

  10. Kliknij w Administrowanie.

  11. Zaznacz „Pozwól na uwierzytelnienie z dowolnej domeny lasu“.

    Jeśli zaznaczone jest „Pozwól na uwierzytelnienie z dowolnej domeny lasu”, można dodać las Active Directory do własnych zasad wyszukiwania komputera dla uwierzytelnienia i kontaktów.

    Gdy dodany jest las Active Directory do własnych zasad wyszukiwania, pojawia się on na liście dostępnych domen katalogowych jako „/Active Directory/Wszystkie domeny.” (Jest to domyślne ustawienie.)

    Jeśli odznaczone jest „Pozwól na uwierzytelnienie z dowolnej domeny lasu”, można dodawać poszczególne domeny Active Directory do własnych zasad wyszukiwania komputera dla uwierzytelnienia i kontaktów.

    Gdy dodamy domeny Active Directory do własnych zasad wyszukiwania, każda domena Active Directory pojawia się oddzielnie na liście dostępnych domen katalogowych.

  12. Następnie kliknij w przycisk OK.

  13. Po zaznaczeniu opcji „Pozwól na uwierzytelnienie z dowolnej domeny lasu“, zmień własną zasadę wyszukiwania na panelach Uwierzytelnienie i Kontakty tak, aby zawierały las Active Directory lub zaznaczone domeny.

    Więcej informacji o zmienianiu własnych zasad wyszukiwania można przeczytać w sekcji Definiowanie własnych zasad wyszukiwania.
