Zmiana kont użytkowników z innych rekordów w Active Directory

Można używać Workgroup Manager do edycji kont użytkowników, kont grupowych, list komputerów oraz innych rekordów w domenie Active Directory. Można również używać Workgroup Manager do usuwania rekordów z domeny Active Directory.

Jeśli schemat Active Directory został rozszerzony i zawiera standardowe typy rekordów (klasy obiektów) i atrybuty Mac OS X, można użyć programu Workgroup Manager do utworzenia i edytowania grup komputerów w domenie Active Directory.

Dodatkową informację można znaleźć w panelu „User Management Overview”.

Do tworzenia kont użytkowników i grup w domenie Active Directory używamy narzędzi administracyjnych Active Directory firmy Microsoft na serwerze administracyjnym Windows.
