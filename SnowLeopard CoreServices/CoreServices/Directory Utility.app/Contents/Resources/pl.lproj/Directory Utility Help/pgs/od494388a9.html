Włączanie lub wyłączanie usługi Active Directory

Narzędzia katalogowego można użyć do włączenia lub wyłączenia używania usług Active Directory dostarczanych przez serwer Windows. Active Directory jest usługą katalogową Windows 2000 i późniejszych serwerów.

Jeśli wyłączymy usługi Active Directory, a domeny Active Directory są częścią własnej zasady wyszukiwania, zostaną one wyświetlone na czerwono w panelu Uwierzytelnienie lub Kontakty w Zasadach wyszukiwania w Narzędziu katalogowym.

Jak włączyć lub wyłączyć dostęp do Active Directory?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Przyłącz lub Edycja.

  4. Kliknij w przycisk „Otwórz Narzędzie katalogowe”.

  5. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  6. Kliknij w Usługi.

  7. Obok Active Directory, zaznacz lub odznacz pole wyboru i kliknij w Zastosuj.
