Dodawanie połączenia do serwera Active Directory

Przy podłączaniu serwera Active Directory musisz znać nazwę serwera lub adres IP, a także nazwę użytkownika i hasło administratora Active Directory.

Jak dodać serwer Active Directory?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Przyłącz lub Edycja.

    Jeśli widzisz przycisk Edycja, oznacza to, że komputer posiada co najmniej jedno połączenie z serwerem katalogowym.

  4. Następnie kliknij w przycisk Dodaj (+).

  5. Z rozwijanego menu „Dodaj nowy katalog typu“, wybierz Active Directory, po czym wpisz następującą informację:

    • Domena Active Directory: To jest nazwa DNS domeny Active Directory (na przykład ads.company.com.)

    • ID komputera:  Opcjonalnie zmień identyfikator, który Active Directory będzie używał dla Twojego serwera. Nazwa NetBIOS serwera. Nazwa powinna zawierać nie więcej niż 15 znaków, bez znaków specjalnych i bez interpunkcji. W praktyce stosuje się nazwę serwera odpowiadającą nazwie DNS. Na przykład, jeśli serwer DNS posiada wpis dla serwera o nazwie „serwer.przykład.com“, nadaj serwerowi nazwę „serwer”.

    • Nazwa użytkownika i hasło administratora AD:  Wpisz nazwę użytkownika i hasło administratora Active Directory.

  6. Następnie kliknij w przycisk OK.
