Określanie preferowanego serwera Active Directory

Na komputerze skonfigurowanym do używania wtyczki Active Directory w Narzędziu katalogowym można określić nazwę DNS serwera, do którego domeny Active Directory komputer ma mieć domyślnie dostęp.

Jeśli serwer stanie się w przyszłości nieosiągalny, wtyczka Active Directory wraca do innego najbliższego serwera w lesie.

Jeżeli ta opcja nie jest zaznaczona, wtyczka Active Directory wyznaczy najbliższą domenę Active Directory w lesie.

Jak określić preferowany serwer dla wtyczki Active Directory?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Edycja.

  4. Kliknij w przycisk „Otwórz Narzędzie katalogowe”.

  5. Kliknij w Usługi.

  6. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  7. Na liście usług, zaznacz Active Directory i kliknij w przycisk Edycja (/).

  8. Jeśli ukryte są opcje zaawansowane, kliknij w Pokaż opcje zaawansowane.

  9. Kliknij w Administrowanie.

  10. Zaznacz „Preferowany serwer domen“ i wpisz nazwę DNS serwera Active Directory.

  11. Następnie kliknij w przycisk OK.
