Konfigurowanie przenośnych kont użytkowników w Active Directory

Można włączyć lub wyłączyć przenośne konta użytkowników Active Directory na komputerze używającym wtyczki Active Directory w Narzędziu katalogowym. Użytkownicy z przenośnymi kontami mogą logować się używając uwierzytelnienia Active Directory, gdy komputer nie jest podłączony do serwera Active Directory.

Podczas logowania użytkownika, wtyczka Active Directory przechowuje uwierzytelnienie dla przenośnego konta użytkownika, gdy komputer jest podłączony do domeny Active Directory. Przechowywanie uwierzytelnienia nie wymaga zmiany schematu Active Directory.

Jeśli schemat Active Directory został rozszerzony tak, aby zawierał atrybuty zarządzalnego klienta Mac OS X, takie ustawienia przenośnego konta są używane zamiast ustawień przenośnego konta wtyczki Active Directory.

Można mieć konta przenośne tworzone automatycznie lub wymagać, żeby użytkownicy Active Directory potwierdzali tworzenie przenośnych kont.

Jak włączyć lub wyłączyć przenośne konta w domenie Active Directory?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Przyłącz lub Edycja.

  4. Kliknij w przycisk „Otwórz Narzędzie katalogowe”.

  5. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  6. Kliknij w Usługi.

  7. Na liście usług, zaznacz Active Directory i kliknij w przycisk Edycja (/).

  8. Jeśli ukryte są opcje zaawansowane, kliknij w „Pokaż opcje zaawansowane”.

  9. Kliknij w Użytkownik, po czym kliknij w „Utwórz konto przenośne podczas logowania“ i opcjonalnie kliknij w „Wymagaj potwierdzenia przed utworzeniem konta przenośnego”.

    • Gdy obie opcje są zaznaczone, każdy użytkownik decyduje, czy tworzyć przenośne konto podczas logowania. Kiedy użytkownik loguje się do Mac OS X używając konta użytkownika Active Directory lub logując się jako użytkownik sieciowy, widzi okno dialogowe z przyciskami do natychmiastowego tworzenia przenośnego konta.

    • Jeżeli zaznaczona jest pierwsza opcja, a druga nie, przenośne konta tworzone są, gdy użytkownik się loguje.

    • Jeżeli pierwsza opcja nie jest zaznaczona, to druga jest wyłączona.

  10. Następnie kliknij w przycisk OK.
