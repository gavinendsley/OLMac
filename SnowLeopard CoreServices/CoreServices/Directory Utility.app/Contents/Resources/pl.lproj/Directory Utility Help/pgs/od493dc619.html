Użycie zaawansowanych ustawień usług katalogowych

Narzędzie katalogowe pokazuje usługi katalogowe, do których Mac OS X może mieć dostęp.Lista zawiera usługi katalogowe, które zapewniają komputerowi z Mac OS X dostęp do informacji o użytkowniku oraz innych danych administracyjnych przechowywanych w domenach katalogowych.

Można włączyć lub wyłączyć dostęp do każdej usługi katalogowej. Jeśli wyłączymy daną usługę w Narzędziu katalogowym, Mac OS X nie będzie miał dostępu do tej usługi katalogowej.
