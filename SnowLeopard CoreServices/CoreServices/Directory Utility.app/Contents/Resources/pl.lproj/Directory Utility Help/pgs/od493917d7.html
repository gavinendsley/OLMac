Przygotowanie katalogu LDAP tylko do odczytu dla Mac OS X

Jeśli komputer Mac OS X ma pobierać dane administracyjne z katalogu LDAP tylko do odczytu, dane muszą istnieć w formacie wymaganym przez Mac OS X. Może zajść potrzeba dodania, zmiany lub przeorganizowania danych katalogu LDAP tylko do odczytu.

Ponieważ Mac OS X nie może zapisywać danych w katalogu tylko do odczytu, trzeba użyć innych narzędzi do dokonania takich zmian. Narzędzia muszą znajdować się na serwerze z katalogiem LDAP tylko do odczytu.

Jak przygotować katalog LDAP tylko do odczytu dla Mac OS X?

  1. Przejdź do serwera z katalogiem LDAP tylko do odczytu i skonfiguruj go tak, aby wspierał uwierzytelnianie w oparciu o LDAP oraz sprawdzanie hasła.

  2. Wedle potrzeby, zmień klasy i atrybuty obiektów katalogu LDAP tak, aby dostarczały danych potrzebnych Mac OS X.

Specyfikacje danych wymaganych przez usługi katalogowe Mac OS X można zobaczyć w „Administrowanie Open Directory”.
