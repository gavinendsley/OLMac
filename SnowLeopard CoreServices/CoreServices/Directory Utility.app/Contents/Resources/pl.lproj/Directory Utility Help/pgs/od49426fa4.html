Odwzorowanie ID grupy w kontach grup na atrybut Active Directory

Na komputerze skonfigurowanym do używania wtyczki Active Directory w Narzędziu katalogowym można określić atrybut Active Directory, który ma być odwzorowany na atrybut ID grupy (GID) Mac OS X w kontach grupowych.

Zwykle, schemat Active Directory musi być rozszerzony tak, aby zawierał odpowiedni atrybut odwzorowania GID:

  • Jeżeli administrator Active Directory rozszerzy schemat Active Directory instalując Microsoft Usługi dla UNIX, można odwzorować GID na atrybut msSFU-30-Gid-Number.

  • Jeżeli administrator Active Directory ręcznie rozszerzył schemat Active Directory tak, żeby zawierał atrybuty RFC 2307, można odwzorować GID na gidNumber.

  • Jeżeli administrator Active Directory ręcznie rozszerzył schemat Active Directory tak, aby zawierał atrybut gidNumber Mac OS X, może na niego odwzorować GID.

Jeżeli wyłączone jest odwzorowanie GID, wtyczka Active Directory generuje GID w oparciu o standardowy atrybut GUID z Active Directory.

UWAGA: Jeśli później zmienimy odwzorowanie GID, użytkownicy mogą utracić dostęp do poprzednio utworzonych plików.
Jak odwzorować GID na atrybut w rozszerzonym schemacie Active Directory?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Edycja.

  4. Kliknij w przycisk „Otwórz Narzędzie katalogowe”.

  5. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  6. Kliknij w Usługi.

  7. Na liście usług, zaznacz Active Directory i kliknij w przycisk Edycja (/).

  8. Jeśli ukryte są opcje zaawansowane, kliknij w Pokaż opcje zaawansowane.

  9. Kliknij w Odwzorowania.

  10. Zaznacz „Odwzorowuj GID na atrybut“ i podaj nazwę atrybutu Active Directory, który ma być odwzorowany na GID w kontach grupowych.

  11. Następnie kliknij w przycisk OK.
