Odłączenie od serwera Active Directory

Jeżeli komputer używa wtyczki Active Directory programu Narzędzie katalogowym do powiązania z serwerem Active Directory, można odłączyć komputer od serwera Active Directory.

Można wymusić usunięcie dowiązania, jeżeli komputer nie może połączyć się z serwerem lub jeżeli rekord komputera został usunięty z serwera.

Jak odłączyć komputer od serwera Active Directory?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Edycja.

  4. Kliknij w przycisk „Otwórz Narzędzie katalogowe”.

  5. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  6. Kliknij w Usługi.

  7. Na liście usług zaznacz Active Directory i kliknij w przycisk Edycja (/).

  8. Kliknij w Odłącz, uwierzytelnij się jako użytkownik posiadający prawa do zakończenia połączenia z domeną Active Directory i kliknij w OK.

    Jeśli pojawi się ostrzeżenie mówiące, że uwierzytelnienie nie zostało zaakceptowane lub komputer nie może połączyć się z Active Directory, kliknij w Wymuś odłączenie, żeby wymusić przerwanie połączenia.

    Jeżeli wymusimy odłączenie, Active Directory będzie dalej zawierał rekord tego komputera. Należy powiadomić administratora Active Directory, iż trzeba usunąć rekord komputera.

  9. W panelu Usługi, odznacz włączenie Active Directory, po czym kliknij w Zastosuj.
