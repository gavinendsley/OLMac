Odwzorowanie atrybutów rekordów katalogów LDAP

Trzeba odwzorować atrybuty RealName oraz DataStamp rekordu typu Config, żeby przechowywać informację dla zarządzanych użytkowników Mac OS X w katalogu LDAP, nie będącego katalogiem LDAP Apple.

Jeśli nie odwzorujesz tych atrybutów, pojawi się następujący komunikat błędu, gdy użyjesz programu Workgroup Manager do zmiany rekordu użytkownika, który znajduje się w katalogu LDAP:

Atrybut o nazwie „dsRecTypeStandard:Config” nie jest odwzorowywany.

Można zignorować ten komunikat, jeśli nie używasz zarządzania klientami Mac OS X, które zależy od atrybutów RealName i DataStamp rekordu typu Config pamięci podręcznej.
