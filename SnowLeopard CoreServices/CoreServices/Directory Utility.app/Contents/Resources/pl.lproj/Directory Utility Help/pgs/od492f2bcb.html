Zarządzanie kontem użytkownika root

Używaj Narzędzia katalogowego (dostępnego w preferencjach konta) do zarządzania kontem root poprzez włączanie oraz wyłączanie użytkownika root. Jeżeli włączymy konto użytkownika root, możemy użyć Narzędzia katalogowego także do zmiany hasła tego konta.

Więcej informacji o zarządzaniu kontem użytkownika root za pomocą Narzędzia katalogowego znajduje się w:

  • Włączanie konta użytkownika root

  • Zmiana hasła konta użytkownika root
