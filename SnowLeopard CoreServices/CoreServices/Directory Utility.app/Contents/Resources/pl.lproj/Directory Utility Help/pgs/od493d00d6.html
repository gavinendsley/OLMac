Odwzorowanie ID grupy podstawowej na atrybut Active Directory

Na komputerze skonfigurowanym do używania wtyczki Active Directory w Narzędziu katalogowym można określić atrybut Active Directory, który ma być odwzorowany na atrybut ID grupy podstawowej (GID) Mac OS X w kontach użytkowników.

Zwykle, schemat Active Directory musi być rozszerzony tak, aby zawierał odpowiedni atrybut odwzorowania podstawowego GID:

  • Jeżeli administrator Active Directory rozszerzy schemat Active Directory instalując Services firmy Microsoft dla UNIX, można odwzorować podstawowy GID na atrybut msSFU-30-Gid-Number

  • Jeżeli administrator Active Directory ręcznie rozszerzył schemat Active Directory tak, żeby zawierał atrybuty RFC 2307, można odwzorować podstawowy GID na gidNumber.

  • Jeżeli administrator Active Directory ręcznie rozszerzył schemat Active Directory tak, aby zawierał atrybut PrimaryGroupID Mac OS X, może na niego odwzorować podstawowy GID.

Jeżeli wyłączone jest odwzorowanie podstawowego GID, wtyczka Active Directory generuje podstawowy GID w oparciu o standardowy atrybut GUID z Active Directory.

UWAGA: Jeśli później zmienimy odwzorowanie podstawowego GID, użytkownicy mogą utracić dostęp do poprzednio utworzonych plików.
Jak odwzorować podstawowy GID na atrybut w rozszerzonym schemacie Active Directory?

  1. Otwórz Preferencje systemowe i kliknij w Konta.

  2. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  3. Kliknij w Opcje logowania, a następnie kliknij w Edycja.

  4. Kliknij w przycisk „Otwórz Narzędzie katalogowe”.

  5. Jeśli ikona kłódki jest zamknięta, otwórz ją klikając w nią i wpisując nazwę oraz hasło administratora.

  6. Kliknij w Usługi.

  7. Na liście usług, zaznacz Active Directory i kliknij w przycisk Edycja (/).

  8. Jeśli ukryte są opcje zaawansowane, kliknij w Pokaż opcje zaawansowane.

  9. Kliknij w Odwzorowania.

  10. Zaznacz „Odwzorowuj GID użytkownika na atrybut“ i podaj nazwę atrybutu Active Directory, który ma być odwzorowany na ID grupy podstawowej w kontach użytkowników.

  11. Następnie kliknij w przycisk OK.
