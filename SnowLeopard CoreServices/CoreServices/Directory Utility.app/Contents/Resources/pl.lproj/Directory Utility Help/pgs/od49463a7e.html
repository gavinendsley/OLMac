Konfigurowanie danych w plikach konfiguracyjnych BSD

Jeśli komputer z Mac OS X ma pobierać dane administracyjne z plików konfiguracyjnych BSD, dane muszą istnieć w plikach i muszą być w formacie wymaganym przez Mac OS X.

Może zajść potrzeba dodania, zmiany lub przeorganizowania danych w plikach. Workgroup Manager nie może zmieniać danych w plikach konfiguracyjnych BSD, więc trzeba dokonać potrzebnych zmian używając edytora tekstu lub innych narzędzi.

W poniższej tabelce znajdują się nazwy plików wraz z opisami ich zawartości.

Plik konfiguracyjny BSD
Zawiera

/etc/master.passwd

Nazwy użytkowników, hasła, ID, ID grupy podstawowej, i tak dalej.

/etc/group

Nazwy grup, ID oraz członków

/etc/fstab

Punkty montowania NFS

/etc/hosts

Nazwy i adresy komputerów

/etc/networks

Nazwy i adresy sieci

/etc/services

Nazwy usług, porty i protokoły

/etc/protocols

Nazwy i numery protokołów IP

/etc/rpcs

Serwery Open Network Computing RPC

/etc/printcap

Nazwy i możliwości drukarek

/etc/bootparams

Ustawienia Bootparam

/etc/bootp

Ustawienia Bootp

/etc/aliases

Aliasy pocztowe oraz listy dystrybucyjne

/etc/netgroup

Nazwy grup sieciowych i ich członkowie
