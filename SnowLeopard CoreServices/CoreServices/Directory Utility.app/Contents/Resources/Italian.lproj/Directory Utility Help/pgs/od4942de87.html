Specificare le impostazioni NIS

Utilizzando Utility Directory, puoi creare una configurazione che definisce la modalità di accesso di Mac OS X a un dominio NIS (Network Information Service).

Per creare una configurazione per l'accesso a un dominio NIS:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Nell'elenco dei servizi, seleziona “Documento Flat BSD e NIS” e fai clic sul pulsante Modifica (/).

  8. Inserisci il nome del dominio NIS o il nome DNS oppure l'indirizzo IP del server su cui risiede il dominio NIS.

    Includi il nome host del server NIS o l’indirizzo IP se è richiesto per sicurezza o se il server non si trova nella stessa sottorete del computer che stai configurando.

    Se non specifichi un server, NIS utilizza il protocollo di broadcast per ricercare il server NIS in una sottorete.

  9. Seleziona “Usa dominio NIS per autenticazione”, quindi fai clic su OK.

    Il dominio NIS viene aggiunto alle politiche di ricerca di autenticazione del computer come /BSD/dominio, dove dominio è il nome inserito nel passo 4.
