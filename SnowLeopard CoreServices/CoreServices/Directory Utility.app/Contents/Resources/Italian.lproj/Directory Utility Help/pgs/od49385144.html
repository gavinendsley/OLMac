Inizio
Impostare le cartelle Inizio per gli account utente di Active Directory

Su un computer configurato per utilizzare il connettore Active Directory di Utility Directory, puoi abilitare o disabilitare le cartelle Inizio del network o le cartelle Inizio locali per account utente di Active Directory.

Con le cartelle Inizio del network, la cartella Inizio del network Windows dell’utente viene attivata come cartella Inizio di Mac OS X quando l'utente esegue il login.

Puoi determinare se la cartella Inizio del network deriva dall'attributo standard homeDirectory di Active Directory oppure dall'attributo HomeDirectory di Mac OS X, se lo schema Active Directory è stato ampliato per includerlo.

Con cartelle Inizio locali, ogni utente Active Directory che esegue il login ha una cartella Inizio sul disco di avvio di Mac OS X. In più, la cartella Inizio del network viene attivata come un volume network, allo stesso modo di un punto di condivisione. L'utente può copiare documenti tra questo volume network e la cartella Inizio locale.

Per impostare le cartelle Inizio per gli account utente di Active Directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Nell'elenco dei servizi, seleziona Active Directory, quindi fai clic sul pulsante Modifica (/).

  8. Se le opzioni avanzate sono nascoste, fai clic su Mostra opzioni avanzate.

  9. Fai clic su Esperienza utente.

  10. Fai clic su “Forza la directory home locale sul disco di avvio” se vuoi che gli account utente di Active Directory abbiano le cartelle home locali nella cartella /Utenti del computer.

    Questa opzione non è disponibile se "Crea account mobile al momento del login" è selezionato.

  11. Per utilizzare l'attributo standard di Active Directory per la posizione della cartella Inizio, seleziona “Usa percorso UNC di Active Directory per derivare la home del network” e scegli i protocolli per accedere alla cartella Inizio:

    • Per utilizzare il protocollo standard SMB di Windows, scegli smb dal menu a comparsa “Protocollo di network da usare”.

    • Per utilizzare il protocollo standard AFP di Macintosh, scegli afp dal menu a comparsa “Protocollo di network da usare”.

  12. Per utilizzare l'attributo di Mac OS X per la posizione della cartella Inizio, deseleziona “Usa percorso UNC di Active Directory per derivare la home del network”.

    Per utilizzare l'attributo di Mac OS X, lo schema Active Directory deve essere ampliato per includerlo.

  13. Fai clic su OK.

Se modifichi il nome di un account utente nel dominio Active Directory, il server crea una cartella Inizio (e le sottocartelle) per l'account utente la prossima volta che viene utilizzato per eseguire il login a un computer Mac OS X. L'utente può ancora consultare la vecchia cartella Inizio e vederne il contenuto nel Finder.

Puoi evitare la creazione di una nuova cartella Inizio rinominando la vecchia cartella prima del successivo login dell'utente.
