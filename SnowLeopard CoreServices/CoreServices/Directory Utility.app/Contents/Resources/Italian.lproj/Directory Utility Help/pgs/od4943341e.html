Specificare un server Active Directory preferito

Su un computer configurato per l'utilizzo del connettore Active Directory di Utility Directory, puoi specificare il nome DNS del server al cui dominio Active Directory vuoi che il computer acceda di default.

Se, in seguito, il server risulta non disponibile, il connettore Active Directory passa a un altro server nella forest che si trova nelle vicinanze.

Se questa opzione è deselezionata, il connettore Active Directory determina il dominio Active Directory più vicino nella forest.

Per specificare un server preferito a cui possa accedere il connettore Active Directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Fai clic su Servizi.

  6. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  7. Nell'elenco dei servizi, seleziona Active Directory, quindi fai clic sul pulsante Modifica (/).

  8. Se le opzioni avanzate sono nascoste, fai clic su Mostra opzioni avanzate.

  9. Fai clic su Amministratore.

  10. Seleziona “Preferisci questo server di dominio” e inserisci il nome DNS del server Active Directory.

  11. Fai clic su OK.
