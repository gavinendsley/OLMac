Configurare dei dati nei documenti di configurazione BSD

Se vuoi un computer Mac OS X per ottenere dati amministrativi da documenti di configurazione BSD, i devono essere presenti nei documenti e devono essere nel formato richiesto da Mac OS X.

Potresti aver bisogno di aggiungere, modificare o riorganizzare i dati nei documenti. Workgroup Manager non può effettuare modifiche ai dati nei documenti di configurazione BSD, pertanto devi effettuare le modifiche necessarie utilizzando un editor di testo o altri strumenti.

La seguente tabella indica i nomi dei documenti e il relativo contenuto.

Documenti di configurazione BSD
Contiene

/etc/master.passwd

Nomi utente, password, ID, ID gruppo primario e così via

/etc/group

Nomi di gruppo, ID e membri

/etc/fstab

Avvii NFS

/etc/hosts

Nomi e indirizzi di computer

/etc/networks

Nomi e indirizzi di network

/etc/services

Nomi di servizi, porte e protocolli

/etc/protocols

Nomi di protocollo IP e numeri

/etc/rpcs

Apri i server Network Computing RPC

/etc/printcap

Nomi e funzionalità stampante

/etc/bootparams

Impostazioni Bootparam

/etc/bootp

Impostazioni Bootp

/etc/aliases

Alias di posta e liste di distribuzione

/etc/netgroup

Nomi di gruppo e membri network-wide
