Assegnare la UID a un attributo Active Directory

Su un computer configurato per l'utilizzo del connettore di Active Directory di Utility Directory, puoi specificare un attributo Active Directory che vuoi assegnare all'attributo UID (unique user ID) di Mac OS X.

Lo schema Active Directory, generalmente, deve essere ampliato per includere un attributo adatto a essere assegnato alla UID.

  • Se l’amministratore Active Directory amplia lo schema Active Directory installando i servizi per UNIX di Microsoft, puoi assegnare l'UID con l'attributo msSFU-30-Uid-Number.

  • Se l'amministratore Active Directory amplia lo schema Active Directory manualmente per includere gli attributi RFC 2307, puoi assegnare l'UID all'attributo uidNumber.

  • Se l'amministratore Active Directory amplia lo schema Active Directory manualmente per includere l'attributo UniqueID di Mac OS X, puoi assegnare l'UID a esso.

Se l'assegnazione dell'UID è disattivata, il connettore Active Directory genera un UID in base all'attributo GUID standard di Active Directory.

ATTENZIONE:    se in seguito modifichi le impostazioni per l'opzione di assegnazione dell'UID, gli utenti potrebbero perdere l'accesso ai documenti creati precedentemente.
Per assegnare l'UID a un attributo in uno schema Active Directory ampliato:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Nell'elenco dei servizi, seleziona Active Directory, quindi fai clic sul pulsante Modifica (/).

  8. Se le opzioni avanzate sono nascoste, fai clic su Mostra opzioni avanzate.

  9. Fai clic su Mappature.

  10. Seleziona “Assegna UID per attribuire” e inserisci il nome dell'attributo Active Directory che vuoi assegnare alla UID.

  11. Fai clic su OK.
