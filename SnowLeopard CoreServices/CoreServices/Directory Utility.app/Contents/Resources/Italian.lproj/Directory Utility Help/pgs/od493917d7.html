Preparare una directory LDAP di sola lettura per Mac OS X

Se vuoi che un computer Mac OS X ottenga dati amministrativi da una directory LDAP di sola lettura, gli stessi dati devono essere presenti nel formato richiesto da Mac OS X. Potresti aver bisogno di aggiungere, modificare o riorganizzare i dati nella directory LDAP di sola lettura.

Dato che Mac OS X non può scrivere dati in una directory di sola lettura, devi utilizzare altri strumenti per effettuare le modifiche. Gli strumenti devono trovarsi sul server che ospita la directory LDAP di sola lettura.

Per preparare una directory LDAP di sola lettura per Mac OS X:

  1. Vai al server che ospita la directory LDAP di sola lettura e configuralo per supportare l'autenticazione basata su LDAP e il controllo password.

  2. Modifica i tipi di oggetti e attributi della directory LDAP come è necessario per fornire i dati di cui ha bisogno Mac OS X.

Per le specifiche dei dati richiesti dai servizi di directory di Mac OS X, consulta “Open Directory Administration”.
