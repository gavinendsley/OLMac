Informazioni su Utility Directory

Utility Directory determina i servizi directory utilizzati da un computer Mac OS X e la modalità di connessione a domini di directory specifici. Utility Directory, inoltre, definisce i criteri di ricerca per trovare le informazioni sui contatti e sull'autenticazione in domini di directory specifici.

Configurazione di Utility Directory

Puoi utilizzare Utility Directory sul tuo computer per configurare e gestire gli accessi del tuo computer ai servizi directory. Puoi inoltre utilizzare Utility Directory per configurare in modalità remota il modo in cui il tuo computer deve accedere ai servizi directory. Utility Directory può essere anche utilizzato per abilitare l'account root o modificare la password dell'account root.

Configurare l'accesso ai servizi

Puoi utilizzare Utility Directory per abilitare e disabilitare i diversi tipi di servizi a cui Mac OS X può accedere su un network. Sono inclusi i servizi directory che forniscono l'accesso ad account utenti e ad altri dati di tipo amministrativo memorizzati nei domini delle directory.

Configurare l'accesso alle directory

Con Utility Directory, puoi configurare i computer per l'accesso agli account utenti e ad altri dati di tipo amministrativo contenuti in directory LDAP specifiche, in un dominio Active Directory e in un dominio NIS.

Configurazione dei criteri di ricerca

Con Utility Directory, puoi definire i criteri di ricerca per l'autenticazione utilizzati da Mac OS X per individuare e recuperare le informazioni di autenticazione dell'utente e altri dati di tipo amministrativo dai domini delle directory. Utility Directory può anche definire i criteri di ricerca per i contatti che Mac OS X utilizza per individuare e recuperare il nome, l'indirizzo e altre informazioni sul contatto dai domini delle directory.
