Rimuovere una connessione server di directory

Prima di rimuovere un server di directory dalle preferenze Account, assicurati di non utilizzare i relativi servizi per altre applicazioni.

Ad esempio, se Mail è configurato per utilizzare il server di directory per ricercare i contatti ed elimini il server di directory, non puoi cercare nessuno dei contatti che era su quel server di directory.

Per eliminare un server di directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Modifica.

  4. Dall'elenco dei server di directory, seleziona il server di directory che desideri eliminare.

  5. Fai clic sul pulsante Elimina (—).

  6. Se sei sicuro di aver selezionato il server di directory corretto, fai clic su “Interrompi utilizzo del server”.

Se un utente si disconnette da un server di directory, l'amministratore directory deve rimuovere manualmente il record del computer dal server di directory. Gli utenti possono anche richiedere che l'amministratore del network disconnetta i loro computer dal server di directory mediante Utility Directory e un account che ha i permessi di amministrazione directory, che rimuove il record del computer dal server di directory.
