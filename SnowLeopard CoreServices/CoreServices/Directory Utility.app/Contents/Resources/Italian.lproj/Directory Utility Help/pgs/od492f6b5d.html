Abilitare l'account utente root

Puoi utilizzare Utility Directory per abilitare l'account utente root. Se abiliti l'account utente root, utilizza una password complessa che contenga caratteri speciali e alfanumerici, per prevenire eventuali compromissioni della password.

ATTENZIONE:    l'account root è un account amministratore non restrittivo utilizzato per eseguire modifiche ai documenti di sistema critici. Anche se hai eseguito il login come amministratore, devi comunque utilizzare l'account root o sudo per eseguire operazioni di sistema critiche.

Non utilizzare mai l'account root per effettuare il login a un computer (in modalità remota o locale). Al contrario, utilizza sudo per eseguire operazioni root. Puoi restringere l'accesso al comando sudo aggiungendo gli utenti al documento /etc/sudoers/.

Per ulteriori informazioni sull'account root, consulta Aiuto Server Admin.

Per abilitare l'account utente root:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

    Se viene visualizzato un pulsante Modifica, significa che il computer dispone di almeno una connessione a un server di directory.

  4. Fai clic su “Apri Utility Directory”.

  5. Scegli Modifica > Abilita utente root.
