Impostare l'accesso LDAP ai domini Active Directory

Utilizzando Utility Directory, puoi impostare la configurazione di un LDAPv3 per accedere a un dominio Active Directory su un server Windows. Una configurazione LDAPv3 ti offre completo controllo sulle assegnazioni dei tipi di record e di attributi Mac OS X alle classi oggetti, basi di ricerca e attributi Active Directory.

Assegnare alcuni tipi di record e attributi importanti di Mac OS X, come ad esempio la UID (unique user ID), richiede l'ampliamento dello schema Active Directory.

Una configurazione LDAPv3 non include le seguenti caratteristiche del connettore Active Directory elencate in Utility Directory.

  • Generazione dinamica di UID (unique user ID) e ID gruppo primario

  • Creazione di una cartella Inizio Mac OS X locale

  • Attivazione automatica della cartella Inizio di Windows

  • Account utente mobile con credenziali di autenticazione con cache

  • Scoperta di tutti i domini in una forest Active Directory

  • Supporto per la replica e il failover di Active Directory

Per ulteriori informazioni, consulta Informazioni sull'accesso ad Active Directory.

Per creare una configurazione server Active Directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Nell'elenco dei servizi, seleziona LDAPv3, quindi fai clic sul pulsante Modifica (/).

  8. Fai clic su Nuovo e inserisci il nome DNS o l’indirizzo IP del server Active Directory.

  9. Seleziona tra le seguenti opzioni per l'accesso alla directory, quindi fai clic su Continua per consentire a Utility Directory di ottenere le informazioni dal server Active Directory.

    • Seleziona “Cripta usando SSL” se desideri che Open Directory utilizzi SSL per le connessioni con il server Active Directory. Prima di selezionare il riquadro SSL, verifica con l'amministratore Open Directory se è necessario utilizzare SSL.

    • Seleziona “Usa per l'autenticazione” se questa directory contiene account utente utilizzati per il login o per l'autenticazione per i servizi.

    • Seleziona “Usa per i contatti” se questa directory contiene indirizzi e-mail e altre informazioni che desideri utilizzare in Rubrica Indirizzi.

    Se Utility Directory non riesce a contattare il server Active Directory, viene visualizzato un messaggio e devi configurare manualmente l’accesso o annullare il processo di configurazione. Per ulteriori informazioni, consulta Configurare manualmente l'accesso a una directory LDAP.

    Se selezioni "Usa per l'autenticazione" o "Usa per i contatti", la connessione LDAPv3 al dominio Active Directory viene aggiunta al criterio di ricerca personalizzato nel pannello Autenticazione o Contatti di Utility Directory.

    Assicurati che LDAPv3 sia abilitato nel pannello Servizi in modo che il computer utilizzi la connessione impostata. Per ulteriori informazioni, consulta Abilitare o disabilitare i servizi di directory LDAP”.

  10. Se la finestra di dialogo si espande per visualizzare le opzioni di mappatura, scegli Active Directory dal menu a comparsa, inserisci la base di ricerca, quindi fai clic su Continua.

    Il modello di mappatura di Active Directory per la configurazione di LDAPv3 assegna alcuni tipi di record e di attributi Mac OS X a classi di oggetti e attributi che non fanno parte dello schema Active Directory standard. Puoi cambiare le assegnazioni definite dal modello o ampliare lo schema Active Directory.

    In alternativa, puoi accedere al dominio Active Directory attraverso il connettore Active Directory al posto di LDAPv3. Per ulteriori informazioni, consulta Configurare l'accesso a un dominio Active Directory.

  11. Quando la finestra di dialogo si espande per visualizzare le opzioni di connessione, inserisci il nome distintivo e la password di un account utente di Active Directory.

  12. Fai clic su OK per terminare la creazione della connessione LDAP.

  13. Fai clic su OK per terminare la configurazione delle opzioni LDAPv3.
