Inizio
Interrompere il binding con il server Active Directory

Se il computer sta utilizzando il connettore Active Directory di Utility Directory per vincolarsi a un server Active Directory, puoi interrompere il binding tra il computer e il server Active Directory.

Puoi forzare l'interruzione del binding se il computer non può contattare il server o se il "record computer" è già stato rimosso dal server.

Per non vincolare il computer dal server Active Directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Nell'elenco dei servizi, seleziona Active Directory, quindi fai clic sul pulsante Modifica (/).

  8. Fai clic su Non vincolare, autenticati come un utente con i diritti di terminare una connessione al dominio Active Directory e fai clic su OK.

    Se vedi un avviso che informa che le credenziali non sono state accettate o che il computer non può contattare Active Directory, fai clic su Forza rimozione vincolo per forzare l’interruzione della connessione.

    Se forzi l'interruzione del binding, Active Directory continua a contenere un "record computer" per questo computer. Avverti l'amministratore di Active Directory in modo che possa definitivamente rimuovere il "record computer".

  9. Nel pannello Servizi, deseleziona l’impostazione per abilitare Active Directory, quindi fai clic su Applica.
