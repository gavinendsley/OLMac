Configurazione di Utility Directory su un server remoto

Puoi utilizzare Directory Utility sul tuo computer per impostare e gestire la modalità utilizzata da Mac OS X Server, su un server remoto, per accedere ai servizi di directory.

Configurare Configurazione Directory su un server remoto:

  1. Apri “Preferenze di Sistema” sul tuo computer, quindi fai clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

    Se è presente il pulsante Modifica, il tuo computer ha almeno una connessione a un directory server.

  4. Fai clic su “Apri Utility Directory”.

  5. Dal menu Archivio, scegli Connetti.

  6. Inserisci le seguenti informazioni su connessione e autenticazione per il server che desideri configurare.

    • Indirizzo: inserisci il nome DNS o l'indirizzo IP del server che vuoi configurare.

    • Nome utente: inserisci il nome utente di un amministratore sul server.

    • Password: inserisci la password per il nome utente che hai inserito.

  7. Fai clic su Connetti.

  8. Fai clic sul pannello Servizi e “Politica di ricerca” e cambia le impostazioni nel modo desiderato.

    Le modifiche che apporti influiscono sul server remoto a cui ti si collegato seguendo i passi precedenti.

  9. Dal menu Documento sul tuo computer, scegli Disconnetti.
