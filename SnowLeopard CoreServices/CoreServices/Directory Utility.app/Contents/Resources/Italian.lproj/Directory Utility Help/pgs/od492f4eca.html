Aggiungere una connessione Server Open Directory

Quando aggiungi un server Open Directory, devi sapere il nome del server o l'indirizzo IP e se il server utilizza SSL (secure socket layer).

Per aggiungere un server Open Directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

    Se viene visualizzato un pulsante Modifica, significa che il computer dispone di almeno una connessione a un server di directory.

  4. Fai clic sul pulsante Aggiungi (+).

  5. Dal menu a comparsa “Aggiungi una nuova directory di tipo”, scegli Servizi Directory.

  6. Nel campo “Nome server o indirizzo IP”, inserisci il nome del server o l'indirizzo IP.

  7. (Condizionale) Prima di selezionare il riquadro “Cripta usando SSL”, chiedi all'amministratore Open Directory se è necessario utilizzare SSL.

Importante:   se modifichi l'indirizzo IP e il nome del computer del server Mac OS X mediante changeip mentre sei connesso a un server di directory, devi disconnetterti e riconnetterti al server di directory affinché la directory venga aggiornata con il nome del computer e l'indirizzo IP nuovi. Se non ti disconnetti e riconnetti al server di directory, la directory non verrà aggiornata e continuerà a utilizzare il nome del computer e l'indirizzo IP precedenti.
