Assegnare l'ID gruppo primario a un attributo Active Directory

Su un computer configurato per l’utilizzo del connettore Active Directory di Utility Directory, puoi specificare un attributo di Active Directory che vuoi assegnare all’attributo ID gruppo primario di Mac OS X negli account utente.

Lo schema Active Directory, generalmente, deve essere ampliato per includere un attributo adatto a essere assegnato al GID primario.

  • Se l’amministratore Active Directory amplia lo schema Active Directory installando i servizi per UNIX di Microsoft, puoi assegnare il GID primario con l'attributo msSFU-30-Gid-Number.

  • Se l'amministratore Active Directory amplia lo schema Active Directory manualmente per includere gli attributi RFC 2307, puoi assegnare il GID primario all'attributo gidNumber.

  • Se l'amministratore Active Directory amplia lo schema Active Directory manualmente per includere l'attributo PrimaryGroupID di Mac OS X, puoi assegnare il GID primario a esso.

Se l’assegnazione dell'ID gruppo primario è disattivata, il connettore Active Directory genera una ID gruppo primario basato su un attributo standard GUID di Active Directory.

ATTENZIONE:    se in seguito modifichi le impostazioni per l'opzione di assegnazione del GID primario, gli utenti potrebbero perdere l'accesso ai documenti creati precedentemente.
Per assegnare il GID primario a un attributo in uno schema Active Directory ampliato:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Nell'elenco dei servizi, seleziona Active Directory, quindi fai clic sul pulsante Modifica (/).

  8. Se le opzioni avanzate sono nascoste, fai clic su Mostra opzioni avanzate.

  9. Fai clic su Mappature.

  10. Seleziona “Assegna GID utente per attribuire” e inserisci il nome dell'attributo di Active Directory che vuoi assegnare all'ID gruppo primario negli account utente

  11. Fai clic su OK.
