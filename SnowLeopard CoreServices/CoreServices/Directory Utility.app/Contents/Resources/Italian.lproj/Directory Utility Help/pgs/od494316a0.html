Configurare le ricerche e le mappature LDAP

Se usi Utility Directory, potrai modificare le mappature, le ricerche e gli ambiti di ricerca in grado di descrivere come Mac OS X deve trovare elementi di dati specifici in una directory LDAP. Potrai modificare una per una tali impostazioni per ogni configurazione di directory LDAP elencata in Utility Directory. Ogni configurazione di directory LDAP descrive come deve accedere Mac OS X ai dati di una directory LDAPv3 o LDAPv2.

Puoi modificare quanto segue:

  • La mappatura di ogni tipo di record Mac OS X su classi di oggetti LDAP

  • La mappatura di tipi di dati Mac OS X, o di attributi, su attributi LDAP per ogni tipo di record

  • Le basi e gli ambiti di ricerca di LDAP che determinano dove Mac OS X deve cercare un tipo di record Mac OS X in una directory LDAP

Quando assegnerai all'utente Mac OS X gli attributi per leggere/scrivere il dominio di directory LDAP (un dominio LDAP che non sia di sola lettura), l'attributo LDAP assegnato a RealName non deve essere uguale al primo attributo di un elenco di attributi LDAP assegnati a RecordName.

Ad esempio l'attributo cn non deve corrispondere al primo attributo assegnato a RecordName, nel caso in cui cn sia anche assegnato a RealName.

L'attributo LDAP assegnato a RealName è lo stesso del primo attributo assegnato a RecordName, i problemi si verificano quando cerchi di modificare il nome completo o il primo nome breve in Workgroup Manager.

Per ulteriori informazioni sui tipi di record e sugli attributi di Mac OS X, consulta “Open Directory Administration” (disponibile all'indirizzo www.apple.com/server/resources/).

Per modificare le basi di ricerca e le mappature per un server LDAP:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Nell'elenco dei servizi, seleziona LDAPv3, quindi fai clic sul pulsante Modifica (/).

  8. Se l'elenco delle configurazioni dei server è nascosto, fai clic su Mostra Opzioni.

  9. Seleziona la configurazione di un server dall'elenco e fai clic su Modifica.

  10. Fai clic su Ricerca & Assegnazione.

  11. Seleziona le mappature che desideri utilizzare come punto di partenza; altrimenti, scegli Personalizzato per iniziare con mappature non predefinite.

    Se scegli uno dei modelli di mappatura LDAP, viene visualizzata l'estensione di una base di ricerca che puoi modificare; altrimenti puoi accettare l'estensione della base di ricerca di default facendo clic su OK.

    Fai clic sul menu a comparsa "Configura l'uso di questo server LDAPv3" e scegli un modello di mappatura per usare le sue mappature come punto di partenza.

  12. Aggiungere tipi di registrazione e cambiare le loro ricerche come necessario.

    • Per aggiungere i tipi di registrazione, fai clic su Aggiungi (nell'elenco “Tipi di registrazione e attributi”); quindi, nel foglio che appare, seleziona “Tipi di registrazione”, seleziona uno o più tipi di registrazione dall'elenco e fai clic su OK.

    • Per modificare la base di ricerca e l'ambito della ricerca di un tipo di registrazione, selezionali nell'elenco “Tipi di record e attributi”; quindi, modifica il campo “Base di ricerca”. Seleziona “tutti i sottoalberi” per impostare l’ambito della ricerca in modo da includere la gerarchia della directory LDAP dalla base di ricerca, oppure seleziona “solo nel primo livello” per impostare l’ambito della ricerca in modo da includere esclusivamente la base di ricerca e un livello sotto di essa nella gerarchia della directory LDAP.

    • Per rimuovere un tipo di registrazione, selezionalo in "Tipi di registrazione e attributi" e fai clic su Elimina.

    • Per aggiungere una mappatura per un tipo di registrazione, seleziona il tipo di registrazione nell'elenco “Tipo di record e attributi, quindi fai clic su Aggiungi (sotto “Assegna a __ elementi nell'elenco”) e inserisci il nome di una classe di oggetti dalla directory LDAP.

      Per aggiungere un'altra classe di oggetti LDAP, premi il tasto A capo, inserisci il nome di una classe di oggetti e specifica se usare le classi di oggetti LDAP elencate utilizzando il menu a comparsa al di sopra dell'elenco.

    • Per modificare una mappatura per un tipo di record, seleziona il tipo di record nell'elenco “Tipi di record e attributi”, fai doppio clic sulla classe di oggetti LDAP che desideri modificare in “Assegna a __ elementi nell'elenco” e modificala. Specifica se utilizzare le classi di oggetti LDAP elencate mediante il menu a comparsa al di sopra dell'elenco.

    • Per rimuovere una mappatura per un tipo di record, nell'elenco “Tipi di record e attributi”, seleziona il tipo di record, quindi la classe di oggetti LDAP che desideri rimuovere da “Assegna a __ elementi nell'elenco” e quindi fai clic su Elimina (sotto "Assegna a __ elementi nell'elenco").

  13. Aggiungere attributi e cambiare le loro assegnazioni correttamente.

    • Per aggiungere attributi a un tipo di record, nell'elenco “Tipi di record e attributi”, seleziona il tipo di record e fai clic su Aggiungi (sotto l'elenco “Tipi di record e attributi”); quindi, nel foglio che appare, seleziona “Tipi di attributo”, seleziona i tipi di attributo desiderati e fai clic su OK.

    • Per aggiungere una mappatura per un attributo, nell'elenco “Tipi di record e attributi”, seleziona l'attributo, fai clic su Aggiungi (sotto “Assegna a __ elementi nell'elenco”) e inserisce il nome di un attributo dalla directory LDAP. Per aggiungere un altro attributo LDAP, premere Invio e inserisci il nome dell'attributo.

    • Per modificare una mappatura per un attributo, nell'elenco “Tipi di record e attributi”, seleziona l'attributo, fai doppio clic sull'elemento che desideri modificare in "Assegna a __ elementi nell'elenco", quindi modifica il nome dell'elemento.

    • Per rimuovere una mappatura per un attributo, nell'elenco “Tipi di record e attributi”, seleziona l'attributo, seleziona l'elemento che desideri rimuovere da "Assegna a __ elementi nell'elenco”, quindi fai clic su Elimina (sotto "Assegna a __ elementi nell'elenco").

    • Per modificare l'ordine degli attributi visualizzati nell'elenco a destra, trascinali verso l'alto o verso il basso nell'elenco.

  14. Per registrare le mappature come un modello, fai clic su "Registra modello".

    I modelli registrati nella posizione di default verranno elencati nel menu a comparsa dei modelli di mappatura LDAP la prossima volta che apri Utility Directory. La posizione di default per i modelli registrati si trova nella tua cartella Inizio in questo percorso:

    ~/Library/Application Support/Directory Access/LDAPv3/Templates

  15. Per archiviare le mappature nella directory LDAP in modo che possa fornirle automaticamente ai propri client, fai clic su "Registra nel server" e quindi inserisci una base di ricerca per archiviare le mappature, un nome distintivo di un amministratore o di un altro utente con i privilegi di scrittura per la base di ricerca (ad esempio, uid=diradmin, cn=users, dc=ods, dc=example, dc=com) e una password.

    Se stai registrando mappature su un server LDAP Open Directory, la base di ricerca corretta sarà cn=config,estensione (dove estensione è l'estensione della base di ricerca del server, come dc=ods,dc=example,dc=com).

La directory LDAP fornisce le proprie mappature ai client Mac OS X il cui criterio di ricerca personalizzato include una connessione configurata per ottenere le mappature dal server LDAP.

La directory LDAP fornisce anche le proprie mappature a tutti i client Mac OS X che dispongono di un criterio di ricerca automatico. Per ulteriori informazioni, consulta Configurare l'accesso a una directory LDAP” e Utilizzare le impostazioni avanzate delle politiche di ricerca”.
