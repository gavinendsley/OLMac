Assegnare attributi e record di configurazione per directory LDAP

Per memorizzare le informazioni degli utenti Mac OS X gestiti in una directory LDAP non Apple, devi assegnare i seguenti attributi del tipo di record Config: RealName e DataStamp.

Se non assegni tali attributi, quando userai Workgroup Manager per modificare un "record utente" che risiede nella directory LDAP, apparirà il seguente messaggio di errore:

L'attributo con nome "dsRecTypeStandard:Config" non è stato assegnato.

Puoi ignorare tale messaggio nel caso non stia usando la gestione clienti Mac OS X, che dipende dagli attributi di buffer RealName e DataStamp del tipo di registrazione di configurazione.
