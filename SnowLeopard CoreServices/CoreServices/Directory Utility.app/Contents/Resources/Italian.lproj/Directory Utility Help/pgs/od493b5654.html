Riempire le directory LDAP con dati per Mac OS X

Dopo aver configurato l'accesso ai domini di directory LDAP e aver impostato la mappatura dei dati, puoi inserire i record e i dati per Mac OS X. Per le directory LDAP che consentono l'amministrazione remota (accesso in lettura/scrittura), puoi utilizzare l'applicazione Workgroup Manager, che è inclusa in Mac OS X Server, come segue:

  • Identifica i punti condivisi e i domini condivisi che vuoi attivare automaticamente nei browser Network degli utenti (ciò che gli utenti vedono quando fanno clic su Network nella barra laterale della finestra del Finder).

    Utilizza Condivisione in Server Admin e i moduli Network di Workgroup Manager.

    Per informazioni aggiuntive, consulta "Anteprima servizi documenti" e "Anteprima gestione utente".

  • Definisci le registrazioni utente e di gruppo e configura le relative impostazioni.

    Utilizza il modulo Account di Workgroup Manager.

    Per informazioni aggiuntive, consulta "Anteprima gestione utente".

  • Definisci gli elenchi dei computer che hanno le stesse impostazioni delle preferenze e sono disponibili per gli stessi utenti e gruppi.

    Utilizza il modulo Computer di Workgroup Manager.

    Per informazioni aggiuntive, consulta "Anteprima gestione utente".

In qualsiasi caso, fai clic sulla piccola icona rotonda sopra l'elenco degli utenti e scegli dal menu a comparsa in Workgroup Manager per aprire il dominio di directory LDAP. Se la directory LDAP non è elencata nel menu a comparsa, scegli Altro dallo stesso menu per selezionare la directory LDAP.

Nota:   per aggiungere registrazioni e dati a una directory LDAP di sola lettura, devi utilizzare gli strumenti presenti sul server in cui risiede la directory LDAP.
