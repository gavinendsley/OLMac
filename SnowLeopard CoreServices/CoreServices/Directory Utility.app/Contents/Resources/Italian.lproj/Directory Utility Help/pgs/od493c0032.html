Abilitare o disabilitare i servizi di directory LDAP

Puoi utilizzare Utility Directory per abilitare o disabilitare l'accesso ai servizi di directory che utilizzano LDAPv2 e LDAPv3. Un singolo plugin di Utility Directory denominato LDAPv3 consente l'accesso a entrambe le versioni LDAP2 e LDAPv3.

I servizi directory forniti da Mac OS X Server utilizzano LDAPv3, come molti altri server. LDAPv3 è uno standard aperto comune nei network misti composti da sistemi Macintosh, UNIX e Windows. Alcuni server utilizzano la versione precedente LDAPv2 per fornire il servizio di directory.

Se disabiliti i servizi directory LDAP e le directory LDAP fanno parte dei criteri di ricerca personalizzati, esse vengono elencate in rosso nel pannello Autenticazione o Contatti di Politica di ricerca in Utility Directory.

Per abilitare o disabilitare i servizi directory LDAP:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Accanto a LDAPv3, seleziona o deseleziona il riquadro e fai clic su Applica.
