Monitoraggio delle connessioni del server di directory

Puoi utilizzare le preferenze Account per monitorare lo stato dei server di directory ai quali il tuo computer è connesso. Queste informazioni possono aiutarti quando cerchi di determinare il motivo per cui non puoi connetterti a un server di directory specifico.

Per monitorare lo stato di un server di directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Modifica.

  4. Controlla il colore del punto di stato a sinistra del server di directory:

    • Verde: indica che il server di directory sta rispondendo al computer.

    • Giallo: indica che il computer sta aspettando una risposta dal server di directory.

    • Rosso: indica che il server di directory non sta rispondendo al computer.
