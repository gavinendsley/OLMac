Impostare una shell UNIX per gli account utente di Active Directory

Su un computer configurato per l'utilizzo del connettore Active Directory di Utility Directory, è possibile impostare la shell linea di comando che gli utenti con account Active Directory utilizzeranno di default durante l'interazione con Mac OS X nell'applicazione Terminale.

La shell di default viene anche utilizzata per l'interazione remota via SSH (Secure Shell) o Telnet. Ogni utente può sovrascrivere la shell di default modificando le preferenze di Terminale.

Per impostare una shell UNIX per gli account utente di Active Directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Nell'elenco dei servizi, seleziona Active Directory, quindi fai clic sul pulsante Modifica (/).

  8. Se le opzioni avanzate sono nascoste, fai clic su Mostra opzioni avanzate.

  9. Fai clic su Esperienza utente.

  10. Seleziona Shell utente di default, quindi inserisci il percorso della shell utente di default.

  11. Fai clic su OK.
