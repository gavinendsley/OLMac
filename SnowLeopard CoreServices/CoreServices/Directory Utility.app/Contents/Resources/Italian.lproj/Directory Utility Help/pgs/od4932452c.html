Aggiungere una connessione Server Active Directory

Quando ti connetti a un server Active Directory, devi conoscere il nome del server o l'indirizzo IP e avere il nome utente e la password dell'amministratore Active Directory.

Per aggiungere un server Active Directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

    Se viene visualizzato un pulsante Modifica, significa che il computer dispone di almeno una connessione a un server di directory.

  4. Fai clic sul pulsante Aggiungi (+).

  5. Dal menu a comparsa “Aggiungi una nuova directory di tipo”, scegli Active Directory, quindi inserisci le seguenti informazioni:

    • Dominio Active Directory: si tratta del nome DNS del dominio Active Directory (ad esempio, ads.company.com.)

    • ID computer: modifica l'ID che Active Directory deve utilizzare per il server (facoltativo). Si tratta del nome NetBIOS del server. Tale nome deve contenere un massimo di 15 caratteri e non deve includere caratteri speciali e segni di punteggiatura. Se possibile, utilizza per il server lo stesso nome non qualificato utilizzato per il relativo host DNS. Se, ad esempio, il server DNS utilizza “server.example.com”, assegna al tuo server il nome “server”.

    • Nome utente e password amministratore AD: inserisci il nome utente e la password dell'amministratore Active Directory.

  6. Fai clic su OK.
