Modificare una connessione server di directory

Puoi utilizzare le preferenze Account per modificare i server di directory a cui sei connesso.

Per modificare una connessione server di directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Modifica.

  4. Dall'elenco di server di directory seleziona il server di directory che desideri modificare.

  5. Fai clic sul pulsante Modifica (/).

  6. Modifica le impostazioni del server di directory.

  7. Fai clic su OK.
