Inizio
Configurazione client automatica

Se il tuo computer Mac OS X v10.6 è connesso a un network ma non a un dominio di directory, il computer Mac OS X v10.6 utilizzerà Bonjour per rilevare server Mac OS X v10.6.

Se viene rilevato un server Mac OS X v10.6 con un master Open Directory che ha in esecuzione almeno un servizio autoconfigurabile (come AFP, SMB, VPN, Mail, iCal, iChat, notifiche) e un account utente sul dominio di directory che corrisponde al nome breve o al nome lungo dell'utente del computer Mac OS X v10.6 al momento connesso, verrà visualizzato un invito sul computer Mac OS X v10.6.

L'invito fornisce assistenti per la configurazione della tua connessione a un server di directory. Puoi accettare o rifiutare la connessione a un server oppure decidere di connetterti in un momento successivo.

Se il computer rileva più di un server di directory Mac OS X v10.6 al quale puoi connetterti, i server vengono elencati in un menu a comparsa di invito. Se decidi di accedere a un server, l'assistente apre, connette e configura le tue applicazioni con i servizi in esecuzione sul server di directory.

Per connetterti a un dominio di directory con l'assistente automatico:

  1. Quando viene visualizzato l'assistente, fai clic su “Configura servizi”.

  2. Nel menu a comparsa Server scegli il dominio di directory al quale desideri accedere.

    Sul tuo network possono essere disponibili più domini di directory.

    Se il server al quale desideri connetterti non è presente nell'elenco, inserisci il nome del server o l'indirizzo IP nel campo Server e fai clic su OK.

  3. Inserisci nome utente e password di un account con privilegi sufficienti affinché il computer possa accedere al dominio di directory desiderato.

    Se non disponi di queste informazioni, contatta l’amministratore di directory.

  4. Inserisci la password per l'account utente che appare in “Inserisci la password per l'account nome utente su questo computer”.

    L'assistente modifica la password dell'account di directory degli utenti sul server per fare corrispondere la password sull'account del computer locale.

  5. Fai clic su Continua.

  6. Al termine dello strumento UpgradeUser, fai clic su Continua.

  7. Quando viene visualizzato il messaggio che spiega i servizi configurati e che richiede di eseguire il logout, fai clic su Logout.

    Una volta effettuato l'accesso, puoi iniziare a utilizzare i nuovi servizi.
