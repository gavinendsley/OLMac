Modificare gli account utente e altri record in Active Directory

Puoi utilizzare Workgroup Manager per effettuare modifiche agli account utente, agli account gruppo, ai gruppi di computer e altri record in un dominio Active Directory. Puoi anche utilizzare Workgroup Manager per eliminare i record in un dominio Active Directory.

Se lo schema Active Directory è stato esteso per includere i tipi di record Mac OS X standard (classi di oggetti) e gli attributi, puoi utilizzare Workgroup Manager per creare e modificare i gruppi di computer nel dominio Active Directory.

Per informazioni aggiuntive, consulta "Anteprima gestione utente".

Per creare degli account utente o gruppo in un dominio Active Directory, utilizza gli strumenti di amministrazione di Microsoft Active Directory su un computer di amministrazione del server Windows.
