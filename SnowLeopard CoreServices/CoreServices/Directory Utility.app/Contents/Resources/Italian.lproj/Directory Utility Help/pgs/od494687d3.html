Definire i criteri di ricerca personalizzata

Se usi Utility Directory, potrai configurare il computer Mac OS X in modo tale da usare un elenco ad hoc di domini di directory per le ricerche di autenticazione e di contatti.

Un elenco ad hoc inizia con il dominio di directory locale del computer a cui potrai aggiungere domini di directory Open Directory (e altri domini LDAP), un dominio Active Directory, domini di directory condivisi, documenti di configurazione BSD e un dominio NIS.

Se il dominio di una directory specificato nel criterio di ricerca personalizzato di un computer non è disponibile, si verifica un ritardo all’avvio del computer.

Per specificare un elenco personalizzato di domini di directory per un criterio di ricerca:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su “Politica di ricerca” e scegli una politica di ricerca.

    • Autenticazione: mostra la politica di ricerca utilizzata per l'autenticazione e la maggior parte dei dati di tipo amministrativo.

    • Contatti: mostra la politica di ricerca utilizzata per le informazioni sui contatti in applicazioni quali “Rubrica Indirizzi”.

  7. Scegli “Percorso personalizzato” dal menu a comparsa Cerca.

  8. Aggiungi i domini di directory facendo clic su Aggiungi, selezionando directory e facendo di nuovo clic su Aggiungi.

  9. Modifica l'ordine dei domini di directory elencati come preferisci trascinandoli nell'elenco.

  10. Rimuovi i domini di directory elencati che non desideri nelle politiche di ricerca selezionandoli e facendo clic sul pulsante Elimina (–).

  11. Conferma la rimozione facendo clic su OK, quindi fai clic su Applica.

Per aggiungere una directory non elencata nelle directory disponibili, assicurati che il computer sia stato configurato per accedere alla directory. Per ulteriori informazioni, consulta:
