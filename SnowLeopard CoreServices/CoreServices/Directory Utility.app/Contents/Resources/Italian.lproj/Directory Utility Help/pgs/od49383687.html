Impostare account utente mobili in Active Directory

Puoi abilitare o disabilitare gli account utente mobili in Active Directory su un computer che è configurato per utilizzare il connettore Active Directory di Utility Directory. Gli utenti con account mobili possono eseguire il login utilizzando le loro credenziali Active Directory quando il computer non è connesso al server Active Directory.

Il connettore Active Directory memorizza le credenziali per un account utente mobile quando l’utente esegue il login mentre il computer è connesso al dominio Active Directory. La memorizzazione delle credenziali non richiede che venga modificato lo schema Active Directory.

Se lo schema Active Directory è stato ampliato in modo da includere gli attributi del client gestito da Mac OS X, quelle impostazioni dell’account mobile sono utilizzate al posto delle impostazioni dell'account mobile del connettore Active Directory.

Puoi creare account mobili automaticamente o puoi richiedere agli utenti di Active Directory di confermare la creazione di un account mobile.

Per abilitare o disabilitare gli account mobili in un dominio Active Directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Nell'elenco dei servizi, seleziona Active Directory, quindi fai clic sul pulsante Modifica (/).

  8. Se le opzioni avanzate sono nascoste, fai clic su Mostra opzioni avanzate.

  9. Fai clic su “Esperienza utente”, quindi seleziona “Crea account mobile al momento del login” ed eventualmente fai clic su “Chiedi conferma prima di creare un account mobile”.

    • Se entrambi le opzioni sono selezionate, ogni utente decide se creare un account mobile durante il login. Quando un utente esegue il login a Mac OS X utilizzando un account utente di Active Directory oppure esegue il login come utente network, l'utente vede una finestra di dialogo con i controlli per creare immediatamente un account mobile.

    • Se la prima opzione è selezionata e la seconda non lo è, gli account mobili sono creati quando gli utenti eseguono il login.

    • Se la prima opzione non è selezionata, la seconda è disabilitata.

  10. Fai clic su OK.
