Abilitare o disabilitare il servizio Active Directory

Puoi utilizzare Utility Directory per abilitare o disabilitare l'utilizzo dei servizi Active Directory forniti da un server Windows. Active Directory è il servizio di directory dei server Windows 2000 e versioni successive.

Se disabiliti i servizi Active Directory e nei criteri di ricerca personalizzati sono presenti dei domini Active Directory condivisi, essi vengono elencati in rosso nel pannello Autenticazione o Contatti di Politica di ricerca in Utility Directory.

Per abilitare o disabilitare l'accesso ad Active Directory:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Accanto ad Active Directory, seleziona o deseleziona il riquadro e fai clic su Applica.
