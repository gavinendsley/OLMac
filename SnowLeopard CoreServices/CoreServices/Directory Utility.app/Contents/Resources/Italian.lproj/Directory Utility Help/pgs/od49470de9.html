Definire criteri di ricerca automatici

Se utilizzi Utility Directory, potrai configurare il computer Mac OS X in modo tale che i criteri di ricerca di autenticazione e di contatti vengano configurati automaticamente.

Una ricerca definita automaticamente include il dominio di directory locale. Può anche includere un server di directory LDAP specificato dal servizio DHCP.

Questa è la configurazione di default per i criteri di ricerca per l'autenticazione e i contatti.

Nota:   alcune applicazioni, come Mail di Mac OS X e “Rubrica Indirizzi”, sono in grado di accedere direttamente alle directory LDAP, senza utilizzare Open Directory. Per configurare una di queste applicazioni in modo tale che accedano direttamente alle directory LDAP, apri l'applicazione e imposta la preferenza opportuna.
Importante:   se configuri Mac OS X per utilizzare politiche di ricerca di autenticazione automatica e un server o un dominio di directory condivisa con DHCP, aumenterai il rischio che un utente malintenzionato assuma il controllo del tuo computer. Il rischio è ancora più alto se il computer è configurato per collegarsi a un network wireless. Per ulteriori informazioni, consulta Proteggere i computer dall'intrusione di un server DHCP.
Per definire in automatico i criteri di ricerca:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Accedi o Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su “Politica di ricerca” e scegli e una politica di ricerca:

    • Autenticazione: mostra la politica di ricerca utilizzata per l'autenticazione e la maggior parte dei dati di tipo amministrativo.

    • Contatti: mostra la politica di ricerca utilizzata per le informazioni sui contatti in applicazioni quali “Rubrica Indirizzi”.

  7. Scegli Automatico dal menu a comparsa Cerca, quindi fai clic su Applica.

  8. In Preferenze di Sistema, accertati che le preferenze del network del computer siano configurate in modo da utilizzare DHCP o DHCP con indirizzo IP manuale.
