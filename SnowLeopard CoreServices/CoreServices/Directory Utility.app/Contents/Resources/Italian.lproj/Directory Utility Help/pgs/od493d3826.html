Modificare le impostazioni di connessione di una directory LDAP

Puoi utilizzare Utility Directory per modificare le impostazioni di connessione di una configurazione in grado di specificare come deve accedere il computer a una directory LDAPv3 o LDAPv2.

Per modificare le impostazioni di connessione per l'accesso a una directory LDAP:

  1. Apri “Preferenze di Sistema” e fare clic su Account.

  2. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  3. Fai clic su “Opzioni login” e quindi su Modifica.

  4. Fai clic su “Apri Utility Directory”.

  5. Se l'icona del lucchetto è bloccata, fai clic su di essa e inserisci il nome e la password di un amministratore per sbloccarla.

  6. Fai clic su Servizi.

  7. Nell'elenco dei servizi, seleziona LDAPv3, quindi fai clic sul pulsante Modifica (/).

  8. Se l'elenco delle configurazioni dei server è nascosto, fai clic su Mostra Opzioni.

  9. Seleziona la configurazione di un server dall'elenco e fai clic su Modifica.

  10. Fai clic su Connessione e modifica una qualsiasi delle seguenti impostazioni.

    • Nome configurazione: identifica questa configurazione nell'elenco delle configurazioni della directory LDAP (puoi anche modificare il nome dall'elenco delle configurazioni della directory LDAP).

    • Nome server o indirizzo IP: specifica il nome DNS del server o il suo indirizzo IP (puoi anche modificarlo dall'elenco delle configurazioni della directory LDAP).

    • Timeout dell'apertura/chiusura tra: specifica la durata massima di tempo di un tentativo di connessione prima che venga annullato.

    • Timeout dell'interrogazione tra: specifica la durata massima di tempo di un'interrogazione prima che venga annullata.

    • Tentativo di ricollegamento tra: specifica il numero di secondi di attesa prima di un nuovo tentativo di connessione se il server LDAP non risponde. Puoi aumentare questo valore per evitare continui tentativi di riconnessione.

    • La connessione diventerà inattiva tra: specifica il numero di minuti in cui si consente a una connessione inattiva o che non risponde di rimanere aperta.

    • Cripta usando SSL: determina se le comunicazioni con la directory LDAP devono essere criptate utilizzando una connessione SSL. (puoi anche modificare questa impostazione dall'elenco delle configurazioni della directory LDAP). Prima di selezionare il riquadro SSL, verifica con l'amministratore Open Directory se è necessario utilizzare SSL.

    • Usa porta personalizzata: specifica un numero di porta diverso da quello della porta standard per le connessioni LDAP (389 senza SSL, 636 con SSL).

    • Ignora i riferimenti del server: determina se ignorare o seguire un riferimento del server LDAP per cercare informazioni su altri server LDAP o repliche. I riferimenti del server possono aiutare un computer nella ricerca di informazioni, ma possono anche rallentare i login o provocare altri ritardi se il computer deve comunicare i riferimenti ad altri server LDAP.

    • Usa LDAPv2 (solo lettura): determina se utilizzare il vecchio protocollo LDAPv2 per l'accesso di sola lettura a una directory LDAP.
