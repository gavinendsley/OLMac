Informazioni sulle connessioni del server di directory

Puoi utilizzare le preferenze Account per connettere i computer ai server di directory. Puoi visualizzare un elenco di server di directory ai quali il computer è connesso, facendo clic sul pannello “Opzioni login” delle preferenze Account. Il tuo computer Mac OS X ha accesso ai server dell'elenco per informazioni sull'utente e altri dati di amministratore archiviati nel dominio della directory dei server di directory.

Quando aggiungi o elimini un server nell'elenco dei server di directory, le voci associate con quel server di directory sono aggiunte o eliminate dall'elenco dei servizi, delle autenticazioni e dei contatti. Tuttavia, se rimuovi le voci associate all'interno degli elenchi dei servizi, delle autenticazioni e dei contatti, il server di directory non viene rimosso dall'elenco dei Server di directory.

I computer Mac OS X v10.6 possono connettersi a un server di directory Open Directory, Active Directory o LDAP. Se non sai a quale server connetterti, chiedi all'amministratore di rete.

Importante:   se il nome del tuo computer contiene un trattino, potresti non essere in grado di collegarti a un dominio di directory quale LDAP o Active Directory. Per stabilire il vincolo, utilizza un nome di computer che non contiene simboli.
