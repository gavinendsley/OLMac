Automatiserad klientkonfiguration

Om en dator med Mac OS X 10.6 är ansluten till ett nätverk men inte till någon katalogdomän, så kommer den att använda Bonjour till att hitta servrar med Mac OS X 10.6.

Om den hittar en server med Mac OS X 10.6 som kör en Open Directory-master med minst en automatkonfigurerad tjänst igång (som AFP, SMB, VPN, Mail, iCal, iChat, meddelanden) och ett användarkonto som stämmer med det långa eller korta namnet på den inloggade Mac OS X 10.6-användaren, så visas en inbjudan på datorn med Mac OS X 10.6.

Denna inbjudan erbjuder hjälp med att ställa in förbindelsen till en katalogserver. Du kan godkänna eller avböja att ansluta till en server, eller bestämma dig för att ansluta senare.

Om datorn hittar mer än en Mac OS X 10.6-katalogserver som den kan ansluta till, så visas dessa i popupmenyn i inbjudan. Om du ansluter till en server så öppnar, ansluter och ställer assistenten in programmen med de tjänster som katalogservern erbjuder.

Anslut till en katalogdomän med den automiserade assistenten så här:

  1. Klicka på ”Setup Services” när assistenten visas.

  2. Välj den katalogdomän som du vill ansluta till från popupmenyn Server.

    Det kan finnas mer än en katalogdomän tillgänglig i nätverket.

    Om servern du vill ansluta till inte finns med i listan, anger du dess servernamn eller IP-adress i fältet Server och klicka på OK.

  3. Ange användarnmnet och lösenordet till ett konto som har behörighet att ansluta en dator till den valda katalogdomänen.

    Kontakta katalogadministratören om du inte har denna information.

  4. Ange lösenordet för det användarkonto som visas under ”Ange lösenordet för kontot användarnamn på den här datorn”.

    Assistenten ändrar lösenordet till användarens katalogkonto på servern så att det överensstämmer med kontot på den lokala datorn.

  5. Klicka på Fortsätt.

  6. Klicka Fortsätt när verktyget UpgradeUser är färdigt.

  7. Klicka på ”Logga ut” när ett meddelande visas om att tjänsterna ställts in och att du ska logga ut.

    Du kan använda de nya tjänsterna när du loggar in.
