Definiera sökpolicyer för lokal katalog

Med Katalogverktyg kan du ställa in att sökpolicyerna för autentisering och kontakter i Mac OS X endast ska använda datorns lokala katalog.

En sökpolicy som endast använder den lokala katalogen begränsar vilken tillgång en dator har till autentiseringsinformation och andra administrativa data.

Begränsas en dators sökpolicy för autentisering till endast den lokala katalogen kan bara användare som har lokala konton logga in.

Så här anger du att en sökpolicy endast ska använda den lokala katalogdomänen (lokal katalog):

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Anslut eller Redigera.

  4. Klicka på Katalogverktyg.

  5. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  6. Klicka på Sökpolicy och välj en sökpolicy:

    • Autentisering: Visar sökpolicyn som används för autentisering och de flesta andra administrativa data.

    • Kontakter: Visar sökpolicyn som används för kontaktinformation i program som Adressbok.

  7. Välj ”Lokal katalog” från popupmenyn Sök och klicka sedan på Utför.
