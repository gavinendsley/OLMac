Ställa in Katalogverktyg på en fjärrserver

Med Katalogverktyg på datorn kan du ställa in och hantera hur en server med Mac OS X Server på en fjärrserver ansluter till katalogtjänster.

Så här konfigurerar du katalogåtkomst på en fjärrserver:

  1. Öppna Systeminställningar i datorn och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Anslut eller Redigera.

    Om knappen Redigera visas så har datorn minst en anslutning till en katalogserver.

  4. Klicka på Katalogverktyg.

  5. Välj Anslut från Arkiv-menyn.

  6. Ange följande anslutnings- och autentiseringsinformation för servern du vill konfigurera.

    • Adress: Ange DNS-namnet eller IP-adressen till servern du vill konfigurera.

    • Användarnamn: Ange användarnamnet på en administratör på servern.

    • Lösenord: Ange lösenordet till det användarnamn du har angivit.

  7. Klicka på Anslut.

  8. Klicka på flikarna Tjänster och Sökpolicy och gör nödvändiga ändringar.

    Ändringarna påverkar den fjärrserver som du anslutit till i föregående steg.

  9. Välj ”Koppla från” från Arkiv-menyn på din dator.
