Lär dig använda Katalogverktyg

Katalogverktyg avgör vilka katalogtjänster en Mac OS X-dator använder och hur den ansluter till specifika katalogdomäner. Katalogverktyg definierar också sökrutiner för att hitta autentiserings- och kontaktinformation i specifika katalogdomäner.

Ställa in Katalogverktyg

Med Katalogverktyg kan du ställa in och hantera hur datorn ansluter till katalogtjänster. Med Katalogverktyg kan du även fjärrkonfigurera hur datorn ansluter till katalogtjänster. Du kan också använda Katalogverktyg till att aktivera root-kontot och ändra lösenordet för root-kontot.

Ställa in åtkomst till tjänster

Med Katalogverktyg kan du aktivera och avaktivera olika typer av tjänster Mac OS X kan komma åt i ett nätverk. Tjänsterna inkluderar katalogtjänster som ger åtkomst till användarkonton och andra administrativa data som sparas i katalogdomäner.

Ställa in åtkomst till kataloger

Med Katalogverktyg kan du konfigurera datorn för åtkomst till användarkonton och annan administrativ information i angivna LDAP-kataloger, en Active Directory-domän och i en NetInfo-domän.

Ställa in sökpolicyer

Med Katalogverktyg kan du definiera den autentiseringssökpolicy som Mac OS X ska använda till att söka och hämta användarautentiseringsinformation och andra administrativa data från katalogdomäner. Katalogverktyg kan också definiera de kontaktsökpolicyer som Mac OS X ska använda till att söka och hämta namn, adresser och annan kontaktinformation från katalogdomäner.
