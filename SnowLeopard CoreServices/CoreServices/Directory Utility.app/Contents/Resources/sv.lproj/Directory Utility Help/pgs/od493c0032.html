Aktivera och avaktivera LDAP-katalogtjänster

Med Katalogverktyg kan du aktivera och avaktivera åtkomst till katalogtjänster som använder LDAP v2 eller LDAP v3. En enda insticksfil för Katalogverktyg, LDAP v3, ger tillgång till både LDAP v2 och LDAP v3.

Katalogtjänsterna som finns i Mac OS X Server använder LDAPv3 och det gäller också för många andra servrar. LDAPv3 är en öppen standard vanlig i blandade nätverk med Mac-, UNIX- och Windows-system. Vissa servrar använder den äldre versionen LDAPv2 för att tillhandahålla katalogtjänster.

Om du avaktiverar LDAP-tjänster och det finns LDAP-kataloger som är en del av en anpassad sökpolicy, visas de med rött i någon av panelerna Autentisering eller Kontakter under Sökpolicy i Katalogverktyg.

Så här aktiverar eller avaktiverar du åtkomst till LDAP-katalogtjänster:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Anslut eller Redigera.

  4. Klicka på Katalogverktyg.

  5. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  6. Klicka på Tjänster.

  7. Markera eller avmarkera kryssrutan LDAPv3 och klicka sedan på Utför.
