Ställa in åtkomst till kataloger

Med Katalogverktyg (i inställningspanelen Konton) kan du ställa in och hantera hur en dator med Mac OS X eller en server med Mac OS X Server ansluter till katalogdomäner.

Ansluta till LDAP-kataloger

Med Katalogverktyg kan du konfigurera en dator med Mac OS X eller en server med Mac OS X Server till att ansluta till bestämda LDAP-kataloger, inklusive LDAP-katalogen för en Mac OS X Server Open Directory-master. Vill du veta mer klickar du på ämnena nedan.

Använda BSD-konfigurationsfiler

Med Open Directory kan du hämta administrativa data från BSD-konfigurationsfiler som /etc/master.passwd. Sådana filer kallas också BSD flat files. Du kan också konfigurera åtkomst till BSD-konfigurationsfiler med Katalogverktyg. Vill du veta mer klickar du på ämnena nedan.
