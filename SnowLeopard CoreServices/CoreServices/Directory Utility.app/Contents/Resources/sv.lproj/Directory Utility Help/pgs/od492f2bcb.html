Hantera root-användarkontot

Med Katalogverktyg (finns i inställningspanelen Konton) kan du hantera root-användarkontot genom att aktivera och avaktivera root-användaren. Om du har aktiverat root-användarkontot kan du också använda Katalogverktyg till att ändra lösenordet för root-kontot.

Vill du veta mer om hur du hanterar root-användarkontot med Katalogverktyg läser du följande:

  • Aktivera root-användarkontot

  • Ändra lösenordet för root-användarkontot
