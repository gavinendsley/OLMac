Om katalogserveranslutning

Med inställningspanelen Konton kan du ansluta datorer till katalogservrar. Du kan visa en lista över katalogservrar som datorn är ansluten till genom att klicka på Redigera i panelen Inloggning i inställningspanelen Konton. Mac OS X-datorn ansluter till servrarna i listan för att hämta användarinformation och andra administrativa data som finns i katalogdomänen på katalogservrarna.

När du lägger till eller tar bort en server i listan Katalogservrar läggs de poster som är kopplade till den katalogservern till i eller tas bort från listorna Tjänster, Autentisering och Kontakter. Om du tar bort de kopplade posterna ur listorna Tjänster, Autentisering och Kontakter tas katalogservern däremot inte bort från listan Katalogservrar.

Datorer med Mac OS X 10.6 kan ansluta till Open Directory-, Active Directory- och Mac OS X Server-katalogservrar. Fråga nätverksadministratören om du inte vet vilken server du ska ansluta till.

Viktigt:   Om datorns namn innehåller ett bindestreck kanske du inte kan ansluta till en katalogdomän, t.ex. av typen LDAP eller Active Directory. Använd ett datornamn som inte innehåller något bindestreck när du ansluter.
