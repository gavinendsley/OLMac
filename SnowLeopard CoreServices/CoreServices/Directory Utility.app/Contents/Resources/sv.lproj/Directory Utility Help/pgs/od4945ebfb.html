Använda avancerade Active Directory-tjänstinställningar

Du kan konfigurera en dator med Mac OS X eller en server med Mac OS X Server så att den kan ansluta till en Active Directory-domän på en Windows 2000- eller Windows 2003-server.

Åtgärdsbeskrivningar och anvisningar finns nedan:

  • Om Active Directory-åtkomst

  • Ställa in åtkomst till en katalogdomän

  • Ställa in flyttbara användarkonton i Active Directory Setting

  • Ställa in hemmappar för användarkonton i Active Directory

  • Ställa in en UNIX-kommandotolk för användarkonton i Active Directory

  • Koppla UID till ett Active Directory-attribut

  • Koppla primärt grupp-ID till ett Active Directory-attribut

  • Koppla grupp-ID i gruppkonton till ett Active Directory-attribut

  • Ange en förvald Active Directory-server

  • Ändra vilka Active Directory-grupper som kan administrera datorn

  • Kontrollera autentisering från alla domäner i Active Directory-trädsamlingen

  • Koppla loss från Active Directory-servern

  • Redigera användarkonton och andra poster i Active Directory

I en del nätverk kan alternativa metoder användas för anslutning till en Active Directory-domän. Se Ställa in LDAP-åtkomst till Active Directory-domäner.
