Aktivera root-användarkontot

Med Katalogverktyg kan du aktivera root-användarkontot. Om du aktiverar root-användarkontot bör du använda ett komplext lösenord som innehåller alfanumeriska tecken och specialtecken så att lösenordet blir säkert.

VARNING: Root-användarkontot är ett obegränsat administratörskonto som används till att ändra filer som är systemkritiska. Även om du är inloggad som administratör måste du använda root-kontot eller kommandot sudo om du vill utföra systemkritiska åtgärder.

Logga aldrig in till en dator (varken fjärrinloggning eller lokalt) med root-kontot. Använd istället sudo till att genomföra åtgärder med root-kontot. Du kan begränsa åtkomsten till kommandot sudo genom att lägga till användare i filen /etc/sudoers.

Du hittar mer information om root-kontot i hjälpen för Server Admin.

Så här aktiverar du root-användarkontot:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Anslut eller Redigera.

    Om knappen Redigera visas så har datorn minst en anslutning till en katalogserver.

  4. Klicka på Katalogverktyg.

  5. Välj Redigera > Aktivera root-användare.
