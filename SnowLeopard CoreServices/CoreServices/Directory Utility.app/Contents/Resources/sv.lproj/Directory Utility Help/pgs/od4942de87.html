Ange NIS-inställningar

Med Katalogverktyg kan du skapa en konfiguration som anger hur Mac OS X ansluter till en NIS-domän (Network Information Service).

Så här skapar du en konfiguration för åtkomst till en NIS-domän:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Anslut eller Redigera.

  4. Klicka på Katalogverktyg.

  5. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  6. Klicka på Tjänster.

  7. Välj ”BSD Flat File och NIS” i listan över tjänster och klicka på redigeraknappen (/).

  8. Ange NIS-domännamnet, DNS-namnet eller IP-adressen för den server där NIS-domänen finns.

    Ta med NIS-serverns värdnamn eller IP-adress om det krävs av säkerhetsskäl, eller om servern inte finns på samma delnät som datorn du konfigurerar.

    Om du inte anger någon server kommer NIS att leta efter en NIS-server på delnätet med hjälp av ett Broadcast-protokoll.

  9. Markera ”Använd NIS-domän för autentisering” och klicka på OK.

    NIS-domänen läggs till i datorns autentiseringssökpolicy som /BSD/domän, där domän är det namn du angav i steg 4.
