Koppla loss från Active Directory-servern

Om datorn använder Katalogverktygs Active Directory-kontakt till att ansluta till en Active Directory-server kan du koppla loss datorn från Active Directory-servern.

Du kan tvinga fram losskopplingen om datorn inte kan ansluta till servern, eller om datorposten redan har tagits bort från servern.

Så här koppar du loss datorn från Active Directory-servern:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Redigera.

  4. Klicka på Katalogverktyg.

  5. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  6. Klicka på Tjänster.

  7. Markera Active Directory i listan över tjänster och klicka sedan på redigeraknappen (/).

  8. Klicka på Koppla loss och autentisera dig som en användare som har behörighet att avsluta en anslutning till Active Directory-domänen. Klicka sedan på OK.

    Om ett varningsmeddelande visas om att ID-informationen inte accepterades eller om datorn inte får kontakt med Active Directory, kan du tvinga fram en avbrytning av anslutningen genom att klicka på Tvinga losskoppling.

    Om du tvingar fram en losskoppling finns datorposten för datorn kvar i Active Directory. Meddela Active Directory-administratören så att administratören kan ta bort datorposten.

  9. I panelen Tjänster avmarkerar du Active Directory-inställningen Aktivera. Klicka sedan på Utför.
