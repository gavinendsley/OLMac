Fylla LDAP-kataloger med data för Mac OS X

När du har konfigurerat åtkomsten till LDAP-katalogdomäner och ställt in datakoppling, kan du fylla dem med poster och data för Mac OS X. För LDAP-kataloger som tillåter fjärradministration (åtkomst att läsa och skriva) kan du använda Workgroup Manager som följer med Mac OS X Server. Gör så här:

  • Identifiera de delningspunkter och delade domäner du vill länka in automatiskt i nätverksnavigeraren på användarnas datorer (det användarna ser när de klickar på Nätverk i sidofältet i ett Finder-fönster).

    Använd Sharing i Server Admin och modulen Network i Workgroup Manager.

    Vill du ha ytterligare information bläddrar du i översikterna för filtjänster och användarhantering.

  • Definiera användar- och grupposter och konfigurera deras inställningar.

    Använd modulen Accounts i Workgroup Manager.

    Vill du ha ytterligare information bläddrar du i översikten för användarhantering.

  • Ange en lista av datorer som har samma inställningar och som kan använda samma användare och grupper.

    Använd modulen Datorer i Workgroup Manager.

    Vill du ha ytterligare information bläddrar du i översikten för användarhantering.

Klicka alltid på nätverkssymbolen ovanför listan med användare och välj att öppna LDAP-katalogdomänen från popupmenyn i Workgroup Manager. Om LDAP-katalogen inte finns med under popupmenyn väljer du Annan från menyn och väljer LDAP-katalogen.

Anm: Om du vill lägga till poster och data till skrivskyddade LDAP-kataloger måste du använda verktygen på servern som LDAP-katalogen ligger på.
