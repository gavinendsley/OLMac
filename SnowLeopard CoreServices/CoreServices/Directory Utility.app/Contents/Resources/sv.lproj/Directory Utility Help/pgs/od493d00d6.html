Koppla primärt grupp-ID till ett Active Directory-attribut

På en dator som är konfigurerad till att använda Katalogverktygs Active Directory-kontakt kan du ange ett Active Directory-attribut som du vill ska kopplas till Mac OS X primära GID-attribut i användarkonton.

I vanliga fall krävs att Active Directory-schemat utökas till att inkludera ett attribut som är lämpligt för koppling till primära GID:

  • Om Active Directory-administratören utökar Active Directory-schemat genom att installera Microsofts tjänster för UNIX kan du koppla primära GID till attributet msSFU-30-Gid-Number.

  • Om Active Directory-administratören manuellt utökar Active Directory-schemat till att inkludera RFC 2307-attribut, kan du koppla primära GID till gidNumber.

  • Om Active Directory-administratören manuellt utökar Active Directory-schemat till att inkludera Mac OS X-attributet PrimaryGroupID, kan du koppla primära GID till det.

Om koppling av primära GID är avaktiverat genererar Active Directory-kontakten ett primärt GID baserat på Active Directorys standard-GUID-attribut.

VARNING: Om du senare ändrar kopplingen av ett primärt GID kan det innebära att användare inte längre kommer åt filer som skapats före ändringen.
Så här kopplar du primärt GID till ett attribut i ett utökat Active Directory-schema:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Redigera.

  4. Klicka på Katalogverktyg.

  5. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  6. Klicka på Tjänster.

  7. Markera Active Directory i listan över tjänster och klicka sedan på redigeraknappen (/).

  8. Om de avancerade alternativen är dolda klickar du på Visa avancerade alternativ.

  9. Klicka på Kopplingar.

  10. Välj ”Koppla användar-GID till attribut” och ange namnet på det Active Directory-attribut du vill koppla till det primära GID i användarkonton.

  11. Klicka på OK.
