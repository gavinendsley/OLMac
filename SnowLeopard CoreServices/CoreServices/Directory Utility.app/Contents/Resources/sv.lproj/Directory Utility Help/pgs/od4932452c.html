Lägga till en Active Directory-serveranslutning

När du ansluter till en Active Directory-server måste du känna till serverns namn eller IP-adress och använda användarnamnet och lösenordet för Active Directory-administratören.

Så här lägger du till en Active Directory-server:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Anslut eller Redigera.

    Om knappen Redigera visas så har datorn minst en anslutning till en katalogserver.

  4. Klicka på lägg till-knappen (+).

  5. Välj Active Directory i popupmenyn ”Lägg till en ny katalog av typen” och ange följande information:

    • Active Directory-domän: Det här är DNS-namnet för Active Directory-domänen (exempelvis ads.company.com.)

    • Dator-ID: Om du vill kan du redigera det ID du vill att Active Directory ska använda för servern. Det här är serverns NetBIOS-namn. Namnet får inte vara längre än 15 tecken eller innehålla specialtecken, å, ä, ö eller interpunktion. Om möjligt låter du servernamnet matcha dess okvalificerade DNS-värdnamn. Om din DNS-server exempelvis har en post för din server som heter ”server.example.com” ger du servern namnet ”server”.

    • AD-administratörsanvändarnamn och -lösenord:  Ange Active Directory-administratörens användarnamn och lösenord.

  6. Klicka på OK.
