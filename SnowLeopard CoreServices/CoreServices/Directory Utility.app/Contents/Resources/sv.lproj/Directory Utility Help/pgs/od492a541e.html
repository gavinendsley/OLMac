Välja ett lösenord

Lösenordet som hör till användarkontot måste anges av användaren när denne autentiserar sig vid inloggning eller andra tjänster. Lösenordet skiljer på små och stora bokstäver (med undantag för lösenord till SMB-LAN Manager) och göms på skärmen när det skrivs in.

Oberoende av typen av lösenord för användaren följer här några råd om att välja lösenord till användarkonton på Mac OS X Server:

  • Ett lösenord bör innehålla bokstäver, siffror och symboler i en kombination som inte lätt kan gissas av ej obehöriga. Lösenord bör inte bestå av ord. Bra lösenord innehåller siffror och symboler (som # eller %), eller så kan de bestå av den första bokstaven av orden i en mening. Använd både stora och små bokstäver.

  • Undvik mellanslag och tecken som kräver alternativ-tangenten.

  • Undvik tecken som inte kan nås på de datorer som användaren har, eller som kan kräva kunskap om en speciell tangentkombination på vissa tangentbord eller plattformar.

  • Vissa nätverksprotokoll hanterar inte lösenord som innehåller mellanslag, när de kommer först, mitt i lösenordet eller sist.

  • Ett tomt lösenord rekommenderas inte. Open Directory och vissa andra system (som LDAP bind) hanterar inte tomma lösenord.

  • Använd endast tecken i ASCII till lösenord, så att de kan hanteras från så många system som möjligt.
