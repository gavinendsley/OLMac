Aktivera och avaktivera Active Directory-tjänster

Med Katalogverktyg kan du aktivera och avaktivera användningen av de Active Directory-tjänster som tillhandahålls av Windows-servrar. Active Directory är den katalogtjänst som servrar med Microsoft Windows 2000 och senare använder.

Om du avaktiverar Active Directory-tjänster och det finns Active Directory-domäner som är en del av en anpassad sökpolicy, visas de med rött i någon av panelerna Autentisering eller Kontakter under Sökpolicy i Katalogverktyg.

Så här aktiverar eller avaktiverar du åtkomst till Active Directory:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Anslut eller Redigera.

  4. Klicka på Katalogverktyg.

  5. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  6. Klicka på Tjänster.

  7. Markera eller avmarkera kryssrutan Active Directory och klicka sedan på Utför.
