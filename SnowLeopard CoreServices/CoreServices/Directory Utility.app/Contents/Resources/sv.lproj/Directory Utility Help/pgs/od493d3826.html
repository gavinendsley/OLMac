Ändra anslutningsinställningarna för en LDAP-katalog

Med Katalogverktyg kan du ändra anslutningsinställningarna för en konfiguration som anger hur datorn ansluter till en LDAPv3- eller LDAPv2-katalog.

Så här ändrar du anslutningsinställningarna för åtkomst till en LDAP-katalog:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Redigera.

  4. Klicka på Katalogverktyg.

  5. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  6. Klicka på Tjänster.

  7. Markera LDAPv3 i listan över tjänster och klicka sedan på redigeraknappen (/).

  8. Om listan över serverkonfigurationer inte visas klickar du på Visa alternativ.

  9. Markera en serverkonfiguration i listan och klicka på Redigera.

  10. Klicka på Anslutning och ändra följande inställningar:

    • Konfigurationsnamn: Identifierar konfigurationen i listan med LDAP-katalogkonfigurationer. (Du kan också ändra namnet i listan med LDAP-katalogkonfigurationer.)

    • Servernamn eller IP-adress: Anger serverns DNS-namn eller dess IP-adress. (Du kan också ändra alternativen i listan med LDAP-katalogkonfigurationer.)

    • Maxtidsgräns för öppning/stängning: Anger hur lång tid som ska gå innan ett försök att ansluta till en LDAP-server avbryts.

    • Maxtidsgräns för förfrågan: Anger hur lång tid som ska gå innan en fråga som skickats avbryts.

    • Återkopplingsförsök efter: Anger hur många sekunder som ska gå innan ett försök att återansluta sker om LDAP-servern inte svarar. Genom att öka värdet kan du förhindra oavbrutna anslutningsförsök.

    • Maxtidsgräns för anslutning: Anger hur många minuter en anslutning som är overksam eller inte svarar förblir öppen.

    • Kryptera med SSL: Anger om anslutningar till LDAP-katalogen ska krypteras med SSL. (Du kan också ändra den här inställningen i listan med LDAP-katalogkonfigurationer.) Fråga Open Directory-administratören om SSL behövs innan du slår på SSL.

    • Använd anpassad port: Anger att du vill använda ett annat portnummer än det förinställda för LDAP-anslutningar (389 utan SSL, 636 med SSL.

    • Ignorera serverremitteringar: Anger om en LDAP-servers remittering att söka i andra LDAP-servrar eller i kopior efter information ska ignoreras eller följas. Serverremitteringar kan hjälpa en dator att hitta information, men de kan också fördröja inloggning eller orsaka andra fördröjningar om datorn måste verifiera remitteringar till andra LDAP-servrar.

    • Använd LDAPv2 (skrivskyddad): Anger om det äldre LDAP v2-protokollet ska användas för skrivskyddad åtkomst till en LDAP-katalog.
