Ändra vilka Active Directory-grupper som kan administrera datorn

På en dator som är konfigurerad till att använda Katalogverktygs Active Directory-kontakten kan du identifiera Active Directory-gruppkonton vars medlemmar du vill ska ha administratörsbehörigheter för datorn.

Användare som är medlemmar i dessa Active Directory-gruppkonton kan utföra administrativa åtgärder som installation av programvara på den Mac OS X-dator du konfigurerar.

Så här lägger du till eller tar bort Active Directory-gruppkonton vars medlemmar har administratörsbehörigheter:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Redigera.

  4. Klicka på Katalogverktyg.

  5. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  6. Klicka på Tjänster.

  7. Markera Active Directory i listan över tjänster och klicka sedan på redigeraknappen (/).

  8. Om de avancerade alternativen är dolda klickar du på Visa avancerade alternativ.

  9. Klicka på Administrativt.

  10. Välj ”Tillåt administration av” och ändra listan över Active Directory-gruppkonton vars medlemmar du vill ska ha administratörsbehörigheter:

    • Lägg till en grupp genom att klicka på lägg till-knappen (+) och ange Active Directory-domännamnet, ett bakstreck och gruppkontonamnet (till exempel ADS\Domain Admins,IL2\Domain Admins).

    • Ta bort en grupp genom att markera den i listan och klicka på ta bort-knappen (–).

  11. Klicka på OK.
