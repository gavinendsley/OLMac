Ställa in flyttbara användarkonton i Active Directory

Du kan aktivera eller avaktivera användningen av flyttbara Active Directory-användarkonton på en dator som är konfigurerad att använda Active Directory-kontakten i Katalogverktyg. Användare med flyttbara konton kan logga in med sin Active Directory-identitetsinformation när datorn inte är ansluten till Active Directory-servern.

Active Directory-kontakten sparar ID-informationen för en användares flyttbara konto i cacheminnet när användaren loggar in medan datorn är ansluten till Active Directory-domänen. Funktionen kräver inte några ändringar i Active Directory-schemat.

Om Active Directory-schemat har utökats och inkluderar Mac OS X-attribut för hanterade klienter, används deras inställningar för flyttbara konton istället för Active Directory-kontaktens inställningar för flyttbara konton.

Du kan låta flyttbara konton skapas automatiskt, eller så kan du kräva att Active Directory-användare måste bekräfta innan ett flyttbart konto skapas.

Så här aktiverar eller avaktiverar du flyttbara användarkonton i en Active Directory-domän:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Anslut eller Redigera.

  4. Klicka på Katalogverktyg.

  5. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  6. Klicka på Tjänster.

  7. Markera Active Directory i listan över tjänster och klicka sedan på redigeraknappen (/).

  8. Om de avancerade alternativen är dolda klickar du på Visa avancerade alternativ.

  9. Klicka på Användarupplevelse och sedan på ”Skapa mobilt konto vid inloggning”. Om du vill kan du också klicka på ”Kräv bekräftelse innan ett mobilt konto skapas”.

    • Om du väljer bägge alternativen kommer varje användare själv att få välja om ett flyttbart konto ska skapas vid inloggningen. När en användare loggar in till Mac OS X med ett Active Directory-användarkonto, eller loggar in som nätverksanvändare, visas en dialogruta med reglage för att skapa ett flyttbart konto omedelbart.

    • Om du väljer det första alternativet men inte det andra skapas flyttbara konton när användarna loggar in.

    • Om du inte har valt det första alternativet är det andra nertonat.

  10. Klicka på OK.
