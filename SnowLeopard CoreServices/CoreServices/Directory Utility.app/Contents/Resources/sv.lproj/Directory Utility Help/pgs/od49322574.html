Övervaka katalogserveranslutningar

Det går att använda inställningspanelen Konton till att övervaka status på de katalogservrar som datorn är ansluten till. Informationen kan vara till hjälp när du försöker ta reda på varför du inte kan ansluta till en viss katalogserver.

Så här övervakar du status för en katalogserver:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Redigera.

  4. Lägg märke till färgen på statuspunkten till vänster om katalogservern:

    • Grön: Katalogservern svarar på frågor från datorn.

    • Gul: Datorn väntar på svar från katalogservern.

    • Röd: Katalogservern svarar inte på frågor från datorn.
