Ange en förvald Active Directory-server

På en dator som är konfigurerad till att använda Katalogverktygs Active Directory-kontakt kan du ange DNS-namn för servern vars Active Directory-domän du vill att datorn ska ansluta till som förval.

Om servern blir otillgänglig vid senare tillfälle använder Active Directory-kontakten en närliggande server i trädsamlingen.

Om alternativet inte är valt använder Active Directory-kontakten den närmaste Active Directory-domänen i trädsamlingen.

Så här anger du vilken server du vill att Active Directory-kontakten ska ansluta till som förval:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Redigera.

  4. Klicka på Katalogverktyg.

  5. Klicka på Tjänster.

  6. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  7. Markera Active Directory i listan över tjänster och klicka sedan på redigeraknappen (/).

  8. Om de avancerade alternativen är dolda klickar du på Visa avancerade alternativ.

  9. Klicka på Administrativt.

  10. Välj ”Föredra denna domänserver” och ange Active Directory-serverns DNS-namn.

  11. Klicka på OK.
