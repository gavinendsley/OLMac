Redigera användarkonton och andra poster i Active Directory

Med Workgroup Manager kan du göra ändringar i användarkonton, gruppkonton, datorgrupper och andra poster i en Active Directory-domän. Du kan också radera poster i en Active Directory-domän med Workgroup Manager.

Om Active Directory-schemat har utökats till att innehålla standard-Mac OS X-posttyper (objektklasser) och -attribut kan du använda Workgroup Manager till att skapa och redigera datorgrupper i Active Directory-domän.

Vill du ha ytterligare information bläddrar du i översikten för användarhantering.

Vill du skapa användarkonton eller gruppkonton i en Active Directory-domän använder du adminstrationsverktyget Microsoft Active Directory på en Windows-serveradministrationsdator.
