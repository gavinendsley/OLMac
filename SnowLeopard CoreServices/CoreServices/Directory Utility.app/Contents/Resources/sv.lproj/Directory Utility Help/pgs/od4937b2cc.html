Ändra en LDAP-anslutnings frågetimeout

Med Katalogverktyg kan du ange hur länge Open Directory väntar innan en fråga skickad till LDAP-servern avbryts.

Så här ändrar du en LDAP-anslutnings frågetimeout:

  1. Öppna Systeminställningar och klicka på Konton.

  2. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  3. Klicka på Inloggning och sedan på Redigera.

  4. Klicka på Katalogverktyg.

  5. Om symbolen med ett hänglås är låst så låser du upp den genom att klicka på den och anger ett administratörsnamn och -lösenord.

  6. Klicka på Tjänster.

  7. Markera LDAPv3 i listan över tjänster och klicka sedan på redigeraknappen (/).

  8. Om listan över serverkonfigurationer inte visas klickar du på Visa alternativ.

  9. Markera en serverkonfiguration i listan och klicka på Redigera.

  10. Klicka på Anslutning och ange sedan ett värde för ”Maxtidsgräns för förfrågan __ sekunder”.

    Det förvalda värdet är 120 sekunder.
