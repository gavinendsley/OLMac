Förbereda en skrivskyddad LDAP-katalog för Mac OS X

Om du vill att en dator med Mac OS X ska kunna få administrativa data från en skrivskyddad LDAP-katalog måste de data som finns i katalogen vara i det format Mac OS X kräver. Du kanske måste lägga till, ändra eller ordna om data i den skrivskyddade LDAP-katalogen.

Eftersom Mac OS X inte kan skriva data till en skrivskyddad katalog måste du använda andra verktyg när du vill göra ändringarna. Verktygen måste finnas på den server där LDAP-katalogen ligger.

Så här förbereder du en skrivskyddad LDAP-katalog för Mac OS X:

  1. Gå till servern som den skrivskyddade LDAP-katalogen ligger på och ställ in den så att den stöder LDAP-baserad autentisering och lösenordskontroll.

  2. Ändra LDAP-katalogens objektklasser och attribut så att de tillhandahåller de uppgifter Mac OS X behöver.

Information om de data som krävs av Mac OS X-katalogtjänster finns i Open Directory Administration.
