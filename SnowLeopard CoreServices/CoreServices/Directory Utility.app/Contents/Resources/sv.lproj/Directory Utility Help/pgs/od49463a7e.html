Göra inställningar för data i BSD-konfigurationsfiler

Om du vill att en dator med Mac OS X ska kunna få administrativa data från BSD-konfigurationfiler, måste dessa data finnas i filerna i det format Mac OS X kräver.

Du kan behöva lägga till, ändra eller ordna om data i filerna. Workgroup Manager kan inte göra ändringar i BSD-konfigurationsfiler, så du måste göra ändringarna i en textredigerare eller liknande verktyg.

Tabellen nedan ger namnen på filerna och vad de innehåller.

BSD-konfigurationsfil
Innehåller

/etc/master.passwd

Användarnamn, lösenord, ID:n, primära grupp-ID:n och så vidare

/etc/group

Gruppnamn, ID:n och medlemmar

/etc/fstab

NFS-inlänkningar

/etc/hosts

Datornamn och -adresser

/etc/networks

Nätverksnamn och -adresser

/etc/services

Tjänstenamn, portar och protokoll

/etc/protocols

IP-protokollnamn och -nummer

/etc/rpcs

Open Network Computing RPC-servrar

/etc/printcap

Skrivarnamn och -funktioner

/etc/bootparams

Bootparam-inställningar

/etc/bootp

Bootp-inställningar

/etc/aliases

E-postalias och -distributionslistor

/etc/netgroup

Gruppnamn och medlemmar inom ett helt nätverk
