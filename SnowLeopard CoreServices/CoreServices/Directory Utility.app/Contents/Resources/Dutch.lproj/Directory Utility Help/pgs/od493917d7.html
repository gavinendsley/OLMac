Een alleen-lezen LDAP-adreslijst voorbereiden voor Mac OS X

Als u vanaf een computer met Mac OS X beheergegevens wilt ophalen uit een alleen-lezen LDAP-adreslijst, moeten deze gegevens een structuur hebben die wordt herkend door Mac OS X. Het is mogelijk dat u gegevens in de alleen-lezen LDAP-adreslijst moet toevoegen, wijzigen of anders indelen.

Aangezien u Mac OS X niet kunt gebruiken om gegevens naar een alleen-lezen adreslijst te schrijven, moet u andere programma's gebruiken om de wijzigingen door te voeren. Deze programma's moeten aanwezig zijn op de server waarop de alleen-lezen LDAP-adreslijst is geplaatst.

Een alleen-lezen LDAP-adreslijst voorbereiden voor Mac OS X

  1. Ga naar de server met de alleen-lezen LDAP-adreslijst en configureer deze voor identiteitscontrole via LDAP en wachtwoordcontrole.

  2. Wijzig de objectklassen en kenmerken van de LDAP-adreslijst, zodat Mac OS X over de benodigde gegevens kan beschikken.

Raadpleeg voor een specificatie van de gegevens die vereist zijn voor Mac OS X-adreslijstvoorzieningen de handleiding Open Directory Administration (Engelstalig).
