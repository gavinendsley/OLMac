Gebruikersaccounts en andere records in Active Directory wijzigen

Gebruik Workgroup Manager om gebruikersaccounts, groepsaccounts, computergroepen en andere records in een Active Directory-domein te wijzigen. U kunt ook records in een Active Directory-domein verwijderen met Workgroup Manager.

Als het Active Directory-schema is uitgebreid met standaardrecordtypen (objectklassen) en -kenmerken van Mac OS X, kunt u Workgroup Manager gebruiken voor het aanmaken en wijzigen van computergroepen in het Active Directory-domein.

Lees voor aanvullende informatie het onderwerp User Management Overview.

Om gebruikers- of groepsaccounts aan te maken in een Active Directory-domein, gebruikt u de Microsoft Active Directory-beheerprogramma's op een beheercomputer van Windows Server.
