Adreslijsthulpprogramma configureren op een externe server

Met Adreslijsthulpprogramma kunt u configureren en beheren op welke wijze Mac OS X Server op een externe server toegang krijgt tot adreslijstvoorzieningen.

Adreslijsttoegang configureren op een externe server

  1. Open Systeemvoorkeuren op uw computer en klik op 'Accounts'.

  2. Als het hangslotsymbool is vergrendeld, ontgrendelt u het door erop te klikken en de naam en het wachtwoord van een beheerder op te geven.

  3. Klik op 'Inlogopties' en vervolgens op 'Verbind' of 'Wijzig'.

    Als u een knop 'Wijzig' ziet, heeft uw computer minimaal één verbinding met een adreslijstserver.

  4. Klik op 'Open Adreslijsthulpprogramma'.

  5. Kies 'Verbind' uit het Archief-menu.

  6. Voer de volgende verbindingsgegevens en gegevens voor identiteitscontrole in voor de server die u wilt configureren.

    • 'Adres': Geef hier de DNS-naam of het IP-adres op van de server die u wilt configureren.

    • 'Gebruikersnaam': Geef hier de gebruikersnaam op van een beheerder op de server.

    • 'Wachtwoord': Geef hier het wachtwoord op voor de gebruikersnaam die u hebt ingevoerd.

  7. Klik op 'Verbind'.

  8. Klik op de tabs 'Voorzieningen' en 'Zoekconfiguratie' en wijzig, indien nodig, de instellingen.

    Wijzigingen die u aanbrengt, zijn van toepassing op de externe server waarmee u in de bovenstaande stappen een verbinding tot stand hebt gebracht.

  9. Kies 'Verbreek verbinding' uit het Archief-menu.
