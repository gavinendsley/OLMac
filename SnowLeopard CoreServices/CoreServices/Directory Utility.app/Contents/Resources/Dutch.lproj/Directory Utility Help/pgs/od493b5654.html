LDAP-adreslijsten vullen met gegevens voor Mac OS X

Nadat u de toegang tot de domeinen van de LDAP-adreslijst hebt geconfigureerd en gegevenskoppelingen hebt ingesteld, kunt u deze vullen met records en gegevens voor Mac OS X. In het geval van LDAP-adreslijsten die extern beheer toestaan (toegang voor lezen/schrijven) kunt u Workgroup Manager van Mac OS X Server gebruiken. Ga hierbij als volgt te werk:

  • Bepaal welke sharepoints en gedeelde domeinen u automatisch wilt activeren in de netwerklijst van gebruikers (wat gebruikers zien wanneer ze op ’Netwerk' klikken in de navigatiekolom van een Finder-venster).

    Gebruik hiervoor de modules Sharing in Server Admin en Network van Workgroup Manager.

    Lees voor aanvullende informatie de onderwerpen File Services Overview en User Management Overview.

  • Definieer gebruikers- en groepsrecords en configureer de bijbehorende instellingen.

    Gebruik hiervoor de module Accounts van Workgroup Manager.

    Lees voor aanvullende informatie het onderwerp User Management Overview.

  • Definieer lijsten met computers die dezelfde voorkeursinstellingen hebben en die beschikbaar zijn voor dezelfde gebruikers en groepen.

    Gebruik hiervoor de module Computers van Workgroup Manager.

    Lees voor aanvullende informatie het onderwerp User Management Overview.

Klik in alle gevallen op de kleine wereldbol boven de lijst met gebruikers en maak een keuze uit het venstermenu in Workgroup Manager om het LDAP-adreslijstdomein te openen. Als de LDAP-adreslijst niet in het venstermenu staat, kiest u 'Other' uit dit menu om de LDAP-adreslijst te selecteren.

Opmerking:  Om records en gegevens toe te voegen aan een alleen-lezen LDAP-adreslijst, moet u programma's gebruiken die aanwezig zijn op de server met de LDAP-adreslijst.
