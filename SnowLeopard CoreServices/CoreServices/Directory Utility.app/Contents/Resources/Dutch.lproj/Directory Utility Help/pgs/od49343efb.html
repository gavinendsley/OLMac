Toegang tot adreslijsten configureren

U kunt Adreslijsthulpprogramma (in het paneel 'Accounts' in Systeemvoorkeuren) gebruiken voor het configureren en beheren van de manier waarop een computer met Mac OS X of een server met Mac OS X Server toegang krijgt tot adreslijstdomeinen.

Toegang krijgen tot LDAP-adreslijsten

Met Adreslijsthulpprogramma kunt u een computer met Mac OS X of een server met Mac OS X Server configureren voor toegang tot specifieke LDAP-adreslijsten, met inbegrip van de LDAP-adreslijst van een Open Directory Master voor Mac OS X Server. Klik op de onderstaande onderwerpen voor meer informatie.

Met Adreslijsthulpprogramma kunt u een configuratie aanmaken om op te geven hoe Mac OS X toegang krijgt tot een NIS-domein. Klik op het onderwerp hieronder voor meer informatie.

BSD-configuratiebestanden gebruiken

Met Open Directory kunt u beheergegevens ophalen uit BSD-configuratiebestanden zoals /etc/master.passwd. Deze bestanden worden ook wel BSD flat files genoemd. Daarnaast kunt u met Open Directory de toegang tot BSD-configuratiebestanden configureren. Klik op de onderstaande onderwerpen voor meer informatie.
