Cómo definir políticas de búsqueda automáticas

Utilizando Utilidad de Directorios, puede configurar las políticas de búsqueda de contactos y de autenticación de un ordenador Mac OS X de forma que se definan automáticamente.

Una política de búsqueda definida automáticamente incluye el dominio de directorio local. También puede incluir un servidor de directorios LDAP especificado por el servicio DHCP.

Esta es la configuración por omisión de las políticas de búsqueda de contactos y de autenticación.

Nota:  algunas aplicaciones como Mail y la Agenda de Mac OS X pueden acceder a los directorios LDAP directamente sin utilizar Open Directory. Para configurar una de dichas aplicaciones de modo que acceda a directorios LDAP directamente, abra la aplicación y ajuste la preferencia adecuada.
Importante:  si configura Mac OS X para que utilice una política de búsqueda de autenticación automática y un servidor LDAP por DHCP o un dominio de directorio compartido por DHCP, aumentará el riesgo de que un usuario malintencionado obtenga el control de su ordenador. El riesgo es si cabe mayor si el ordenador está configurado para conectarse a una red inalámbrica. Para obtener más información, consulte Cómo proteger los ordenadores frente a un servidor DHCP malintencionado.
Para que una política de búsqueda se defina automáticamente:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Acceder o Editar.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  6. Haga clic en “Política de búsqueda” y seleccione una política de búsqueda:

    • Autenticación: muestra la política de búsqueda utilizada para la autenticación y para la mayor parte del resto de datos administrativos.

    • Contactos:muestra la política de búsqueda utilizada para la información de contactos en aplicaciones como la Agenda.

  7. En el menú local Buscar, seleccione Automáticamente y, a continuación, haga clic en Aplicar.

  8. En Preferencias del Sistema, compruebe que las preferencias de red del ordenador están configuradas para utilizar DHCP o DHCP con una dirección IP manual.
