Información acerca de las conexiones de servidores de directorios

Puede utilizar el panel de preferencias Cuentas para conectar ordenadores a servidores de directorios. Para ver las listas de servidores de directorios a los que está conectado su ordenador, haga clic en Editar en el panel “Opc. inicio sesión” del panel de preferencias Cuentas. El ordenador Mac OS X accede a los servidores de la lista para obtener información de usuario y otros datos administrativos almacenados en el dominio de directorio de servidores de directorios.

Cuando añada o elimine un servidor en la lista "Servidores de directorios", las entradas asociadas a ese servidor de directorios se añadirán o eliminarán en las listas Servicios, Autenticación y Contactos. Sin embargo, si elimina las entradas asociadas en las listas Servicios, Autenticación y Contactos, el servidor de directorios no se eliminará de la lista "Servidores de directorios".

Los ordenadores con Mac OS X 10.6 se pueden conectar a Open Directory, Active Directory o a servidores de directorios LDAP. Si no sabe a qué ordenador conectarse, pregunte a su administrador de red.

Importante:  si el nombre de su ordenador contiene un guión, es posible que no pueda acceder o enlazarse a un dominio de directorio como LDAP o Active Directory. Para establecer un enlace, utilice un nombre de ordenador que no contenga un guión.
