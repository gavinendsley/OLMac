Cómo usar ajustes avanzados de servicios de directorio

Utilidad de Directorios muestra una lista de los servicios de directorio a los que puede acceder Mac OS X. Esa lista incluye servicios de directorio que ofrecen a Mac OS X acceso a información de los usuarios y otros datos administrativos almacenados en los dominios de directorio.

Puede activar o desactivar el acceso a cada servicio de directorio. Si desactiva un servicio en Utilidad de Directorios, Mac OS X ya no puede acceder a ese servicio de directorio.
