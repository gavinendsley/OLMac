Cómo añadir una conexión de servidor Open Directory

Cuando añada un servidor Open Directory, debe saber el nombre del servidor o la dirección IP y si el servidor utiliza SSL (Secure Socket Layer).

Para añadir un servidor Open Directory:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Acceder o Editar.

    Si hay un botón Editar, su ordenador tiene al menos una conexión a un servidor de directorios.

  4. Haga clic en el botón Añadir (+).

  5. En el menú local “Añada un nuevo directorio del tipo”, seleccione "Open Directory".

  6. En el campo “Nombre del servidor o dirección IP”, introduzca el nombre o la dirección IP del servidor.

  7. (Condicional) Antes de seleccionar la opción “Encriptar con SSL”, pregunte al administrador de Open Directory si es necesario utilizar SSL.

Importante:  si cambia la dirección IP y el nombre de ordenador de Mac OS X Server mediante changeip mientras está conectado a un servidor de directorios, deberá desconectarse y conectarse de nuevo al servidor de directorios para actualizar el directorio con la dirección IP y el nombre de ordenador nuevos. De lo contrario, el directorio no se actualizará y seguirá utilizando el nombre y la dirección IP antiguos.
