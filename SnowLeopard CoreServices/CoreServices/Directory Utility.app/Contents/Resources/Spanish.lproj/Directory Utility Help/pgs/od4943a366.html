Asignación del Identificador de usuario (UID) a un atributo de Active Directory

En un ordenador configurado para utilizar el conector de Active Directory de Utilidad de Directorios, puede especificar el atributo de Active Directory que desee asignar al atributo UID (ID de usuario único) de Mac OS X.

Normalmente, el esquema de Active Directory debe ampliarse para incluir un atributo adecuado para asignar al UID.

  • Si el administrador de Active Directory amplía el esquema de Active Directory mediante la instalación de servicios para UNIX de Microsoft, podrá asignar el UID al atributo msSFU-30-Uid-Number.

  • Si el administrador de Active Directory amplía el esquema de Active Directory manualmente para incluir atributos RFC 2307, podrá asignar el UID al atributo uidNumber.

  • Si el administrador de Active Directory amplía el esquema de Active Directory manualmente para incluir el atributo UniqueID de Mac OS X, podrá asignar el UID a dicho atributo.

Si la asignación del UID está desactivada, el conector de Active Directory genera un UID basado en el atributo GUID estándar de Active Directory.

ADVERTENCIA:  si se modifica la asignación del atributo UID posteriormente, los usuarios pueden perder el acceso a los archivos creados con anterioridad.
Para asignar el UID a un atributo en un esquema ampliado de Active Directory:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Editar.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  6. Haga clic en Servicios.

  7. En la lista de servicios, seleccione "Active Directory" y haga clic en el botón Editar (/).

  8. Si las opciones avanzadas están ocultas, pulse "Mostrar opciones avanzadas".

  9. Haga clic en Asociaciones.

  10. Seleccione “Asociar UID a atributo” y, a continuación, introduzca el nombre del atributo de Active Directory que desea asignar al UID.

  11. Haga clic en Aceptar.
