Gestión de la cuenta de usuario root

Puede activar o desactivar el usuario root y usar Utilidad de Directorios (en el panel de preferencias Cuentas) para gestionar la cuenta de usuario root. Si ha activado la cuenta de usuario root, también puede utilizar Utilidad de Directorios para cambiar la contraseña de la cuenta root.

Para obtener más información acerca de cómo gestionar la cuenta de usuario root con Utilidad de Directorios, consulte los temas siguientes:

  • Activación de la cuenta de usuario root

  • Cambio de la contraseña de la cuenta de usuario root
