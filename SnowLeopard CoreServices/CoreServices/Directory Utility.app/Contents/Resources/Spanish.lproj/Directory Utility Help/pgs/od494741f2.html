Cómo cambiar una configuración para acceder a un directorio LDAP

Puede utilizar Utilidad de Directorios para modificar los ajustes de la configuración de un directorio LDAP. Los ajustes de configuración especifican cómo accede Open Directory a un directorio LDAPv3 o LDAPv2.

Si la configuración de LDAP la proporcionó DHCP, no se puede cambiar, por lo que este tipo de configuración no estará disponible en la lista de configuraciones de LDAP.

Para editar una configuración para acceder a un directorio LDAP:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Editar.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  6. Haga clic en Servicios.

  7. En la lista de servicios, seleccione LDAPv3 y haga clic en el botón Editar (/).

  8. Si la lista de configuraciones del servidor está oculta, haga clic en "Mostrar opciones".

  9. Haga los cambios que precise a los ajustes siguientes:

    • Activar: haga clic en una casilla para activar o desactivar el acceso a un servidor de directorio LDAP.

    • Nombre de la configuración: haga doble clic en un nombre de configuración para editarlo.

    • Nombre o dirección IP del servidor: haga doble clic en un nombre o una dirección IP de servidor para modificarlos.

    • Asignaciones LDAP: seleccione una plantilla en el menú local, introduzca el sufijo de la base de búsqueda para el directorio LDAP y haga clic en Aceptar.

      Si eligió una plantilla, debe introducir un sufijo de la base de búsqueda o el ordenador no podrá encontrar información en el directorio LDAP. Normalmente, el sufijo de la base de búsqueda deriva del nombre del DNS del servidor. Por ejemplo, para un servidor cuyo nombre DNS sea ods.ejemplo.com, el sufijo de la base de búsqueda sería “dc=ods,dc=ejemplo,dc=com”.

      Si selecciona "Del servidor" en lugar de una plantilla, no se necesita el sufijo de la base de búsqueda. En tal caso, Open Directory considera que el sufijo de la base de búsqueda es el del primer nivel del directorio LDAP.

      Si selecciona Personalizado, tendrá que configurar las asignaciones entre los tipos de registro de Mac OS X y los atributos y las clases del directorio LDAP al que se esté conectando. Para obtener más información, consulte Cómo configurar búsquedas y mapas LDAP.

    • SSL: haga clic en esta opción para activar o desactivar las comunicaciones encriptadas mediante el protocolo SSL. Antes de seleccionar la opción SSL, pregunte al administrador de Open Directory si es necesario utilizar SSL.

  10. Para modificar los ajustes por omisión siguientes de esta configuración LDAP, haga clic en Editar para que se muestren las opciones de la configuración LDAP seleccionada, realice los cambios pertinentes y haga clic en Aceptar cuando termine de editar las opciones de configuración LDAP.

    • Haga clic en Conexión para ajustar las opciones de tiempo, especificar un puerto personalizado, ignorar las referencias del servidor o forzar el uso del protocolo LDAPv2 (solo lectura). Para obtener más información, consulte Cómo cambiar los ajustes de conexión de un directorio LDAP.

    • Haga clic en "Búsqueda y mapas" para configurar búsquedas y asignaciones de un servidor LDAP. Para obtener más información, consulte Cómo configurar un vínculo fiable con un directorio LDAP.

    • Haga clic en Seguridad para configurar una conexión autenticada (en lugar de un vínculo fiable) y otras opciones de las políticas de seguridad. Para obtener más información, consulte Cómo cambiar la política de seguridad de una conexión LDAP.

    • Haga clic en Enlazar para configurar el vínculo fiable o haga clic en "Anular enlace" para detenerlo. (En caso de que el directorio LDAP no permita vínculos fiables, es posible que estos botones no aparezcan). Para obtener más información, consulte Cómo configurar un vínculo fiable con un directorio LDAP.

  11. Para finalizar el proceso de cambio de configuración para acceder a un directorio LDAP, haga clic en Aceptar.
