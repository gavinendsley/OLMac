Cómo configurar cuentas de usuario móviles en Active Directory

Puede activar o desactivar cuentas de usuario móviles de Active Directory en un ordenador que esté configurado para utilizar el conector de Active Directory de Utilidad de Directorios. Los usuarios que dispongan de cuentas móviles podrán iniciar la sesión utilizando sus credenciales de Active Directory cuando el ordenador no esté conectado al servidor Active Directory.

El conector de Active Directory almacena en caché las credenciales de la cuenta móvil de un usuario cuando el usuario inicia la sesión mientras el ordenador está conectado al dominio de Active Directory. Este almacenamiento en caché de las credenciales no requiere la modificación del esquema de Active Directory.

Si el esquema de Active Directory se ha ampliado para incluir atributos de cliente gestionado de Mac OS X, se utilizarán los ajustes de la cuenta móvil en lugar del ajuste de cuenta móvil del conector de Active Directory.

Puede crear cuentas móviles automáticamente o puede solicitar a los usuarios de Active Directory que confirmen la creación de una cuenta móvil.

Para activar o desactivar las cuentas móviles en un dominio de Active Directory:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Acceder o Editar.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  6. Haga clic en Servicios.

  7. En la lista de servicios, seleccione "Active Directory" y haga clic en el botón Editar (/).

  8. Si las opciones avanzadas están ocultas, pulse "Mostrar opciones avanzadas".

  9. Haga clic en “Opciones del usuario”. A continuación, haga clic en “Crear una cuenta móvil al iniciar la sesión” y, opcionalmente, puede hacer clic en “Solicitar confirmación antes de crear una cuenta móvil”.

    • Si ambas opciones están seleccionadas, los usuarios podrán decidir si crear una cuenta móvil durante el inicio de sesión. Cuando un usuario inicia la sesión en Mac OS X utilizando una cuenta de usuario de Active Directory, o cuando inicia la sesión como un usuario de red, el usuario puede ver un cuadro de diálogo con controles para crear una cuenta móvil de forma inmediata.

    • Si la primera de las opciones está seleccionada pero la segunda no, las cuentas móviles se crearán cuando el usuario inicie la sesión.

    • Si la primera opción no está seleccionada, entonces la segunda opción estará desactivada.

  10. Haga clic en Aceptar.
