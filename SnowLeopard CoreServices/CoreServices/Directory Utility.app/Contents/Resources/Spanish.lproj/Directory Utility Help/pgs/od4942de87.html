Especificación de ajustes NIS

Mediante Utilidad de Directorios, puede crear una configuración que especifique la forma en que Mac OS X debe acceder a un dominio NIS (Network Information Service, servicio de información de la red).

Para crear una configuración para acceder a un dominio NIS:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Acceder o Editar.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  6. Haga clic en Servicios.

  7. En la lista de servicios, seleccione “BSD Flat File y NIS” y haga clic en el botón Editar (/).

  8. Introduzca el nombre del dominio NIS, el nombre DNS o la dirección IP del servidor donde reside el dominio NIS.

    Incluya el nombre del servidor NIS o su dirección IP si se requiere por seguridad o si el servidor no está en la misma subred que el ordenador que está configurando.

    Si no especifica ningún servidor, NIS utiliza un protocolo de transmisión para buscar un servidor NIS en la subred.

  9. Seleccione “Usar el dominio NIS para autenticar” y, a continuación, haga clic en Aceptar.

    El dominio NIS se añadirá a la política de búsqueda de autenticación del ordenador como /BSD/dominio, donde dominio es el nombre que introdujo en el paso 4.
