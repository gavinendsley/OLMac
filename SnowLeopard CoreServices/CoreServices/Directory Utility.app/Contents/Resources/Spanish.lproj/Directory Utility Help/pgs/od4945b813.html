Cómo asociar atributos de registro de configuración para directorios LDAP

Para almacenar información de usuarios de Mac OS X gestionados en un directorio LDAP que no sea de Apple, debe asignar los siguientes atributos del tipo de registro Config: RealName y DataStamp.

Si no asigna dichos atributos, aparecerá el siguiente mensaje de error cuando utilice Workgroup Manager para modificar un registro de usuario ubicado en el directorio LDAP:

El atributo con el nombre "dsRecTypeStandard:Config" no está asignado.

Puede ignorar el mensaje si no está utilizando la gestión de clientes de Mac OS X, que depende de los atributos RealName y DataStamp Config del tipo de registro para una caché.
