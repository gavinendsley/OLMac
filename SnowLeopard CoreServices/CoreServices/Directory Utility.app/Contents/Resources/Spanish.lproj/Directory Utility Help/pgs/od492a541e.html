Cómo crear una contraseña

La contraseña asociada a una cuenta de usuario debe ser introducida por el usuario cuando se autentique para iniciar sesión u otros servicios. La contraseña distingue entre mayúsculas y minúsculas (excepto las contraseñas de SMB-LAN Manager) y, cuando se introduce, se oculta en la pantalla.

Independientemente del tipo de contraseña que se seleccione para un usuario, aplique estas directrices para formar contraseñas para las cuentas de usuario de Mac OS X Server:

  • Una contraseña debe contener letras, números y símbolos combinados de forma que los usuarios no autorizados no puedan adivinarla fácilmente. Las contraseñas no deben estar formadas por palabras. Una buena contraseña incluye dígitos y símbolos (como # o $), o está formada por la primera letra de las palabras de una frase. Use tanto mayúsculas como minúsculas.

  • Evite combinaciones con espacios y la tecla Opción.

  • Evite aquellos caracteres que el usuario no podrá introducir en su ordenador o que requieran conocer una combinación de teclas especial para introducirlos correctamente en distintos en teclados y plataformas.

  • Algunos protocolos de red no admiten contraseñas con espacios iniciales, espacios intermedios o espacios finales.

  • No se recomienda usar una contraseña de longitud cero. Open Directory y algunos sistemas (como enlace LDAP) no admiten contraseñas de longitud cero.

  • Para lograr la máxima compatibilidad con ordenadores y servicios a los que pudieran acceder los usuarios, use solamente caracteres ASCII para las contraseñas.
