Configuración de datos en los archivos de configuración BSD

Si desea que un ordenador equipado con Mac OS X obtenga datos de administración a partir de archivos de configuración BSD, dicha información debe estar almacenada en dichos archivos en el formato requerido por Mac OS X.

Es posible que deba añadir, cambiar o reorganizar los datos de los archivos. Workgroup Manager no puede efectuar cambios en los datos de los archivos de configuración BSD, por lo que las modificaciones necesarias deberán realizarse mediante un editor de texto u otra herramienta.

En la siguiente tabla figuran los nombres de los archivos y su contenido.

Archivo de configuración BSD
Contenido

/etc/master.passwd

Nombres de usuario, contraseñas, ID, ID de grupos primarios, etc.

/etc/group

Nombres de grupo, ID y miembros

/etc/fstab

Montajes NFS

/etc/hosts

Nombres y direcciones de ordenadores

/etc/networks

Nombres y direcciones de red

/etc/services

Nombres de servicio, puertos y protocolos

/etc/protocols

Nombres y números de protocolo IP

/etc/rpcs

Servidores Open Network Computing RPC

/etc/printcap

Nombres y funciones de impresora

/etc/bootparams

Ajustes Bootparam

/etc/bootp

Ajustes de Bootp

/etc/aliases

Alias de correo y listas de distribución

/etc/netgroup

Nombres y miembros de grupos de toda la red
