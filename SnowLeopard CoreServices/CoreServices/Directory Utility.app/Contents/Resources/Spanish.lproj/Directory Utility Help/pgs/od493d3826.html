Cómo cambiar los ajustes de conexión de un directorio LDAP

Puede utilizar Utilidad de Directorios para modificar los ajustes de conexión de una configuración que especifique cómo debe acceder el ordenador a un directorio LDAPv3 o LDAPv2.

Para cambiar los ajustes de conexión para acceder a un directorio LDAP:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Editar.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  6. Haga clic en Servicios.

  7. En la lista de servicios, seleccione LDAPv3 y haga clic en el botón Editar (/).

  8. Si la lista de configuraciones del servidor está oculta, haga clic en "Mostrar opciones".

  9. Seleccione una configuración del servidor en la lista y, a continuación, haga clic en Editar.

  10. Haga clic en Conexión y cambie cualquiera de los ajustes siguientes:

    • Nombre de la configuración: identifica esta configuración en la lista de configuraciones de directorio LDAP. (También puede modificar el nombre en la lista de configuraciones del directorio LDAP).

    • Nombre o dirección IP del servidor: especifica el nombre DNS del servidor o su dirección IP. (También puede modificar dicho ajuste en la lista de configuraciones del directorio LDAP).

    • Tiempo máximo para abrir/cerrar conexión: especifica el tiempo máximo que puede durar un intento de conexión antes de que se cancele.

    • Tiempo máximo de consulta: especifica el tiempo máximo que puede durar una consulta antes de que se cancele.

    • Intento de reconexión tras: especifica el número de segundos que hay que esperar antes de intentar volver a conectarse en caso de que el servidor LDAP no responda. Para evitar constantes intentos de reconexión, aumente este valor.

    • Conexión inactiva tras: especifica el número de minutos que se permite que permanezca abierta una conexión inactiva o una conexión que no responde.

    • Encriptar con SSL: determina si se deben encriptar las comunicaciones con el directorio LDAP utilizando una conexión SSL. (También puede modificar dicho ajuste en la lista de configuraciones del directorio LDAP). Antes de seleccionar la opción SSL, pregunte al administrador de Open Directory si es necesario utilizar SSL.

    • Usar puerto personalizado: especifica un número de puerto diferente del puerto estándar empleado para las conexiones LDAP (389 sin SSL o 636 con SSL).

    • Ignorar referencias del servidor: determina si se va a ignorar o seguir la referencia de un servidor LDAP para buscar información en otros servidores o réplicas LDAP. Las referencias del servidor pueden ayudar a un ordenador a encontrar información, pero también pueden producir retardos a la hora de iniciar sesiones u otro tipo de retardos si el ordenador debe verificar las referencias a otros servidores LDAP.

    • Usar LDAPv2 (solo lectura): determina si se va a utilizar el antiguo protocolo LDAPv2 para el acceso de solo lectura a un directorio LDAP.
