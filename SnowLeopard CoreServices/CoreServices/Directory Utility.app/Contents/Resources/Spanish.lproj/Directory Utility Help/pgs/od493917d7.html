Preparación de un directorio LDAP de solo lectura para un ordenador Mac OS X

Si desea que un ordenador con Mac OS X obtenga datos de administración de un directorio LDAP de solo lectura, dicha información debe estar en el formato requerido por Mac OS X. Es posible que tenga que añadir, modificar o reorganizar la información en el directorio LDAP de solo lectura.

Dado que Mac OS X no puede escribir datos en un directorio de solo lectura, deberá utilizar otras herramientas para efectuar los cambios. Las herramientas deben residir en el servidor que aloja el directorio LDAP de solo lectura.

Para preparar un directorio LDAP de solo lectura para un ordenador Mac OS X:

  1. Vaya al servidor que aloja el directorio LDAP de solo lectura y configúrelo para que admita la autenticación basada en LDAP y la verificación de contraseñas.

  2. Cambie las clases y los atributos de objetos del directorio LDAP de la manera adecuada para suministrar los datos requeridos por Mac OS X.

Para ver las especificaciones de los datos requeridos por los servicios de directorio de Mac OS X, consulte el manual Open Directory Administration (Administración de Open Directory).
