Anulación del enlace al servidor Active Directory

Si el ordenador está utilizando el conector de Active Directory de Utilidad de Directorios para enlazarse a un servidor Active Directory, puede anular el enlace del ordenador con el servidor Active Directory.

Puede forzar la anulación del enlace si el ordenador no puede establecer contacto con el servidor en estos momentos o si el registro del ordenador ya se ha eliminado del servidor.

Para anular el enlace del ordenador de un servidor Active Directory:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Editar.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  6. Haga clic en Servicios.

  7. En la lista de servicios, seleccione "Active Directory" y haga clic en el botón Editar (/).

  8. Haga clic en "Anular enlace", autentíquese como usuario con derechos para finalizar una conexión con el dominio de Active Directory y, a continuación, haga clic en Aceptar.

    Si ve un aviso que dice que las credenciales no se han aceptado o que el ordenador no puede establecer contacto con Active Directory, haga clic en "Forzar anulación del enlace" para forzar la ruptura de la conexión.

    Si fuerza la ruptura del enlace, Active Directory sigue manteniendo un registro de dicho ordenador. Notifíqueselo al administrador de Active Directory para que lo sepa y así elimine el registro del ordenador.

  9. En el panel Servicios, anule la selección del ajuste Activar de Active Directory y, a continuación, haga clic en Aplicar.
