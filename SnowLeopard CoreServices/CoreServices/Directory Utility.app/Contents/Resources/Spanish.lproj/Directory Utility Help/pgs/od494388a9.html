Cómo activar o desactivar el servicio Active Directory

Puede utilizar Utilidad de Directorios para activar o desactivar el uso de servicios de Active Directory proporcionados por un servidor Windows. Active Directory es el servicio de directorio de los servidores Windows 2000 y posteriores.

Si desactiva los servicios de Active Directory y hay algún dominio de Active Directory que forme parte de una política de búsqueda personalizada, dichos dominios aparecerán en rojo en los paneles Autenticación o Contactos de "Política de búsqueda" en Utilidad de Directorios.

Para activar o desactivar el acceso a Active Directory:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Acceder o Editar.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  6. Haga clic en Servicios.

  7. Seleccione o anule la selección de la casilla junto a Active Directory y haga clic en Aplicar.
