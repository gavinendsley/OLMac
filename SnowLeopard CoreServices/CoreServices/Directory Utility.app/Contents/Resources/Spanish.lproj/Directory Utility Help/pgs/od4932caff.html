Cambio de la contraseña de la cuenta de usuario root

Puede usar Utilidad de Directorios (situada en el panel de preferencias Cuentas) para cambiar la contraseña de la cuenta root. Cuando cambie la contraseña de usuario root, utilice una contraseña compleja que contenga caracteres alfanuméricos y especiales para evitar poner en peligro su seguridad.

ADVERTENCIA:  la cuenta root es una cuenta de administrador no restringida utilizada para efectuar cambios en archivos de sistema críticos. Aunque haya iniciado sesión como administrador, deberá utilizar la cuenta root, o sudo, para efectuar las tareas más importantes del sistema.

Nunca utilice la cuenta root para iniciar sesión en un ordenador (remota o localmente). En lugar de ello, use sudo para realizar tareas de root. Si desea restringir el acceso al comando sudo, añada usuarios al archivo /etc/sudoers/.

Para obtener más información acerca de la cuenta root, consulte la ayuda de la aplicación Server Admin. Para obtener más información acerca de cómo crear contraseñas, consulte Cómo crear una contraseña.

Para cambiar la contraseña de la cuenta de usuario root:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Acceder o Editar.

    Si hay un botón Editar, su ordenador tiene al menos una conexión a un servidor de directorios.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Seleccione Editar > Cambiar contraseña de usuario root.

  6. Cuando el sistema se lo solicite, introduzca la nueva contraseña de usuario root en los campos Contraseña y Verificar.

  7. Haga clic en Aceptar.
