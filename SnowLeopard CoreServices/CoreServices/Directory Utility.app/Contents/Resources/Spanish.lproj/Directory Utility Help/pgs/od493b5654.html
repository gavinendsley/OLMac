Cómo llenar directorios LDAP con datos para un ordenador Mac OS X

Después de configurar el acceso a dominios de directorio LDAP y configurar la asignación de datos, puede llenarlos con registros y datos para Mac OS X. Para los directorios LDAP que permiten administración remota (acceso de lectura/escritura), puede utilizar la aplicación Workgroup Manager, incluida en Mac OS X Server, del siguiente modo:

  • Identifique los puntos de volúmenes compartidos y los dominios compartidos que desee montar automáticamente en los navegadores de red de los usuarios (lo que los usuarios ven al hacer clic en Red en la barra lateral de una ventana del Finder).

    Utilice los módulos Compartir de Server Admin y Red de Workgroup Manager.

    Para información adicional, localice Visión general de servicios de archivo y Visión general de gestión de usuarios.

  • Defina registros de usuario y de grupo, y configure sus ajustes.

    Utilice el módulo Cuentas de Workgroup Manager.

    Para información adicional, localice Visión general de gestión de usuarios.

  • Defina listas de ordenadores que tengan los mismos ajustes de preferencias y que estén disponibles para los mismos usuarios y grupos.

    Utilice el módulo Ordenadores de Workgroup Manager.

    Para información adicional, localice Visión general de gestión de usuarios.

En todos los casos, haga clic en el pequeño icono de globo situado encima de la lista de usuarios y seleccione abrir el dominio de directorio LDAP en el menú local en Workgroup Manager. Si el directorio LDAP no aparece en la lista del menú local, seleccione Otro en este menú para seleccionar el directorio LDAP.

Nota:  para añadir registros y datos a un directorio LDAP de solo lectura, debe utilizar las herramientas del servidor que alberga el directorio LDAP.
