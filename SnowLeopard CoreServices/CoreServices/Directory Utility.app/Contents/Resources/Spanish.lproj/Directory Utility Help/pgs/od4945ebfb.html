Cómo usar ajustes avanzados del servicio Active Directory

Puede configurar un servidor con Mac OS X Server o un ordenador con Mac OS X para acceder a un dominio de Active Directory en un servidor Windows 2000 o Windows 2003.

  • Cómo definir una shell de UNIX para cuentas de usuario de Active Directory

  • Asignación del Identificador de usuario (UID) a un atributo de Active Directory

  • Asignación del ID de grupo primario a un atributo de Active Directory

  • Asignación del ID de grupo de cuentas de grupo a un atributo de Active Directory

  • Especificación del servidor Active Directory preferido

  • Cambio de los grupos de Active Directory que pueden administrar el ordenador

  • Control de autenticaciones desde todos los dominios del bosque (dominio root) de Active Directory

  • Anulación del enlace al servidor Active Directory

  • Edición de cuentas de usuario y otros registros en Active Directory

Con determinadas redes hay otros métodos alternativos que puede utilizar para acceder a un dominio de Active Directory. Consulte Configuración del acceso LDAP a dominios de Active Directory.
