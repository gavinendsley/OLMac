Configuración del acceso a directorios

Puede usar Utilidad de Directorios (en el panel de preferencias Cuentas) para configurar y gestionar la forma en que un ordenador con Mac OS X o un servidor con Mac OS X Server acceden a dominios de directorio.

Acceso a directorios LDAP

Mediante Utilidad de Directorios, puede configurar un ordenador con Mac OS X o un servidor con Mac OS X Server para acceder a directorios LDAP específicos, incluido el directorio LDAP de un máster de Open Directory de Mac OS X Server. Para obtener más información, haga clic en los temas siguientes.

Uso de archivos de configuración BSD

Puede utilizar Open Directory para recuperar datos administrativos de archivos de configuración BSD, como "/etc/master.passwd", también denominados "BSD flat files". También puede configurar el acceso a los archivos de configuración BSD utilizando Utilidad de Directorios. Para obtener más información, haga clic en los temas siguientes.
