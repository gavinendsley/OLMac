Edición de cuentas de usuario y otros registros en Active Directory

Puede utilizar Workgroup Manager para efectuar cambios a cuentas de usuario, cuentas de grupo, listas de ordenadores y otros registros en un dominio de Active Directory. También puede utilizar Workgroup Manager para eliminar registros en un dominio de Active Directory.

Si se ha ampliado el esquema de Active Directory para incluir atributos y tipos de registro (clases de objetos) de Mac OS X estándar, puede utilizar Workgroup Manager para crear y editar grupos de ordenadores en el dominio de Active Directory.

Para información adicional, localice Visión general de gestión de usuarios.

Para crear cuentas de usuario o de grupo, utilice las herramientas de administración de Microsoft Active Directory en un ordenador de administración del servidor Windows.
