Cómo configurar carpetas de inicio para cuentas de usuario de Active Directory

En un ordenador que esté configurado para utilizar el conector de Active Directory de Utilidad de Directorios, puede activar o desactivar las carpetas de inicio de red o las carpetas de inicio locales de las cuentas de usuario de Active Directory.

En el caso de las carpetas de inicio de red, cuando un usuario inicia la sesión se monta una carpeta de inicio de red de Windows del usuario como carpeta de inicio de Mac OS X.

Puede determinar si la ubicación de la carpeta de inicio de red debe obtenerse del atributo estándar homeDirectory de Active Directory o del atributo homeDirectory de Mac OS X, en caso de que se haya ampliado el esquema de Active Directory para incluirlo.

En el caso de las carpetas de inicio locales, todos los usuarios de Active Directory que inicien una sesión dispondrán de una carpeta de inicio en el disco de arranque de Mac OS X. Además, la carpeta de inicio de red del usuario se monta como un volumen de red, es decir, como un punto de volumen compartido. El usuario puede copiar archivos entre este volumen de red y la carpeta de inicio local.

Para configurar carpetas de inicio para cuentas de usuario de Active Directory:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Acceder o Editar.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  6. Haga clic en Servicios.

  7. En la lista de servicios, seleccione "Active Directory" y haga clic en el botón Editar (/).

  8. Si las opciones avanzadas están ocultas, pulse "Mostrar opciones avanzadas".

  9. Haga clic en "Opciones del usuario".

  10. Haga clic en “Forzar la carpeta de inicio local en el disco de arranque” si desea que las cuentas de usuario de Active Directory dispongan de carpetas de inicio locales en la carpeta /Usuarios del ordenador.

    Esta opción no estará disponible si la opción "Crear una cuenta móvil al iniciar la sesión" está seleccionada.

  11. Para utilizar el atributo estándar de Active Directory para la ubicación de la carpeta de inicio, seleccione “Usar la ruta UNC para derivar la ubicación de la carpeta de inicio a la red” y, a continuación, elija uno de los siguientes protocolos para acceder a la carpeta de inicio:

    • Para utilizar el protocolo SMB estándar de Windows, seleccione smb en el menú local “Protocolo de red”.

    • Para utilizar el protocolo AFP estándar de Macintosh, seleccione afp en el menú local “Protocolo de red”.

  12. Para utilizar el atributo de Mac OS X para la ubicación de la carpeta de inicio, anule la selección de “Usar la ruta UNC para derivar la ubicación de la carpeta de inicio a la red”.

    Para utilizar el atributo de Mac OS X, debe haberse ampliado el esquema de Active Directory para incluirlo.

  13. Haga clic en Aceptar.

Si cambia el nombre de una cuenta de usuario en el dominio de Active Directory, el servidor creará una carpeta de inicio (y subcarpetas) para la cuenta de usuario la próxima vez que se utilice para iniciar la sesión en un ordenador Mac OS X. El usuario todavía puede examinar la antigua carpeta de inicio y ver su contenido en el Finder.

Puede impedir la creación de una nueva carpeta de inicio renombrando la antigua carpeta antes de los próximos inicios de sesión del usuario.
