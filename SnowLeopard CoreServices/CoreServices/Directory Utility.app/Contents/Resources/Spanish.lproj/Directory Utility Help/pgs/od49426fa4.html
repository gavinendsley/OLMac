Asignación del ID de grupo de cuentas de grupo a un atributo de Active Directory

En un ordenador configurado para utilizar el conector de Active Directory de Utilidad de Directorios, puede especificar el atributo de Active Directory que desee asignar al atributo ID de grupo (GID) de Mac OS X en las cuentas de grupo.

Normalmente, el esquema de Active Directory debe ampliarse para incluir un atributo adecuado para asignar el GID:

  • Si el administrador de Active Directory amplía el esquema de Active Directory mediante la instalación de servicios para UNIX de Microsoft, podrá asignar el GID al atributo msSFU-30-Gid-Number.

  • Si el administrador de Active Directory amplía el esquema de Active Directory manualmente para incluir atributos RFC 2307, podrá asignar el GID a gidNumber.

  • Si el administrador de Active Directory amplía el esquema de Active Directory manualmente para incluir el atributo gidNumber de Mac OS X, podrá asignar el GID a dicho atributo.

Si la asignación del GID está desactivada, el conector de Active Directory generará un GID basado en el atributo GUID estándar de Active Directory.

ADVERTENCIA:  si se modifica la asignación del atributo GID posteriormente, los usuarios pueden perder el acceso a los archivos creados con anterioridad.
Para asignar el GID a un atributo en un esquema ampliado de Active Directory:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Editar.

  4. Haga clic en “Abrir Utilidad de Directorios”.

  5. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  6. Haga clic en Servicios.

  7. En la lista de servicios, seleccione "Active Directory" y haga clic en el botón Editar (/).

  8. Si las opciones avanzadas están ocultas, pulse "Mostrar opciones avanzadas".

  9. Haga clic en Asociaciones.

  10. Seleccione “Asociar GID de grupo a atributo” y, a continuación, introduzca el nombre del atributo de Active Directory que desee asignar al GID de las cuentas de grupo.

  11. Haga clic en Aceptar.
