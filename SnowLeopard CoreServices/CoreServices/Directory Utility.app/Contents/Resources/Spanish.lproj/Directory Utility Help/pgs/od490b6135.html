Información acerca de Utilidad de Directorios

La aplicación Utilidad de Directorios determina qué servicios de directorio utiliza un ordenador Mac OS X y cómo se conecta a dominios de directorio específicos. También define las políticas de búsqueda para buscar información de autenticación y contactos en dominios de directorio específicos.

Configuración de Utilidad de Directorios

Puede utilizar Utilidad de Directorios en su ordenador para configurar y gestionar cómo su ordenador accede a servicios de directorio. También puede utilizar esta aplicación para configurar de forma remota cómo su ordenador accede a servicios de directorio. Utilidad de Directorios también permite activar la cuenta root o cambiar la contraseña de la cuenta root.

Configuración del acceso a servicios

Puede utilizar Utilidad de Directorios para activar y desactivar los distintos tipos de servicios a los que Mac OS X puede acceder en una red. Entre estos servicios se incluyen los servicios de directorio, que proporcionan acceso a las cuentas de usuario y a otros datos administrativos almacenados en dominios de directorio.

Configuración del acceso a directorios

Mediante Utilidad de Directorios, puede configurar el ordenador para que acceda a cuentas de usuario y a otros datos administrativos de directorios LDAP específicos, dominios de Active Directory y dominios NIS.

Configuración de políticas de búsqueda

Mediante Utilidad de Directorios, puede definir la política de búsqueda de autenticación que Mac OS X utiliza para localizar y recuperar la información de autenticación del usuario y otros datos administrativos en los dominios de directorio. También define la política de búsqueda de contactos que Mac OS X utiliza para ubicar y encontrar el nombre, la dirección y otros datos de contacto en los dominios de directorio.
