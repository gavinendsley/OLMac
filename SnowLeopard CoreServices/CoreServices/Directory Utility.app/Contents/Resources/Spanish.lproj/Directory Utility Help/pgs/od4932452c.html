Cómo añadir una conexión de servidor Active Directory

Para conectarse a un servidor Active Directory, debe saber el nombre o la dirección IP del servidor, así como el nombre de usuario y la contraseña de administrador de Active Directory.

Para añadir un servidor Active Directory:

  1. Abra Preferencias del Sistema y haga clic en Cuentas.

  2. Si el icono del candado está cerrado, desbloquéelo haciendo clic en él e introduciendo el nombre y la contraseña de un administrador.

  3. Haga clic en "Opc. inicio sesión" y luego haga clic en Acceder o Editar.

    Si hay un botón Editar, su ordenador tiene al menos una conexión a un servidor de directorios.

  4. Haga clic en el botón Añadir (+).

  5. En el menú local “Añada un nuevo directorio del tipo”, seleccione "Active Directory" y, a continuación, introduzca la información siguiente:

    • Dominio de Active Directory: es el nombre DNS del dominio de Active Directory (por ejemplo, ads.empresa.com.)

    • ID del ordenador: opcionalmente, puede editar el ID que desea que Active Directory utilice para el servidor. Es el nombre NetBIOS del servidor. Este nombre no puede tener más de 15 caracteres y no puede contener ni caracteres especiales ni signos de puntuación. Si le resulta práctico, haga coincidir el nombre del servidor con su nombre de servidor DNS sin calificar. Por ejemplo, si el servidor DNS tiene una entrada para el servidor como “servidor.ejemplo.com”, dé al servidor el nombre “servidor”.

    • Nombre de usuario y contraseña de administrador de Active Directory:introduzca el nombre de usuario y la contraseña de administrador de Active Directory.

  6. Haga clic en Aceptar.
