Bruge avancerede indstillinger til bibliotekstjenester

Biblioteksværktøj viser de forskellige bibliotekstjenester, som Mac OS X kan bruge. Listen omfatter bibliotekstjenester, som giver Mac OS X adgang til brugeroplysninger og andre administrative data, som er lagret i biblioteksdomæner.

Du kan slå adgang til hver enkelt bibliotekstjeneste til og fra. Hvis du slår en tjeneste fra i Biblioteksværktøj, bruger Mac OS X ikke længere den pågældende bibliotekstjeneste.

  • Slå Active Directory-tjeneste til og fra

  • Slå LDAP-bibliotekstjenester til og fra
