Konfigurere adgang til biblioteker

Du kan bruge Biblioteksværktøj (i vinduet Konti) til at indstille og administrere, hvordan en Mac OS X-computer eller en server med Mac OS X Server bruger biblioteksdomæner.

Adgang til LDAP-biblioteker

Med Biblioteksværktøj kan du konfigurere en computer med Mac OS X eller en server med Mac OS X Server til at bruge bestemte LDAP-biblioteker, inklusive LDAP-biblioteket på en Mac OS X Server Open Directory-master. Du kan læse mere, hvis du klikker på emnerne nedenfor.

Bruge BSD-konfigurationsarkiver

Du kan bruge Open Directory til at hente administrative data fra BSD-konfigurationsarkiver som f.eks. /etc/master.passwd, også kaldet "BSD flat files". Du kan også konfigurere brug af BSD-konfigurationsarkiver vha. Biblioteksværktøj. Du kan læse mere, hvis du klikker på emnerne nedenfor.
